The SessionStart hook in `src/hooks/session-start.ts` triggers when a Claude Code session begins. It loads all memories from disk via MemoryManager.listMemories() and returns the 5 most recent memories sorted by occurred_at. The vector store is not used here to avoid slow initialization on every session start - it uses direct file-based memory retrieval.
