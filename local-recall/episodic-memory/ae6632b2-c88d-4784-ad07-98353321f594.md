# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by dashes and URL-encoded.

Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`

## Transcript Storage

Transcripts are stored in `~/.claude/projects/<project-folder>/transcripts/` as JSONL files with format: `<session-id>-<timestamp>.jsonl`

Example: `01HQY8jXfJm7zh35BcYYiE1u-2025-12-01T16-32-09-279Z.jsonl`

This location is critical for the transcript-collector subsystem to sync transcripts for memory extraction.
