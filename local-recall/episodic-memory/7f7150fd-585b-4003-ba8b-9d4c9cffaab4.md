Claude stores project transcripts in `~/.claude/projects/` with a URL-encoded path pattern. The directory name is constructed from the project path: `/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/` contains session transcripts as JSONL files. The transcript path structure is: `~/.claude/projects/<url-encoded-project-path>/transcripts/<session-id>.jsonl`
