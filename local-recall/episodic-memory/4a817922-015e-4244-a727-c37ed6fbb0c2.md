Claude stores project transcripts in `~/.claude/projects/` directory. Key details:

1. **Path Format**: Project folders use URL-encoded path format
   - Example: `-Users-joe-Code-Syntessera-local-recall/` (forward slashes converted to hyphens)
   - This encoding allows multiple projects with same names from different directories

2. **File Structure**: Each project folder contains transaction logs
   - Files are named with UUIDs (e.g., `086dd5...`)
   - These contain the actual transcript data

3. **Discovery Strategy**: To find transcripts for a project at path `/Users/joe/Code/Syntessera/local-recall`, construct the cache path by:
   - Taking the absolute project path
   - URL-encoding forward slashes to hyphens
   - Looking in `~/.claude/projects/<encoded-path>/`

This is important for the MCP background daemon that syncs transcripts for memory extraction.
