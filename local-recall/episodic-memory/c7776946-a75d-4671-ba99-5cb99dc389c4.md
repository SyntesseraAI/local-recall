# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the workspace path by:
1. Converting the path to lowercase
2. Replacing forward slashes `/` with dashes `-`
3. Using this as the project folder name

Example:
- Workspace: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `users-joe-code-syntessera-local-recall`
- Transcript path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

Transcripts are stored as `.jsonl` files in this directory, with one JSON object per line representing each transcript entry.

This path format is used by the `transcript-collector.ts` module to locate and read Claude's cached transcripts for memory extraction.
