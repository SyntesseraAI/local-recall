# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the project path by replacing slashes with dashes:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files in the `transcripts/` directory, one file per session.

## Implementation Impact

When implementing `transcript-collector.ts`, account for this path transformation to correctly locate and read transcript files from Claude's cache.
