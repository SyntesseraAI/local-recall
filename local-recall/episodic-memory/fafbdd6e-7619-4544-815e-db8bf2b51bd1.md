# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the workspace name, with spaces replaced by underscores and slashes replaced by dashes. For example:
- Workspace: `Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Syntessera-local-recall/transcripts/`
- Transcript file format: `.jsonl` (JSON Lines)

## Transcript Structure

Transcript files contain:
- `role`: Either "user" or "assistant"
- `content`: The message text
- `timestamp`: ISO 8601 format
- `toolUse`: Object containing tool invocations (when present)
  - `name`: Tool name (e.g., "Read", "Edit", "Bash")
  - `input`: Tool parameters as a JSON object

## Usage in transcript-collector.ts

The transcript collector reads these files to extract information about coding sessions for memory creation. The path construction is critical for finding the correct transcript files.
