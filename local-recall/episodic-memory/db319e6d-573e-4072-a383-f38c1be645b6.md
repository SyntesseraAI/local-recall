Memory deduplication in local-recall uses two mechanisms: the `occurred_at` timestamp (for temporal grouping) and a SHA-256 content hash prefix (16 chars) stored in each memory's frontmatter. The `findDuplicate()` method checks for existing memories with the same hash and timestamp to prevent duplicate memory creation.
