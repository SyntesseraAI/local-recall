Local Recall uses the BGE-small-en-v1.5 embedding model from fastembed (~133MB) for semantic search. Important details:
- First run downloads the model (30-60 seconds) to local_cache/fast-bge-small-en-v1.5/
- Subsequent runs load from cache
- If tokenizer error occurs, delete local_cache/fast-bge-small-en-v1.5* to re-download
- The cache directory is gitignored
