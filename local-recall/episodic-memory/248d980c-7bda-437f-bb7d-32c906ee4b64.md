# Claude Code Transcript Storage Location

Claude Code stores session transcripts in `~/.claude/projects/` with the following naming convention:

- Directory name: URL-encoded project path (e.g., `Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall`)
- Transcript files: Named with timestamps and session IDs (e.g., `1732880340000-session-abcd.jsonl`)
- Format: JSONL (JSON Lines) where each line is a complete message event

This is used by `src/core/transcript-collector.ts` to locate and process transcripts for memory extraction. The transcript-collector needs to URL-decode the project path from the `cwd` to find the correct transcript directory.
