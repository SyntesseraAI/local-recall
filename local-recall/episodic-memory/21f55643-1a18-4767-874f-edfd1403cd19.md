The UserPromptSubmit hook extracts keywords from user prompts by calling Claude Haiku (`claude -p --model haiku`) to analyze the text. This is used to search the memory index for relevant memories before Claude processes the prompt. The extracted keywords are used with fuzzy matching to find related memories.
