# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored in JSONL format (newline-delimited JSON) at:
- `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each line in a transcript file is a complete JSON object representing one event (user message, assistant response, tool call, or tool result).

## Importance

This location is crucial for the MCP server background daemon that:
- Syncs transcripts from Claude's cache
- Processes transcripts using `claude -p` to extract memories
- Tracks processed transcripts with content hashes for change detection
