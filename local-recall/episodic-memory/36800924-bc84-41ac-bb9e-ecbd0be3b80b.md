# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored as JSONL format with sequential numbering:
- Format: `transcript_001.jsonl`, `transcript_002.jsonl`, etc.
- Each line is a JSON object representing a single event
- Events include user messages, assistant responses, and tool invocations

## Key Details

- The transcript path is passed to hooks via the `transcript_path` JSON input
- Transcript collector uses this path to locate and process transcript files
- The file structure allows multiple transcripts per project session
