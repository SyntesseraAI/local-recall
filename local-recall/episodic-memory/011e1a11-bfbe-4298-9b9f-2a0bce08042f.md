All memory files use YAML frontmatter for metadata (id, subject, keywords, applies_to, occurred_at, content_hash) followed by markdown content. The id field is a unique identifier, keywords are searchable terms, applies_to defines scope (global, file:<path>, area:<name>), and content_hash is a SHA-256 prefix for deduplication. This standardized format enables consistent memory storage and retrieval across the system.
