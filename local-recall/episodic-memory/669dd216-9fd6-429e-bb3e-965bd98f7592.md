Removed `memory_update` and `memory_delete` MCP tools. Available tools now:

- `memory_create`: Create idempotent memories (returns existing if duplicate)
- `memory_get`: Retrieve specific memory by ID
- `memory_search`: Semantic search
- `memory_list`: List all memories

Memory management is now read-only for agents; only users can manually delete.
