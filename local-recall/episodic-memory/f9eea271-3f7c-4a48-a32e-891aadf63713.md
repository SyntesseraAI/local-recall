Local Recall structure:
- Project root IS the plugin root
- `.claude-plugin/plugin.json` contains plugin metadata
- `hooks.json` configures hook handlers
- `.mcp.json` configures MCP server
- `scripts/` contains bundled executables (build output, gitignored)
- Hooks use `${CLAUDE_PLUGIN_ROOT}` variable and receive `cwd` via stdin
