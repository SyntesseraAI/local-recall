# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `-Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format) in the `transcripts/` directory. Each line is a valid JSON object representing a single message in the conversation.

## Key Insight

The transcript collector needs to reconstruct the original working directory path from the processed project folder name by replacing dashes with slashes and prepending the expected prefix.
