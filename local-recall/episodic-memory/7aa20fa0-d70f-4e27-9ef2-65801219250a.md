# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name encodes the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored as JSONL files within each project folder

## Importance for Transcript Collection

The `transcript-collector.ts` module needs to construct the correct path by:
1. Taking the working directory (cwd) from the session
2. Replacing all `/` with `-` to match Claude's encoding scheme
3. Looking in `~/.claude/projects/{encoded-path}/transcripts/`

This is crucial for the MCP server's background daemon which syncs transcripts for memory extraction.
