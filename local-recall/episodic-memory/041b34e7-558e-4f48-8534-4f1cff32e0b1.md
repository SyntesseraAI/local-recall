# Transcript Collector Module

The `transcript-collector.ts` module is responsible for:
1. Discovering transcript files in Claude's project directories
2. Reading and parsing JSONL (JSON Lines) format transcript files
3. Extracting transcript events for memory processing

## Key Functions

The module needs to:
- Scan `~/.claude/projects/` for transcript files matching the current project path
- Read JSONL files and parse each line as a separate JSON object
- Convert raw transcript events into a format suitable for memory extraction

## Integration Points

This module is used by:
- MCP server background daemon for automatic transcript processing
- Memory extraction pipeline for creating memories from session data

## Current Status

The module exists but may need refinement for robust error handling and performance optimization when dealing with large transcript files.
