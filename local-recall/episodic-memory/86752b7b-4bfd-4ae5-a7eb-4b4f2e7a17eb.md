# Memory Extraction Prompt Update

Updated `buildMemoryExtractionPrompt()` in `src/prompts/memory-extraction.ts` to:

1. **Accept condensed data** - Takes pre-parsed, minimal transcript format instead of raw JSONL
2. **Clear instructions** - Specifies extraction from already-condensed content
3. **Consistent format** - Maintains expected JSON output structure for memories

The prompt now receives a much smaller input while maintaining the semantic information needed for accurate memory extraction. This improves:
- Token efficiency (fewer input tokens)
- Speed (faster Claude processing)
- Cost (less API usage)
- Quality (focused content, less noise)
