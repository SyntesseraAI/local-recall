Use UUID validation with the regex pattern `/^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}\.jsonl$/i` to validate transcript filenames. This ensures only proper UUID-named transcript files are processed across all transcript collection methods.
