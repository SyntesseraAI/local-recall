# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path:
1. Remove leading slash from absolute path
2. Replace remaining slashes with dashes
3. Create folder under `~/.claude/projects/`

### Example

Project path: `/Users/joe/Code/Syntessera/local-recall`
Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

This is important for `transcript-collector.ts` when syncing transcripts from Claude's cache during MCP server operations.
