Claude Code hooks for Local Recall receive structured JSON input via stdin containing:
- `session_id`: Unique session identifier
- `transcript_path`: Path to the session transcript
- `cwd`: Current working directory of the user's project
- `prompt`: User's prompt (for UserPromptSubmit hook only)

Hooks output memory content to stdout which is injected into Claude's context. SessionStart hook loads the memory index at startup, while UserPromptSubmit hook performs dynamic keyword extraction and search before processing each user prompt.
