The `src/core/` directory contains the memory management system:
- **memory.ts**: CRUD operations (no update - memories are idempotent)
  - `createMemory(data)` - Creates or returns existing duplicate
  - `deleteMemory(id)` - Delete by ID
  - `getMemory(id)` - Retrieve specific memory
  - `listMemories(filter?)` - List with optional filtering
  - `findDuplicate(occurredAt, contentHash)` - Deduplication check
- **index.ts**: Maintains searchable keyword index
  - `buildIndex()` - Scan memory files and build keyword index
  - `getIndex()` - Retrieve current index
  - `refreshIndex()` - Rebuild from scratch
- **search.ts**: Fuzzy search with multiple strategies
  - `searchByKeywords(query)` - Fuzzy matching on keywords
  - `searchBySubject(query)` - Search subject lines
  - `searchByScope(scope)` - Filter by scope
- **types.ts**: TypeScript interfaces
