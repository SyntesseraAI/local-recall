# Claude Project Transcripts Storage

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Project Folder Naming Convention

The project folder name is derived from the absolute working directory path:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Forward slashes are replaced with hyphens
- Transcripts stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL files (one JSON object per line), with naming pattern: `<timestamp>_<uuid>.jsonl`

Each transcript contains message events with: `type`, `timestamp`, `role`, `content`, and `tool_use` information.

## MCP Server Transcript Processing

The MCP server's background daemon syncs these transcripts:
1. Reads from Claude's transcript cache
2. Extracts memories using `claude -p` prompt processing
3. Tracks processed transcripts with content hashes for change detection
4. Deletes and recreates memories when transcripts change
5. Runs every 5 minutes

This allows asynchronous memory extraction without blocking the user's workflow.
