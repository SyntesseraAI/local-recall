Vector similarity search scoring:
- **Score Range**: 0.0 (no match) to 1.0 (identical)
- **Precision**: Rounded to 2 decimal places (e.g., 0.65)
- **Sorting**: Results ordered by score descending
- **Tie-breaking**: When scores are equal, more recent memories (by occurred_at) rank first

This ensures consistent and predictable ranking even with many similar results.
