The logger utility in src/utils/logger.ts expects string-only arguments for debug/info/error methods. When adding logging statements to memory-extractor, avoid passing structured data as second arguments. Use string interpolation or JSON.stringify() within the string argument to format complex objects.

Example:
- Wrong: `logger.debug('Parsed structure:', parsedData)`
- Correct: `logger.debug('Parsed structure: ' + JSON.stringify(parsedData))`
