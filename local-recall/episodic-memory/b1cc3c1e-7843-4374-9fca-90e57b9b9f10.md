# Keyword Extraction Using Claude Haiku

The local-recall system uses the Claude Haiku API to extract keywords from text for memory indexing.

## Implementation Details

- Uses `claude -p --model haiku` command-line interface
- Prompts Claude to extract 5-10 keywords from given text
- Returns keywords as a JSON array of strings
- More accurate than simple regex-based keyword extraction
- Useful for both user prompts (UserPromptSubmit hook) and transcript analysis

## Benefits

- Natural language understanding vs keyword matching
- Captures semantic intent rather than literal words
- Better memory retrieval accuracy
- Can extract implicit concepts from text
