Version 0.1.3 of the Local Recall plugin was released with significant improvements:

- **New Skills Added**: Two new memory-related skills were introduced:
  - `check-memories`: Allows checking/searching memories
  - `proactive-recall`: Enables proactive memory retrieval based on context

- **Improvements Made**:
  - MCP tool descriptions enhanced with usage guidance
  - Plugin metadata updated for better discoverability in marketplaces
  - 8 thinking memory files added to improve contextual reasoning

- **Key Files Modified**:
  - `dev-marketplace/local-recall-plugin/` directory structure
  - Skills directory reorganized for clarity

This release improves the plugin's visibility and usability for end-users discovering it through plugin marketplaces.
