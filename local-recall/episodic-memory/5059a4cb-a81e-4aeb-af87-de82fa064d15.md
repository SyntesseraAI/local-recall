The memory storage structure has two categories of files:
- `local-recall/index.json` - Auto-generated keyword index (gitignored for performance)
- `local-recall/episodic-memory/*.md` - Individual memory files (tracked in git)

This allows memories to be shared among team members while keeping the generated index local. The .gitignore file in the local-recall directory automatically excludes the index and recall.log.
