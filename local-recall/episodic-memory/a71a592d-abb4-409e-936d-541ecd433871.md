# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons removed:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files, where each line is a separate JSON object representing a message or event in the session.

Key fields in transcript entries:
- `role`: `user` or `assistant`
- `content`: The actual message content
- `type`: Message type identifier
- `timestamp`: When the message was created

This location is important for the transcript-collector component which needs to sync these transcripts for memory extraction.
