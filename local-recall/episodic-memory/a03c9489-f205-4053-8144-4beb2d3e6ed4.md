Claude stores transcript files in `~/.claude/projects/` with encoded directory names (using hyphens for path separators, e.g., `-Users-joe-Code-Syntessera-local-recall`). Each project directory contains a transcript JSON file. The MCP server's background daemon syncs these transcripts for memory extraction.
