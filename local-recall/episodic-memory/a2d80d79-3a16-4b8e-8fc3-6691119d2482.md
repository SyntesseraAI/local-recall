# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Takes the full path: `/Users/joe/Code/Syntessera/local-recall`
- Replaces `/` with `-`: `Users-joe-Code-Syntessera-local-recall`
- Example: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension, one event per line. Each line contains a JSON object representing a Claude Code event (user message, assistant response, tool invocation, etc.).

This is important for the `transcript-collector` module which needs to read and parse these files to extract session data.
