# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Conversion Pattern

The project folder name converts the absolute working directory path:
- Replace all forward slashes `/` with dashes `-`
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in the project folder. This pattern is used by the transcript-collector to locate and sync transcripts for memory extraction.
