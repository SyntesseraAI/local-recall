# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension. Each line is a complete JSON object representing a transcript message or event.

## Usage Context

This path is used by the MCP server daemon to sync transcripts from Claude's cache for memory extraction and processing.
