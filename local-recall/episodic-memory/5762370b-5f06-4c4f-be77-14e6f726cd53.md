# Transcript Collector Implementation

The `transcript-collector.ts` module is responsible for syncing and processing Claude Code transcripts.

## Current Functionality

- Reads JSONL transcript files from `~/.claude/projects/<encoded-project-name>/transcripts/`
- Parses transcript entries and processes them
- Used by the MCP server daemon for background memory extraction

## Key Methods

- Handles transcript file discovery
- Reads JSONL format transcripts
- Provides methods for accessing transcript data

## Integration Points

- Used by MCP server background daemon (runs every 5 minutes)
- Part of the memory extraction pipeline
- Works with the path encoding pattern specific to macOS Claude installation
