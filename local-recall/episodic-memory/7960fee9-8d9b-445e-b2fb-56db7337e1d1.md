# MCP Server Background Daemon

The MCP server runs a background daemon that processes Claude Code transcripts asynchronously.

## Key Details

- **Frequency**: Runs every 5 minutes
- **Source**: Syncs transcripts from `~/.claude/projects/<project>/transcripts/`
- **Processing**: Uses `claude -p` to extract memories from transcripts
- **Change Detection**: Tracks processed transcripts with content hashes for change detection
- **Memory Management**: Deletes and recreates memories when transcripts change

## Implementation Location

See `src/mcp-server/server.ts` for the background daemon implementation.

## Relationship to Hooks

The Stop hook is currently disabled. All transcript processing and memory extraction is handled by this background daemon instead.
