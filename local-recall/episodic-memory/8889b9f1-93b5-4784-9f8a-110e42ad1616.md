# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and the last path component appended. For example:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript file format: `*.jsonl` (JSON Lines format, one event per line)

This path format is consistent and predictable, making it suitable for programmatic access.
