# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as `.jsonl` files (JSON Lines format), one entry per line, each containing:
- `type`: Event type ("user_message", "assistant_message", "tool_call", "tool_result", etc.)
- `content`: Event content
- `timestamp`: ISO-8601 timestamp
- `role`: "user" or "assistant"

## Implementation

The `transcript-collector.ts` uses this path to:
1. Watch for new transcript files
2. Parse transcript events
3. Extract memories from user interactions and tool results
4. Deduplicate based on `occurred_at` and `content_hash`

This is critical for the MCP server's background daemon that syncs transcripts every 5 minutes.
