The MemoryManager provides:
- `createMemory(data)`: Creates new memory or returns existing if duplicate
- `deleteMemory(id)`: Delete by ID
- `getMemory(id)`: Retrieve specific memory
- `listMemories(filter?)`: List all with optional filtering
- `findDuplicate(occurredAt, contentHash)`: Check for duplicates

Memories are idempotent - there's no update operation. Duplicates are detected using occurred_at timestamp and content hash.
