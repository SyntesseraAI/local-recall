Claude stores project transcripts at `~/.claude/projects/<encoded-project-path>/transcripts/`. The project directory name is the absolute path with slashes replaced by dashes and URL-encoded. For example, `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`. Each session creates a `<session-id>.jsonl` file containing the transcript events.
