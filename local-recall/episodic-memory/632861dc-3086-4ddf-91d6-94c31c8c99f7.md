The memory extractor in `src/core/memory-extractor.ts` uses Claude Haiku model via the `claude` CLI tool to process transcripts and extract valuable memories. The extractor reads transcript files, condenses them, and sends them to Claude Haiku with a system prompt from `src/prompts/memory-extraction.ts` to generate structured memories in JSON format.
