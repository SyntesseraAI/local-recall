# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path with forward slashes replaced by dashes.

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files: JSONL format with names like `transcript-001.jsonl`, `transcript-002.jsonl`, etc.

## Transcript Format

Each transcript file contains JSONL (JSON Lines) format with sequential message entries.

This convention is important for the Local Recall MCP server which needs to locate and sync transcripts for memory extraction.
