# Session Start Hook

The SessionStart hook is triggered when a Claude Code session begins and loads the memory index to provide context.

## Functionality

1. Receives JSON input via stdin with:
   - `session_id`: Unique session identifier
   - `transcript_path`: Path to current session transcript
   - `cwd`: Current working directory

2. Loads the memory index from `local-recall/index.json`

3. Retrieves relevant memories based on current context
   - May use project path or other contextual clues

4. Outputs memory content to stdout (injected into Claude's context)

## Purpose

Provides Claude with relevant background knowledge at the start of each session, improving context awareness and consistency across sessions.
