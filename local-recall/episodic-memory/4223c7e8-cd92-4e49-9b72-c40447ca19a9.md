# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and the trailing dash removed.

**Example:**
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts.jsonl`

## File Format

Transcripts are stored as JSONL (JSON Lines) format, with each line containing a JSON object representing a message or event in the conversation.
