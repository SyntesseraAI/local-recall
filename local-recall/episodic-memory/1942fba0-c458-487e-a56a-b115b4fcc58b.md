The transcript condenser module (src/core/transcript-condenser.ts) is responsible for reducing transcript size before memory extraction. It processes the full transcript and condenses it to focus on the most important parts while removing redundant information. This condensed transcript is then passed to the memory extractor to improve efficiency and reduce token usage in Claude API calls.
