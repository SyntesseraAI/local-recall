Each memory file has YAML frontmatter with required fields: `id` (UUID), `subject` (brief description), `keywords` (searchable array), `applies_to` (global|file:<path>|area:<name>), `occurred_at` (ISO-8601 timestamp), and `content_hash` (SHA-256 prefix for deduplication). The actual memory content follows the frontmatter in markdown format.
