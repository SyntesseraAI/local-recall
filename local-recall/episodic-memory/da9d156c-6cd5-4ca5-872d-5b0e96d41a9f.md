# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in the project folder:
- Location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`
- Files are numbered: `transcript_001.jsonl`, `transcript_002.jsonl`, etc.

## Transcript Content Structure

Each line in a transcript file is a JSON object with:
- `type`: Event type ("user", "assistant", "tool")
- `content`: The actual message or tool invocation details
- `timestamp`: When the event occurred

This information is useful when implementing transcript collection and processing features.
