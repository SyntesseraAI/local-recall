# Memory Extraction Keyword Extraction

The user-prompt-submit hook and MCP daemon use Claude Haiku (`claude -p --model haiku`) to extract keywords from:
- User prompts (in user-prompt-submit hook)
- Transcript content (in MCP daemon for memory extraction)

This approach allows for intelligent keyword extraction rather than simple keyword matching, improving memory relevance and discoverability.
