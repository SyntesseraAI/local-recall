# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes.

**Example:**
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/*.jsonl`

## Implementation in transcript-collector.ts

The `transcript-collector.ts` module handles:
1. Computing the project folder name from `cwd` by replacing `/` with `-`
2. Reading `.jsonl` files from the transcripts directory
3. Parsing JSONL format (each line is a separate JSON object)
4. Processing transcripts to extract memories

See `src/transcript-collector.ts` for implementation details.
