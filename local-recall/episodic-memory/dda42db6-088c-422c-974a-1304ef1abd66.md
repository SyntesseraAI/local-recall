# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_users_joe_code_syntessera_local-recall`
- Full transcript directory: `~/.claude/projects/_users_joe_code_syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files:
- Each line is a separate JSON object representing an event/message
- Files contain `timestamp`, `type`, `content`, and other fields
- Files are named with timestamps or session IDs

## Importance

This is critical for the transcript-collector component which needs to locate and parse Claude's cached transcripts for memory extraction via the MCP server daemon.
