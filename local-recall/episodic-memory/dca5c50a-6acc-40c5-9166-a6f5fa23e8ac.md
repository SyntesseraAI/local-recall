Configuration options:
- `memoryDir` (default: ./local-recall) - Directory for memory storage
- `maxMemories` (default: 1000) - Maximum number of memories
- `indexRefreshInterval` (default: 300 seconds) - Between index refreshes
- `fuzzyThreshold` (default: 0.6) - Minimum fuzzy match score (0-1)

Can be set via environment variables or .local-recall.json file in project root.
