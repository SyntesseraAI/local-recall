The `analyzeForMemories()` function in `src/utils/transcript.ts` is responsible for converting transcript messages into memory format. It:
1. Iterates through messages
2. Skips empty messages
3. Formats each message with role prefix
4. Returns formatted content for storage

This function is the primary interface between transcript parsing and memory creation.
