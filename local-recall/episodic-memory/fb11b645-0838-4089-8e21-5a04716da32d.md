The UserPromptSubmit hook extracts keywords from user prompts using `claude -p --model haiku` to intelligently identify relevant search terms. This same approach is used in the MCP server background daemon to extract memories from full session transcripts. The `-p` flag indicates prompt processing mode for Claude CLI.
