Important distinction: Memory markdown files in `local-recall/episodic-memory/*.md` ARE version-controlled and will be committed to git. The `local-recall/.gitignore` file excludes:
- `memory.sqlite` (auto-generated vector store database)
- `recall.log` (debug log file)

This allows team members to share and synchronize memories across the repository while keeping generated artifacts out of version control.
