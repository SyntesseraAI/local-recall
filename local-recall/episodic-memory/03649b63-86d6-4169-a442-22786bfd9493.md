# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project directory, with `/` replaced by `_`.

Example:
- Project at `/Users/joe/Code/Syntessera/local-recall`
- Transcripts stored at `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) in the `transcripts/` subdirectory:
- `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/YYYY-MM-DD.jsonl`

Each line contains a transcript entry with fields like `type`, `content`, `timestamp`, etc.

## Related Code

The `transcript-collector.ts` module handles reading and parsing these transcript files. When converting project paths to Claude's directory format, the `/` separators must be replaced with `_`.
