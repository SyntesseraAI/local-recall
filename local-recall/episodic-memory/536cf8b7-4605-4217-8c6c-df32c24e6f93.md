The configuration system uses Zod for schema validation and default values. All configuration defaults are defined in `src/core/types.ts` using the `.default()` method on Zod fields. When making configuration changes, both the schema defaults in `src/core/types.ts` AND the documentation in `CLAUDE.md` must be updated to remain in sync.
