# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with the filename format `YYYYMMDD-HHMMSS-xxxxxxxx.jsonl`

## Content Structure

Each JSON object in the transcript contains:
- `type`: Event type (e.g., "user", "assistant", "tool_use", "tool_result")
- `text`: The content of the event
- Additional fields depending on event type

This is used by the MCP server daemon to sync and process transcripts for memory extraction.
