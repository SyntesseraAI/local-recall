All new features must have:
- Unit tests in `tests/unit/`
- Integration tests in `tests/integration/`
- Documentation in `docs/` folder
- Existing docs updated when modifying functionality
- Tests covering both happy paths and edge cases

NOTE: Memory files ARE version-controlled and will be committed to git. The SQLite database and vector model cache are gitignored.
