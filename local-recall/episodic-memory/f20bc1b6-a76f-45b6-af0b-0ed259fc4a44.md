Memory extraction output must follow a specific JSON structure with a 'memories' array. Each memory object requires: subject (brief one-line description, max 200 chars), keywords (array of 1-10 lowercase searchable terms), applies_to (scope: 'global', 'file:<path>', or 'area:<name>'), and content (detailed markdown description). The applies_to field determines memory scope and is critical for proper memory indexing and retrieval.
