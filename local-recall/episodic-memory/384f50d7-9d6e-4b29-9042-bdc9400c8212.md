# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Conversion

The project folder name converts absolute paths to folder names by:
1. Starting with the absolute path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replacing all slashes with dashes (e.g., `Users-joe-Code-Syntessera-local-recall`)
3. Storing transcripts in `~/.claude/projects/{converted-path}/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format) in the transcripts directory.

## Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Each transcript file: `<uuid>.jsonl`
