# Keywords Extraction Format

When extracting keywords from text using Claude's API:
- Return format: JSON array of strings
- No explanation text, no markdown formatting
- Only the JSON array itself
- Example: `["keyword1", "keyword2", "keyword3"]`

This is used in the transcript collector to extract searchable keywords from transcript content for memory indexing.
