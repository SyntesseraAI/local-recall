The local-recall system processes transcripts asynchronously through an MCP server daemon that runs every 5 minutes. The daemon:
- Syncs transcripts from Claude's cache directory
- Uses `claude -p` to extract structured memories from transcripts
- Tracks processed transcripts with content hashes for change detection
- Automatically deletes and recreates memories when transcripts change

This enables automatic memory creation without requiring explicit Stop hook execution.
