# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is created by encoding the project's file path:
- Slashes are replaced with hyphens: `/` → `-`
- Colons are replaced with hyphens: `:` → `-`
- The result is the folder name under `~/.claude/projects/`

## Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
1. Remove leading slash: `Users-joe-Code-Syntessera-local-recall`
2. Folder path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
3. Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (newline-delimited JSON) files with `.jsonl` extension. Each line contains a complete event object from the Claude Code session.

## Implications for transcript-collector

When implementing `transcript-collector.ts`:
1. Calculate the encoded project path from `process.cwd()`
2. Construct the full path: `~/.claude/projects/{encoded-path}/transcripts/`
3. Use `~` expansion (e.g., via `path.expandUser()` or `os.homedir()`)
4. Read JSONL files, parsing each line as a JSON object
5. Handle missing transcript directory gracefully (new projects may not have transcripts yet)
