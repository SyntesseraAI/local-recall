# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

Project folder names are derived from the project's working directory path:
- Original path: `/Users/joe/Code/Syntessera/local-recall`
- Converted to: `users-joe-code-syntessera-local-recall` (dashes replace slashes and spaces)
- Transcripts stored in: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a valid JSON object representing a transcript event.

## Relevance to TranscriptCollector

The `TranscriptCollector` class needs to construct this path to find and read transcript files for the current project. Understanding this path format is essential for the MCP server's background daemon that syncs transcripts and extracts memories.
