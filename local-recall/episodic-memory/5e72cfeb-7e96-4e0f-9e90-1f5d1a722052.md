# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON lines), where each line is a complete JSON object representing an event.

## Implications for Local Recall

The transcript collector daemon should:
1. Normalize project paths by replacing `/` with `-`
2. Look for transcript files in the `~/.claude/projects/` directory structure
3. Handle JSONL format when parsing transcripts
4. Account for the fact that transcript paths are deterministic based on the working directory
