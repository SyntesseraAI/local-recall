# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format) in the `transcripts/` subdirectory:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Format: One JSON object per line
- Each object contains: `role` ("user" or "assistant"), `content` (text content), `timestamp`

## Implementation Notes

The `src/core/transcript-collector.ts` module handles reading and parsing these transcript files. When implementing transcript processing:
1. Use the working directory from Claude's context to construct the folder name
2. Replace all slashes with dashes in the absolute path
3. Look for `.jsonl` files in the `transcripts/` subdirectory
4. Parse each line as separate JSON objects
5. Handle missing or malformed transcript files gracefully
