Claude Code stores project transcripts at `~/.claude/projects/<encoded-path>/` where the encoded path replaces `/` with `-`. For example, a project at `/Users/joe/Code/Syntessera/local-recall` stores transcripts at `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`.

Transcripts are stored in JSONL format with entries containing:
- `timestamp`: ISO-8601 format
- `session_id`: Unique session identifier
- `type`: Event type ("user", "assistant", "tool", etc.)
- `content`: The actual message or tool output

This location is important for the MCP server's background daemon which processes transcripts asynchronously every 5 minutes to extract memories.
