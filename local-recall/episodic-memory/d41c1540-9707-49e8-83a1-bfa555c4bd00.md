# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by underscores and periods.

## Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Transcripts folder: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`
- Transcript files: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`
- File format: JSONL (JSON Lines)

## Accessing Transcripts

Use `transcript-collector.ts` to parse and process these transcripts. The transcripts contain detailed event logs of Claude Code sessions including user prompts, assistant responses, and tool invocations.
