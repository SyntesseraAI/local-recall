The MCP server runs a background daemon that processes transcripts asynchronously every 5 minutes. It extracts memories from session transcripts using memory-extractor, which uses claude-haiku to analyze the transcript content and generate structured memories. This allows automatic memory creation without requiring Stop hook execution.
