# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the project directory path:
- The project directory is taken relative to the home directory
- Forward slashes (`/`) are replaced with dashes (`-`)
- The resulting folder name is used directly under `~/.claude/projects/`

### Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Home directory: `/Users/joe`
- Relative path: `Code/Syntessera/local-recall`
- Project folder name: `Code-Syntessera-local-recall`
- Full path to transcripts: `~/.claude/projects/Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files in this location.
