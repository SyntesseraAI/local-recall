The transcript-collector module (src/core/transcript-collector.ts) processes Claude Code transcripts (stored as .jsonl files) and converts them into structured memory events.

It reads transcript files line by line, parses each JSON object, and extracts relevant information to help create memories from Claude's conversation history.

This is a key component for the MCP server's background daemon that syncs and processes transcripts to create memories.
