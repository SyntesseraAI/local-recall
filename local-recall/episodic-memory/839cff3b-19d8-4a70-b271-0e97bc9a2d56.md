The memory-extractor module uses the Anthropic Claude API to analyze session transcripts and extract valuable memories. It implements streaming responses for efficient processing of potentially large transcripts. The extractor accepts condensed transcript events and outputs structured memory objects with subject, keywords, applies_to scope, and markdown content.
