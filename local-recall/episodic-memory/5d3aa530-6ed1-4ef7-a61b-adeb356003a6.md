# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons replaced with underscores:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Encoded project folder: `_Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) with names like:
- `transcript_20250101_120000.jsonl`
- `transcript_20250101_120500.jsonl`

Each line contains a JSON object representing a transcript event.

## Implementation Notes

When implementing transcript collection in the MCP server daemon:
1. Detect the Claude project folder name from the working directory by encoding the path (slashes to dashes, colons to underscores)
2. Read transcripts from `~/.claude/projects/{encoded-path}/transcripts/`
3. Process JSONL files by reading line-by-line and parsing each line as JSON
4. Track which transcripts have been processed to avoid re-processing unchanged files
