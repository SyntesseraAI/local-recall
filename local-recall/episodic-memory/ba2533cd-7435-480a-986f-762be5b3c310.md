# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and URL-encoded special characters.

### Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Encoded path: `Users-joe-Code-Syntessera-local-recall`
- Transcripts directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- File format: `*.jsonl` (JSON Lines format)

## Usage in Local Recall

The transcript-collector uses this location to sync and process transcripts for memory extraction. The path encoding is important for locating the correct project transcripts.
