# Claude Transcripts Storage Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Conversion

The project folder name is derived by converting the absolute working directory path:
- Replace forward slashes with hyphens
- Convert to lowercase

For example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) where each line represents a transcript entry with content and metadata.
