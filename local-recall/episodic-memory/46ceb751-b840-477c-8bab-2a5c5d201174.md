# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is created by taking the absolute path and replacing slashes with dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `_Users_joe_Code_Syntessera_local-recall`
- The transcripts are stored as JSONL files in the project folder

## Usage in Code

This is important for the `transcript-collector.ts` module which needs to find and read Claude's transcripts. The path encoding must be exact to locate the correct project folder.

- `src/core/transcript-collector.ts` - Implements transcript discovery and collection
- `src/core/types.ts` - Defines transcript-related TypeScript interfaces
