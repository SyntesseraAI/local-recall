Project structure:

- `src/core/` - Memory management (memory.ts, index.ts, search.ts, types.ts)
- `src/hooks/` - Claude Code hooks (session-start.ts, user-prompt-submit.ts, stop.ts)
- `src/mcp-server/` - MCP server implementation (server.ts, tools.ts)
- `src/utils/` - Utilities (markdown.ts, transcript.ts, logger.ts, fuzzy.ts)
- `scripts/` - Bundled build output (gitignored)
- `local-recall/` - Memory storage with episodic-memory/ subdirectory

Building creates JavaScript bundles in scripts/ from src/ TypeScript sources.
