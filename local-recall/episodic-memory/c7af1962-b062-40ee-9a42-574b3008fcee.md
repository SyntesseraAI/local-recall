# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Contents

Each project folder contains:
- `transcripts/` - Directory containing transcript files
- Individual JSONL files for each session

## Implications for local-recall

The MCP server daemon needs to locate and process transcripts from this Claude cache directory for memory extraction. The transcript-collector needs to handle the path transformation when accessing Claude's project cache.
