The user-prompt-submit hook and memory extraction process uses Claude Haiku (`claude -p --model haiku`) to extract keywords from prompts and transcripts. This is done via the `claude` CLI with the `-p` flag for parsing/processing mode, and the `--model haiku` flag to specify the model.
