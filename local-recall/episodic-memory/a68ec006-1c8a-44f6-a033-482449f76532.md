Each memory file follows a standard format: YAML frontmatter with fields id (UUID), subject (brief description), keywords (array), applies_to (global/file:/path/area:name), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix). Content goes below the frontmatter in markdown format.
