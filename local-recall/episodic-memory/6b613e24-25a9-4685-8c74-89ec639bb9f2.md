# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `users-joe-code-syntessera-local-recall`
- Full path: `~/.claude/projects/users-joe-code-syntessera-local-recall/`

## Transcript Storage

Transcripts are stored as JSONL files in `transcripts/` subdirectory:
- File format: `.jsonl` (JSON Lines - one JSON object per line)
- Location: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`
- Naming: Sequential files like `1.jsonl`, `2.jsonl`, etc.

## Key Insight

This path transformation is essential for the transcript-collector to locate and process Claude Code transcripts. The slash-to-dash conversion is consistent and reproducible.
