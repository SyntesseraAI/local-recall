# Claude Transcripts Storage Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Directory Naming

Project folders use dash-separated names derived from the working directory:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Path separators (`/`) are converted to dashes (`-`)
- The first forward slash is omitted in the naming

## Transcript Format

Transcripts are stored as JSONL files within the project directory:
- Location: `~/.claude/projects/<dash-separated-project-name>/transcripts/`
- Format: JSONL (JSON Lines - one JSON object per line)
- Each line contains a single event in the transcript

## Relevance

This path format is critical for the transcript-collector to locate and process transcripts from Claude Code sessions. Understanding the naming convention allows for programmatically constructing the correct path to transcript files.
