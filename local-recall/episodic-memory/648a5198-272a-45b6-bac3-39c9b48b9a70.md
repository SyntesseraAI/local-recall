# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS in JSONL format.

## Path Format

The project folder name converts the absolute working directory path by replacing slashes with dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines - one JSON object per line), making them suitable for streaming and incremental processing.

## Relevance to Local Recall

This is critical for the transcript-collector component which needs to:
1. Locate and read Claude transcripts from the cache
2. Parse JSONL format for transcript events
3. Process transcripts to extract memories automatically

The MCP server background daemon relies on this path pattern to sync transcripts every 5 minutes.
