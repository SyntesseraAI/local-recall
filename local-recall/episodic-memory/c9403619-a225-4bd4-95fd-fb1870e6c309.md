# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is created by replacing path separators with dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder `Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored in `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Implementation Details

This encoding scheme is used in `src/core/transcript-collector.ts` when building paths to locate and sync transcripts. The transcript collector needs to handle this path transformation to correctly identify project transcripts.
