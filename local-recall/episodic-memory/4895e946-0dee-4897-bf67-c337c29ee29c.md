# Memory Extraction Process

The memory extraction workflow:

1. **Transcript Collection**: Transcripts are automatically collected from `~/.claude/projects/<project>/transcripts/`
2. **Keyword Extraction**: Use Claude Haiku to extract keywords from transcript content
3. **Memory Creation**: Create memory files with:
   - Unique ID (UUID)
   - Subject line (brief description)
   - Keywords array (for search/indexing)
   - Content (detailed information from transcript)
   - Metadata (applies_to scope, occurred_at timestamp, content_hash)

## Key Implementation Detail

The transcript collector must process JSONL files and handle multiple messages per transcript.
