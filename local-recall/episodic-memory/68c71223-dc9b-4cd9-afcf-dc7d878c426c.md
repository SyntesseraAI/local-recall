# JSONL Transcript Format

Claude Code stores transcripts as JSONL files (JSON Lines format), where each line is a valid JSON object.

## Parsing Implications

The transcript collector must:
1. Read transcript files with `.jsonl` extension
2. Parse each line as a separate JSON object
3. Not attempt to parse transcripts as plain text or single JSON documents

This affects how the `transcript-collector.ts` module reads and processes transcript files from `~/.claude/projects/*/transcripts/`.
