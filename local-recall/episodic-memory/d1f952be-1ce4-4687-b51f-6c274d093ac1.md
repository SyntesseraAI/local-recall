# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files with naming pattern:
- Format: `{session-id}.jsonl`
- Each line is a valid JSON object representing one event in the session
- Files are appended to throughout the session

## Key Discovery

This path transformation is critical for the transcript-collector to accurately locate and parse Claude Code session transcripts. The MCP server daemon relies on this to find and process transcripts asynchronously.
