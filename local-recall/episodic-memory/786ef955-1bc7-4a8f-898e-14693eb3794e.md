Hooks are configured in `.claude/settings.json` with different paths depending on installation:

**npm package**: `node ./node_modules/local-recall/dist/hooks/{session-start,user-prompt-submit,stop}.js`
**local project**: `node ./dist/hooks/{session-start,user-prompt-submit,stop}.js`

Hooks receive JSON via stdin with session_id, transcript_path, cwd, and (for UserPromptSubmit) prompt. They output memory content to stdout for injection into Claude's context. Timeouts: SessionStart 30s, UserPromptSubmit 30s, Stop 60s.
