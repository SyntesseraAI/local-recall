The SessionStart hook performs a full memory reload (not incremental) on every session start:
1. Receives JSON with session_id, transcript_path, cwd
2. Loads all memories from disk via MemoryManager.listMemories()
3. Returns 5 most recent memories (sorted by occurred_at)
4. Outputs memory content to stdout for context injection

Important: Vector store is NOT used here to avoid slow initialization on every session start. This is a performance optimization.
