Memory files are stored as markdown with YAML frontmatter containing: id (UUID), subject (brief description), keywords (searchable terms), applies_to (scope: global/file/area), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix for deduplication). This structure enables semantic search and prevents duplicate memory creation.
