All memories are stored as markdown files in the `local-recall/episodic-memory/` directory with YAML frontmatter containing metadata. Required fields in frontmatter:
- `id`: UUID
- `subject`: Brief description
- `keywords`: Array of searchable terms
- `applies_to`: Scope (global, file:path, or area:name)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix

Memory files are version-controlled in git.
