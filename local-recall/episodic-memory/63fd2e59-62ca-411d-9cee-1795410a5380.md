# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

- Files are `.jsonl` (JSON Lines) format
- Each line is a separate JSON object representing a transcript event
- Filenames are timestamps like `2024-12-01-1234.jsonl`

## Key Details

- The `cwd` field in transcript data always reflects the working directory when the session started
- This is important for memory extraction and context management
- The path mapping uses a predictable pattern that allows reverse-lookup of project directories
