# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path to the project directory. The conversion replaces path separators (`/`) with underscores (`_`) and removes the leading underscore.

Example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with one JSON object per line representing a conversation turn.

## Relevant Code

The `transcript-collector.ts` module handles converting the absolute path to the folder name format using `path.sep` and string manipulation.
