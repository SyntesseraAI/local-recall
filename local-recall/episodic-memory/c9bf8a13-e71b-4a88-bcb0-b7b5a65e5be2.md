Local Recall provides two integration methods:
1. **Claude Code Plugin**: Uses hooks (SessionStart, UserPromptSubmit) to inject memories into context. The plugin root is the project directory itself.
2. **MCP Server**: Exposes memory tools (memory_create, memory_get, memory_search, memory_list, index_rebuild) and runs a background daemon that syncs transcripts every 5 minutes.

Hooks are configured in `hooks.json` using `${CLAUDE_PLUGIN_ROOT}` variable for cross-platform compatibility.
