# Claude Haiku for Keyword Extraction

## Context

The `UserPromptSubmit` hook uses Claude Haiku to extract keywords from user prompts before searching the memory index.

## Implementation Detail

The hook invokes Claude Haiku using the `claude -p --model haiku` command to:
1. Analyze the user's prompt
2. Extract relevant keywords
3. Return keywords that will be used for memory search

## Rationale

- **Cost-efficient**: Haiku is the fastest and cheapest Claude model
- **Speed**: Needed for sub-second response in hook execution
- **Sufficient capability**: Keyword extraction is a straightforward task that Haiku handles well
