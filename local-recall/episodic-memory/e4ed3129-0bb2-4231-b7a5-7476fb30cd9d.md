# Local Recall Architecture

Local Recall is a persistent memory layer enabling AI assistants to retain context across sessions through structured markdown files stored in `local-recall/episodic-memory/`.

## Key Components

1. **Memory Storage**: Markdown files with YAML frontmatter in `local-recall/episodic-memory/`
2. **Claude Code Integration**: Hooks for SessionStart, UserPromptSubmit, and Stop events
3. **MCP Server**: Exposes memory tools to MCP-compatible clients
4. **Index System**: Keyword-based search index stored at `local-recall/index.json` (gitignored)
5. **Plugin Structure**: Project root is the plugin root with `.claude-plugin/plugin.json`

## Memory File Format

Each memory has YAML frontmatter with fields: `id`, `subject`, `keywords`, `applies_to`, `occurred_at`, `content_hash`
