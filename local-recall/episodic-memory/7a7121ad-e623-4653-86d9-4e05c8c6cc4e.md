# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses the absolute path with slashes replaced by dashes and prefixed with a hash:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `8d5a0.../Users-joe-Code-Syntessera-local-recall/`
- Full path: `~/.claude/projects/8d5a0.../Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with one JSON object per line, containing individual events from the session.
