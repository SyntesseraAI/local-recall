The SessionStart hook retrieves the 5 most recent memories (sorted by occurred_at) from disk directly, avoiding vector store initialization to prevent slow startup. This is a full reload, not incremental. The vector store is lazy-initialized only in UserPromptSubmit hook for semantic search.
