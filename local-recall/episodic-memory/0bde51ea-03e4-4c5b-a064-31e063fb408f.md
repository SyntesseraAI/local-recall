Added `parseTranscriptForMemories(rawContent: string): MemorySuggestion[]` function in transcript.ts that:
1. Parses raw JSONL content (as read from transcript files)
2. Filters for user/assistant message entries
3. Extracts text and thinking from content blocks
4. Calls analyzeForMemories() to convert messages to memory suggestions
5. Returns array of MemorySuggestion objects ready to be saved

This is the main entry point for processing transcripts and centralizes all parsing logic.
