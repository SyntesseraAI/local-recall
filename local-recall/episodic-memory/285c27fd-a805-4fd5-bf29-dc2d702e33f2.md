# Transcript Collector Implementation

The `transcript-collector.ts` module is responsible for reading and processing Claude Code session transcripts.

## Key Responsibilities

1. **Locate transcripts** - Find transcript files in `~/.claude/projects/<encoded-path>/transcripts/`
2. **Parse JSONL format** - Read transcript files line-by-line as JSON objects
3. **Extract conversation turns** - Process user prompts and assistant responses
4. **Generate memory content** - Convert transcripts into structured memories

## Integration with Local Recall

This module is used by:
- MCP server background daemon (processes transcripts every 5 minutes)
- Memory extraction pipeline (converts raw transcripts to memory files)
- Deduplication logic (uses content hashes to prevent duplicate memories)

The module was undergoing refactoring to improve code consistency (standardized quotes, improved logging) as of the last session.
