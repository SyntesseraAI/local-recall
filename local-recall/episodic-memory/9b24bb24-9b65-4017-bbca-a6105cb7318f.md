The memory-extractor.ts module uses Claude Haiku model to analyze transcripts and extract structured memories. It sends transcripts to Claude with a detailed system prompt that instructs the model to identify key learnings, patterns, and decisions. The extractor returns an array of extracted memories with subject, keywords, applies_to scope, and content fields.
