Each extracted memory must be a JSON object with:
- `subject`: Brief one-line description (max 200 chars)
- `keywords`: Array of 1-10 searchable keywords (lowercase, specific)
- `applies_to`: Scope string - `global`, `file:<path>`, or `area:<name>`
- `content`: Detailed memory in markdown format

Memories are stored as markdown files with YAML frontmatter in `local-recall/episodic-memory/` directory. The content field should be detailed and use markdown formatting for clarity.
