Local Recall configuration supports `memoryDir` (default: `./local-recall`) and `maxMemories` (default: 1000). Settings can be provided via `.local-recall.json` file or environment variables. The MCP server reads `LOCAL_RECALL_DIR` environment variable to find the memory directory.
