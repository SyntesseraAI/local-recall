The MCP server provides five tools for memory management: `memory_create` (idempotent memory creation), `memory_get` (retrieve specific memory by ID), `memory_search` (semantic search using vector embeddings), `memory_list` (list with optional filtering), and `index_rebuild` (sync vector store with memory files). These tools are exposed to any MCP-compatible client including Claude Code and Claude Desktop.
