# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS in JSONL format.

## Path Structure

Project folders use the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `_Users_joe_Code_Syntessera_local-recall`
- Transcripts stored at: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with each line representing a single event or message in the conversation.
