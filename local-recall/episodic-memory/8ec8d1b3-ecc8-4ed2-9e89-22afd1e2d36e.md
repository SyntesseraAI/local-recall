# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line:
- Each line is a valid JSON object
- Can be parsed by reading line-by-line
- Contains complete session history

## Relevance

This is critical for `transcript-collector.ts` which needs to locate and read transcripts to extract memories from Claude Code sessions.
