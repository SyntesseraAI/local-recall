# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by hyphens:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format), where each line is a separate JSON object representing a single message in the conversation.

## Usage in Local Recall

The `transcript-collector` component reads these transcripts to extract memories from Claude Code sessions. The MCP server background daemon syncs transcripts from this location every 5 minutes.
