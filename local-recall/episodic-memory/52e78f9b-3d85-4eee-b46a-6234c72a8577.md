The local-recall system uses Claude Haiku model (via `claude -p --model haiku`) to extract keywords from user prompts and transcripts. This is configured in the UserPromptSubmit and Stop hooks to intelligently identify relevant keywords for memory search and extraction without requiring manual keyword specification.
