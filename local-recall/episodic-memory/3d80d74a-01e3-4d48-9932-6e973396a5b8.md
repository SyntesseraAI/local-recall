# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL files (one JSON object per line):
- Each line is a separate transcript event
- Events contain `type`, `timestamp`, `content`, and other metadata
- Files are named with timestamps (e.g., `session-1733124000.jsonl`)

## Key Details

- Location is consistent across Claude Code sessions
- Transcripts persist after session ends
- Used by MCP server to extract memories asynchronously
- Content hashes track changes to detect new/modified transcripts
