Claude Code transcript files use JSONL format where each line is a JSON object containing a message in the conversation. Each message object has:
- `type`: Message type (user, assistant, etc.)
- `content`: The actual text content of the message
- Potentially other metadata fields

The transcript represents a complete conversation history that can be processed to extract memories about what was discussed, problems solved, and decisions made.
