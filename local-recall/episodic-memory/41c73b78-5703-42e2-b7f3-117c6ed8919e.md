Design decision: Episodic memories default to OFF while thinking memories default to ON. This reflects that thinking memories are more consistently useful for future sessions, while episodic memories require more careful curation. Users can enable episodic memories via `LOCAL_RECALL_EPISODIC_ENABLED=true` environment variable.
