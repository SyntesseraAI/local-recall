Memory deduplication in `findDuplicate()` uses two fields: `occurred_at` (ISO-8601 timestamp) and `content_hash` (SHA-256 prefix). This allows the system to detect when the same memory is created twice (useful when processing transcripts that may be processed multiple times) and avoid duplication.
