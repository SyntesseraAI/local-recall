Configuration can be set via environment variables or `.local-recall.json` file in the project root.

**Available Options:**
- `memoryDir` (default: `./local-recall`) - Directory for memory storage
- `maxMemories` (default: `1000`) - Maximum number of memories to store

This allows projects to customize where memories are stored and set memory limits based on their needs.
