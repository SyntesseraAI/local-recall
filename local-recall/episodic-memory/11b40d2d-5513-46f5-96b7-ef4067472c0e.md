The memory extraction system uses `claude -p --model haiku` to process transcripts and extract memories. The prompt is passed via stdin and uses a structured format defined in `src/prompts/memory-extraction.ts`. This approach allows efficient processing of transcripts to identify valuable memories for future sessions.
