# Keyword Extraction in Memory Processing

When the MCP server's background daemon processes transcripts to extract memories, it uses Claude Haiku to extract keywords from the transcript content.

## Process

1. Read transcript from `~/.claude/projects/<project>/transcripts/`
2. Extract keywords using `claude -p --model haiku` with a prompt asking for keyword extraction
3. Parse the returned keywords
4. Use keywords to create memory entries

## Important Notes

- The keyword extraction happens in the background daemon (every 5 minutes)
- This requires Claude Haiku API access
- Keywords are extracted from the full transcript content
- The extraction process is asynchronous from the session itself
