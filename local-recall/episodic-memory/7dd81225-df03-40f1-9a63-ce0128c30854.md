# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

Project folder names are derived from the absolute working directory path with slashes replaced by underscores and colons removed:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `Users_joe_Code_Syntessera_local-recall`
- Full transcript path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension.

## Usage in Local Recall

The transcript-collector module reads from this location to extract memories. Important when syncing transcripts from Claude's cache for background processing.
