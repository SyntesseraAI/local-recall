# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with forward slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), where each line represents a transcript entry with:
- `timestamp`: ISO-8601 timestamp
- `type`: Event type (e.g., 'user_message', 'assistant_response', 'tool_call')
- `content`: The actual message or event data

This location is important for the MCP server's background daemon which syncs transcripts every 5 minutes.
