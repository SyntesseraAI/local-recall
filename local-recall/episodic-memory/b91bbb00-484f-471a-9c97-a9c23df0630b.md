MCP server exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild. Background daemon syncs transcripts from `~/.claude/projects/<project>/transcripts/`, processes them to extract memories, tracks processed transcripts with content hashes, and runs every 5 minutes. Memory extraction uses claude -p to generate from transcripts.
