The config system loads settings with this precedence:

1. Environment variables (highest priority)
2. `.local-recall.json` file in the project root
3. Default values (lowest priority)

Key configuration options:
- `memoryDir` - Location of memory storage (default: `./local-recall`)
- `maxMemories` - Maximum memory count (default: 1000)
- `episodicEnabled` / `thinkingEnabled` - Toggle memory types
- `episodicMaxTokens` / `thinkingMaxTokens` - Token budget per injection
- `episodicMinSimilarity` / `thinkingMinSimilarity` - Similarity thresholds (0.0-1.0)

Environment variables follow the pattern `LOCAL_RECALL_*` (e.g., `LOCAL_RECALL_EPISODIC_ENABLED`).
