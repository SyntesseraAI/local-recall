The memory extraction system in `memory-extractor.ts` spawns a Claude Haiku subprocess using `spawnSync` to analyze transcripts. The process pipes the transcript and extraction prompt to Claude's stdin, then parses the JSON response from stdout. This approach allows for efficient memory extraction without blocking the main process. The extraction prompt is defined in `src/prompts/memory-extraction.ts`.
