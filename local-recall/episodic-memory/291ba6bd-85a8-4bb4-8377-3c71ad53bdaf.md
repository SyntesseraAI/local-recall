# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path of the project directory:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall` (slashes replaced with hyphens)
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as `.jsonl` (JSON Lines) files where each line is a valid JSON object representing a single turn in the conversation.

## Significance for transcript-collector

The transcript-collector component needs to accurately construct this path to find and read Claude Code session transcripts. This path construction is critical for the memory extraction pipeline that processes conversation history to extract actionable memories.
