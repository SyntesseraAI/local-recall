# Keyword Extraction Request Format

When user asks to extract keywords from text:
1. Return ONLY valid JSON array of strings
2. No explanation, markdown, or code blocks
3. Format: `["keyword1", "keyword2", "keyword3"]`
4. Keywords should be lowercase and specific to the content

This is used for processing memory extraction and understanding user intent.
