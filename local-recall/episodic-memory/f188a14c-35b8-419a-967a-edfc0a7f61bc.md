# Transcript Collector JSONL Parsing

The `TranscriptCollector` class needs to parse JSONL (JSON Lines) format files.

## Format Details

- Each line is a separate JSON object
- No JSON array wrapper
- Lines can be parsed independently
- Files located at `~/.claude/projects/<project>/transcripts/<session-id>.jsonl`

## Implementation Notes

- Must handle streaming or line-by-line parsing
- Each line represents a discrete transcript event/message
- Need to handle potential malformed lines gracefully
