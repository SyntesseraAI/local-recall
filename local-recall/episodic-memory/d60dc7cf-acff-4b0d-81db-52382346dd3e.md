# Transcript Collector Implementation

The `transcript-collector.ts` module is responsible for reading and parsing Claude Code transcripts.

## Key Functionality

- Reads JSONL transcript files from Claude's cache
- Parses transcript events (user prompts, assistant responses, tool invocations)
- Extracts structured information from transcripts
- Handles transcript validation and error cases

## Usage Context

This module is used by the MCP server's background daemon to:
1. Sync transcripts from `~/.claude/projects/<project>/transcripts/`
2. Extract memories from transcript content
3. Track processed transcripts using content hashes
4. Detect transcript changes for re-processing
