# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts are stored as: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files, where each line is a separate JSON object representing an event in the conversation.

## Key Details

- This is used by the `transcript-collector` in `src/core/transcript-collector.ts` to locate and read transcripts
- The path construction must account for the macOS-specific location and the slash-to-dash conversion
- Transcripts accumulate over sessions and can be used to extract memories for the memory system
