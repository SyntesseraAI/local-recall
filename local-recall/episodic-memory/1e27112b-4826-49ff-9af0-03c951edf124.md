Vector embeddings are generated via Ollama with the 'nomic-embed-text' model producing 768-dimensional vectors. Ollama must be installed and running (ollama serve). The OLLAMA_BASE_URL defaults to http://localhost:11434. When migrating from fastembed (384 dimensions), vector indexes must be deleted and rebuilt.
