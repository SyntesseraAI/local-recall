# Transcript Collector Implementation

The `transcript-collector.ts` module handles extracting keywords and memories from transcripts.

## Key Components

- **Claude API Integration**: Uses `claude -p` command to extract keywords via streaming
- **Keyword Extraction**: Processes transcripts to identify searchable keywords
- **Memory Creation**: Converts extracted data into memory records
- **Error Handling**: Robust error handling for API failures and malformed data

## Important Pattern

When calling Claude's keyword extraction, wrap the content in a proper JSON structure and use streaming mode (`-p` flag) for efficiency.
