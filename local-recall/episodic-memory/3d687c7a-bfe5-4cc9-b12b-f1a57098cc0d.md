# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute work directory path using a specific encoding:
- Replace slashes with hyphens
- Convert to lowercase
- Remove special characters

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in the project folder:
- Location: `~/.claude/projects/{project-name}/transcripts/`
- Filename format: Sequential files like `0.jsonl`, `1.jsonl`, etc.
- Each line contains a JSON object representing a single interaction

This is important for the transcript-collector daemon that needs to locate and process transcripts for memory extraction.
