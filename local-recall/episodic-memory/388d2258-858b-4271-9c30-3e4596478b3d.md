# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with timestamp-based filenames:
- Example: `2025-12-01T12-30-45-UTC.jsonl`
- Each line contains a transcript event object

## Transcript Event Structure

Each event has:
- `type`: Event type (e.g., 'message', 'tool_call', 'tool_result')
- `timestamp`: ISO-8601 formatted timestamp
- `role`: 'user' or 'assistant'
- `content`: Event content (varies by type)

This information is essential for building features that read and process Claude's session transcripts.
