# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name converts absolute paths by replacing slashes with dashes:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored as JSONL files in `~/.claude/projects/<converted-path>/transcripts/`

## Implementation Notes

- The `transcript-collector.ts` utility handles this path conversion
- This is critical for the MCP server daemon that processes transcripts asynchronously
- Transcripts track conversation history across sessions
- The daemon runs every 5 minutes to sync and process new transcripts
