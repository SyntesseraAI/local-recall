The Memory Manager (`src/core/memory.ts`) provides core operations:

- `createMemory(data)` - Creates new memory or returns existing if duplicate
- `deleteMemory(id)` - Deletes a memory by ID
- `getMemory(id)` - Retrieves a specific memory
- `listMemories(filter?)` - Lists all memories with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Checks for existing duplicates

Memories are idempotent - there's no update operation. Duplicate detection uses both `occurred_at` timestamp and SHA-256 content hash prefix (16 chars) to prevent duplicates from being created.
