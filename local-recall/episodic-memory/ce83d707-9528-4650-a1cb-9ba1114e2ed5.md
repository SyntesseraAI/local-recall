# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Slashes (`/`) are replaced with dashes (`-`)
- Leading dash is removed for root-level projects
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

Transcripts are stored as JSONL files within each project folder, with filenames following Claude's internal naming convention.

## Implementation Detail

The `transcript-collector.ts` module handles:
- Path conversion from working directory to Claude's transcript folder naming scheme
- Scanning `~/.claude/projects/` for matching project transcripts
- Parsing JSONL transcript files
- Extracting memory content from transcripts
