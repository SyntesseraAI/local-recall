Claude stores projects in ~/.claude/projects/ using a path-to-dashes naming convention. For example, `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`. The TranscriptCollector must convert the current working directory path using `.replace(/\//g, '-').replace(/^-/, '')` to find the corresponding project folder. Transcripts are stored directly in the project folder (not in a `transcripts` subfolder).
