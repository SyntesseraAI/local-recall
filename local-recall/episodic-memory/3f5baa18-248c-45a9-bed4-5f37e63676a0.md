# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are stored as JSONL files (one JSON object per line) with naming pattern: `transcript_*.jsonl`

## Implementation Notes

For the `transcript-collector`, the path conversion logic needs to:
1. Take the absolute working directory path
2. Replace all forward slashes with dashes
3. Look for transcripts in `~/.claude/projects/{converted-path}/transcripts/`

This is critical for `TranscriptCollector.collectTranscripts()` to find and process Claude's session transcripts.
