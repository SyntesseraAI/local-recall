The transcript collector in `src/core/transcript-collector.ts` reads JSONL-formatted transcript files from Claude's cache directory. Each line is a JSON object representing an event in the session. The collector parses these events to extract relevant information for memory creation. This process requires proper error handling for malformed JSONL and handling of different event types.
