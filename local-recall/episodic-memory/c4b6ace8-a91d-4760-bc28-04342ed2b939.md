The markdown.ts utility contains functions for parsing YAML frontmatter from memory files. Memory files use YAML frontmatter to store metadata like id, subject, keywords, applies_to, occurred_at, and content_hash before the markdown content. This utility is critical for extracting metadata when reading memory files.
