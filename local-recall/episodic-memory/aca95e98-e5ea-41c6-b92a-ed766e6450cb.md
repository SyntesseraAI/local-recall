# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude projects folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files with names like `transcript_1732968149638.jsonl`.

## Relevance

This is critical for the transcript-collector component which needs to locate and read Claude's transcript files to extract memories. The path transformation (slashes to dashes) is essential for correct file discovery.
