# Embedding Model and Semantic Search

Local Recall integrates an embedding model for semantic search capabilities:

- **Model**: BGE-small-en-v1.5 from fastembed library (~133MB)
- **Cache Location**: `local_cache/` directory (auto-downloaded on first use)
- **First Run**: Initial startup takes 30-60 seconds while the model downloads
- **Subsequent Runs**: Model loads from cache (fast startup)

## Files Added for Embedding Support
- `src/core/embedding.ts`: Handles embedding generation
- `src/core/vector-store.ts`: Manages vector storage and similarity search

This enables semantic search beyond keyword matching for better memory retrieval.
