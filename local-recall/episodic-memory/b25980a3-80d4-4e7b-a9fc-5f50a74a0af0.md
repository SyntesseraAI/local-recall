The SessionStart hook executes when a Claude Code session begins. It:
1. Loads the memory index from `local-recall/index.json`
2. Retrieves relevant memories based on current context (cwd, file paths)
3. Outputs memory content to stdout for injection into Claude's initial context

This ensures Claude starts each session with relevant project knowledge and memories from previous work.
