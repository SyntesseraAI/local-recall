The MCP server (src/mcp-server/server.ts) runs a background daemon that syncs transcripts from Claude's cache every 5 minutes. It processes transcripts using `claude -p` to extract memories, tracks processed transcripts with content hashes for change detection, and handles deleting/recreating memories when transcripts change. Exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild.
