Transcripts are stored as JSONL (JSON Lines) format where each line is a valid JSON object. Each object represents a single turn containing:
- Role (user/assistant)
- Content of the message
- Optional metadata about tools used
- Timestamps

This format allows efficient streaming and line-by-line processing.
