# Claude Project Transcripts Location

Claude Code stores project transcripts in a cache directory on macOS:

## Path Structure

- **Base directory**: `~/.claude/projects/`
- **Project folder naming**: The absolute path with slashes replaced by dashes
  - Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- **Transcript storage**: `~/.claude/projects/<project-folder>/transcripts/`
- **Filename format**: JSONL files (JSON Lines - newline-delimited JSON)

## File Format

Transcript files are stored in JSONL format where each line is a separate JSON object representing a transcript event.

## Relevance

This is crucial for the MCP server daemon that processes transcripts to extract memories. The daemon needs to sync transcripts from this cache location and track processed transcripts using content hashes.
