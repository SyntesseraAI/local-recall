# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the working directory path by replacing forward slashes with dashes and removing leading dashes.

### Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files:
- One JSON object per line
- Each line represents a single transcript event
- Files are named with timestamps or sequential numbers

## Key Implementation Detail

When implementing transcript collection (src/core/transcript-collector.ts), account for:
1. The dash-based path transformation (not slash-based)
2. Multiple transcript files may exist in the transcripts directory
3. Must handle the home directory (`~`) expansion to the actual user home path
