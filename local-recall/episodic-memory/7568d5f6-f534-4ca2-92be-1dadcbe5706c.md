# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Path Format

The project folder name encodes the absolute working directory with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This naming convention allows multiple projects to coexist without conflicts and makes it possible to map transcripts back to their source projects.

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a complete JSON object representing a session event.
