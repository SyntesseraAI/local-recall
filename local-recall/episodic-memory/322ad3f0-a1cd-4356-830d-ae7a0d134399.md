Session transcripts are condensed into a specific event format:
- `[User]` - User requests or questions
- `[Assistant]` - Claude responses or explanations
- `[Tool: Name]` - Tool invocations (Read, Edit, Write, Bash, Grep, etc.)
- `[Result: OK/ERROR]` - Outcomes of tool invocations

This condensed format is used for memory extraction analysis.
