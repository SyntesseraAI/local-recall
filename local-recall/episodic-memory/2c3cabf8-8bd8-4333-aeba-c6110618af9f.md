# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with names like:
- `transcript_<timestamp>.jsonl`

Each line in a JSONL file is a complete JSON object representing a single transcript event.

## Relevance

This is critical for the transcript-collector functionality - it needs to know exactly where to find transcripts to process them for memory extraction.
