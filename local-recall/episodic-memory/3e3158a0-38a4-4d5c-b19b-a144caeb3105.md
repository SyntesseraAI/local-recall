Development requirements:
- Node.js 18+
- TypeScript 5.9+
- npm scripts: `build`, `dev` (watch mode), `test`, `lint`, `mcp:start`
- Testing: `test` (all), `test:unit`, `test:integration`
- ALL new features must have unit tests in `tests/unit/` and integration tests in `tests/integration/`
- Documentation must be placed in `docs/` folder
- Tests should cover both happy paths and edge cases
