The `transcript-collector.ts` module is responsible for:
1. Discovering transcript files from Claude's cache directory
2. Converting OS-specific paths to the correct format for each platform
3. Parsing JSONL files where each line is a separate turn object
4. Extracting conversation content for memory extraction

Key considerations:
- Must handle OS-specific path conventions (Windows backslashes vs Unix forward slashes)
- Project folder name is the basename of the working directory, not the full path
- Transcripts are line-delimited JSON, requiring line-by-line parsing
