# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts JSONL file: `transcripts.jsonl`

## File Format

Transcripts are stored in JSONL (JSON Lines) format, with each line containing:
- `type`: Event type (e.g., "assistant", "user", "tool_call", "tool_result")
- `timestamp`: When the event occurred
- `content`: The actual content of the message/event
- Additional metadata depending on event type

This is important for the transcript-collector to parse and process transcripts for memory extraction.
