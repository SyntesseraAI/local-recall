# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path of the project with special character handling:
- Slashes `/` are converted to underscores `_`
- Colons `:` are converted to underscores `_`
- Other special characters may also be converted

Example:
- Project at: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`

## Transcript Files

Each transcript is a JSONL file (JSON Lines format) stored in the `transcripts/` subdirectory:
- File format: `.jsonl`
- Location: `~/.claude/projects/<project-folder>/transcripts/`
- Each line is a valid JSON object representing a transcript event

This is important for the transcript-collector which needs to locate and parse transcripts from Claude's cache.
