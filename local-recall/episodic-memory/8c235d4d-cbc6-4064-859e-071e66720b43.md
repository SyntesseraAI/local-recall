Local Recall functions as both a Claude Code plugin and an MCP server. The plugin root contains:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook configuration for SessionStart, UserPromptSubmit, and Stop events
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (build output)

Hooks use `${CLAUDE_PLUGIN_ROOT}` for relative references and receive `cwd` via stdin to operate on the user's project directory.
