The user-prompt-submit hook extracts keywords from the user's prompt using `claude -p --model haiku` to run a prompt pipeline. These keywords are then used to search the memory index with fuzzy matching to find relevant memories. The extracted keywords improve search accuracy by using Claude's understanding rather than simple string matching.
