# Hook Execution via stdin/stdout

Claude Code hooks communicate with scripts through stdin/stdout using JSON format.

## How It Works

1. Claude Code invokes the hook script as a subprocess
2. Input data (session info, transcript path, prompt, etc.) is passed via stdin as JSON
3. The hook script reads from stdin, processes the data, and outputs results via stdout
4. Output is also JSON formatted and returned to Claude

## Key Points

- Hooks receive `cwd` parameter to identify the project directory
- Multiple hooks can be chained (SessionStart, UserPromptSubmit, Stop)
- Timeout is configurable per hook
- This is the standard integration point for Claude Code plugins

## Files Involved

- Hook implementation: `src/hooks/`
- Compiled scripts: `scripts/hooks/`
