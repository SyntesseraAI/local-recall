The UserPromptSubmit hook (src/hooks/user-prompt-submit.ts:71) limits search results to 5 memories maximum. Each result includes the full memory content plus similarity percentage. This limit balances context size with retrieval quality - enough results to find relevant information but not so many as to bloat the context window.
