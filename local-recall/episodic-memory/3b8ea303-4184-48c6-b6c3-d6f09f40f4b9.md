# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Files are named with timestamps or session identifiers
- Format: JSONL (JSON Lines) - one JSON object per line
- Each line represents a transcript event

## Key Details

- This is how the MCP server's background daemon finds and processes transcripts
- The path is deterministic based on the working directory
- Useful for debugging or manually accessing transcript history
