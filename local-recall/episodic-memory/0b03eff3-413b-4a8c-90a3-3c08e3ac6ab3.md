# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a separate JSON object representing a transcript event.

## Location Strategy

When collecting transcripts:
1. Get the working directory from context
2. Replace `/` with `-` to create the project folder name
3. Look in `~/.claude/projects/{project-folder}/transcripts/`
4. Process all `.jsonl` files in that directory
