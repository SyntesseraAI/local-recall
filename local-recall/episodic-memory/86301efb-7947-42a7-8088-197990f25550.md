# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in the `transcripts/` directory:
- Filename format: `<session-id>.jsonl`
- Each line is a complete JSON object representing a transcript event
- Objects contain: `type`, `timestamp`, `content`, and other metadata depending on event type

## Implementation Note

When implementing transcript collection (e.g., in `transcript-collector.ts`), use this exact path format for reliable transcript discovery across different project directories.
