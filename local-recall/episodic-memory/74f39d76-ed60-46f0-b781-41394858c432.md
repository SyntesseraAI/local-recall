The MCP server in `src/mcp-server/` exposes memory management tools:

Available tools:
- `memory_create` - Create new memory (idempotent)
- `memory_delete` - Delete memory
- `memory_search` - Search by keywords
- `memory_list` - List all memories
- `index_rebuild` - Rebuild keyword index

Background daemon functionality:
- Syncs transcripts from `~/.claude/projects/<project>/transcripts/`
- Processes transcripts using `claude -p` to extract memories
- Uses content hashes for change detection
- Runs every 5 minutes
- Deletes and recreates memories when transcripts change

Configuration via environment variable `LOCAL_RECALL_DIR` points to memory storage directory.
