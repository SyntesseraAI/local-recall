Claude Code stores project transcripts at `~/.claude/projects/<encoded-path>/` where the encoded path matches the repository folder name. Example: `/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/` for the local-recall project at `/Users/joe/Code/Syntessera/local-recall`. Transcripts are stored as JSONL files in the `transcripts/` subdirectory and can be processed to extract memories.
