# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with:
- All leading/trailing slashes removed
- Internal slashes converted to dashes
- Path converted to lowercase

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/users-joe-code-syntessera-local-recall/`
- Transcripts stored as: `*.jsonl` files

This is important for the transcript-collector to correctly locate and read transcripts from Claude Code sessions.
