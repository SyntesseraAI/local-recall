# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Taking the absolute path
2. Replacing forward slashes with underscores
3. Removing leading underscores

**Example:** Working directory `/Users/joe/Code/Syntessera/local-recall` maps to `Users_joe_Code_Syntessera_local-recall`

Full transcript path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

Transcripts are stored as JSONL files with ISO-8601 timestamps in the filename.
