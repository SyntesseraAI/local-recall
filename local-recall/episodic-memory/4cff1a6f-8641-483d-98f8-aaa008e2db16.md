Memory files are stored in the `local-recall/episodic-memory/` directory as markdown files. The directory structure includes:
- `local-recall/index.json` - Keyword index cache (gitignored)
- `local-recall/recall.log` - Debug log file (gitignored)
- `local-recall/episodic-memory/` - Individual memory files (version-controlled)

Memory files use a consistent naming pattern and YAML frontmatter for metadata.
