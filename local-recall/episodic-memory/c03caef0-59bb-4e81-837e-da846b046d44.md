# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude projects folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts are stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Each transcript is a JSONL file named with a session ID

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files where each line is a JSON object representing a transcript event.

## Usage in local-recall

The `transcript-collector.ts` reads from this location to extract memories asynchronously, allowing the MCP server daemon to process transcripts independently of the hooks system.
