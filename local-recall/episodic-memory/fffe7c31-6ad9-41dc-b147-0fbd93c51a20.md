# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall`
- Transcripts folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one event per line.

## Implementation

The `transcript-collector.ts` module handles:
1. Building the correct path from the current working directory
2. Reading JSONL transcript files
3. Parsing individual transcript events
4. Extracting memories from transcript content

This is used by the MCP server's background daemon to process transcripts every 5 minutes.
