# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Format: `.jsonl` (JSON Lines - one JSON object per line)
- Each entry contains the complete session transcript with all tool calls and results

## Usage in Code

This information is critical for the transcript-collector component which needs to:
1. Locate transcripts programmatically
2. Parse JSONL format correctly
3. Handle macOS-specific path conventions
