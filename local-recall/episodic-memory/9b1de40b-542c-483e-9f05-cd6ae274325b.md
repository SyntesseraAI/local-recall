# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension, not `.json`. Each line is a separate JSON object representing an event in the transcript.

## Implementation Note

When implementing transcript collection in `src/core/transcript-collector.ts`, use `path.resolve(os.homedir(), '.claude/projects/')` and replace forward slashes with dashes when constructing the project folder name.
