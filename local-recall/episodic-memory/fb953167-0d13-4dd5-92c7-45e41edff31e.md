# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one per session.

## Relevance to Local Recall

This is critical for the MCP server's background daemon which syncs transcripts from this location to extract memories automatically. The daemon scans this directory every 5 minutes to find new transcripts for processing.
