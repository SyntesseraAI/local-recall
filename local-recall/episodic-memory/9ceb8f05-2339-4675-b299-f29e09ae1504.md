# Embedding and Vector Store Implementation

Local Recall uses semantic embeddings for memory search via:

1. **Model**: BGE-small-en-v1.5 (via fastembbed library)
2. **Dimension**: 384-dimensional vectors
3. **Storage**: Local vector store implementation
4. **Use case**: Enables semantic similarity search beyond keyword matching

Implementation files:
- `src/core/embedding.ts` - Embedding model integration
- `src/core/vector-store.ts` - Vector storage and retrieval

The embedding model (~133MB) is cached locally in `local_cache/` and downloads automatically on first use. This provides memory search capabilities beyond simple fuzzy keyword matching.
