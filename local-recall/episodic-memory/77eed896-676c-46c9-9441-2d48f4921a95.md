The MCP server daemon syncs transcripts from Claude's cache (~/.claude/projects/<project>/transcripts/), processes them using 'claude -p' to extract memories, and tracks processed transcripts with content hashes for change detection. Deletes and recreates memories when transcripts change. Runs every 5 minutes automatically, enabling asynchronous memory management without blocking Claude sessions.
