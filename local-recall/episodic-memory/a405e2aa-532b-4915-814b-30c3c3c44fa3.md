# Transcript File Format

Claude Code stores transcripts as JSONL files (one JSON object per line) in `~/.claude/projects/<project>/transcripts/`.

## File Structure

Each line is a JSON object containing transcript events with:
- `type`: Event type (e.g., 'user', 'assistant', 'tool')
- `role`: 'user' or 'assistant'
- `content` or `text`: Message content
- `timestamp`: ISO-8601 timestamp
- Additional metadata depending on event type

This structure allows transcript-collector to parse and process individual transcript events for memory extraction.
