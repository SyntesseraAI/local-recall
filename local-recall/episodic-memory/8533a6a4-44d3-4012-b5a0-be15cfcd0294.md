# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Work directory: `/Users/joe/Code/Syntessera/local-recall`
- Cache folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with naming pattern:
- Format: `transcript_<timestamp>.jsonl`
- Example: `transcript_1733154600002.jsonl`
- Content: Line-delimited JSON events

## Implementation Details

The `transcript-collector.ts` module handles:
1. Building the cache folder path from `cwd` by replacing `/` with `-`
2. Reading JSONL transcript files from the transcripts directory
3. Parsing events for memory extraction

This is critical for the MCP server's background daemon that syncs transcripts every 5 minutes.
