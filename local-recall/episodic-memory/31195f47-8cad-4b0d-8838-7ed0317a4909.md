Local Recall has three core memory management components in `src/core/`:

1. **Memory Manager** (`memory.ts`) - CRD operations (no update):
   - `createMemory(data)` - Returns existing if duplicate
   - `deleteMemory(id)`
   - `getMemory(id)`
   - `listMemories(filter?)`
   - `findDuplicate(occurredAt, contentHash)`

2. **Index Manager** (`index.ts`) - Keyword indexing:
   - `buildIndex()` - Scan all memory files
   - `getIndex()` - Retrieve current index
   - `refreshIndex()` - Rebuild from scratch
   - Index stored at `local-recall/index.json`

3. **Search** (`search.ts`) - Fuzzy matching:
   - `searchByKeywords(query)`
   - `searchBySubject(query)`
   - `searchByScope(scope)`
