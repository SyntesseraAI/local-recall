# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Each transcript line contains: `{"role": "user"|"assistant", "content": "...", "timestamp": "..."}` (or similar structure)

## Key Insights

- This is important for the MCP server background daemon that syncs transcripts
- The daemon needs to read these JSONL files to extract memories
- Understanding this path structure is critical for implementing transcript processing
- The path transformation (slashes to dashes) must be handled correctly when constructing the path
