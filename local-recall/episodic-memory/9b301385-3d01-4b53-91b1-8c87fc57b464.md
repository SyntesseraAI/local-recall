# Transcript Condensing for Memory Extraction

The `TranscriptCollector` class in `src/core/transcript-collector.ts` condenses verbose JSONL transcripts into a compact format suitable for memory extraction.

## Purpose

Reduce token usage when sending transcripts to Claude for memory extraction by:
- Removing verbose metadata
- Compressing event information
- Maintaining essential context

## Format

Condensed transcripts use a simplified notation:
- `[User]` - User messages
- `[Assistant]` - Assistant responses
- `[Tool: Name]` - Tool invocations with tool name
- `[Result: OK/ERROR]` - Tool execution outcome

This condensed format makes transcripts more token-efficient while preserving the key information needed for memory extraction.
