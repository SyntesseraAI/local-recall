Local Recall uses fastembed with BGE-small-en-v1.5 model for semantic search. First run downloads ~133MB to `local_cache/fast-bge-small-en-v1.5/`. Initial startup takes 30-60 seconds; subsequent runs load from cache. If tokenizer.json missing error occurs, remove `local_cache/fast-bge-small-en-v1.5*` to re-download.
