When duplicate functionality is discovered in multiple modules, extract it to a shared utility module. This improves maintainability, reduces bugs, and makes future changes easier. For the gitignore case: create a utility function that handles the creation and update of gitignore files, then use it from both memory-extractor.ts and index.ts.
