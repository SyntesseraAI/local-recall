The MCP server daemon automatically syncs transcripts from Claude's cache (~/.claude/projects/<project>/transcripts/), processes them to extract memories, and tracks processed transcripts with content hashes for change detection. Memory deletion/recreation occurs when transcripts change. The daemon runs every 5 minutes.
