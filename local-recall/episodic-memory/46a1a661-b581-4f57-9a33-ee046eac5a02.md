# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path with slashes converted to hyphens.

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall`

## Transcript Storage

Transcripts are stored as JSONL files in `~/.claude/projects/<project-name>/transcripts/`

Each transcript file contains condensed session events that can be parsed to extract memories for the MCP server daemon.
