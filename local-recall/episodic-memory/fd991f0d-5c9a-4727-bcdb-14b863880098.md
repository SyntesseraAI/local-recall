The `MemoryExtractor` class in `src/core/memory-extractor.ts` provides an `extract()` method that:
1. Takes a transcript string and optional user ID as parameters
2. Invokes Claude Haiku process with `execCommand()` using the extraction prompt
3. Parses the returned JSON containing memories array
4. Returns extracted memories or empty array on failure
5. Includes logging for debugging extraction process
6. Handles edge cases where Claude output may not be valid JSON
