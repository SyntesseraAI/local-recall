# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using a hash or encoding scheme. For a project at `/Users/joe/Code/Syntessera/local-recall`, the transcripts directory would be located at:

`~/.claude/projects/<encoded-project-name>/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` (JSON Lines) format files, with each line representing a single event in the conversation.

## Key Insight for Memory Extraction

When implementing transcript collection and memory extraction, the `transcript-collector.ts` module needs to correctly:
1. Calculate or resolve the project folder name from the working directory
2. Locate the correct `.claude/projects/` subdirectory
3. Parse `.jsonl` files line-by-line for event processing

This is critical for the background daemon and MCP server memory extraction features.
