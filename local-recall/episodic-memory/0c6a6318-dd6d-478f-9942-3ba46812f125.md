Local Recall project root IS the plugin root. Required structure: `.claude-plugin/plugin.json` for metadata, `hooks.json` for hook event handlers, `.mcp.json` for MCP configuration, and `scripts/` directory (build output, gitignored) containing bundled hooks and MCP server. Plugin can be installed by path or as npm package.
