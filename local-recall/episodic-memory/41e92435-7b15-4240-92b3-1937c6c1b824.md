The stop hook (src/hooks/stop.ts) processes transcripts to extract memories. It uses the rake-pos library for keyword extraction (RAKE algorithm with POS tagging). The hook:
1. Reads transcript from path provided in hook input
2. Parses transcript content
3. Extracts keywords using RAKE + POS tagging
4. Creates memory records with these keywords
5. Requires npm dependencies to be installed (rake-pos not available without npm install)
