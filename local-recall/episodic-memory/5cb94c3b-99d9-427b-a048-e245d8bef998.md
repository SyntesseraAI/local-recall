# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Files: Each transcript is a separate `.jsonl` file
- Format: Each line is a JSON object representing a conversation turn

This is relevant for the transcript collector functionality that needs to read and process Claude Code session transcripts.
