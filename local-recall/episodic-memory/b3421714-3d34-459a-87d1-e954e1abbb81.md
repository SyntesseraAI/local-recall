Memory files are stored in `local-recall/episodic-memory/` directory with filenames based on their UUID (e.g., `03c44340-51b3-49c6-923b-ea795fe55ecc.md`). The `.gitignore` file in `local-recall/` auto-generates to exclude `memory.sqlite` and `recall.log` but includes memory markdown files.
