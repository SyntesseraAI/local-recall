The memory extraction system uses Claude Haiku API to analyze session transcripts and extract valuable memories. The system:

1. Uses a structured prompt from `src/prompts/memory-extraction.ts` to guide Claude's analysis
2. Calls Claude Haiku to analyze transcript content
3. Returns a JSON object with a `memories` array containing extracted memories

Key implementation details:
- Uses `anthropic.messages.create()` API
- Processes transcript content as system context
- Returns structured JSON with memory objects containing: subject, keywords, applies_to, and content fields
