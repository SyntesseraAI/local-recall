# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Path Encoding Convention

Project folder names encode the absolute path with:
- Forward slashes `/` replaced with dashes `-`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files in the project folder, with one JSON object per line representing transcript events.

## Reference

This location and format is used by `transcript-collector.ts` to sync and process Claude sessions.
