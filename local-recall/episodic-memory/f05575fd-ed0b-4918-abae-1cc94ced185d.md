# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path using a consistent algorithm:
- Each directory in the path is converted to a "slug" format
- Slugs remove special characters and use hyphens as separators
- The full slug is computed by joining slugified path components with hyphens

## Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Path components: `users`, `joe`, `code`, `syntessera`, `local-recall`
- Result: `~/.claude/projects/users-joe-code-syntessera-local-recall/`
- Transcripts stored in: `transcripts/` subdirectory as JSONL files

## Relevance to TranscriptCollector

The `transcript-collector.ts` module should handle dynamic path computation for accessing Claude's transcript cache. This is needed for the MCP server daemon to sync and process transcripts asynchronously.
