# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files inside the project folder:
- Location: `~/.claude/projects/{encoded-path}/transcripts/`
- Format: JSONL (JSON Lines - one JSON object per line)
- Extension: `.jsonl`
- Each line contains a transcript event

## Usage in Local Recall

The transcript collector in `src/core/transcript-collector.ts` needs to:
1. Take the working directory (passed via stdin as `cwd`)
2. Encode it with slashes replaced by dashes
3. Read from `~/.claude/projects/{encoded-path}/transcripts/`
4. Parse JSONL format to extract transcript events

This is essential for the Stop hook and background daemon to process transcripts and extract memories.
