# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Taking the full absolute path
2. Converting forward slashes `/` to dashes `-`
3. Removing the leading dash

**Example:**
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), where each line represents a single message or event in the conversation.

## Implementation Note

This path format needs to be properly handled in `transcript-collector.ts` when building the path to fetch transcripts. The implementation should construct the path from the working directory using this dash-conversion logic.
