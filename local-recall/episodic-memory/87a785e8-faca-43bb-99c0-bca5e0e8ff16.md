# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes converted to underscores. For example:

- Project at `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`
- Transcript files: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/*.jsonl`

## Implementation Details

When implementing transcript collection, the `transcript-collector.ts` module needs to handle this path conversion properly. The folder structure uses underscores as separators for the absolute path to create a valid filesystem directory name.

## Key Points

- Transcripts are stored in JSONL format
- The path conversion is crucial for locating transcripts correctly
- This applies to macOS; other platforms may have different locations
- Used in the SessionStart hook to load and process project transcripts
