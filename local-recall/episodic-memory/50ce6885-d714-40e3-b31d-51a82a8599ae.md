# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with `.jsonl` extension.

## Implementation

The `transcript-collector.ts` module handles:
1. Converting working directory paths to Claude's project folder name format
2. Locating transcripts in the `~/.claude/projects/` directory
3. Reading and parsing JSONL transcript files

This is critical for the MCP server's background daemon that syncs transcripts for memory extraction.
