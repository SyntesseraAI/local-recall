The MCP server (src/mcp-server/) exposes tools:
- memory_create, memory_get, memory_search, memory_list, index_rebuild

It also runs a background daemon that every 5 minutes:
- Syncs transcripts from Claude cache
- Processes transcripts to extract memories
- Tracks processed transcripts by content hash for change detection
- Deletes and recreates memories when transcripts change
