Memory files are structured markdown documents with YAML frontmatter containing:
- `subject`: One-line description (max 200 chars)
- `keywords`: Array of searchable terms (lowercase, specific)
- `applies_to`: Scope indicator (`global`, `file:<path>`, or `area:<name>`)
- `content`: Detailed memory in markdown format

This structured format enables semantic search and filtering across the memory system.
