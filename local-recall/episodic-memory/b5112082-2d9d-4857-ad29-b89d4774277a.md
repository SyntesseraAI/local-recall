The MCP server implements memory tools exposed to MCP-compatible clients. Available tools:
- `memory_create`: Create new memory (idempotent)
- `memory_delete`: Delete a memory by ID
- `memory_search`: Search memories by keywords (fuzzy)
- `memory_list`: List all memories with optional filtering
- `index_rebuild`: Rebuild memory index

The server runs a background daemon that syncs transcripts from Claude's cache every 5 minutes, processes them with claude -p to extract memories, and tracks changes using content hashes.
