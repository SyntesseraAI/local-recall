Two main integration points:
1. **SessionStart Hook**: Loads 5 most recent memories on session start (full reload, no vector store to avoid slow initialization)
2. **UserPromptSubmit Hook**: Performs semantic search using vector embeddings before Claude processes user input
3. **MCP Server**: Exposes memory tools (create, get, search, list, rebuild index) and runs background daemon that syncs transcripts every 5 minutes
