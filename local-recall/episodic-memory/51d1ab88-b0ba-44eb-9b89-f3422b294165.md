Each memory file must include:
- `id`: UUID
- `subject`: Brief one-line description
- `keywords`: Array of searchable terms
- `applies_to`: Scope (global, file:<path>, area:<name>)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix for deduplication

Memories are stored in `local-recall/episodic-memory/` and are tracked in git.
