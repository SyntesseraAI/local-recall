# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name in the path uses **dashes instead of slashes** to represent the file system path:

- Physical path: `/Users/joe/Code/Syntessera/local-recall`
- Claude projects folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each transcript is a JSONL file with a timestamp-based name like `1733212800000.jsonl`.

## Implementation Note

When building path converters for `transcript-collector.ts`, account for this dash-conversion behavior when looking up transcripts from a given working directory path.
