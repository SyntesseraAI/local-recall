Each memory file is a markdown document with YAML frontmatter containing: id (UUID), subject (brief description), keywords (array for search), applies_to (scope: global/file:/path/area:name), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix for deduplication). The content_hash field enables duplicate detection to prevent creating duplicate memories from the same event.
