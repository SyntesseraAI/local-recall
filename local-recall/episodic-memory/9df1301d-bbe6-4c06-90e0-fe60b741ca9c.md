# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with timestamps:
- Format: `transcript_<timestamp>.jsonl`
- Example: `transcript_2025-12-01T10-30-45.jsonl`

## Content Format

Each line in the JSONL file is a JSON object representing a transcript event with:
- `type`: Event type (e.g., 'user_message', 'assistant_response')
- `content`: The actual message or response
- `timestamp`: When the event occurred

This location is used by the transcript-collector to sync transcripts for memory extraction.
