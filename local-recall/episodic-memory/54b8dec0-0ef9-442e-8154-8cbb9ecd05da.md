The memory extraction prompt templates in src/prompts/memory-extraction.ts define the instructions and guidelines that Claude uses when extracting memories from transcripts. These templates specify what to extract (architectural decisions, bug fixes, patterns), what not to extract (generic knowledge, temporary debugging), and the output format (JSON with subject, keywords, applies_to, content).
