Each memory is a markdown file with YAML frontmatter containing:
- `id`: UUID
- `subject`: Brief one-line description
- `keywords`: Searchable terms
- `applies_to`: Scope (global, file:<path>, area:<name>)
- `occurred_at`: ISO-8601 timestamp for deduplication
- `content_hash`: SHA-256 prefix for duplicate detection

Memories are idempotent - `createMemory()` returns existing if duplicate.
