# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path converted to a valid directory name:
- Forward slashes `/` are converted to underscores `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users_joe_Code_Syntessera_local-recall`

## Transcript Files

Transcripts are stored as JSONL files in the format:
`~/.claude/projects/{project-folder-name}/transcripts/{transcript-id}.jsonl`

Each line in the JSONL file is a JSON object representing an event in the transcript.
