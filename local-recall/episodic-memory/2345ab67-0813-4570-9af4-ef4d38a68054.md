# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are JSONL format (JSON Lines - one JSON object per line):
- Files are named with timestamps: `transcript-{timestamp}.jsonl`
- Each line contains a JSON object representing one event/message
- Files are numbered sequentially

## Implementation Notes

- This is used by the transcript-collector to find and process transcripts
- The path conversion from slashes to dashes is critical for correctly locating transcripts
- Understanding this path format is essential for the MCP server's background daemon that syncs transcripts
