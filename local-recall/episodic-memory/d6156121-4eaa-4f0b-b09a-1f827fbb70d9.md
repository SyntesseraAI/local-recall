The marketplace.json file at `.claude-plugin/marketplace.json` controls the version displayed in the Claude Code marketplace. When updating Local Recall, ensure marketplace.json version matches package.json version (currently 0.1.9). If not updated together, users will see stale version information when checking for plugin updates in Claude Code.
