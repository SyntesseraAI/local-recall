# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with `/` replaced by `_`. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `_Users_joe_Code_Syntessera_local-recall`
- Transcripts stored as `.jsonl` files (JSON Lines format)

## Key Implementation Details

- Each line in the JSONL file is a JSON object representing a single event in the transcript
- The transcript collector needs to handle this path conversion to locate project transcripts
- This is essential for the MCP server's background daemon that syncs and processes transcripts

## Path Conversion Logic

When implementing transcript collection:
1. Take the absolute working directory path
2. Replace all `/` with `_`
3. Add leading `_` to complete the folder name format
4. Look in `~/.claude/projects/{converted-path}/` for transcript files
