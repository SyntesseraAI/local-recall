# Transcript Collector Implementation

The `transcript-collector.ts` module handles reading and processing Claude Code transcripts to extract memories.

## Purpose

It scans the Claude projects transcript directory and processes JSONL files to:
1. Identify relevant sessions and transcripts
2. Extract information that should become memories
3. Integrate with the memory creation system

## Key Functions

- `collectTranscripts()` - Main function to gather transcript data
- Integration point between transcript data and memory system
