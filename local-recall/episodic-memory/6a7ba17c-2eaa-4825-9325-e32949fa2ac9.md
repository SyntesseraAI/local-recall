# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons replaced by underscores.

### Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `_Users_joe_Code_Syntessera_local_recall`
- Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local_recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with naming pattern: `transcript_*.jsonl`

Each line in a JSONL file is a JSON object representing a conversation event.
