# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and colons removed:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files with events like:
- `UserPromptSubmit` - User submitted a prompt
- `AssistantThought` - Assistant's internal reasoning
- `ToolStart` / `ToolComplete` - Tool invocations
- `AssistantMessage` - Claude's response

## Important

Claude Code maintains these transcripts internally for session context and memory extraction. The transcript collector in local-recall uses this location to sync and process transcripts for memory creation.
