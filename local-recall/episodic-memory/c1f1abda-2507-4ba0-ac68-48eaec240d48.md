Claude stores project transcripts at `~/.claude/projects/<encoded-project-path>/`. The project directory name is the absolute path with slashes replaced by dashes, e.g., `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`. Transcripts are stored as JSONL files in this directory and can be processed to extract memories for the local-recall system.
