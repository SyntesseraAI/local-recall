# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and URL-encoded characters.

### Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Replace `/` with `-` to get: `Users-joe-Code-Syntessera-local-recall`
- Apply URL encoding for any special characters
- Final path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (newline-delimited JSON) with names matching session identifiers:
- Format: `{session-id}.jsonl`
- Each line is a complete JSON object representing a transcript event
- Files can be read and parsed line-by-line

## Practical Notes

- This path encoding is deterministic and consistent
- The transcript collector in `transcript-collector.ts` uses this pattern to locate transcripts
- Understanding this path structure is crucial for implementing transcript syncing and memory extraction
