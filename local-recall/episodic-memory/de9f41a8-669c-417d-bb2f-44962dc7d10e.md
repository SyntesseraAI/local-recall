# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and a UUID appended:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Pattern: `~/.claude/projects/<project-folder>/transcripts/`
- Format: JSONL (one JSON object per line)
- Each line represents a single message or event in the transcript

This information is crucial for the transcript collector to locate and process Claude Code session transcripts.
