# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) with naming pattern:
- Format: `transcript_<timestamp>.jsonl`
- Example: `transcript_2024-12-01_10-30-45.jsonl`
- Each line contains a complete JSON event object

## Implications for transcript-collector

The transcript-collector must:
1. Construct the project folder name by replacing `/` with `-` in the working directory path
2. Read JSONL files sequentially, parsing one JSON object per line
3. Handle the specific macOS path structure (`~/.claude/projects/`)
