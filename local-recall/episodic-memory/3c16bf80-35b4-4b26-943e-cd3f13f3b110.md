Local Recall's plugin root IS the project root. The structure follows Claude Code's plugin format:
- .claude-plugin/plugin.json for metadata
- hooks.json for event handlers using ${CLAUDE_PLUGIN_ROOT}
- .mcp.json for MCP server configuration
- scripts/ for bundled executables (build output, gitignored)
- src/ for TypeScript source
- local-recall/ for memory storage

Hooks receive `cwd` via stdin to operate on the user's project directory.
