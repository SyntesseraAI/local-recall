Claude stores project transcripts in `~/.claude/projects/` directory. Project directories are named with URL-encoded paths matching the repo folder name. Example: `/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`. This is where the MCP server's background daemon syncs transcripts from for memory extraction.
