Each memory file has YAML frontmatter containing: `id` (UUID), `subject` (brief description), `keywords` (array for search), `applies_to` (global|file:<path>|area:<name>), `occurred_at` (ISO-8601 for deduplication), and `content_hash` (SHA-256 prefix for duplicate detection). The markdown content follows the frontmatter, allowing atomic, searchable memories.
