# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. Example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension. Each line is a valid JSON object representing a transcript event.

## Relevance

This path structure is critical for the transcript collector and MCP server daemon which sync and process transcripts from Claude's cache.
