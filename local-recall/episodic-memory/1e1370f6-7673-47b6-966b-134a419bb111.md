# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files, one event per line:
- Each line is a complete JSON object
- Lines can be parsed independently
- Use `readFileSync()` and split by newline, then `JSON.parse()` each line

## JSON Structure

Each event contains:
- `type`: Event type (e.g., "message", "tool_call", "tool_result")
- `timestamp`: ISO-8601 timestamp
- `data`: Event-specific data structure

This is critical for transcript processing in the memory extraction pipeline.
