# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path with:
- Leading slashes removed
- Path separators (/) converted to dashes (-)

### Example
For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each transcript is a JSONL file containing session events with timestamps.
