# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and prefixed with underscore:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with each line being a separate JSON object representing a message in the conversation.
