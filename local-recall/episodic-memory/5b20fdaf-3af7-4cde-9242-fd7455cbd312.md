Claude Code stores project transcripts in ~/.claude/projects/ with URL-encoded directory paths. The format is: ~/.claude/projects/<url-encoded-path>/<session-id>.jsonl

Example: /Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/ contains transcripts for the local-recall project.

Path encoding replaces '/' with '-' in the directory structure.
