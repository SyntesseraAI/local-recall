Memories are deduplicated using a two-part strategy:
1. **occurred_at timestamp**: Prevents duplicate memories from the same event
2. **content_hash**: SHA-256 prefix comparison ensures semantic uniqueness

The `findDuplicate(occurredAt, contentHash)` method in memory.ts checks for existing memories before creation. This prevents accumulation of duplicate memories across multiple transcript processing runs while preserving semantically different memories from the same time period.
