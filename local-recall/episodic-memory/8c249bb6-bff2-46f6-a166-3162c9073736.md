The MCP server daemon automatically syncs transcripts from Claude's cache directory (~/.claude/projects/<project>/transcripts/), processes them using `claude -p` to extract memories, and tracks processed transcripts with content hashes for change detection. The daemon runs every 5 minutes and handles deletion and recreation of memories when transcripts change.
