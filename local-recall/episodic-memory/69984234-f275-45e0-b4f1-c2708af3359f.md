# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path of the project directory with special characters encoded. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users_joe_Code_Syntessera_local-recall` (slashes replaced with underscores)

## Transcript Files

Transcripts are stored as JSONL files in `transcripts/` subdirectory:
- Path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`
- Files: `*.jsonl` format, one event per line

## Relevance

This is used by the transcript-collector module to sync and process Claude's session transcripts for memory extraction.
