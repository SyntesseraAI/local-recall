Memory files require YAML frontmatter with: `id` (UUID), `subject` (brief description), `keywords` (array), `applies_to` (global/file/area), `occurred_at` (ISO-8601), `content_hash` (SHA-256 prefix). Content follows after frontmatter in markdown. Memories are idempotent and deduplicated using occurred_at + content_hash.
