Search results use cosine distance similarity (0.0-1.0 range, rounded to 2 decimals). Results are sorted by score descending. When scores are equal, more recent memories (by `occurred_at` timestamp) rank first. This recency tie-breaker ensures older, duplicate memories don't shadow newer insights.
