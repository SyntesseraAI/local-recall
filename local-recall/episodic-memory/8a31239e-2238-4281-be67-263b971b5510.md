# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project directory with special character replacements:
- Slashes `/` are replaced with dashes `-`
- Colons `:` are replaced with dashes `-`
- Other special characters may also be replaced with dashes

## Example

Project path: `/Users/joe/Code/Syntessera/local-recall`
Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
Transcripts file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/current.jsonl`

## Implementation Note

When implementing `transcript-collector.ts`, use the `path.join()` function to build paths properly. The transcript files are in JSONL format (JSON Lines - one JSON object per line).
