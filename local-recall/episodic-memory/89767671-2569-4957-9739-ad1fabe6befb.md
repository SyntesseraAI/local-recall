# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

Project folder names use absolute paths with slashes replaced by dashes:
- Example: Project at `/Users/joe/Code/Syntessera/local-recall` maps to folder `_Users_joe_Code_Syntessera_local-recall`
- This encoding preserves the full project path as a valid filesystem name

## Transcript Files

Transcripts are stored as `.jsonl` files within each project folder, making them queryable and processable by the transcript collector.
