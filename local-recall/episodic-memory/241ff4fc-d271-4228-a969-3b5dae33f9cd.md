# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files where each line is a complete JSON object representing one event.

## Key Insight

The transcript collector needs to properly construct this path by:
1. Taking the current working directory
2. Replacing all forward slashes with dashes
3. Looking in `~/.claude/projects/<transformed-path>/transcripts/`

This is critical for the transcript-collector to find and process Claude Code transcripts for memory extraction.
