# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcript files are JSONL format stored in the `transcripts/` subdirectory:
- Path: `~/.claude/projects/<project-folder>/transcripts/`
- Format: Each line is a JSON object representing a transcript event
- Files are named with timestamps or session identifiers

## Importance

This path is essential for the transcript collector functionality in `src/core/transcript-collector.ts`. The MCP server and hooks need to locate and process transcripts from this cache location to extract memories.
