The memory.sqlite database is gitignored and auto-generated from memory files. Individual memory markdown files in local-recall/episodic-memory/ ARE version-controlled and will be committed to git. The .gitignore file in local-recall/ also excludes recall.log. This design allows team sharing of memories while keeping generated indexes local.
