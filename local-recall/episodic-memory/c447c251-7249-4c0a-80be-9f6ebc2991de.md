The memory extraction workflow follows: 1) Read transcript file, 2) Send to Claude Haiku with extraction prompt, 3) Parse returned JSON memories, 4) Validate extracted data, 5) Check for duplicates, 6) Store valid memories to `local-recall/episodic-memory/`. This multi-step approach ensures quality and prevents duplicate memories from being stored.
