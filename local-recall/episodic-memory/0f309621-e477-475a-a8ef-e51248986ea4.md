# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and the trailing slash removed. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `_Users_joe_Code_Syntessera_local-recall`
- Full path to transcripts: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcript files are stored as `.jsonl` format (JSON Lines - one JSON object per line).

## Relevance

This is critical for the transcript-collector functionality in local-recall, which needs to accurately locate and parse transcript files from Claude Code projects.

## Source

Discovered through investigation of actual file system structure and confirmed by examining the transcript collector code.
