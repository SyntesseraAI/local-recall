Claude's transcript files (.jsonl format) contain JSON objects with fields:
- `message_id`: Unique identifier for the message
- `session_id`: Session identifier
- `role`: Either 'user' or 'assistant'
- `content`: Markdown/text content of the message
- `created_at`: ISO-8601 timestamp
- `metadata`: May include tool usage information

Tool invocations and results are encoded in the assistant's content field using markdown formatting.
