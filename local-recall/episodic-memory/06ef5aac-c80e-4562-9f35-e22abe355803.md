MCP server runs background daemon syncing transcripts every 5 minutes from ~/.claude/projects/<project>/transcripts/. Processes transcripts using `claude -p` for memory extraction. Tracks processed transcripts with content hashes for change detection. Deletes and recreates memories when transcripts change. This replaced the disabled Stop hook for asynchronous memory extraction.
