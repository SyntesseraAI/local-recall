# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files within the transcripts directory. Each file contains a JSON object per line representing events in the session.

## Key Pattern

To find transcripts for a given working directory:
1. Replace all `/` with `-`
2. Look in `~/.claude/projects/{converted-path}/transcripts/`
3. Read JSONL files to extract session events

This is essential for the transcript collector to locate and process transcripts.
