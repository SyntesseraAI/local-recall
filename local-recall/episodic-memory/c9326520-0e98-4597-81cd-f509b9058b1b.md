# Claude Project Transcripts Path Format

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Directory Naming Convention

The project folder name is created by taking the absolute working directory path and:
1. Replacing all forward slashes (/) with dashes (-)
2. Removing the leading dash from the root /

Example transformation:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files within the project folder at:
- Path: `~/.claude/projects/{project-folder}/transcripts/{file}.jsonl`
- Format: JSONL (JSON Lines - one JSON object per line)
- Content: Claude Code session transcripts

This location and naming convention is critical for the transcript-collector to locate and process transcripts from the correct project directory.
