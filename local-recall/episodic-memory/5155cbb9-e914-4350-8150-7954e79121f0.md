The local-recall directory IS the plugin root. Plugin structure includes:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook configuration using `${CLAUDE_PLUGIN_ROOT}` variable
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (gitignored build output)
  - `scripts/hooks/` - Compiled hook entry points
  - `scripts/mcp-server/` - Compiled MCP server

Hooks receive `cwd` via stdin to operate on user's project directory. This allows hooks to be project-aware.
