# Claude Project Transcripts Path Format

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

Project folder names use the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This encoding is used for organizing transcripts across different project paths on the same machine. When building paths programmatically, remember to replace all `/` with `-` in the absolute path.
