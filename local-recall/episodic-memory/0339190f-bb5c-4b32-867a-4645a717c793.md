# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local_recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local_recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with format: `transcript-{timestamp}.jsonl`
- Example: `transcript-20250102T120000Z.jsonl`
- Each line is a JSON object representing a single message event
- Files can be quite large (hundreds of MB) containing full session history

## Key Implementation Detail

The `transcript-collector.ts` module needs to handle this path transformation when locating transcripts for processing.
