# Transcript Collector Implementation

The `transcript-collector.ts` module handles parsing and processing of Claude Code transcripts for memory extraction.

## Responsibilities

- Reads transcript JSONL files from Claude's transcript cache
- Parses transcript events into structured format
- Extracts actionable information for memory creation
- Handles file I/O and error cases

## Key Details

- Uses synchronous file reading operations
- Parses JSONL format (one JSON object per line)
- Should handle edge cases like malformed JSON and file access errors
- Memory extraction logic depends on properly structured transcript events
