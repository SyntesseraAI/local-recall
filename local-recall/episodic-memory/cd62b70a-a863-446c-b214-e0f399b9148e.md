# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and a UUID suffix.

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/transcripts/`

Transcript files are stored as `.jsonl` (JSON Lines) format.

This information is critical for the transcript-collector component which needs to locate and read Claude's session transcripts for memory extraction.
