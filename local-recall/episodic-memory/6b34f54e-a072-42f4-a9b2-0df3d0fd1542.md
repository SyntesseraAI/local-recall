# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: Named as `<session-id>.jsonl`

## File Format

Transcripts are stored as JSONL (JSON Lines) format, where each line is a separate JSON object representing an event in the session.

## Implementation Notes

The `transcript-collector.ts` module is responsible for:
1. Reading transcript files from the Claude projects directory
2. Parsing JSONL format
3. Processing transcript events for memory extraction
4. Handling file system operations for transcript discovery and synchronization
