# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with `.jsonl` extension. Each line is a JSON object representing a single message or tool invocation in the session.

## Importance

This is critical for the transcript-collector component which needs to:
1. Locate transcripts for the current project
2. Parse JSONL format correctly
3. Extract memories from transcript content
4. Detect changes using content hashes for deduplication
