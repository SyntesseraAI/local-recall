Configuration options for episodic memory are defined in `src/core/types.ts` using zod validation schema. The `episodicEnabled` option defaults to `false` and is controlled via the `LOCAL_RECALL_EPISODIC_ENABLED` environment variable. The memory extraction logic is implemented in `src/core/memory-extractor.ts` and runs as a background daemon every 5 minutes when enabled.
