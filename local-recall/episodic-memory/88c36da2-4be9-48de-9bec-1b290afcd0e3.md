The MCP server runs a background daemon that syncs transcripts from Claude's cache (~/.claude/projects/<project>/transcripts/), processes them using `claude -p` to extract memories, and tracks processed transcripts with content hashes for change detection. The daemon runs every 5 minutes and handles both new memories and updates when transcripts change.
