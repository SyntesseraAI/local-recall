Both episodic and thinking memories have independent configuration: episodicMaxTokens (default 1000), episodicMinSimilarity (default 0.5), thinkingMaxTokens (default 1000), thinkingMinSimilarity (default 0.5). These can be set via environment variables or .local-recall.json. The similarity threshold filters results by cosine distance (0.0-1.0), and token budget limits context injection.
