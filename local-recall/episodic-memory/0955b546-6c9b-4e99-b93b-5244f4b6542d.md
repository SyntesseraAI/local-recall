Claude Code stores project transcripts in `~/.claude/projects/` directory. The project folder name is derived from the absolute path with slashes replaced by dashes. For example:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored in JSONL format (one JSON object per line), with each line containing a message event with `role`, `content`, and `timestamp` fields.
