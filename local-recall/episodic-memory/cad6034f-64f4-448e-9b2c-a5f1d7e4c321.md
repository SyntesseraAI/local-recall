# Local Recall Project Architecture

Local Recall is a Claude Code plugin that provides a persistent memory system for AI coding assistants.

## Key Architecture Points

- Memories are stored as markdown files with YAML frontmatter in `local-recall/episodic-memory/`
- Uses keyword-based search and fuzzy matching for memory retrieval
- Includes hooks for SessionStart, UserPromptSubmit, and Stop events
- Integrates with Claude Code via `.claude-plugin/` structure
- Provides an MCP server for memory access
- Memories are version-controlled as git artifacts

## Core Components

- **Memory Manager** (`src/core/memory.ts`): CRUD operations for memories
- **Index Manager** (`src/core/index.ts`): Maintains searchable keyword index
- **Search** (`src/core/search.ts`): Fuzzy search implementation
- **Hooks** (`src/hooks/`): SessionStart, UserPromptSubmit, Stop event handlers
- **MCP Server** (`src/mcp-server/`): Exposes memory tools to Claude and other clients
