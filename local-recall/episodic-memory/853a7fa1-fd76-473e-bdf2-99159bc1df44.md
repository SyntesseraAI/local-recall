The vector store has been migrated from Orama with JSON index files (`orama-episodic-index.json`, `orama-thinking-index.json`) to SQLite with embedding storage. The new implementation:

- Stores embeddings in SQLite tables with `memory_embeddings` table containing normalized distance values
- Performs vector similarity search using SQLite's vector search capabilities
- No longer generates or persists `orama-*.json` index files
- Uses the same Ollama service for generating embeddings (nomic-embed-text, 768 dimensions)
