Each memory file is stored as markdown with YAML frontmatter containing: id (UUID), subject, keywords (array), applies_to (scope), occurred_at (ISO-8601), and content_hash (SHA-256 prefix). The applies_to field determines scope: 'global', 'file:<path>', or 'area:<name>'. Content hash is used for deduplication.
