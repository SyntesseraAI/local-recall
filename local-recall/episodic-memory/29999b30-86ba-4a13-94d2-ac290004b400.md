# Memory Extraction Process

The local-recall system uses `claude -p --model haiku` to extract keywords from transcript content.

## Implementation Details

- The UserPromptSubmit hook uses Claude Haiku to extract keywords from user prompts before searching the memory index
- This allows for intelligent keyword extraction beyond simple string matching
- The stop hook (disabled) would have used this same approach to extract keywords from full transcripts
- The MCP server's background daemon uses Claude to process transcripts and create memories

## Location

Implementation can be found in:
- `src/hooks/user-prompt-submit.ts` - Uses `claude -p --model haiku` for keyword extraction
- Background processing uses the same approach for transcript analysis
