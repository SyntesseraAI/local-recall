Local Recall can be configured through:
1. Environment variables
2. `.local-recall.json` file in project root

Configurable options:
- `memoryDir`: Directory for memory storage (default: `./local-recall`)
- `maxMemories`: Maximum number of memories to keep (default: `1000`)

The config utility handles loading from environment variables and JSON file with appropriate defaults.
