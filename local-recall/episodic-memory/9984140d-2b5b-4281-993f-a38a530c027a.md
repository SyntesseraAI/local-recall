# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This naming convention allows Claude to uniquely identify project directories and store their transcripts separately.

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), where each line is a separate JSON object representing an event in the transcript.
