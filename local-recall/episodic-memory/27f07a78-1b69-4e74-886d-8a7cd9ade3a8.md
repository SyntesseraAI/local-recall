The memory extraction system in `src/core/memory-extractor.ts` uses Claude Haiku model via the Claude API to analyze transcripts and extract valuable memories. The extractor processes condensed transcript events (User, Assistant, Tool, Result) to identify learnings, architectural decisions, and bug fixes. This enables the MCP server daemon to automatically create memories from session transcripts every 5 minutes.
