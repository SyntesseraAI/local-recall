MCP server exposes memory tools: memory_create, memory_get, memory_search, memory_list, and index_rebuild. Server runs as background daemon processing transcripts every 5 minutes. Can be configured in Claude Code or Claude Desktop via mcpServers setting. Requires LOCAL_RECALL_DIR environment variable pointing to memory directory.
