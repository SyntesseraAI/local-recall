The memory-extractor.ts module is responsible for analyzing Claude Code transcripts and extracting valuable memories. It uses the Claude API (claude-3-5-sonnet) with a dedicated memory extraction prompt to identify and create memories from session transcripts. The extractor processes transcript content and returns structured memory data that can be persisted to memory files.
