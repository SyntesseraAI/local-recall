The `local-recall/` directory structure:
- `.gitignore`: Auto-generated, excludes memory.sqlite and recall.log
- `memory.sqlite`: SQLite database (gitignored, auto-generated)
- `recall.log`: Debug log (gitignored)
- `episodic-memory/*.md`: Individual memory files (tracked in git)

Only memory markdown files are version-controlled. Database and logs are regenerated.
