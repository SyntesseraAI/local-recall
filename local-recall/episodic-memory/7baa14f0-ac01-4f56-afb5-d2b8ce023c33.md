The `MemoryExtractor` computes SHA-256 content hashes to detect duplicate memories. When the same transcript is processed multiple times, existing memories are returned (idempotent behavior) rather than creating duplicates. The `findDuplicate` method uses `occurred_at` timestamp and `content_hash` to identify existing memories.
