# Transcript File Format

Claude Code stores session transcripts as JSONL (JSON Lines) files within the project's transcripts directory.

## File Naming

Transcript files follow a timestamp-based naming pattern:
- Format: `YYYY-MM-DDTHH-mm-ss-SSSZ.jsonl`
- Example: `2025-12-01T15-30-45-123Z.jsonl`
- Each file represents one Claude Code session

## File Location

Full path example: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/2025-12-01T15-30-45-123Z.jsonl`

This allows transcript-collector to scan and process transcripts chronologically by filename.
