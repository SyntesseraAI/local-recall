Memory files are markdown files in local-recall/episodic-memory/ with YAML frontmatter containing: id (UUID), subject, keywords (array), applies_to (global/file:/path/to/file/area:component-name), occurred_at (ISO-8601), and content_hash (SHA-256 prefix). Files are tracked in git and formatted as structured markdown with these metadata fields.
