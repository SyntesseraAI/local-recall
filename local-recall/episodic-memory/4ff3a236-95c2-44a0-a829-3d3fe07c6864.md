The local-recall directory IS the plugin root. Structure: `.claude-plugin/plugin.json` (metadata), `hooks.json` (hook handlers), `.mcp.json` (MCP config), `scripts/` (bundled output, gitignored), `src/` (TypeScript source), and `local-recall/` (memory storage). Install by adding path to `.claude/settings.json` under plugins.installed array.
