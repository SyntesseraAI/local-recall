# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (JSON Lines format), with one message per line. Each line contains:
- `role`: "user" or "assistant"
- `content`: Message text or tool invocations
- `timestamp`: ISO-8601 timestamp

This is useful for:
- Extracting memories from session history
- Analyzing conversation patterns
- Processing transcripts in the MCP server daemon
