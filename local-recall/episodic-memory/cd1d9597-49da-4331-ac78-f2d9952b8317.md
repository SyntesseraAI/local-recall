# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Format: `*.jsonl` (JSON Lines)
- Location: `~/.claude/projects/<encoded-path>/transcripts/`
- Each line is a complete JSON object
- Files are named by session timestamp or identifier

## Content Structure

Each transcript line contains:
- `role`: "user" or "assistant"
- `content`: Message text or tool invocation details
- `timestamp`: When the message was sent
- Additional metadata fields may be present
