# MCP Server Background Daemon

The Local Recall MCP server runs a background daemon that continuously processes Claude Code transcripts.

## Core Responsibilities

1. **Transcript Syncing**: Monitors and syncs transcripts from Claude's cache at `~/.claude/projects/<project>/transcripts/`
2. **Memory Extraction**: Uses Claude Haiku to extract memories from processed transcripts
3. **Change Detection**: Tracks processed transcripts using content hashes to detect when transcripts change
4. **Memory Management**: Deletes and recreates memories when transcripts are modified
5. **Scheduled Execution**: Runs every 5 minutes to process new or updated transcripts

## Benefits

- Asynchronous processing prevents blocking user sessions
- Handles long transcripts without timeout issues
- Allows multiple passes over transcript content
- Can detect and respond to transcript changes
