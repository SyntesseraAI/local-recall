Important git configuration:

- **Memory files** (`local-recall/episodic-memory/*.md`) ARE tracked in git - they should be committed
- **SQLite database** (`local-recall/memory.sqlite`) is gitignored and auto-generated
- **recall.log** debug log is gitignored
- **local_cache/** embedding model cache is gitignored
- `.gitignore` in memory storage directory is auto-generated and excludes `memory.sqlite` and `recall.log`

This allows team sharing of memories while keeping generated artifacts local.
