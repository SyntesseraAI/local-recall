# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory:
- Replace `/` with `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `_Users_joe_Code_Syntessera_local-recall`

## File Structure

Transcripts are stored as JSONL (JSON Lines) format:
- Location: `~/.claude/projects/<project-folder>/transcripts/`
- Files: `*.jsonl`
- Each line is a complete JSON object representing one event in the transcript

The `transcript-collector.ts` module is responsible for reading and processing these transcript files to extract memories and analyze session data.
