Claude Code hooks (SessionStart and UserPromptSubmit) are configured as shell commands in hooks.json. They receive JSON input via stdin containing `session_id`, `transcript_path`, `cwd`, and optional `prompt`. Hooks output memory content to stdout which Claude automatically injects into context. This allows hooks to enrich Claude's context with relevant memories without requiring direct API calls.
