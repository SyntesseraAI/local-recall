The system condenses transcripts into structured events using a specific format:
- `[User]` - User requests or questions
- `[Assistant]` - Claude's responses or explanations
- `[Tool: Name]` - Tool invocations (Read, Edit, Write, Bash, Grep, etc.)
- `[Result: OK/ERROR]` - Outcome of tool operations

This condensed format is what gets passed to the memory extraction API for analysis, making it easier for the extractor to identify important moments and create relevant memories.
