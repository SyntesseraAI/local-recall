When processing transcripts to extract memories, the system uses Claude's API to generate embeddings.

## Key Requirements

- Claude API key must be available in the environment (via `ANTHROPIC_API_KEY`)
- The `claude` CLI must be configured with valid credentials
- Processing will fail silently if credentials are missing
- This is needed for memory extraction and semantic analysis

## Implementation Detail

The `transcript-collector.ts` module handles transcript processing and needs valid API credentials to function properly. Ensure the environment is configured before running transcript collection processes.
