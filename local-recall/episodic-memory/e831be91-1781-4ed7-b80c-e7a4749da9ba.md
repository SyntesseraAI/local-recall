# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), with the current session transcript at:
`~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/current.jsonl`

## Memory Extraction Context

This is essential for the transcript-collector and MCP server background daemon which needs to:
1. Locate Claude's cached transcripts
2. Sync them for processing
3. Extract memories from session history
4. Track processed transcripts to avoid re-processing unchanged content
