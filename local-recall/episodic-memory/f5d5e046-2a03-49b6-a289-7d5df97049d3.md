The project root IS the plugin root following Claude Code's plugin format:
- `.claude-plugin/plugin.json`: Contains plugin name, version, description
- `hooks.json`: Hook event handlers using `${CLAUDE_PLUGIN_ROOT}` environment variable
- `.mcp.json`: MCP server configuration
- `scripts/`: Contains bundled build outputs (hooks and mcp-server), gitignored
- `src/`: Source TypeScript files bundled into scripts/ during build

Hooks receive `cwd` via stdin to operate on user's project directory rather than plugin directory.
