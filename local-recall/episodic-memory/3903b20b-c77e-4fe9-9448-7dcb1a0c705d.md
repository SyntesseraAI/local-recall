# Path Normalization for Transcript Lookup

When implementing the transcript collector to find Claude's project transcripts, ensure proper path conversion:

1. Take the working directory path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replace all forward slashes with dashes
3. Look in `~/.claude/projects/` for the converted folder name

This allows the transcript collector to reliably locate transcripts regardless of the user's project location.
