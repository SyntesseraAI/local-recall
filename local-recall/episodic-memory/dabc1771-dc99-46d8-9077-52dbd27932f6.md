Claude Code stores session transcripts in `~/.claude/projects/` directory. The project folder name is derived from the working directory path with slashes replaced by dashes.

Example path pattern:
`/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/086dd549-7e86-4477-8c3a-7e49b1a1f3d6.jsonl`

This pattern is important for the transcript-collector to locate and process transcripts from previous sessions.
