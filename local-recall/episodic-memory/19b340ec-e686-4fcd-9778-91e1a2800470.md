# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Each transcript is stored as a JSONL file (JSON Lines format) with the naming pattern:
- `transcript_<timestamp>.jsonl`
- Example: `transcript_20250101120000.jsonl`

Each line in the JSONL file is a JSON object representing a message or event in the transcript.

## Usage in local-recall

The `transcript-collector` module reads from this location to extract and process transcripts for memory creation. This is critical for the MCP server's background daemon which syncs and processes transcripts every 5 minutes.
