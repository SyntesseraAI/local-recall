Memory operations are designed to be idempotent:
- `createMemory(data)` - Creates new memory, returns existing if duplicate (checks occurred_at + content_hash)
- `deleteMemory(id)` - Removes a memory by ID
- `getMemory(id)` - Retrieves specific memory
- `listMemories(filter?)` - Lists all memories with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Check for existing duplicate

No update operation exists - memories are immutable once created.
