# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` (JSON Lines) files in the transcripts directory.

## Key Details

- Transcripts are in the Claude cache directory, not in the project itself
- Project folder names use dashes as path separators
- Each transcript is a separate JSONL file
- This is specific to macOS (Windows uses `%APPDATA%`)
