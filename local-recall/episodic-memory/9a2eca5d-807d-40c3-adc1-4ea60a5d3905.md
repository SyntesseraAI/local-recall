# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line). Each object has:
- `role`: Either "user" or "assistant"
- `content`: The message content
- `timestamp`: ISO-8601 timestamp

## Key Details

- Transcripts accumulate in the directory, not replaced
- Each session creates new transcript entries
- Useful for tracking conversation history and extracting memories
- The transcript-collector module likely uses this path for processing
