# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files, with each session in a separate file.

## Relevance to Local Recall

The MCP server background daemon syncs transcripts from this location:
- Reads from: `~/.claude/projects/<project-folder>/transcripts/`
- Processes transcripts every 5 minutes
- Extracts memories using `claude -p` for memory extraction
- Tracks processed transcripts with content hashes for change detection
