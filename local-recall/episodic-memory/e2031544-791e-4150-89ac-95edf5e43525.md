# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by periods and underscores:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users.joe.Code.Syntessera.local-recall`
- Full transcripts path: `~/.claude/projects/Users.joe.Code.Syntessera.local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one per session. Each line is a JSON object containing:
- `type`: Event type ("user_message", "assistant_message", "tool_call", "tool_result", etc.)
- `content`: The actual content/data
- `timestamp`: When the event occurred

This format allows for efficient streaming and processing of transcript data.
