The session-start hook (src/hooks/session-start.ts) reads memories directly from filesystem without using vector search. It:
1. Loads episodic memory files from local-recall/episodic-memory/
2. Sorts by occurred_at timestamp (most recent first)
3. Returns up to 5 most recent memories for context injection
4. Uses direct file reading (no Orama dependency) for faster startup
5. Outputs formatted memory content to stdout for Claude context
