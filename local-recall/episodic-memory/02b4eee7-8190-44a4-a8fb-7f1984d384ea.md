# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Forward slashes (`/`) are converted to underscores (`_`)
- The leading slash is removed
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users_joe_Code_Syntessera_local-recall`

## Transcript Files

Transcripts are stored as JSONL files in `~/.claude/projects/<project-id>/transcripts/`

The path format helps locate transcripts for the current working directory when processing memories.

## Reference

See `src/core/transcript-collector.ts` for implementation details on how paths are converted and transcripts are collected.
