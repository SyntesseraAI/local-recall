# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Transcripts folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with names like:
- `transcript_2025-12-01_15-56-21.jsonl`
- Naming pattern: `transcript_YYYY-MM-DD_HH-mm-ss.jsonl`

Each line in the JSONL file represents a single message or event in the transcript with metadata including timestamps and message type.
