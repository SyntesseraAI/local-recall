The MCP server daemon runs in the background and processes Claude Code transcripts every 5 minutes. It syncs transcripts from Claude's cache, extracts memories automatically using the extraction system, and tracks processed transcripts with content hashes to detect changes. This allows memories to be created asynchronously without blocking the user's coding workflow.
