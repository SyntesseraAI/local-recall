The daemon uses `claude -p` to analyze transcripts and extract episodic memories. The process works by sending a specially formatted prompt that includes:

1. The transcript content
2. Instructions to identify valuable memories
3. A marker `[LOCAL_RECALL_INTERNAL]` to prevent recursive memory extraction

The extracted memories should follow the standard memory file format with YAML frontmatter (id, subject, keywords, applies_to, occurred_at, content_hash).
