Memory markdown files in `local-recall/episodic-memory/` ARE version-controlled and will be committed to git. However, `local-recall/memory.sqlite` and `local_recall_cache/` are gitignored and auto-generated. The `.gitignore` in the memory directory excludes only database and log files.
