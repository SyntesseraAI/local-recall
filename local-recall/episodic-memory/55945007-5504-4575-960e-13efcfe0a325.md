Local Recall integrates with Claude Code through: 1) SessionStart hook - loads 5 most recent memories on session start, 2) UserPromptSubmit hook - performs semantic search on user prompts, 3) MCP server - exposes memory tools and runs background daemon processing transcripts every 5 minutes.
