# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with forward slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Files stored as `.jsonl` format with timestamps

This path-as-name convention allows Claude to track transcripts for different projects without collisions.
