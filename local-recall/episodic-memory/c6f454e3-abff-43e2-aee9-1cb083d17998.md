# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one event per line. Each file represents a single session.

## Implementation Notes

When implementing `transcript-collector.ts`, the path construction should:
1. Replace forward slashes with dashes in the absolute working directory path
2. Look in `~/.claude/projects/{slugified-path}/transcripts/`
3. Parse `.jsonl` files where each line is a complete JSON event object

This is used by the MCP daemon to sync and process transcripts for memory extraction.
