Local Recall uses a dedicated transcript-collector module to extract memories from Claude Code session transcripts.

**Process:**
1. Monitors `~/.claude/projects/` for new/updated transcript files
2. Reads JSONL-formatted transcript files
3. Condenses/chunks transcript content into manageable segments
4. Uses Claude Haiku to extract memories from condensed transcripts
5. Creates episodic memory files in `local-recall/episodic-memory/`

**Key files:**
- `src/core/transcript-collector.ts` - Main transcript collection and parsing logic
- `src/utils/transcript.ts` - Transcript parsing utilities
- Background daemon runs every 5 minutes to process new transcripts

This enables the system to automatically capture and store valuable context from coding sessions without manual intervention.
