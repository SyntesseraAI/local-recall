# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path of the project directory, converted to a URL-encoded format.

Example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall` (slashes converted to hyphens)
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (JSON Lines format) in the transcripts directory. Each line is a JSON object representing an event in the session.
