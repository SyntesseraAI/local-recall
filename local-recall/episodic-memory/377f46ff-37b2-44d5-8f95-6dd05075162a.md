# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Claude folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`

## Transcript Files

Transcript files are stored in the `transcripts/` subdirectory with numeric names:
- Format: `<number>.jsonl`
- Example: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/1.jsonl`
- Content: JSONL format (one JSON object per line)

## Session Structure

Each JSONL file contains messages in chronological order with:
- `role`: "user" or "assistant"
- `content`: The message text
- `timestamp`: ISO 8601 format timestamp

This is critical for the `transcript-collector.ts` implementation to properly parse and extract memories from Claude sessions.
