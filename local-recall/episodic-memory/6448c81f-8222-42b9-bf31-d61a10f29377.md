# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with each line being a complete JSON object representing a single transcript event or message.

## Usage in MCP Server

The MCP server's background daemon syncs these transcripts to extract memories every 5 minutes. This allows the system to automatically capture insights from Claude Code sessions without manual intervention.
