When a user submits a prompt, the UserPromptSubmit hook extracts keywords by:
1. Sending the user's prompt text to Claude Haiku via `claude -p --model haiku`
2. Requesting a JSON array of relevant keywords
3. Using those keywords to search the memory index
4. Returning matching memories for context injection

This approach allows for semantic keyword extraction rather than simple string matching, improving memory relevance.
