# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, where each line is a separate JSON object representing a transcript event.

## Accessing Transcripts

The MCP server daemon uses these transcripts to extract memories and process session data. The transcript collector needs to handle this specific file structure and location.
