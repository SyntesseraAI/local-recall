Memory files use YAML frontmatter with these key fields for deduplication:
- `occurred_at`: ISO-8601 timestamp of when the original event occurred
- `content_hash`: SHA-256 hash prefix (16 chars) of the memory content

The `findDuplicate()` function in memory.ts checks for existing memories with matching `occurred_at` and `content_hash` to prevent duplicates. If a duplicate is found, it returns the existing memory instead of creating a new one.
