MCP server exposes five tools for memory operations:
- `memory_create`: Create new memory (idempotent)
- `memory_get`: Retrieve by ID
- `memory_search`: Semantic search
- `memory_list`: List with optional filtering
- `index_rebuild`: Sync vector store with files

Server runs background daemon that syncs transcripts every 5 minutes using content hashing for change detection.
