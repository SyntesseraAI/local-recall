# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with one JSON object per line. Each line represents a transcript event.

## Relevant Code

The `transcript-collector.ts` module handles reading and parsing these transcripts:
- `getTranscriptPath()` - Constructs the path to the transcripts directory
- `readTranscriptFile()` - Reads and parses JSONL files
- `condenseTranscript()` - Processes transcript events into a condensed format
