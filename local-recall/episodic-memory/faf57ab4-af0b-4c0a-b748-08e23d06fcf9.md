The MCP server runs a background daemon that syncs transcripts from Claude cache, processes them to extract memories, and tracks processed transcripts using content hashes. Daemon runs every 5 minutes. Exposes 5 tools: memory_create, memory_get, memory_search, memory_list, index_rebuild.
