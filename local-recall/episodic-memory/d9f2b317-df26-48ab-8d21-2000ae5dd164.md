# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the working directory path:
- Slashes `/` are replaced with dashes `-`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- The transcript file is stored as `transcript.jsonl` inside this directory

## Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcript.jsonl`

This allows the MCP server daemon to locate and process transcripts from Claude Code sessions.
