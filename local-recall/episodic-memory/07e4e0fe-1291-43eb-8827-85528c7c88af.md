# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by underscores and the full path as a hash component.

## Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Path is converted to `/Users/joe/Code/Syntessera/local-recall`
- Transcripts stored at: `~/.claude/projects/<derived-folder-name>/transcripts/`
- Transcript files use JSONL format (one JSON object per line)

## File Structure

Each transcript file contains:
- `type`: Event type ("user", "assistant", "tool", etc.)
- `content`: The actual content or message
- `timestamp`: When the event occurred
- Additional metadata depending on event type

## Relevance to Local Recall

The MCP server's background daemon syncs transcripts from this location to extract memories automatically. The path format is important for properly locating and parsing project-specific transcripts.
