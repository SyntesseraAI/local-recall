# JSONL File Format for Transcripts

Claude Code stores transcripts as JSONL (JSON Lines) files, not standard JSON.

## Format Details

- Each line in a .jsonl file is a separate, complete JSON object
- Not wrapped in an array
- Must be parsed line-by-line, not as a single JSON document
- Cannot use `JSON.parse()` on the entire file content

## Parsing Requirement

When implementing transcript collection in `src/core/transcript-collector.ts`, ensure:
1. Read file as text
2. Split by newlines
3. Parse each line individually with `JSON.parse()`
4. Filter out empty lines

This is critical for correctly extracting memories from transcript files.
