Technical requirements and setup:

- **Runtime**: Node.js 18+ required
- **Build**: TypeScript 5.9+
- **Embedding Model**: Uses `fastembed` library with BGE-small-en-v1.5 (~133MB)
- **Model Caching**: Downloaded to `local_cache/` on first use (30-60 seconds)
- **First Run**: May be slow due to model download; subsequent runs use cached version
- **Troubleshooting**: If tokenizer error occurs, remove `local_cache/fast-bge-small-en-v1.5*` to re-download

Scripts:
- `npm run build` - Compile TypeScript
- `npm run dev` - Watch mode
- `npm run test` - Run tests
- `npm run mcp:start` - Start MCP server
