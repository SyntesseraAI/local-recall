Local Recall uses a flexible configuration system supporting both environment variables and `.local-recall.json` file:
- `LOCAL_RECALL_DIR` / `memoryDir` (default: ./local-recall)
- `LOCAL_RECALL_EPISODIC_ENABLED` / `episodicEnabled` (default: true)
- `LOCAL_RECALL_EPISODIC_MAX_TOKENS` / `episodicMaxTokens` (default: 1000)
- `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY` / `episodicMinSimilarity` (default: 0.5)
- `LOCAL_RECALL_THINKING_ENABLED` / `thinkingEnabled` (default: true)
- `LOCAL_RECALL_THINKING_MAX_TOKENS` / `thinkingMaxTokens` (default: 1000)
- `LOCAL_RECALL_THINKING_MIN_SIMILARITY` / `thinkingMinSimilarity` (default: 0.5)
- `LOCAL_RECALL_LOG_LEVEL` (default: error)

Environment variables take precedence over config file.
