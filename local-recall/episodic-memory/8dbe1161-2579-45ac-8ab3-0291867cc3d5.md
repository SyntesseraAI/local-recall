MCP server provides 5 tools for memory management: memory_create (idempotent), memory_get (retrieve by ID), memory_search (semantic search), memory_list (with optional filtering), and index_rebuild (sync vector store with files). Background daemon syncs transcripts from Claude cache, processes them to extract memories, and deletes/recreates memories when transcripts change, running every 5 minutes.
