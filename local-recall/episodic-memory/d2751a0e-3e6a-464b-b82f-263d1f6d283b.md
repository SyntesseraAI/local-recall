# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Format

Transcripts are stored as JSONL files in the `transcripts/` subdirectory.

Example path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each line in a JSONL file is a JSON object representing a message event in the conversation.
