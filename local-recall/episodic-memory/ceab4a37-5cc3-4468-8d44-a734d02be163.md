# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name encodes the absolute working directory:
- Slashes are replaced with dashes
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files in the project's `transcripts/` directory:
- Each line is a valid JSON object
- Each object represents a single transcript event/turn
- Enables efficient streaming and line-by-line parsing

## Implementation Location

Transcript collection logic is in `src/core/transcript-collector.ts` and handles syncing transcripts from Claude's cache for memory extraction.
