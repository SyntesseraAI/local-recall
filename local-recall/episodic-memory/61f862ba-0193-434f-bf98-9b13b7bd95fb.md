The memory extractor in src/core/memory-extractor.ts uses Claude Haiku with the `-p` flag to enable prompt caching for memory extraction. Before extracting memories, it condenses the transcript using src/core/transcript-condenser.ts to reduce token usage while preserving essential information. This is a performance optimization that reduces API costs while maintaining extraction quality.
