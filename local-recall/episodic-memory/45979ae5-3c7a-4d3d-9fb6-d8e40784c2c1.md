Memory files are stored as markdown files in `local-recall/episodic-memory/` and ARE version-controlled (committed to git). The index (`local-recall/index.json`) and debug log (`local-recall/recall.log`) are gitignored and auto-generated. A `.gitignore` file in the memory directory excludes only index.json and recall.log, keeping individual memory markdown files tracked.
