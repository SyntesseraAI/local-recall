The memory extraction process is implemented in `src/core/memory-extractor.ts`. This module processes session transcripts to automatically extract and create memories using the Claude API. The extraction uses a dedicated prompt defined in `src/prompts/memory-extraction.ts` to guide Claude in identifying valuable memories from transcript content.
