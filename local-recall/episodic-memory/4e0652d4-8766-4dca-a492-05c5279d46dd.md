Removed `memory_update` and `memory_delete` MCP tools from the exposed API. Memory mutations are not agent-initiated - only manual user deletion is permitted. The API now exposes: `memory_create` (idempotent), `memory_search`, `memory_list`, and `index_rebuild`. The create operation is idempotent and returns existing memory if duplicate detected.
