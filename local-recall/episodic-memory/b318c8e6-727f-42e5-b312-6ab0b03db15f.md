All memory files follow a consistent markdown format with YAML frontmatter containing: id (UUID), subject (brief description), keywords (array), applies_to (global|file:<path>|area:<name>), occurred_at (ISO-8601 timestamp), content_hash (SHA-256 prefix). Memory content is markdown below the frontmatter.
