# Project Configuration

## TypeScript Setup

The local-recall project uses TypeScript 5.9+ with strict mode enabled.

## Build System

- Compilation target: ES2020
- Module system: commonjs
- Output directory: `dist/`
- Source directory: `src/`

## Dependencies Philosophy

The project aims to minimize external dependencies for core functionality:
- Core memory operations use Node.js built-ins
- Vector embeddings use `fastembed` library
- Database uses `better-sqlite3` with `sqlite-vec` extension
- Testing framework: Jest

## Project Structure

Key directories:
- `src/core/` - Core memory management
- `src/hooks/` - Claude Code hooks
- `src/mcp-server/` - MCP server implementation
- `src/utils/` - Utility functions
- `tests/` - Unit and integration tests
- `local-recall/` - Memory storage (version-controlled)

Remember that all memory files are version-controlled and committed to git, but the SQLite database and vector cache are gitignored.
