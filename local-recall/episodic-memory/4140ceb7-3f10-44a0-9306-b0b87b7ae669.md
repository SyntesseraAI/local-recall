# Transcript JSONL Format

Claude transcripts are stored in JSONL format with each line containing a JSON object representing a session event.

## Event Types

1. **User Messages**
   - Contains: `role: 'user'`, `content: string`
   - May include nested tool calls and results

2. **Assistant Messages**
   - Contains: `role: 'assistant'`, `content: string`
   - May include nested tool invocations with names and parameters

## Structure Notes

- Each line is a complete JSON object (not an array)
- User and assistant messages can span multiple JSONL lines
- Tool invocations are embedded within assistant messages
- Tool results are embedded within user messages

This format is important for the transcript parser in `src/utils/transcript.ts` which extracts information for memory creation.
