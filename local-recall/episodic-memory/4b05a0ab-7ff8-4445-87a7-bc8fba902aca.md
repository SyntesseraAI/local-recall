Memory extraction must return ONLY a valid JSON object with no explanation, markdown formatting, or code blocks - just raw JSON.

The JSON object must have a `memories` array where each memory contains:
- `subject`: Brief one-line description (max 200 chars)
- `keywords`: Array of 1-10 searchable keywords (lowercase, specific)
- `applies_to`: Scope string (`global`, `file:<path>`, or `area:<name>`)
- `content`: Detailed memory content in markdown format

If no valuable memories exist, return: `{ "memories": [] }`
