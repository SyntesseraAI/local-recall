MemoryManager (src/core/memory.ts) provides: createMemory() returns existing memory if duplicate (uses occurred_at + content_hash for detection), deleteMemory(), getMemory(), listMemories(), and findDuplicate(). No update operation - memories are idempotent. This prevents duplicate memory creation when the same information is extracted multiple times.
