Key core components:
- `memory.ts`: CRD operations (Create/Read/Delete, no update) for memory files
- `vector-store.ts`: SQLite-backed semantic search, initialization, add/remove/search operations
- `embedding.ts`: Embedding service using fastembed
- `search.ts`: Search implementation using vector store
- `transcript-collector.ts`: Collects transcripts from Claude cache
- `transcript-condenser.ts`: Condenses transcripts for processing
- `memory-extractor.ts`: Extracts memories from transcripts
