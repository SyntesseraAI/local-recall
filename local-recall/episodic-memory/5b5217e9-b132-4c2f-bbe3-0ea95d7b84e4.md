# Memory Extraction with Claude Haiku

When processing transcripts to extract memories, use `claude -p --model haiku` to:

1. Analyze transcript content and identify valuable memories
2. Extract relevant details that should be preserved for future sessions
3. Generate appropriate keywords and metadata for memories

The model processes JSON input from stdin and outputs structured memory data.

This is used in:
- The MCP server's background daemon (processes transcripts every 5 minutes)
- The stop hook (disabled, but previously handled memory creation)
- Future transcript-based memory extraction workflows
