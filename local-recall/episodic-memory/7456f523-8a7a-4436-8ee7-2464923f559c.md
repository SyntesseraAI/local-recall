# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name encodes the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format:
- Location: `~/.claude/projects/<encoded-path>/transcripts/`
- Format: One JSON object per line, each representing a turn in the conversation
- File naming: Uses timestamps or sequential numbering

This is important for the transcript-collector to locate and parse Claude Code session history.
