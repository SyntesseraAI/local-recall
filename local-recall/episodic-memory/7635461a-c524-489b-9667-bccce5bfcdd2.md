# Tool Parameter Precision

When making function calls using tools that accept array or object parameters, always use exact values provided by the user or inferred from context. Never use placeholders or guess missing parameters.

This ensures accuracy and prevents unnecessary tool iterations or failed calls.

Example of correct approach:
- Read files first to understand structure before making modifications
- Use exact file paths from grep/glob results
- Specify exact values in JSON parameters rather than `{"key": "value-here"}`
