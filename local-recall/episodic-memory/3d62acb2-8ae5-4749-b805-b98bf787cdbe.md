# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with a timestamp prefix:
- Example: `2025-12-02T12-34-56.123Z.jsonl`
- Each line is a separate JSON object representing a message or event
- Transcripts accumulate over multiple sessions in the same directory

## Usage in Local Recall

The `transcript-collector.ts` utility reads from this location to extract memories from session transcripts. This is essential for the MCP background daemon which processes transcripts asynchronously.
