# Claude Transcript File Location

Claude Code stores project transcripts in `~/.claude/projects/` with directory names that are URL-encoded versions of the project path.

## Path Format
- Base directory: `~/.claude/projects/`
- Directory name: URL-encoded version of the project's absolute path
- Transcripts stored as: `transcripts/*.jsonl` files

## Example
For project at `/Users/joe/Code/Syntessera/local-recall`:
- Encoded path: `file%3A%2F%2FUsers%2Fjoe%2FCode%2FSyntessera%2Flocal-recall`
- Full transcript directory: `~/.claude/projects/file%3A%2F%2FUsers%2Fjoe%2FCode%2FSyntessera%2Flocal-recall/transcripts/`

## Usage in Code
The `transcript-collector` module needs to URL-encode project paths when constructing file paths to locate transcripts.

## Implementation Detail
URL encoding replaces:
- `/` → `%2F`
- `:` → `%3A`
- Other special characters with their percent-encoded equivalents
