When extracting memories, the system returns ONLY a valid JSON object with a 'memories' array. Each memory contains: subject (max 200 chars), keywords (array of 1-10 searchable lowercase keywords), applies_to (global/file:<path>/area:<name>), and content (markdown format). No explanation, markdown formatting, or code blocks should be included in the output - only raw JSON.
