Claude Code stores project transcripts in a specific location:

- **Base directory**: `~/.claude/projects/`
- **Project folder**: Sanitized version of the working directory path (slashes replaced with dashes)
- **Example**: Project at `/Users/joe/projects/my-app` stores transcripts in `~/.claude/projects/-Users-joe-projects-my-app/transcripts/`

The transcript collector uses this path to locate and sync transcripts from Claude's cache into the local-recall memory system.
