# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory with slashes replaced by dashes:
- Example: Working directory `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full transcript directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (one JSON object per line) with the filename format: `<session_id>.jsonl`

Each line contains a message object with:
- `role`: 'user' or 'assistant'
- `content`: The message text
- Other metadata fields

This is used by the transcript-collector to sync and process Claude Code sessions.
