# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format) in the transcripts directory. Each line is a JSON object representing a message or event in the session.

## Implementation

The `transcript-collector.ts` module handles syncing and processing these transcripts for memory extraction. When collecting transcripts:
1. Construct the project folder name by replacing slashes with dashes
2. Read transcript files from `~/.claude/projects/{project-folder}/transcripts/`
3. Parse each transcript as JSON Lines format
4. Process for memory extraction using Claude Haiku
