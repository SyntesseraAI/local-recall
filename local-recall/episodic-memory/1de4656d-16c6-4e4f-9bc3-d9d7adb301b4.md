The index manager maintains a searchable index of all memory keywords stored in `local-recall/index.json` (gitignored). The index maps keywords to memory IDs for fast lookups. The `buildIndex()` method scans all memory files and builds the keyword index, while `refreshIndex()` rebuilds it from scratch. This enables efficient fuzzy search across memories.
