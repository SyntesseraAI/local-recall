Memory extraction must return valid JSON with specific structure:
- `memories` array containing memory objects
- Each memory requires: `subject` (one-line, max 200 chars), `keywords` (array, lowercase), `applies_to` (scope: global/file:/area:), `content` (markdown format)
- No markdown code blocks or explanations - just raw JSON
- If no memories found, return: `{ "memories": [] }`
