The local-recall project follows Claude Code's plugin format where the project root IS the plugin root. The structure includes:
- `.claude-plugin/plugin.json`: Plugin metadata
- `hooks.json`: Hook configuration using `${CLAUDE_PLUGIN_ROOT}` variable
- `.mcp.json`: MCP server configuration
- `scripts/`: Bundled JavaScript executables (build output, gitignored)
- `src/`: TypeScript source code

The `${CLAUDE_PLUGIN_ROOT}` variable is replaced at runtime to reference scripts relative to plugin installation location.
