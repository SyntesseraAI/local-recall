The npm build process outputs compiled hook scripts to `dev-marketplace/local-recall-plugin/scripts/hooks/` rather than to a `scripts/` directory at the project root. This is important for hook configuration - the actual paths must point to the dev-marketplace location, not a root scripts/ folder.
