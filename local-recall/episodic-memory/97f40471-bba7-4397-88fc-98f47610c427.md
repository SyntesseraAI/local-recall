Memory files require: id (UUID), subject (brief description), keywords (array), applies_to (global/file/area), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix, 16 chars). The content_hash is used for deduplication and must be computed from the memory content.
