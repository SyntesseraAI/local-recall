# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one file per session.

Each line in the file is a JSON object containing a transcript event:
- `type`: The event type (e.g., "user", "assistant", "tool_call", "tool_result")
- `content`: The content of the event
- Other fields depend on the event type

## Usage in transcript-collector

When implementing transcript collection in the MCP server or hooks, use this path pattern to locate and read transcript files.
