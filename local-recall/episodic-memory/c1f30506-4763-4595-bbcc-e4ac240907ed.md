Development setup requires Node.js 18+ and TypeScript 5.9+. Key npm scripts:
- `npm run build` - Compile TypeScript
- `npm run dev` - Watch mode
- `npm test` - Run tests
- `npm run lint` - Lint code
- `npm run mcp:start` - Start MCP server

Tests include unit tests (`tests/unit/`) and integration tests (`tests/integration/`).
