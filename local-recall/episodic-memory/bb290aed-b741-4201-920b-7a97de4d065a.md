Local Recall is a markdown-powered memory system for AI coding assistants. Project root IS the plugin root. Key directories:
- `src/core/` - Memory management, vector store, embeddings, search
- `src/hooks/` - Claude Code hook implementations (session-start, user-prompt-submit, stop)
- `src/mcp-server/` - MCP server implementation
- `local-recall/` - Memory storage directory with episodic-memory/ for tracked markdown files
- `scripts/` - Bundled build output (gitignored)

Architecture uses hooks.json and .mcp.json for configuration. Plugin uses ${CLAUDE_PLUGIN_ROOT} for relative script paths.
