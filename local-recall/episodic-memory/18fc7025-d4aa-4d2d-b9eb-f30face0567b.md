The local-recall project has a modular architecture for semantic search:
- `src/core/embedding.ts` - Handles embedding generation using BGEm (or similar embedding model)
- `src/core/vector-store.ts` - Manages vector storage and similarity search
These modules support semantic search capabilities beyond keyword-based fuzzy matching. The embedding model is fastembed with BGE-small-en-v1.5 (~133MB), automatically cached in `local_cache/`.
