The memory-extractor module uses the Anthropic Claude API (via @anthropic-ai/sdk) to analyze session transcripts and generate memories. Key implementation details:

- Uses `Anthropic` client from `@anthropic-ai/sdk` package
- Sends transcript content to Claude API for analysis
- Extracts structured memory data from Claude's responses
- Implements retry logic and error handling for API calls
- Parses Claude's output to create memory objects with frontmatter

This is the core mechanism for converting raw transcripts into structured, searchable memories.
