# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory with specific conversion rules:
- Forward slashes `/` are converted to underscores `_`
- Colons `:` are removed entirely
- The result uses the full path components

## Example

Working directory: `/Users/joe/Code/Syntessera/local-recall`

Transcript location: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Implementation Details

This conversion logic is critical for the transcript-collector to locate transcripts correctly. The path conversion must handle:
- macOS-specific paths with user directories
- Multi-level directory structures
- Proper escaping of special characters

See `src/core/transcript-collector.ts` for the implementation details of transcript discovery and collection from Claude's cache.
