If you encounter 'Tokenizer file not found' error at `local_cache/fast-bge-small-en-v1.5/tokenizer.json`, it indicates corrupted or incomplete cache (usually from interrupted download). Solution: Remove the corrupted cache with `rm -rf local_cache/fast-bge-small-en-v1.5*` - the model will re-download automatically on next run.
