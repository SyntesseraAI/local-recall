# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format with naming pattern `transcript-YYYYMMDD-HHMMSS.jsonl`.

Each line in the JSONL file contains a complete JSON object representing an event in the conversation.

## Implications for Local Recall

When implementing transcript collection or processing:
1. Must handle path encoding: convert `/` to `-` when constructing paths
2. Transcripts are in JSONL format, not plain JSON
3. Location is platform-specific (macOS: `~/.claude/projects/`, likely different on Windows/Linux)
4. Sessions have timestamps in the filename
5. This is where the MCP server should read from when syncing transcripts
