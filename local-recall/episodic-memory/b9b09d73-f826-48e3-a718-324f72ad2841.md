# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by underscores. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full transcript path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) with one JSON object per line, each representing a message or event in the session.

## Relevance

This is critical for the transcript collector which syncs transcripts from Claude's cache to process for memory extraction.
