Two installation options:

1. **Clone and Build**
   - `git clone`, `npm install`, `npm run build`
   - Add to `.claude/settings.json` plugins array

2. **NPM Package**
   - Install via npm
   - Configure hooks in `.claude/settings.json` to point to `./node_modules/local-recall/scripts/hooks/`
   - Configure MCP server similarly

Uses `${CLAUDE_PLUGIN_ROOT}` in hooks.json for relative paths to bundled scripts.
Hooks receive `cwd` via stdin to operate on user's project directory.
