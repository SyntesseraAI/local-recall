# Local Recall Embedding Model

Local Recall implements semantic search using the `fastembed` library with the BGE-small-en-v1.5 model for vector embeddings.

## Key Details

- **Model:** BGE-small-en-v1.5
- **Library:** fastembed
- **Size:** ~133MB
- **Cache Location:** `local_cache/` directory
- **Auto-download:** Model is automatically downloaded on first use
- **First Run:** Initial startup may take 30-60 seconds while the model downloads
- **Subsequent Runs:** Load from local cache

## Files

- `src/core/embedding.ts` - Embedding model initialization and usage
- `src/core/vector-store.ts` - Vector storage and semantic search implementation

## Note

If the tokenizer cache becomes corrupted, remove `local_cache/fast-bge-small-en-v1.5*` to trigger a fresh download on next run.
