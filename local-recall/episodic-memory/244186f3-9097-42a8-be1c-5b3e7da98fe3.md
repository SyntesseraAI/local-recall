# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory `/Users/joe/Code/Syntessera/local-recall` becomes folder `_Users_joe_Code_Syntessera_local-recall`
- Transcripts are stored as JSONL files in the project folder
- Each line is a complete JSON object representing a single message/event

## Transcript Format

Each transcript entry contains:
- `role`: Either "user" or "assistant"
- `content`: The message text
- `timestamp`: ISO 8601 timestamp
- `type`: Message type ("text", "tool_call", etc.)

This structure is used by transcript-collector.ts to process Claude Code transcripts for memory extraction.
