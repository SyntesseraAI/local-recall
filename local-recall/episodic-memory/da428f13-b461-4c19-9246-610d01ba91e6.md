# KeywordExtractor Uses Claude CLI Local API

The `KeywordExtractor` class in `src/core/keyword-extractor.ts` uses the Claude CLI tool `claude -p` to extract keywords from text.

## Implementation Details

- Uses local API calls via `claude -p` command
- This allows running Claude Haiku locally without network calls
- The `-p` flag enables prompt mode for text processing
- Used by the `UserPromptSubmit` hook to extract keywords from user prompts

## Benefits

- Faster keyword extraction (local processing)
- No external API calls needed
- Reduces latency compared to making HTTP calls to Claude API

## Requirement

The Claude CLI must be installed and available in the PATH for this to work.
