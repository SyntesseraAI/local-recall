# Memory Extraction Process

The local-recall project uses Claude Haiku (`claude -p` with `--model haiku`) to:

1. **Extract keywords** from user prompts and assistant responses
2. **Generate structured memories** from transcript content
3. **Identify relevant context** for future sessions

## API Usage

The process calls Claude programmatically using:
- `claude -p` - Pipe prompt via stdin
- `--model haiku` - Uses the faster, cheaper Haiku model

This is used in:
- `user-prompt-submit.ts` hook for keyword extraction from current prompts
- MCP server daemon for batch memory extraction from transcripts

## Key Insight

Using Haiku keeps the system responsive and cost-effective while still leveraging Claude's understanding for intelligent memory generation.
