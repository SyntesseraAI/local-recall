# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name encodes the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcripts are stored as `.jsonl` files (JSON Lines format)

## Key Details

- Each session generates a transcript file named with a timestamp or session ID
- Transcripts contain the full conversation history in JSON format
- One message per line in JSONL format for efficient parsing
- This enables external tools (like Local Recall's MCP server) to access and process session transcripts asynchronously

## Usage in Local Recall

The MCP server background daemon:
1. Scans the transcripts directory for new/changed `.jsonl` files
2. Uses content hashes to detect changes
3. Processes transcripts to extract memories
4. Runs on a 5-minute interval
