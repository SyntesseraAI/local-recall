Development requirements and structure:
- Node.js 18+ and TypeScript 5.9+ required
- `npm install && npm run build` for setup
- Test structure: `tests/unit/` for unit tests, `tests/integration/` for integration tests
- All new features must have corresponding tests covering happy paths and edge cases
- Documentation should be placed in `docs/` folder
- Scripts: `npm run build`, `npm run dev`, `npm test`, `npm run lint`, `npm run mcp:start`
