The memory extraction system uses the `claude -p` CLI with the `-c` flag to specify context files. The process:
1. Reads transcript files from the project's transcript directory
2. Passes transcript content to Claude Haiku via stdin
3. Uses a structured prompt template (from `src/prompts/memory-extraction.ts`) to generate memories
4. Parses the JSON response containing extracted memories
5. Validates and deduplicates memories before storage

Implemented in `src/core/memory-extractor.ts` with the `MemoryExtractor` class.
