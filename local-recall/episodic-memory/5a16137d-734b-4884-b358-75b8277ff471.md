Established testing workflow for hooks:
1. Verify hooks are compiled in dist/hooks/ (results of `npm run build`)
2. Check existing memory files in local-recall/episodic-memory/ to understand baseline state
3. Check memory index (orama-episodic-index.json) to confirm vector store state
4. Test session-start hook by simulating hook input with session_id, transcript_path, and cwd
5. Test stop hook similarly

Hooks expect JSON input via stdin with standard hook parameters and output results to stdout.
