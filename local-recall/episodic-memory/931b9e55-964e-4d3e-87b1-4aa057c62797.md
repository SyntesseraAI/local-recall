Development standards for this project:
- All new features must have unit tests in `tests/unit/`
- Integration tests go in `tests/integration/`
- Documentation must be in `docs/` folder
- Update existing docs when modifying functionality
- Tests should cover happy paths and edge cases
- Run with: `npm test`, `npm run test:unit`, `npm run test:integration`
