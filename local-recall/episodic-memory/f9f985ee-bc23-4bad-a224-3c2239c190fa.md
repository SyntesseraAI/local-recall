# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and a UUID appended. For example:

- Project at `/Users/joe/Code/Syntessera/local-recall` →  `_Users_joe_Code_Syntessera_local-recall-01HQY8jXfJm7zh35BcYYiE1u`
- Project at `/path/to/project` → `_path_to_project-<uuid>`

## Transcript Storage

Transcripts are stored as JSONL files in `~/.claude/projects/<project-folder>/transcripts/` directory.

## Relevance to Local Recall

The transcript-collector needs to scan this directory to find and process transcripts for memory extraction. The path pattern and JSONL format are important for implementing the transcript sync mechanism.
