# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with one JSON object per line. Each object contains:
- `type`: The type of event (e.g., "user_message", "assistant_response", "tool_use")
- `timestamp`: ISO-8601 timestamp of when the event occurred
- Other fields depend on the event type

This path convention is used by the `transcript-collector` module to locate and process Claude Code session transcripts.
