Memory extraction prompts are isolated in `src/prompts/memory-extraction.ts` for maintainability. The prompt framework includes:

1. **System prompt**: Defines the role and instructions for memory extraction
2. **User prompt template**: Takes transcript content and generates extraction instructions
3. **Structured output**: Guides Claude to return memories in the expected JSON format

This separation allows for prompt iteration and testing without modifying the core extractor logic. Keep prompts focused on guiding Claude to identify actionable, specific memories that help future sessions.
