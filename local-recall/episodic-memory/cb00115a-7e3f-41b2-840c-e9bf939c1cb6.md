# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `.jsonl` format in the `transcripts/` subdirectory

## Implementation Details

When implementing transcript collection for the MCP server:
1. Get the working directory from stdin input (`cwd` field)
2. Convert absolute path to folder name: replace `/` with `-` and strip leading `/`
3. Read transcript files from `~/.claude/projects/<converted-path>/transcripts/`
4. Parse `.jsonl` format (one JSON object per line)
5. Each line represents a single message in the transcript

## Related Code

This is used in the MCP server's background daemon (`src/mcp-server/server.ts`) to sync transcripts and extract memories from them.
