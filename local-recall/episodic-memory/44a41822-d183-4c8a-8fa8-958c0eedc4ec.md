The UserPromptSubmit hook extracts keywords from user prompts by calling `claude -p --model haiku --no-hooks` to leverage Claude's language understanding. This provides more intelligent keyword extraction than simple text parsing.

The hook receives the user prompt via stdin and uses Claude to identify relevant keywords, which are then used to search the memory index.
