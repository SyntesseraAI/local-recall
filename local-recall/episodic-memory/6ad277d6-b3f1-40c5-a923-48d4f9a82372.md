# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

Project folders use the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored at `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Filename Convention

Transcript files are named with timestamps:
- Format: `YYYYMMDD_HHmmss.json`
- Example: `20251202_120500.json`
- One file per session

## Relevance to Local Recall

This is critical for the MCP server background daemon which needs to locate and process transcripts from Claude's cache directory for memory extraction.
