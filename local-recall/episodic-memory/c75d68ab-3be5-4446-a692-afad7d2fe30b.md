The `.gitignore` file generated in the local-recall folder must exclude:
- `transcripts/` - Claude transcript files that should not be version-controlled
- `processed-log.json` - Processed transcript tracking file that's auto-generated
- `orama-*-index.json` - Vector index files (already excluded)
- `recall.log` - Debug log file (already excluded)

Two locations maintain this template:
1. `src/core/index.ts` - Initial gitignore generation (initializeMemoryDir)
2. `src/core/memory.ts` - Fallback gitignore creation

Both must stay in sync when updating gitignore rules.
