# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name converts the absolute working directory path by replacing slashes with dashes:
- Original path: `/Users/joe/Code/Syntessera/local-recall`
- Converted name: `Users-joe-Code-Syntessera-local-recall`
- Full transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files, one event per line.

## Use Cases

This information is critical for:
- The transcript collector (`src/core/transcript-collector.ts`) to locate and process transcripts
- MCP server background daemon to sync and process project transcripts
- Memory extraction from Claude Code sessions
