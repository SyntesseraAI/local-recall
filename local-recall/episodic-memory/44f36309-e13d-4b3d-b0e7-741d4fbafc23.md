# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored as JSONL files in `~/.claude/projects/{encoded-path}/transcripts/`
- Each transcript file is named with a timestamp: `{timestamp}.jsonl`

## Transcript Format

Transcripts are JSONL (JSON Lines) format containing conversation events with:
- `type`: Event type (e.g., 'message', 'tool_call', 'tool_result')
- `content`: Message or tool content
- `role`: 'user' or 'assistant'
- `timestamp`: ISO-8601 formatted timestamp
- `tool_name`: Present for tool invocations
- `tool_input`: Parameters passed to the tool

This is important for transcript parsing in the transcript-collector component.
