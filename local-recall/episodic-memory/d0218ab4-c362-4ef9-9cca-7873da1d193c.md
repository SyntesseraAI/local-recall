MCP server daemon:
1. Syncs transcripts from Claude cache (`~/.claude/projects/<project>/transcripts/`)
2. Processes transcripts using `claude -p` to extract memories
3. Tracks processed transcripts with content hashes for change detection
4. Deletes and recreates memories when transcripts change
5. Runs every 5 minutes in background

Key files: transcript-collector.ts, transcript-condenser.ts, memory-extractor.ts, processed-log.ts
