Transcripts are stored as JSONL (one JSON object per line) with events including User messages, Assistant responses, and Tool invocations. Transcripts can be condensed into high-level events (User, Assistant, Tool: Name, Result: OK/ERROR) for efficient memory extraction and analysis.
