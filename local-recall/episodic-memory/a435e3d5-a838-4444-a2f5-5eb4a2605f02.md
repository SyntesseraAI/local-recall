# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

Transcript files are stored as JSONL (JSON Lines) format with a `.jsonl` extension.

## Relevance

This is critical for the transcript-collector component which needs to locate and process Claude's session transcripts for memory extraction.
