# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with forward slashes replaced by dashes and leading slash removed. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format) in the `transcripts/` subdirectory. Each line is a complete JSON object representing a transcript event.

## Usage

This information is critical for the `TranscriptCollector` when syncing transcripts from Claude's cache. The path construction must handle:
- Platform-specific home directory expansion (`~`)
- Correct slash replacement logic
- Proper subdirectory navigation to the `transcripts/` folder
