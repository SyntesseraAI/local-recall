# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall`
- Transcripts folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Individual transcript files: `<uuid>.jsonl`

## Key Details

- Each transcript is a JSONL file (JSON Lines format)
- Files are ordered by modification time
- The transcript-collector reads and processes these files
- This is a critical path for the memory extraction pipeline to function correctly
