# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension.

## Key Details

- Each line in the JSONL file represents one event/message
- Transcript files are newline-delimited JSON
- The transcript-collector uses this location to find and process transcripts
- This is essential for the MCP server's background daemon that syncs transcripts every 5 minutes
