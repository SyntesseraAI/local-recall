# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes.

### Example
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Encoded project folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a JSON object representing a transcript event.

## Purpose

This is important for the transcript-collector functionality which syncs and processes transcripts from Claude's cache to extract memories.
