# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with leading slashes removed and internal slashes replaced with underscores.

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`

## Transcript Storage

Transcripts are stored as JSONL files in the `transcripts/` subdirectory with filenames like `transcript_1.jsonl`, `transcript_2.jsonl`, etc.

Each line in the JSONL file is a JSON object representing a transcript event or message.
