Memory files follow a strict format with required fields: id (UUID), subject, keywords array, applies_to (scope), occurred_at (ISO-8601), and content_hash (SHA-256 prefix). The applies_to field supports three scope types: 'global', 'file:<path>', or 'area:<name>'.
