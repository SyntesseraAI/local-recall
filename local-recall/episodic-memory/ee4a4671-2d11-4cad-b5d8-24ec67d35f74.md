The BGE-small-en-v1.5 embedding model is automatically downloaded to local_cache/fast-bge-small-en-v1.5/ on first use. This takes 30-60 seconds and is cached for subsequent runs. If the cache becomes corrupted, delete the directory and the model will re-download automatically.
