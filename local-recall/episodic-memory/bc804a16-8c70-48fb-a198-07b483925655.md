The TranscriptCollector class in src/core/transcript-collector.ts syncs transcripts from Claude's local cache directory (~/.claude/projects/<project>/transcripts/) to the local-recall memory system. It tracks processed transcripts using content hashes for change detection, allowing incremental updates when transcripts are modified. This enables the memory extraction system to process transcripts asynchronously without re-processing unchanged content.
