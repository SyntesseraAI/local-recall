# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Path Format

Project folder names encode the absolute path with slashes replaced by dashes:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) with content hashes as filenames:
- Example: `52359c9b9d6ce8e7.jsonl`
- Each line is a complete JSON object representing a message/event
- Files grow as the session progresses

## File Cleanup

Transcript files are removed when their content is fully processed and memories are extracted. This prevents reprocessing.

Relevant code: `src/core/transcript-collector.ts`
