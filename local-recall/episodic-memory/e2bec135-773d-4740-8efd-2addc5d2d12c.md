# Claude Project Naming Convention

Claude Code converts absolute filesystem paths to project folder names by replacing forward slashes with dashes.

## Example

- **Filesystem path**: `/Users/joe/Code/Syntessera/local-recall`
- **Claude project folder**: `Users-joe-Code-Syntessera-local-recall`
- **Full transcript path**: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Usage in Code

When implementing transcript collection:
1. Get the current working directory
2. Replace all `/` with `-` (and remove leading `/`)
3. Use this as the project folder name in `~/.claude/projects/`

This is important for the transcript-collector to correctly locate Claude transcripts.
