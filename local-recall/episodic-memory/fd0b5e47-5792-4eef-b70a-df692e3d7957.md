Memory manager provides CRD operations (no update - memories are idempotent):
- `createMemory(data)` - Creates new memory or returns existing if duplicate
- `deleteMemory(id)` - Delete by ID
- `getMemory(id)` - Retrieve specific memory
- `listMemories(filter?)` - List with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Check for existing duplicate

Deduplication uses both occurrence timestamp and content hash.
