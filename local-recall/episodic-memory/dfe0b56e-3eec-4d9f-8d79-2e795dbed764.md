Local Recall can be configured via environment variables or .local-recall.json file with options:
- memoryDir: Directory for memory storage (default: ./local-recall)
- maxMemories: Maximum number of memories (default: 1000)

The LOCAL_RECALL_DIR environment variable controls where memories are stored when used as MCP server.
