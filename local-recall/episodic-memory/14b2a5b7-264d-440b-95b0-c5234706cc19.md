# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with format: `{session_id}.jsonl`

Each line in the file is a JSON object representing a message event:
- Contains session metadata (session_id, timestamp, etc.)
- Each message is a complete JSON object on a single line
- Ordered chronologically

## Accessing Transcripts Programmatically

Path can be constructed at runtime using `os.path.expanduser()` or equivalent for dynamic access to latest transcripts.
