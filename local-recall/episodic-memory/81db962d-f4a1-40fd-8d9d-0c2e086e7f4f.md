Local Recall is a markdown-powered memory system for Claude Code and MCP-compatible clients. The project root IS the plugin root with:

**Key directories:**
- `src/core/` - Memory CRUD operations, indexing, fuzzy search
- `src/hooks/` - Claude Code hooks (SessionStart, UserPromptSubmit, Stop)
- `src/mcp-server/` - MCP server implementation
- `local-recall/` - Memory storage (version-controlled markdown files)
- `local-recall/index.json` - Gitignored keyword index cache
- `local-recall/recall.log` - Gitignored debug log

**Memory file format:** YAML frontmatter with id, subject, keywords, applies_to (global/file:/path/area:name), occurred_at, content_hash, followed by markdown content.

**Key components:**
- Memory Manager (`src/core/memory.ts`) - CRD operations (no update)
- Index Manager (`src/core/index.ts`) - Keyword indexing
- Search (`src/core/search.ts`) - Fuzzy search implementation
- Background MCP daemon - Syncs transcripts every 5 minutes, extracts memories using `claude -p`
