When extracting memories from transcripts, the output must be ONLY a valid JSON object with no explanation, no markdown formatting, and no code blocks. Return raw JSON directly: `{ "memories": [...] }`. This format is critical for proper parsing by downstream processes that consume the memory extraction output.
