# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Replace `/` with `_`
- Replace `:` with `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `_Users_joe_Code_Syntessera_local-recall`

## File Structure

Transcripts are stored in `~/.claude/projects/<folder>/transcripts/` as JSONL files:
- One transcript per session
- Each transcript contains multiple events (user prompts, assistant responses, tool calls)
- Files are named with timestamps or session IDs

## Accessing Transcripts

The `transcript-collector.ts` module reads from this location to process transcripts and extract memories asynchronously.
