When npm run build is executed, the compiled hook scripts are output to `dev-marketplace/local-recall-plugin/scripts/hooks/` rather than to a `scripts/` directory at the project root. This is configured in the build setup and hooks must be referenced from this location in `.claude/settings.json`.
