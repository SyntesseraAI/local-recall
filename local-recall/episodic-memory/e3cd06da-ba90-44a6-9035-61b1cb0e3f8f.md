# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS (and likely other platforms).

## Path Format

Project folders are named by replacing slashes in the absolute path with dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with the naming pattern: `{session_id}.jsonl`

## Usage in local-recall

The `transcript-collector.ts` module syncs these transcripts from the Claude cache to the local-recall project for processing and memory extraction.
