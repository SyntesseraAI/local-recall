# Claude Transcript Storage Location

Claude Code stores project transcripts in `~/.claude/projects/` with directory names that are URL-encoded versions of the project path.

## Path Format
- Base directory: `~/.claude/projects/`
- Subdirectory: URL-encoded project path (e.g., `/Users/joe/Code/Syntessera/local-recall` â†’ `Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall`)
- Transcripts stored in: `~/.claude/projects/<encoded-path>/transcripts/`
- Format: JSONL files with one JSON object per line

## Usage in Local Recall
The MCP server and hooks use this path to sync and process transcripts asynchronously. The daemon processes transcripts every 5 minutes to extract memories.
