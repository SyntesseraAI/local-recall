The transcript-collector module needs to:
1. List all .jsonl files in `~/.claude/projects/<encoded-path>/transcripts/`
2. Parse each file as newline-delimited JSON
3. Extract message objects with role, content, and metadata
4. Handle multiple sessions (each session is a separate .jsonl file)
5. Process transcripts chronologically

The directory encoding scheme (slashes to dashes) is critical for correctly locating transcripts for the current working directory.
