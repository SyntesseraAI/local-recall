The stop hook implements automatic memory extraction from transcripts:

- **Transcript Parsing**: Uses `src/utils/transcript.ts` with `analyzeForMemories()` function
- **Auto-extraction**: The stop hook automatically analyzes transcripts to identify learnings and create memories
- **Scope Assignment**: Memories are assigned appropriate scope (`global`, `file:`, `area:`)
- **Keyword Generation**: Keywords are automatically extracted from the memory content

This enables passive memory creation without manual intervention. The pipeline appears in `src/hooks/stop.ts` and processes transcript data to identify what should be remembered.
