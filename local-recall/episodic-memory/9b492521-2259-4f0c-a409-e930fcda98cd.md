# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Encoded path: `/Users-joe-Code-Syntessera-local-recall`
- Full transcript location: `~/.claude/projects//Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with `.jsonl` extension. Each line is a valid JSON object representing a transcript event.

## Integration with Local Recall

The transcript-collector module should use this path structure when syncing transcripts from Claude's cache to the local-recall memory system. This is critical for the background MCP server daemon which processes transcripts every 5 minutes to extract memories.
