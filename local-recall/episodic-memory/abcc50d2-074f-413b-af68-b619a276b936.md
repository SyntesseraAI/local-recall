# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with names like `transcript_<session-id>_<timestamp>.jsonl`.

Each line is a transcript entry with `type` and `content` fields:
- `type`: "user_message", "assistant_message", "tool_use", "tool_result", etc.
- `content`: The actual message or tool data

This is important for the transcript-collector to locate and parse transcripts for memory extraction.
