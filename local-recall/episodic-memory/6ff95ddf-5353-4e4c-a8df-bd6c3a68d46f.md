# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format, where each line is a complete JSON object representing a session event or turn in the conversation.

## Importance

This location is critical for the transcript-collector to find and process Claude Code session transcripts for memory extraction.
