# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and colons removed:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `Users_joe_Code_Syntessera_local-recall`

## Transcript Files

Each session creates a JSONL file in `~/.claude/projects/<project-folder>/transcripts/` with format:
- Filename: `<timestamp>-<random-suffix>.jsonl` (e.g., `2025-01-15T14-30-45-abc123.jsonl`)
- Format: Each line is a JSON object representing a transcript event
- Event types: user messages, assistant responses, tool invocations, etc.

This is critical for the transcript-collector to properly locate and parse Claude Code transcripts.
