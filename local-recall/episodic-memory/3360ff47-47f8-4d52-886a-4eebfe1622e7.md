Transcripts are condensed into key events: `[User]` (requests), `[Assistant]` (explanations), `[Tool: Name]` (invocations), `[Result: OK/ERROR]` (outcomes). This format preserves actionable information while reducing token usage, making transcripts suitable for memory extraction with Claude Haiku.
