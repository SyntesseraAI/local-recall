# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with names like:
- `<session-id>.jsonl` - Individual session transcripts

Each line in the JSONL file is a JSON object representing a transcript event.

## Implementation Note

The `transcript-collector.ts` module needs to:
1. Take the current working directory as input
2. Compute the transcript path by replacing `/` with `-`
3. Look in `~/.claude/projects/<path>/transcripts/`
4. Read and parse JSONL files to extract transcript events
