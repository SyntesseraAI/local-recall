Transcripts are condensed before being passed to the memory extraction process. The condensed format includes events like `[User]`, `[Assistant]`, `[Tool: Name]`, and `[Result: OK/ERROR]`. This condensed format is more efficient for Claude to process than raw transcripts and reduces token usage.
