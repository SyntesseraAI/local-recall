The MemoryExtractor class in src/core/memory-extractor.ts processes Claude transcripts to extract valuable memories. It uses the Claude API (via claude -p command) with a prompt from src/prompts/memory-extraction.ts to identify and create memories from session transcripts. The extractor handles transcript reading, calls the Claude model to analyze content, parses the JSON response, and creates memory files using the memory manager.
