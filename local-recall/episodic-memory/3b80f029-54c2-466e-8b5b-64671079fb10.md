# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (JSON Lines format) with one event per line.

## Relevance to Local Recall

The MCP server's background daemon needs to access transcripts from this location to:
1. Extract memories from completed sessions
2. Track processed transcripts using content hashes
3. Detect changes and regenerate memories when transcripts change

This path format is crucial for the `transcript-collector.ts` implementation to correctly locate and process transcript files.
