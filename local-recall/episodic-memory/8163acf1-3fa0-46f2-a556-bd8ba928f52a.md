# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files in the transcripts directory, with one event per line. Each transcript event contains:
- `type`: The event type (e.g., 'user_message', 'assistant_message', 'tool_call')
- `timestamp`: ISO-8601 formatted timestamp
- `content`: The event data

## Key Discovery

When building the transcript collector utility, this path convention is critical for:
1. Locating the correct transcripts directory
2. Parsing the folder name to derive the working directory path
3. Syncing transcripts from Claude's cache for background processing

The transcript collector in the MCP server uses this pattern to find and process transcripts from Claude's project cache.
