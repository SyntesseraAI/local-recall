# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude projects folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as `.jsonl` files in the `transcripts/` subdirectory:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Format: JSONL (JSON Lines) - one JSON object per line
- Each object represents a single message/event in the transcript

## Key Fields in Transcript Objects

- `type`: The message type (e.g., 'message', 'tool_use', etc.)
- `role`: Who sent the message ('user' or 'assistant')
- `content`: The actual message content
- `timestamp`: When the message was created

## Implications for Memory Extraction

When implementing transcript processing for the MCP server or hooks, Claude's transcripts are:
1. Stored locally on the user's machine
2. In JSONL format (line-delimited JSON)
3. Located at a predictable path based on the working directory
4. Can be read and parsed to extract memories from session history
