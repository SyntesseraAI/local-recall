# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Replace `/` with `_` (forward slashes become underscores)
- Remove leading underscore if the path starts with `/`

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Folder name: `Users_joe_Code_Syntessera_local-recall`
- Transcripts location: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcript files are JSONL (JSON Lines) format, with one JSON object per line representing an event in the session.

This information is crucial for the transcript-collector module which syncs and processes transcripts from Claude's cache.
