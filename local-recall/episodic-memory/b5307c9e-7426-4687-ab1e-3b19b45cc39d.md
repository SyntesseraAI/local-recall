# Claude Project Transcripts Location

Claude Code stores project transcripts in the user's cache directory on macOS at `~/.claude/projects/`.

## Project Folder Naming

The project folder name uses the absolute path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This naming convention allows Claude to uniquely identify projects across different machines and ensures transcripts are organized by project.
