# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with the naming convention: `transcript-TIMESTAMP.jsonl`

## Transcript Structure

Each line in a transcript file is a JSON object representing an event:
- `role`: Either 'user' or 'assistant'
- `content`: The text content of the message
- `type`: Optional field indicating message type

This format allows for efficient streaming and parsing of large transcripts.
