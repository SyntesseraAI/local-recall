Three hooks configured in hooks.json:

1. **SessionStart**: Triggered on session begin, loads 5 most recent memories (no vector store, full disk reload for speed)
2. **UserPromptSubmit**: Triggered before Claude processes prompt, performs semantic search using vector embeddings
3. **Stop**: Currently disabled - memory extraction handled by MCP server daemon every 5 minutes

Hooks are shell commands receiving JSON via stdin. Configure in `.claude/settings.json` under hooks section.
