The memory system maintains a searchable index of all memory keywords for fast retrieval.

**How it works**:
- Index is stored at `local-recall/index.json` (gitignored)
- Built by scanning all memory files and extracting keywords
- Uses fuzzy matching to find memories matching user queries
- Fuzzy match threshold is configurable (default 0.6)

**Key functions**:
- `buildIndex()` - Scans memory files and builds keyword index
- `getIndex()` - Retrieves current index
- `refreshIndex()` - Rebuilds index from scratch
- `searchByKeywords(query)` - Fuzzy search implementation

**Configuration**: Fuzzy threshold can be set via `.local-recall.json` or environment variables.
