# Transcript Keyword Extraction

When extracting keywords from Claude transcripts for memory indexing, use the Claude API with prompt caching enabled.

## Implementation Details

- Use Claude Haiku model (`claude-3-5-haiku-20241022`) for keyword extraction
- Enable prompt caching to optimize repeated extraction calls
- Request keywords as a JSON array of strings
- Keep keywords lowercase and specific for better fuzzy matching

## API Call Pattern

The extraction process should:
1. Send the transcript text to Claude
2. Request only keywords as output (no explanations)
3. Parse the JSON array response
4. Filter for relevance (typically 1-10 keywords per transcript)

This approach leverages Claude's language understanding while keeping costs low via Haiku and caching.
