# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the file system path:
- Folder path: `/Users/joe/Code/Syntessera/local-recall`
- Stored as: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`
- All slashes converted to underscores
- Transcripts stored in `transcripts/` subdirectory as JSONL files

This naming convention allows Claude to maintain separate transcript histories for each project across the file system.
