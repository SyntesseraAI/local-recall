Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Converted to: `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format), with one JSON object per line representing a turn in the conversation.

## Relevance

The transcript-collector needs to know this path structure to locate and process Claude Code transcripts for memory extraction.
