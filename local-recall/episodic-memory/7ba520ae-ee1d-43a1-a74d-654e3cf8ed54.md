Local Recall has three core components: (1) Memory Manager (src/core/memory.ts) - CRD operations for memory files, createMemory() returns existing if duplicate, deleteMemory() removes by id, listMemories() with optional filtering, findDuplicate() checks for duplicates; (2) Index Manager (src/core/index.ts) - maintains searchable keyword index in local-recall/index.json, buildIndex() scans memories, refreshIndex() rebuilds; (3) Search (src/core/search.ts) - fuzzy search with searchByKeywords(), searchBySubject(), searchByScope().
