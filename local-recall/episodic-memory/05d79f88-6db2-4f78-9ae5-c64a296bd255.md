# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored as `.jsonl` (JSON Lines) format, one event per line.

## Important Notes

- The transcript path is passed to hooks via `transcript_path` in the JSON input
- This information is crucial for the MCP server's background daemon to locate and process transcripts
- The path structure follows a consistent pattern that can be reconstructed from the working directory
