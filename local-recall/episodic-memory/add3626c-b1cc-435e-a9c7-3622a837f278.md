# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes followed by the project hash.

### Example
For project at `/Users/joe/Code/Syntessera/local-recall`:
- Folder: `Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) with names like:
- `session-1733062956000.jsonl`
- `session-1733049600000.jsonl`

Each line contains a message with fields like `role`, `content`, `type`, `timestamp`.

## Key Details

- Transcripts are automatically created and stored by Claude
- The project hash in the folder name appears to be a session or project identifier
- Multiple transcript files can exist for the same project (one per session)
- Timestamp in filename represents the session creation time in milliseconds
