MCP server (src/mcp-server/server.ts) exposes: memory_create (idempotent), memory_get (by ID), memory_search (vector embeddings), memory_list (with filtering), and index_rebuild (sync vector store). Server runs background daemon syncing transcripts from ~/.claude/projects/<project>/transcripts/ and processing with 'claude -p' every 5 minutes, tracking processed transcripts with content hashes.
