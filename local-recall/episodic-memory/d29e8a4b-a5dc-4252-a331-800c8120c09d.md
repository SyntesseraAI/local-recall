# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path by replacing slashes with dashes:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/transcripts.jsonl`

This path-to-dash conversion is important for the transcript-collector to accurately locate and sync Claude's transcript history for memory extraction.
