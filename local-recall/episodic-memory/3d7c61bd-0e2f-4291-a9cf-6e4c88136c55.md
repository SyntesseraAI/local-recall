The `.gitignore` file created in the `local-recall` folder should exclude:
- `transcripts/` - Directory containing Claude Code session transcripts
- `processed-log.json` - Log file tracking processed transcripts
- `orama-*-index.json` - Vector store indexes (auto-generated)
- `recall.log` - Debug log file

These are auto-generated or temporary files that should not be committed to git. The gitignore template is maintained in two locations:
- `src/core/index.ts` - Primary template definition
- `src/core/memory.ts` - Duplicate that should be kept in sync

Both templates must be updated together when adding new exclusions.
