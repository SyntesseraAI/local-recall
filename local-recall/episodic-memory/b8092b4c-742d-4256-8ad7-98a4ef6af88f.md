Configuration can be set via environment variables (e.g., `LOCAL_RECALL_DIR`, `LOCAL_RECALL_EPISODIC_MAX_TOKENS`) or a `.local-recall.json` file in the project root. Environment variables take precedence over the config file. Common settings include memory directory, max memories limit, and threshold settings for episodic/thinking memory retrieval.
