Each memory file contains YAML frontmatter with:
- `id` - UUID for unique identification
- `subject` - Brief one-line description
- `keywords` - Array of searchable terms
- `applies_to` - Scope (global, file:<path>, area:<name>)
- `occurred_at` - ISO-8601 timestamp for chronological sorting
- `content_hash` - SHA-256 prefix for deduplication detection

Memories are idempotent - creating a duplicate returns the existing memory instead of creating a new file.
