The memory extractor in `src/core/memory-extractor.ts` uses Claude Haiku model with streaming enabled (`stream: true`) to extract memories from transcripts cost-effectively. This is a deliberate choice to minimize token consumption while maintaining quality memory extraction from session transcripts.
