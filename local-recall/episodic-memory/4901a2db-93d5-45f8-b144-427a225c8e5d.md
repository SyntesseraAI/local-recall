# Path Conversion Implementation

The `src/core/transcript-collector.ts` file contains a private function `convertPathToDashes` that:
1. Takes an absolute file path as input
2. Replaces all forward slashes (/) with underscores (_)
3. Returns the converted path for use in Claude's project directory naming

This function is essential for correctly locating transcript files in `~/.claude/projects/` on macOS where the project folder name encodes the original path.
