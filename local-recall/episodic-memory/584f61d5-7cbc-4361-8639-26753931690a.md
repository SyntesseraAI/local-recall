MCP server implementation:
- Exposes 5 main tools: memory_create, memory_delete, memory_search, memory_list, index_rebuild
- Runs background daemon (every 5 minutes) that:
  - Syncs transcripts from Claude's cache
  - Uses `claude -p` with Haiku model to extract memories
  - Tracks processed transcripts with content hashes
  - Deletes/recreates memories when transcripts change
- Configurable via `.mcp.json` and environment variables
- Can run standalone with `npm run mcp:start`
