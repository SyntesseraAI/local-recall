MCP server exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild. Background daemon syncs transcripts from Claude cache, processes them to extract memories, tracks processed transcripts with content hashes for change detection, and deletes/recreates memories when transcripts change. Runs every 5 minutes.
