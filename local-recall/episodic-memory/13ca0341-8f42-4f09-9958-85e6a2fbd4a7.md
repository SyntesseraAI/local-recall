# Claude Transcript Storage

Claude Code stores project transcripts in `~/.claude/projects/` with URL-encoded directory names based on the project path.

## Path Structure

- Base directory: `~/.claude/projects/`
- Directory naming: URL-encoded project path (spaces become `%20`, slashes become `%2F`, colons become `%3A`)
- Transcript files: Stored in `transcripts/` subdirectory
- File format: JSONL (JSON Lines), one event per line

## Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Encoded name: `Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall`
- Path: `~/.claude/projects/Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall/transcripts/`

## Related Code

- `src/core/transcript-collector.ts` - Collects and processes transcripts from this location
- The URL encoding must be considered when constructing file paths in transcript collection logic
