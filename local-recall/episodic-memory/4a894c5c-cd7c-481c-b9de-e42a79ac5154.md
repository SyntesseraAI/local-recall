# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format), one per Claude Code session.

## Implementation

The `transcript-collector.ts` module handles discovering and reading these transcripts:
- Uses `os.homedir()` to get the home directory
- Constructs the path by replacing all `/` in the working directory with `-`
- Reads and parses the JSONL files to extract conversation events
