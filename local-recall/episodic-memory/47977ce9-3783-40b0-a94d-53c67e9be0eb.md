# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with the pattern `transcript_YYYY-MM-DD_HHmmss.jsonl`

## Implementation Notes

When implementing transcript collection:
1. Replace all forward slashes in the absolute path with dashes
2. Look for the `.claude/projects/` directory in the user's home directory
3. Files are newline-delimited JSON (JSONL format)
4. Each line represents a transcript event with timestamp and content
