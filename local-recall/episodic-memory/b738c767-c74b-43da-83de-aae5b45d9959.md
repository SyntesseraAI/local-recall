Claude Code stores session transcripts in `~/.claude/projects/` directory. The project folder name is derived from the working directory path with slashes replaced by dashes.

Example path pattern:
`/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/086dd549-7e86-4477-a7e8-...` (project directory)

Within each project directory:
- `transcripts/` folder contains JSONL files
- Each transcript file is named with a UUID pattern like `086dd549-7e86-4477-a7e8-dc959b4b6e0b.jsonl`
- Files contain newline-delimited JSON records

This path structure is used by the transcript-collector to find and process session transcripts for memory extraction.
