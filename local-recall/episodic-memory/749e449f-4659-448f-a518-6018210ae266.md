# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL files with:
- Filename format: `YYYY-MM-DD.jsonl` (date-based)
- Each line is a JSON object representing a turn in the conversation
- Files accumulate throughout the day

## Implementation Note

This information is useful for the MCP server's transcript collection daemon which needs to sync transcripts from Claude's cache for memory extraction purposes.
