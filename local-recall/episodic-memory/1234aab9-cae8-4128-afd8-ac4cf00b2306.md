Claude stores transcripts in `~/.claude/projects/<project>/transcripts/`. The Local Recall MCP server background daemon syncs these transcripts, processes them to extract memories every 5 minutes, and tracks processed transcripts with content hashes for change detection. This enables asynchronous memory extraction without blocking the main Claude session.
