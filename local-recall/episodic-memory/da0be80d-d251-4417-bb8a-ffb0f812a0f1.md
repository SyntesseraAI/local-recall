The transcript condensing logic is located in `src/core/transcript-collector.ts`. This module converts full transcripts into a condensed event format with [User], [Assistant], [Tool: Name], and [Result: OK/ERROR] markers. The condensed format is then used as input to the memory extraction prompt.
