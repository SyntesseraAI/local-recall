Memory files are stored in `local-recall/episodic-memory/` as individual markdown files and ARE version-controlled in git. The directory structure:
- `local-recall/` - Project memory storage root
- `.gitignore` - Auto-generated, excludes index.json and recall.log
- `index.json` - Keyword index cache (gitignored)
- `recall.log` - Debug log file (gitignored)
- `episodic-memory/` - Individual memory markdown files (tracked in git)

This makes memories shareable across team members.
