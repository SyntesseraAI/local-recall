# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Encoded folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored in JSONL format with sequential numbering:
- File format: `transcript_<number>.jsonl` (e.g., `transcript_0.jsonl`, `transcript_1.jsonl`)
- Each line in the file is a JSON object representing a single event
- Transcripts accumulate over time, with new sessions creating new files

## Key Implementation Detail

When collecting transcripts, the path encoding must match Claude's naming convention exactly:
- Replace all `/` with `-` in the absolute path
- Include the full path from root (starting with `Users` on macOS, not `~`)
- This allows the transcript-collector to locate the correct transcript files
