# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using a conversion scheme:
- Replace forward slashes with underscores
- Replace colons with underscores (for drives on Windows)
- Preserve the rest of the path structure

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `_Users_joe_Code_Syntessera_local-recall`

## Transcript Format

Transcripts are stored as JSONL files (one JSON object per line) in the project folder. Each line represents a single event in the session.

## Relevance to Local Recall

The `transcript-collector.ts` module needs to build the correct path to locate and process Claude Code transcripts for memory extraction. Understanding this path conversion is critical for the MCP server's background daemon to sync transcripts correctly.
