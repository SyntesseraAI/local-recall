# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Conversion

The project folder name converts the absolute working directory path:
1. Take the absolute path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replace forward slashes (`/`) with dashes (`-`)
3. Remove leading dash from the start

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript Files

Inside the project folder, transcripts are stored as:
- Pattern: `*.jsonl` files (JSON Lines format)
- Location: `~/.claude/projects/[converted-path]/transcripts/`
- Each transcript is a session recording with tool outputs and exchanges
