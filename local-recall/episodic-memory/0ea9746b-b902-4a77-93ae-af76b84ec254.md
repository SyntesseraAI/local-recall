The memory extractor (`src/core/memory-extractor.ts`) uses the Claude API (via `claude` command with `-p` flag for parsing) to analyze transcripts and extract structured memories. It processes transcript files and generates memory objects that can be stored in the episodic-memory directory. The extractor handles the conversion of raw transcript data into semantic memories with proper metadata.
