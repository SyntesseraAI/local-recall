# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a complete JSON object representing a message/event in the conversation.

## Key Details

- Multiple transcript files can exist for different sessions
- Files are named with timestamps or session identifiers
- Each line in a `.jsonl` file is independently valid JSON
- This is important for the transcript-collector module which needs to parse these files to extract memories
