Claude Code stores project transcripts in `~/.claude/projects/` directory on macOS. The project folder name is derived from the absolute path with slashes replaced by dashes (e.g., `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`). Transcript files are stored in JSONL format in the `transcripts/` subdirectory within each project folder.
