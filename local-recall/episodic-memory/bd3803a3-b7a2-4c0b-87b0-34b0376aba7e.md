The local-recall system extracts memories from Claude Code session transcripts to persist knowledge across sessions. The extraction process analyzes what was learned, discovered problems and solutions, and codebase-specific patterns. Memories should be specific with file paths and function names, concise with one concept per memory, and actionable for future assistants.
