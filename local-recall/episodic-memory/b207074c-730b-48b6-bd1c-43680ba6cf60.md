# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name in the path uses dashes instead of slashes to represent the original file path:
- Original path: `/Users/joe/Code/Syntessera/local-recall`
- Claude folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This format is used by the transcript collector to locate and sync transcripts from Claude's cache.

## Related Code

The `transcript-collector.ts` uses this path format when looking up transcripts from Claude's project directories.
