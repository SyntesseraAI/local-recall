# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the base64-encoded project path with:
- `/` (slashes) replaced with `-` (dashes)
- No padding characters

**Example:**
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Base64 encoded: `L1VzZXJzL2pvZS9Db2RlL1N5bnRlc3NlcmEvbG9jYWwtcmVjYWxs`
- Base64 (no padding): `L1VzZXJzL2pvZS9Db2RlL1N5bnRlc3NlcmEvbG9jYWwtcmVjYWxs`
- Transcripts folder: `~/.claude/projects/L-VzZXJzL-pvZS9Db2RlL-1N5bnRlc3NlcmEvbG9jYWwtcmVjYWxs/transcripts/`

## Related Code

The `transcript-collector.ts` module handles reading and parsing these transcript files for memory extraction in the MCP server background daemon.
