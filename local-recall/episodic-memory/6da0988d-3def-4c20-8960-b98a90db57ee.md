# MCP Server Background Processing

The MCP server runs a background daemon that automatically processes Claude transcripts:

## Processing Cycle
- Runs every 5 minutes
- Syncs transcripts from `~/.claude/projects/<project>/transcripts/`
- Extracts memories from transcript content using `claude -p`
- Tracks processed transcripts with content hashes for change detection
- Automatically deletes and recreates memories when transcripts change

This asynchronous processing keeps the memory system in sync without blocking Claude Code sessions.
