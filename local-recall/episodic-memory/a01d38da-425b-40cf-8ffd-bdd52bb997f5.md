The `local-recall/` directory structure separates version-controlled memories from auto-generated files:

- **episodic-memory/** - Version-controlled individual memory files (*.md)
- **memory.sqlite** - Gitignored, auto-generated vector store
- **recall.log** - Gitignored, debug log file
- **.gitignore** - Auto-generated, explicitly excludes memory.sqlite and recall.log

This separation allows memory files to be tracked in git while keeping the SQLite database and logs local to each developer's machine. The project automatically maintains and updates the .gitignore file.
