# Claude Project Transcripts Location

Claude stores project transcripts at `~/.claude/projects/` on macOS.

## Path Conversion

Project folder names are derived from the working directory path:
- Forward slashes in paths are converted to dashes
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `users-joe-code-syntessera-local-recall`

## File Format

Transcripts are stored as JSONL files (one JSON object per line):
- Location: `~/.claude/projects/{project-folder}/transcripts/`
- Format: `.jsonl` (JSON Lines)
- Each line is a complete JSON object representing a transcript event

## Relevance to Local Recall

The transcript collector (`src/core/transcript-collector.ts`) reads from this location to:
1. Find project-specific transcripts
2. Process them for memory extraction
3. Track changes using content hashes for deduplication
