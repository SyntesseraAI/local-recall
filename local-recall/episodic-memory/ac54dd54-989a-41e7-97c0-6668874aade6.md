# Transcript Condensing for Memory Extraction

The transcript-collector needs to condense full JSONL transcripts into a simplified format for memory extraction.

## Requirements

1. **Input**: Full JSONL transcript from Claude cache
2. **Output**: Condensed JSON with events:
   - `[User]` - User prompts
   - `[Assistant]` - Claude responses
   - `[Tool: Name]` - Tool invocations
   - `[Result: OK/ERROR]` - Tool outcomes

## Implementation Notes

- Filter out low-value entries (timestamps, status messages)
- Preserve tool names and invocation context
- Include only relevant assistant responses (not metadata)
- Maintain chronological order
- Keep the condensed format machine-readable for downstream processing
