# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with leading slash removed and remaining slashes replaced with colons.

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Folder name: `Users:joe:Code:Syntessera:local-recall`
- Full path: `~/.claude/projects/Users:joe:Code:Syntessera:local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files where each line is a JSON event object.

## Implementation

This is used in `src/core/transcript-collector.ts` to locate and process transcripts for memory extraction in the MCP server daemon.
