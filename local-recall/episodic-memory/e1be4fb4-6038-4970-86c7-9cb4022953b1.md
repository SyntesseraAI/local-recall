# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Converting the path to a slug format
2. Replacing path separators with dashes
3. Using lowercase characters

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `users-joe-code-syntessera-local-recall`
- Full transcript path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files (one JSON object per line), with filenames following the format `transcript-{timestamp}.jsonl`.

This is used by the transcript-collector in local-recall to sync transcripts from Claude's cache for memory extraction.
