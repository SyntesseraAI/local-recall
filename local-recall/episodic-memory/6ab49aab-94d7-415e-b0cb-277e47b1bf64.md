# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in `transcripts/` subdirectory:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Format: JSONL (JSON Lines - one JSON object per line)
- Each entry contains: `role` ("user" or "assistant"), `content`, `timestamp`

## Integration Notes

The `transcript-collector.ts` module uses this path format to sync and process transcripts for memory extraction. The path encoding is important for correctly locating transcript files across different projects.
