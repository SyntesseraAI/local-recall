The MemoryManager in `src/core/memory.ts` includes a `findDuplicate(occurredAt, contentHash)` method for deduplication. Memory creation is idempotent - if a duplicate is detected, it returns the existing memory instead of creating a new one. Content hash uses SHA-256 prefix (16 chars).
