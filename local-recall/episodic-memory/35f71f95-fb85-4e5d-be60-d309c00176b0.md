MCP server runs a background daemon that syncs transcripts from Claude's cache (`~/.claude/projects/<project>/transcripts/`), processes them to extract memories using claude -p, tracks processed transcripts with content hashes for deduplication, and runs every 5 minutes. Available tools: memory_create, memory_delete, memory_search, memory_list, index_rebuild.
