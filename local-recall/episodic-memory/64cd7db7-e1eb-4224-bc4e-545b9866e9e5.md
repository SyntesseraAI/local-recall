# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

Project folder names use absolute paths with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcripts are stored as `.jsonl` files (JSON Lines format)

## Transcript Sync Process

The transcript collector needs to:
1. Identify the current working directory
2. Convert it to the Claude cache path format
3. Read `.jsonl` transcript files from that location
4. Process transcripts to extract memories

This is crucial for the background daemon and transcript collection functionality.
