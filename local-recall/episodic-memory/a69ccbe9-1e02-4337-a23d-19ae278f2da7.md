The project is organized as:
- `src/core/` - Core memory management (memory.ts, vector-store.ts, embedding.ts, search.ts, transcript handling, thinking memories)
- `src/hooks/` - Claude Code hooks (session-start.ts, user-prompt-submit.ts, stop.ts)
- `src/mcp-server/` - MCP server implementation (server.ts, tools.ts)
- `src/prompts/` - Prompt templates for memory extraction
- `src/types/` - Type definitions
- `src/utils/` - Utilities (markdown, transcript, logger, fuzzy, config, gitignore, summarize)
- `local-recall/` - Memory storage directory (version-controlled episodic and thinking memories)
- `local_cache/` - Embedding model cache (gitignored)
