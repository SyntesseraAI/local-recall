# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name encodes the absolute working directory path:
- Replace forward slashes `/` with underscores `_`
- Replace colons `:` with underscores `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users_joe_Code_Syntessera_local-recall`

## Transcript Files

Within each project folder:
- Transcripts are stored in the `transcripts/` subdirectory
- Files are named with UUID identifiers: `<uuid>.md`
- Each transcript contains the full conversation history in markdown format

## Usage in MCP Server

The MCP server can access transcripts from:
- The actual session transcript passed via `transcript_path` in hook inputs
- The cached transcript directory for background processing
- Transcripts are used for memory extraction and processing
