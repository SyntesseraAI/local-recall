The SessionStart hook is triggered when a Claude Code session begins. It receives JSON input with `session_id`, `transcript_path`, and `cwd`, then: 1) Loads the memory index from `local-recall/index.json`, 2) Retrieves relevant memories based on current context, 3) Outputs memory content to stdout for injection into Claude's context. This provides instant access to relevant project history.
