# TranscriptCollector Path Handling

The `transcript_path` input can be either:
- A single file path (JSONL transcript file)
- A directory path containing multiple transcript files

## Required Handling

The TranscriptCollector should:
1. Check if the path is a file or directory using `fs.statSync()`
2. If file: process that single file
3. If directory: use glob pattern to find all JSONL files (e.g., `*.jsonl` or `/**/*.jsonl`)
4. Filter out test files (paths containing "__tests__" or ".test")
5. Process all matching files

This flexibility allows the MCP server and hooks to pass either specific transcripts or entire transcript directories for batch processing.
