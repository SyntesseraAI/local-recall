Claude Code hooks (SessionStart, UserPromptSubmit, Stop) receive JSON input via stdin with the following fields:
- `session_id`: Identifier for the current session
- `transcript_path`: Path to the transcript file
- `cwd`: Current working directory of the project
- `prompt`: (UserPromptSubmit only) The user's prompt text

Hooks parse this JSON, perform their operations, and output results to stdout.
