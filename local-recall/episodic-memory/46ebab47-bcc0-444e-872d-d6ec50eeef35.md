# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by replacing slashes with dashes and removing colons.

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcript location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are stored in JSONL format with event objects.

## Relevance to Local Recall

The transcript-collector component needs to read transcripts from this location to extract memories. The path transformation logic is critical for correctly locating project transcripts.
