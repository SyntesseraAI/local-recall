## Functionality
The user-prompt-submit hook:
1. Receives the user's prompt as input
2. Searches the memory index for relevant memories based on prompt keywords
3. Outputs matching memories to stdout
4. These memories are automatically injected into Claude's context before processing the prompt

## Purpose
Enhances Claude's awareness of project-specific context by providing relevant previous learnings before processing each user request.
