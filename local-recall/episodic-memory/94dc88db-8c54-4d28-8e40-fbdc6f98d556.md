The system prevents duplicate memories using two fields:
1. `occurred_at`: ISO-8601 timestamp of when the event occurred
2. `content_hash`: SHA-256 hash prefix (16 characters) of the memory content

The `MemoryManager.findDuplicate(occurredAt, contentHash)` method checks for existing duplicates. If a duplicate is found, the existing memory is returned instead of creating a new one, making memory creation idempotent.
