# Memory Extraction via Claude Haiku API

The user-prompt-submit hook uses the Claude CLI to extract keywords from user prompts:
- Command: `claude -p --model haiku`
- Input: User prompt text (piped via stdin)
- Output: JSON array of keywords

This allows the system to dynamically identify relevant keywords without hard-coding search terms, enabling intelligent memory retrieval based on user input.
