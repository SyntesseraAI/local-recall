# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS in JSONL format.

## Project Folder Naming

Project folders are named by converting the absolute path:
- Replace all forward slashes `/` with dashes `-`
- The project folder name is the absolute path with slashes converted to dashes
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`

## Transcript Access

To access transcripts for a specific project:
1. Identify the absolute working directory path
2. Convert slashes to dashes
3. Navigate to `~/.claude/projects/{encoded-path}/transcripts/`
4. Transcripts are stored as individual JSONL files

This encoding scheme allows Claude to uniquely identify and organize transcripts by project location.
