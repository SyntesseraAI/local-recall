# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This path format is used by the transcript collector to locate and process transcript files. The conversion replaces `/` with `-` in the absolute path.

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension in the transcripts directory.
