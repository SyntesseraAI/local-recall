The memory extractor in memory-extractor.ts uses the `claude -p` command to invoke Claude Haiku for extracting memories from transcripts. The `-p` flag enables prompt mode. The extracted memories are parsed from the stdout output and converted from markdown to structured JSON format with metadata fields (id, subject, keywords, applies_to, occurred_at, content_hash).
