# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using SHA-256 hashing. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Hash: First 12 chars of SHA-256(/Users/joe/Code/Syntessera/local-recall)
- Project folder: `~/.claude/projects/01HQY8jXfJm7zh35/`
- Transcripts location: `~/.claude/projects/01HQY8jXfJm7zh35/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with one event per line. Events are newline-delimited JSON objects that can be parsed incrementally.

## Relevant Files

The `transcript-collector.ts` file handles reading and parsing these transcripts from Claude's cache.
