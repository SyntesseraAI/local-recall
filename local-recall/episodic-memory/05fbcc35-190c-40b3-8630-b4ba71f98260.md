# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- JSONL file format: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/<session-id>.jsonl`

Each transcript is stored as JSONL (JSON Lines) where each line is a separate JSON object representing a transcript event.

## Implementation

When implementing transcript collection, account for this path format:
1. Get absolute working directory path
2. Replace all slashes with dashes
3. Construct path as `~/.claude/projects/{transformed-path}/transcripts/`
4. Parse JSONL format (each line is valid JSON)

See `src/core/transcript-collector.ts` for reference implementation.
