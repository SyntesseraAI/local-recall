Added new environment variables and configuration schema fields:

**Token counts (default: 1000 each):**
- `LOCAL_RECALL_EPISODIC_MAX_TOKENS` - max tokens of episodic memories to inject
- `LOCAL_RECALL_THINKING_MAX_TOKENS` - max tokens of thinking memories to inject

**Similarity thresholds (default: 0.8 = 80% each):**
- `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY` - minimum similarity score for episodic memories
- `LOCAL_RECALL_THINKING_MIN_SIMILARITY` - minimum similarity score for thinking memories

Each memory type now has independent configuration allowing users to fine-tune how many tokens and what similarity threshold to use for episodic vs thinking memory retrieval.
