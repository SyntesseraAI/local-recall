# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and dots. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Format

Transcripts are stored as JSONL files in the project folder:
- File: `~/.claude/projects/<project-folder>/transcripts/<transcript-id>.jsonl`
- Each line is a complete JSON event object
- Events have `type`, `timestamp`, and type-specific fields

## Implementation

The `transcript-collector.ts` module handles:
- Computing project folder name from working directory
- Reading and parsing JSONL transcript files
- Extracting messages and tool invocations from transcript events

This is used by the MCP server background daemon to sync transcripts from Claude's cache for memory extraction.
