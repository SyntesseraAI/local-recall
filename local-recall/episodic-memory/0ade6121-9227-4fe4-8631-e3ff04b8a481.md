# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses base64 encoding of the absolute path:
- Example: Project at `/Users/joe/Code/Syntessera/local-recall` is stored at `~/.claude/projects/L1VzZXJzL2pvZS9Db2RlL1N5bnRlc3NlcmEvbG9jYWwtcmVjYWxs/`
- The base64 string represents the absolute path with a `/` prefix

## Transcript Files

Transcripts within each project are stored as:
- `.jsonl` format (JSON Lines)
- Named with session IDs or timestamps
- Located in the project's transcripts subdirectory

## Important for Integration

When building transcript collection functionality, the `transcript-collector.ts` must:
1. Calculate base64 encoding of the working directory path
2. Query the correct location in `~/.claude/projects/`
3. Handle path conversion properly to match Claude's internal storage format
