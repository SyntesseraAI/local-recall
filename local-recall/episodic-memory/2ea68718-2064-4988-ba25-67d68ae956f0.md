# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Located at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format):
- One JSON object per line
- Filenames use ISO 8601 timestamps: `2024-01-15T10-30-45.jsonl`
- Each object contains session events with `role` and `content` fields

## Accessing Transcripts

The `transcript-collector.ts` module provides utilities for:
- Building the correct path from the working directory
- Reading and parsing JSONL transcript files
- Extracting session data for memory creation

This is critical for the MCP server's background daemon which syncs transcripts every 5 minutes.
