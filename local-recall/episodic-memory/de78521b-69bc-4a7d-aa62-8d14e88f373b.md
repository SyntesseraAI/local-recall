Memory files include required frontmatter fields: id (UUID), subject (brief description), keywords (array), applies_to (scope), occurred_at (ISO-8601 timestamp for deduplication/sorting), and content_hash (SHA-256 prefix for deduplication). The applies_to field can be 'global', 'file:/path/to/file', or 'area:component-name' to specify scope.
