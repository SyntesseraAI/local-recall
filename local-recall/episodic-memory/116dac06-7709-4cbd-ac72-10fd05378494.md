# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension:
- Each line is a valid JSON object representing a single event
- Files are named with timestamps: `YYYYMMDDHHMM.jsonl`
- Events are chronologically ordered within each file

## Transcript Events

Each event object contains:
- `type`: Event type (e.g., 'user_message', 'assistant_response', 'tool_call', etc.)
- `timestamp`: ISO-8601 timestamp
- `content`: Event-specific data

This pattern is consistent across all projects using Claude Code.
