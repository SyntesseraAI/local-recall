# MCP Server Background Daemon

The Local Recall MCP server includes a background daemon that runs independently of Claude Code hooks.

## Functionality
- **Sync Interval**: Processes transcripts every 5 minutes
- **Source**: Syncs from `~/.claude/projects/<project>/transcripts/`
- **Processing**: Uses Claude Haiku to extract keywords and memories from transcripts
- **Deduplication**: Tracks processed transcripts with content hashes to detect changes
- **Lifecycle**: Deletes and recreates memories when transcripts change

## Why Async?
- Decouples memory extraction from interactive Claude Code sessions
- Prevents blocking the user interface during transcript processing
- Allows for more comprehensive memory extraction from full conversation context
