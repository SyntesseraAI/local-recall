- Uses `condenseTranscript()` to summarize transcript content before extraction
- Calls `extractMemoriesFromTranscript()` to generate structured memories using Claude Haiku
- Handles transcript processing with proper error handling and logging
- Generated memories include metadata (subject, keywords, applies_to, occurred_at, content_hash)
- Memory extraction is part of the MCP server's background daemon (runs every 5 minutes)
