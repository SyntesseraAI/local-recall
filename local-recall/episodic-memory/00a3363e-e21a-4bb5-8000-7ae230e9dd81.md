# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with forward slashes replaced by dashes. For example:

- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files in the transcripts directory with naming pattern: `<session-id>.jsonl`

Each line in the JSONL file contains a JSON object representing a transcript event with `type` and `content` fields.
