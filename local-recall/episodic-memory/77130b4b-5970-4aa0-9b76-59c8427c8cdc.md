# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Work directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (JSON Lines format) with `.jsonl` extension.

## Implementation Note

This path construction is critical for `transcript-collector.ts` to locate and process transcripts from Claude sessions. The path transformation (slashes to dashes) is essential for correct file discovery.
