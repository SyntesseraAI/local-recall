# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files, with each line being a complete JSON object representing a single event in the conversation.

## Implications for Local Recall

The transcript-collector needs to:
1. Calculate the project path based on the current working directory
2. Look for transcripts in the `~/.claude/projects/<path>/transcripts/` directory
3. Process JSONL format files to extract memories
