# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path of the project directory:
- Replace leading `/Users/` with empty string
- Replace `/` with `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `joe_Code_Syntessera_local-recall`

## Transcript Files

Within each project folder, transcripts are stored as `.jsonl` files:
- Location: `~/.claude/projects/{project-folder}/transcripts/`
- Format: `.jsonl` (JSON Lines - one JSON object per line)
- Each line contains a message object from the transcript

## Usage in Local Recall

The `transcript-collector.ts` module reads these transcripts to extract memories:
- Scans the transcripts directory for `.jsonl` files
- Reads and parses each line as JSON
- Processes transcripts to extract meaningful memories
- Tracks processed transcripts via content hashes to avoid duplicate processing

## Relevance

This discovery is critical for the MCP server daemon which syncs transcripts asynchronously and processes them to extract memories every 5 minutes.
