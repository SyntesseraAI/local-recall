# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with ISO timestamps:
- Example: `2025-12-01T14-23-45.jsonl`
- One JSON object per line representing conversation events
- Each event has type, content, timestamp, and other metadata

## Implementation

The `src/core/transcript-collector.ts` file handles finding and collecting transcripts programmatically using this path format.
