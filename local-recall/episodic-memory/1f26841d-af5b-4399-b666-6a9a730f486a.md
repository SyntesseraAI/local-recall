# UserPromptSubmit Hook Workflow

## Current Implementation

1. Hook receives user prompt text
2. Extracts keywords using `claude -p --model haiku`
3. Searches memory index using extracted keywords with fuzzy matching
4. Returns matching memories as context

## Purpose

Provides Claude with relevant memories from previous sessions before processing each user prompt, improving context awareness and consistency across sessions.
