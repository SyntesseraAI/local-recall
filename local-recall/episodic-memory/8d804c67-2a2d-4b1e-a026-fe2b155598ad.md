# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts.jsonl`

## File Format

Transcripts are stored in JSONL (JSON Lines) format, one event per line.

## Usage

This is used by the transcript-collector to sync transcripts from Claude's cache for memory extraction via the MCP server daemon.
