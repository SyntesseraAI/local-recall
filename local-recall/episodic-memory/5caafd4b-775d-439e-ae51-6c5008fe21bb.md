The `.mcp.json` plugin configuration uses `${CLAUDE_PLUGIN_ROOT}/scripts/mcp-server/server.js` to reference the bundled server script. This variable is resolved at plugin runtime to point to the actual deployed plugin cache directory, allowing the same config to work across different installation locations.
