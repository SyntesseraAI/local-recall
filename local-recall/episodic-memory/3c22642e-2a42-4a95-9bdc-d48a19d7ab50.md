# Claude Project Transcripts Path Resolution

Claude Code stores project transcripts in `~/.claude/projects/` directory on macOS.

## Folder Naming Convention

The project folder name is **not** simply the base directory name. Instead:

1. Take the absolute working directory path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Convert slashes to hyphens and remove leading slash: `Users-joe-Code-Syntessera-local-recall`
3. This becomes the project folder name in `~/.claude/projects/`

Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Implementation Notes

- Must handle path conversion correctly when building transcript paths
- The `transcript-collector.ts` component needs to replicate this path conversion logic
- Test with various path depths and special characters
- This pattern ensures unique folders for different project locations (even if project names are identical)
