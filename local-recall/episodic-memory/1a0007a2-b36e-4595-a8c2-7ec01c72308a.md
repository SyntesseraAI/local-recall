# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL files (one JSON object per line). Each event/message in the transcript is a separate JSON object.

Key fields in transcript objects:
- `type`: Type of event (e.g., "user_message", "assistant_message", "tool_use")
- `timestamp`: ISO-8601 timestamp when the event occurred
- `content`: The actual message or tool invocation details

## Implementation Location

The transcript collection logic is implemented in `src/core/transcript-collector.ts` and is used by the MCP server to process transcripts for memory extraction.
