# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (newline-delimited JSON) files, with one transcript per file.

## Implications for transcript-collector.ts

When implementing the transcript collector:
1. Use `os.homedir()` to get the user's home directory
2. Replace all forward slashes in the absolute path with dashes to construct the folder name
3. Read from `~/.claude/projects/<project-folder>/transcripts/`
4. Parse as JSONL format (one JSON object per line)

This is used by the MCP server's background daemon to sync and process transcripts for memory extraction.
