# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in:
- `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each line contains a transcript event with metadata like:
- `timestamp`: ISO-8601 timestamp
- `event_type`: Type of event (user_message, assistant_response, tool_invocation, etc.)
- `content`: The actual message or tool data

This is important for the transcript-collector to locate and process transcripts for memory extraction.
