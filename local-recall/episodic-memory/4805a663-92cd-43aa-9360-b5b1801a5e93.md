# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (one JSON object per line) with the naming pattern:
- `transcript-<timestamp>.jsonl`
- Example: `transcript-2025-12-01T15-30-45-123.jsonl`

Each line contains a JSON object representing a session event or message.

## Key Files

- `src/core/transcript-collector.ts` - Handles transcript collection and processing
- The transcript collector reads from the Claude cache and processes JSONL files

This is important for the MCP server's background daemon which syncs transcripts from Claude's cache every 5 minutes.
