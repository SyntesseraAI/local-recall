The MCP server in `src/mcp-server/server.ts` includes a background daemon (documented in comments at line 6) that processes transcripts asynchronously. This daemon runs every 5 minutes to extract memories from Claude's transcript cache. The daemon respects the `episodicEnabled` configuration to determine whether to process episodic memories.
