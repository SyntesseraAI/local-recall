MCP server exposes five memory tools:
- `memory_create` - Create new memory (idempotent)
- `memory_delete` - Delete memory
- `memory_search` - Search memories by keywords
- `memory_list` - List all memories
- `index_rebuild` - Rebuild memory index

Server also runs background daemon that syncs transcripts every 5 minutes, processes them using Claude, detects changes via content hashes, and recreates memories when transcripts change.
