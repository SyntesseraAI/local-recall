# Keyword Extraction for Memory Indexing

The user-prompt-submit hook uses Claude Haiku to extract keywords from user prompts for memory searching.

## Implementation

Using `claude -p --model haiku` with a prompt to extract keywords:
- Model: claude-haiku (smallest, fastest model)
- Input: User prompt text
- Output: JSON array of relevant keywords
- Purpose: Enable fuzzy search matching against stored memories

## Why Haiku

Haiku is lightweight and fast, making it suitable for real-time hook execution during user prompt submission without adding latency.
