# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by three dots ("..."). For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder `...Users...joe...Code...Syntessera...local-recall`

## Transcript Files

Transcripts are stored as JSONL files in `~/.claude/projects/<folder>/transcripts/`
- Each line is a valid JSON object representing a transcript event
- Files are named with timestamps or sequential identifiers
- The MCP server's background daemon should sync these into memory files

## Implementation Details

When implementing transcript collection:
1. Expand `~` to user's home directory using `os.homedir()` or similar
2. URL-encode or escape the path (three dots replace slashes)
3. Read JSONL files line-by-line for memory processing
4. Track processed transcripts by content hash to detect changes
5. Re-process when transcripts are modified or new ones appear
