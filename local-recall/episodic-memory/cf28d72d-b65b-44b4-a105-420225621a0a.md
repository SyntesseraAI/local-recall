Local Recall is a markdown-powered memory system for AI coding assistants. Memory files are stored in `local-recall/episodic-memory/` and use YAML frontmatter with these required fields:

- `id`: UUID identifier
- `subject`: Brief one-line description
- `keywords`: Array of searchable keywords
- `applies_to`: Scope - 'global', 'file:<path>', or 'area:<name>'
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix (16 chars) for deduplication

Memory files are version-controlled in git. The SQLite database (memory.sqlite) and recall.log are gitignored and auto-generated.
