# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes.

### Example

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project ID folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript File Format

Transcripts are stored as JSONL files in the `transcripts/` subdirectory with filenames like:
- `transcript_20240120_143022.jsonl`
- Format: Line-delimited JSON (JSONL)
- Each line is a separate JSON object representing a transcript event

## Key Points

- Transcripts are stored per-project
- File format is JSONL for streaming/sequential processing
- The path encoding scheme uses dash-separated path components
