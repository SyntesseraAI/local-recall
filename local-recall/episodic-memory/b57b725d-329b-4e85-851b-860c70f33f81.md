Local Recall is a markdown-powered memory system stored in `local-recall/episodic-memory/` directory. Each memory is a markdown file with YAML frontmatter containing:

- `id`: UUID unique identifier
- `subject`: Brief one-line description
- `keywords`: Array of searchable keywords
- `applies_to`: Scope (global, file:path, or area:name)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix (16 chars) for deduplication

Memory files are version-controlled in git, while the SQLite database (`memory.sqlite`) and logs (`recall.log`) are gitignored.
