The MemoryExtractor class:
- Processes transcripts asynchronously using LLM streaming
- Uses Claude Sonnet (not Haiku) for memory extraction to ensure quality
- Implements streaming response parsing for efficient processing
- Handles deduplication of memories using content hashing
- Tracks processing state to avoid reprocessing same transcripts

Key methods:
- `extractMemories(transcript)` - Main entry point for extraction
- `parseStreamingResponse()` - Parses streamed JSON responses
- Integrates with core memory system for persistence
