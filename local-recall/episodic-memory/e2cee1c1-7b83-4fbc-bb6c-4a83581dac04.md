Each memory must have: id (UUID), subject (brief description), keywords (array), applies_to (global|file:/path|area:name), occurred_at (ISO-8601 for deduplication), and content_hash (SHA-256 prefix for detecting duplicates). The content_hash and occurred_at fields enable deduplication to prevent duplicate memories from being stored.
