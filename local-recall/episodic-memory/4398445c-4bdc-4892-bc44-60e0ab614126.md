# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files:
- File naming: `<session-id>.jsonl`
- Each line is a JSON object representing a single message or interaction
- Files accumulate over time, with new messages appended

## Key Details

- Transcripts are JSONL format (one JSON object per line), not single JSON files
- The path construction uses absolute working directory path with slashes replaced by dashes
- This is important for the transcript-collector to locate and process transcripts correctly
