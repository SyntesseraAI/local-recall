# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and a UUID appended.

### Example

For working directory: `/Users/joe/Code/Syntessera/local-recall`

Transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-<UUID>/transcripts/`

## Transcript File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with one event per line.

## Implementation Location

The `transcript-collector.ts` module handles reading and processing these transcripts for memory extraction.
