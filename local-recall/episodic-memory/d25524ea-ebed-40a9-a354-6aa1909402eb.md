# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the project folder:
- Location: `~/.claude/projects/{project-folder}/transcripts/`
- Format: JSONL (one JSON object per line)
- Each line represents a single message in the conversation
- Timestamp: Each message has a `timestamp` field

## Message Structure

Each JSONL line contains:
- `type`: Message type (e.g., 'user', 'assistant')
- `content`: Message content (markdown text)
- `timestamp`: ISO-8601 timestamp when message was sent
- `tool_calls`: Array of tool invocations (if present)
- `tool_results`: Array of tool results (if present)

This is used by the transcript-collector to extract memories from conversations.
