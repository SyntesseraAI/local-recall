The MCP server daemon processes transcripts asynchronously every 5 minutes:
1. Syncs transcripts from Claude's cache
2. Processes transcripts using Claude Haiku to extract memories
3. Uses content hashes to detect changes (prevents duplicate processing)
4. Deletes and recreates memories when transcripts change
5. Transcripts are condensed before processing to remove noise and focus on meaningful content
