The system uses SHA-256 content hashes to detect and prevent duplicate memories. When processing transcripts, the `findDuplicate(occurredAt, contentHash)` function checks if a memory with the same occurred_at timestamp and content hash already exists, returning the existing memory instead of creating a duplicate.
