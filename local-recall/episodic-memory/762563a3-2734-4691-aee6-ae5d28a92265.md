The memory extraction process receives a condensed transcript with events marked as:
- `[User]` - User requests or questions
- `[Assistant]` - Claude's responses and explanations
- `[Tool: Name]` - Tool invocations
- `[Result: OK/ERROR]` - Tool outcomes

This format is designed for efficient analysis of session activity to extract valuable memories for future AI assistants.
