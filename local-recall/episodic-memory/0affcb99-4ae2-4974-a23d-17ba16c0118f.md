# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name in the path is derived from the working directory path:
- Forward slashes `/` in the path are replaced with dashes `-`
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files where each line is a complete JSON event object.

## Relevance

The `transcript-collector.ts` module needs to use this path format when retrieving transcripts from Claude's cache. The path construction is critical for the memory extraction pipeline that processes session transcripts to create memories.
