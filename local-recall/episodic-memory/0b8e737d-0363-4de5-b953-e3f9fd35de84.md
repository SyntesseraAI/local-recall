# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with characters converted:
- Forward slashes `/` → underscores `_`
- Colons `:` → underscores `_`
- Other special characters → underscores `_`

## Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Directory: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`
- Transcripts: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension, not `.json`. Each line is a separate JSON object representing a transcript event.

## Implementation Note

The `src/utils/transcript-collector.ts` needs to handle this path conversion when reading Claude's transcript files.
