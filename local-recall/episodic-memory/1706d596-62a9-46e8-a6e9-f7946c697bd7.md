In the `local-recall/` directory:
- `orama-episodic-index.json` and `orama-thinking-index.json` are gitignored (auto-generated from memory files)
- `episodic-memory/*.md` and `thinking-memory/*.md` are version-controlled
- `.gitignore` file is auto-generated by the system
- `recall.log` is gitignored (debug log)

This allows sharing memory content via git while keeping indexes local for performance.
