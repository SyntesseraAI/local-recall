The memory extraction process uses a prompt template defined in `src/prompts/memory-extraction.ts`. This separates the extraction logic from the prompt definition, making it easier to modify extraction behavior without changing core code.

The prompt should specify the memory format (YAML frontmatter with fields like id, subject, keywords, applies_to, occurred_at, content_hash) and guide Claude on how to identify valuable memories from transcripts.
