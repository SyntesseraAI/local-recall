# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (JSON Lines - one JSON object per line).

## Key Insight

The transcript collector tool needs to:
1. Get the working directory from stdin or environment
2. Convert slashes to dashes for the project folder name
3. Read from the correct transcripts directory
4. Parse JSONL format (not plain JSON)

This is critical for the MCP server's background daemon which syncs transcripts to extract memories.
