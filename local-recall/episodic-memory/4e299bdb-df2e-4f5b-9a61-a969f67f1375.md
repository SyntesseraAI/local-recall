SessionStart and UserPromptSubmit hooks are configured in hooks.json and execute as shell commands. They receive JSON input via stdin containing session_id, transcript_path, and cwd. The hooks use `claude -p --model haiku` to extract keywords from prompts for semantic searching of memories.
