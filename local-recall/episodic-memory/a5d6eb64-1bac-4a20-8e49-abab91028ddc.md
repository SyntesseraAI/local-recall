# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in:
`~/.claude/projects/<project-folder>/transcripts/`

Each line is a JSON object representing a single turn in the conversation.

## Relevance to Local Recall

The MCP server background daemon uses this path structure to:
1. Locate and sync transcripts from Claude's cache
2. Process transcripts using `claude -p` to extract memories
3. Track processed transcripts with content hashes for change detection
4. Delete and recreate memories when transcripts change
