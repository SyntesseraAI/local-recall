The memory index (`local-recall/index.json`) maintains a searchable keyword map across all memories. Fuzzy search is implemented to match keywords with configurable thresholds (default 0.6). The index is gitignored and auto-generated from scanning all memory files. Fast keyword lookup enables efficient memory retrieval during hook execution.
