The extraction prompt in `memory-extraction.ts` specifies:
- Required memory fields: subject, keywords, applies_to, content
- Clear guidelines for what to extract (architectural decisions, bug fixes, patterns, configuration quirks)
- Clear guidelines for what NOT to extract (generic knowledge, temporary debugging, obvious info, sensitive data)
- Scope guidance: global, file:<path>, or area:<name>

- Requirement for actionable, specific, and concise memories
