The Local Recall MCP server uses Claude Haiku (`claude -p --model haiku`) to analyze transcripts and extract valuable memories. The transcript condenser converts raw transcripts into a summarized event format with `[User]`, `[Assistant]`, `[Tool]`, and `[Result]` entries before passing to Haiku for memory extraction.
