# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Converting the path to lowercase
2. Replacing `/` with `-`
3. Removing the leading `-`

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `users-joe-code-syntessera-local-recall`
- Full path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files where each line is a complete JSON object representing one turn in the conversation.

This information is useful for the transcript-collector and other components that need to process Claude's project transcripts.
