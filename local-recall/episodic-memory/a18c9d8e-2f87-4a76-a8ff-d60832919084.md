# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a separate JSON object representing a turn in the conversation.

## Important for Memory Collection

The MCP server background daemon syncs these transcripts to collect memories. The daemon:
1. Scans `~/.claude/projects/<project>/transcripts/` directory
2. Reads and processes JSONL files
3. Uses content hashes to detect changes
4. Extracts memories using claude -p prompt processing
5. Runs every 5 minutes
