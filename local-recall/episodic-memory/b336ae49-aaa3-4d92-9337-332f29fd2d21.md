# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files:
- Each line is a valid JSON object representing a transcript event
- Files use `.jsonl` extension
- Multiple transcript files may exist for different sessions

## Usage in Local Recall

The `transcript-collector.ts` component reads from this location to:
1. Discover available transcripts for a project
2. Parse transcript events to extract memories
3. Track processed transcripts using content hashes to detect changes
4. Create episodic memories from significant transcript events
