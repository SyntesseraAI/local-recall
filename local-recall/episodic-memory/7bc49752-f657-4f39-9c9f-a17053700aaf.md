# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line contains a complete JSON object representing a single transcript event or message.

## Implications for Local Recall

The transcript collector must handle this path format to properly sync transcripts from Claude's cache. The `transcript-collector.ts` module should compute the expected project folder name based on the working directory path by replacing slashes with dashes.
