The `analyzeForMemories()` function in transcript.ts:
1. Parses transcript messages to identify speaker and content
2. Filters to only assistant messages (skips user messages)
3. Checks for multi-line content (minimum 2 non-empty lines)
4. Returns message content for memory creation
5. Role prefixes are removed from stored content to keep memories clean
