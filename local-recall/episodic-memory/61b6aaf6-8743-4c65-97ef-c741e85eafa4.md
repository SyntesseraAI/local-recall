MCP tools follow a naming convention using double underscores to separate namespaces: `mcp__<plugin>_<plugin>__<tool_category>_<tool_name>`. However, the tool may not be available in all contexts. The actual tool names for memory operations should be verified in the MCP server configuration (`src/mcp-server/tools.ts`).
