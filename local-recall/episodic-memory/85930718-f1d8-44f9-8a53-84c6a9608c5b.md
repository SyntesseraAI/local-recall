# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with leading slashes removed and forward slashes replaced with underscores.

**Example:**
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall`
- Transcript files: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` (JSON Lines) files in the `transcripts/` subdirectory.

Each line is a JSON object representing a conversation turn or event in the Claude Code session.

## Usage

The transcript-collector module reads these transcripts to extract memories and analyze session history. This is critical for the background daemon that processes transcripts asynchronously every 5 minutes.
