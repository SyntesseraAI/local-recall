Memory deduplication uses a combination of `occurred_at` timestamp and `content_hash` (SHA-256 prefix, 16 chars). The `findDuplicate()` method checks for existing memories before creation. This prevents duplicate memories from the same event. Memories are idempotent - attempting to create the same memory twice returns the existing one.
