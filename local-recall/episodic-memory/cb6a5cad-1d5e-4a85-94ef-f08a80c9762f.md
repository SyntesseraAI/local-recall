The user-prompt-submit hook in `src/hooks/user-prompt-submit.ts` extracts keywords from user prompts using `claude -p --model haiku` to leverage Claude's understanding for semantic keyword extraction. The `-p` flag enables structured/JSON output mode. These extracted keywords are then used to search the memory index for relevant context before Claude processes the prompt, improving context awareness across sessions.
