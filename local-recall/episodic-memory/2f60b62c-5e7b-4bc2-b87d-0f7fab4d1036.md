Claude Code stores project transcripts at `~/.claude/projects/` with project directories named using a path-encoding pattern.

**Pattern:** `/Users/<user>/.claude/projects/-Users-<user>-<path-segments>/`

For example, a project at `/Users/joe/Code/Syntessera/local-recall` has transcripts stored at:
`~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`

Each transcript is a JSONL file with filename pattern `<timestamp>-<session-id>.jsonl` containing Claude Code session events and interactions.
