The MCP server daemon:

- Syncs transcripts from Claude's cache (`~/.claude/projects/<project>/transcripts/`)
- Processes transcripts using `claude -p` (Haiku model) every 5 minutes
- Tracks processed transcripts with content hashes for change detection
- Deletes and recreates memories when transcripts change
- Exposes tools: `memory_create`, `memory_delete`, `memory_search`, `memory_list`, `index_rebuild`

This replaces the Stop hook for more reliable, asynchronous memory extraction.
