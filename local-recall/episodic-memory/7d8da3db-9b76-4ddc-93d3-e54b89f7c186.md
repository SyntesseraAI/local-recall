# Transcript Collection Implementation

The `transcript-collector.ts` module handles:
1. Discovering transcript files in Claude's project folders
2. Parsing `.json` transcript files that contain session conversation history
3. Tracking which transcripts have been processed using content hashes
4. Feeding transcript content to the memory extraction pipeline

Transcripts are stored as `.json` files in `~/.claude/projects/<project-folder>/transcripts/` and contain the full conversation history from each session.

This is part of the background MCP daemon workflow that runs every 5 minutes to extract memories from ongoing sessions.
