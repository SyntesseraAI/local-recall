The project builds TypeScript source files into JavaScript bundles in the `scripts/` directory:
- `scripts/hooks/session-start.js` - Session initialization hook
- `scripts/hooks/user-prompt-submit.js` - Prompt processing hook
- `scripts/hooks/stop.js` - Session cleanup hook
- `scripts/mcp-server/server.js` - MCP server executable

These bundled files are the actual executables referenced in hooks.json and .mcp.json, not the TypeScript source files in src/.
