Configuration can be set via environment variables or `.local-recall.json`:
- `memoryDir`: Directory for memory storage (default: `./local-recall`)
- `maxMemories`: Maximum number of memories (default: `1000`)

LOCAL_RECALL_DIR environment variable also used for MCP server configuration.
