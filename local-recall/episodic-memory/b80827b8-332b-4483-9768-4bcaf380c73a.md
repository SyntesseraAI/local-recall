# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files where:
- Each line is a complete JSON object (not an array)
- Files follow naming pattern: `<date>T<time>.jsonl`
- Example: `2025-12-01T143022.jsonl`

## Content Structure

Each line in the JSONL file contains:
- `type`: Event type ('user', 'assistant', 'tool', 'result', etc.)
- `role`: 'user' or 'assistant'
- `content`: The actual message/event content
- `timestamp`: ISO-8601 timestamp
- Additional metadata depending on event type

This is useful for the transcript-collector module which needs to read and parse these files.
