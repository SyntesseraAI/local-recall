# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON lines format), where each line is a separate JSON object representing an event in the transcript.

## Relevance to Local Recall

The MCP server's background daemon syncs transcripts from this location to extract memories. The exact path structure is critical for the transcript collector functionality.
