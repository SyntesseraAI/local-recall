# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and colons removed. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) format with `.jsonl` extension. Each line is a JSON object representing a conversation event.

## Relevance

This is critical for `TranscriptCollector` which reads these files to extract memories from Claude Code sessions.
