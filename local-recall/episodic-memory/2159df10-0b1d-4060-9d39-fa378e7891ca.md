# Transcript Structure

## Messages Array

Transcript objects contain a `messages` array where each element represents a turn in the conversation.

## Message Object Format

Each message has:
- `role`: Either "user" or "assistant"
- Other properties depending on the role

## Usage for Memory Extraction

When processing transcripts for memory extraction:
1. Iterate through the `messages` array
2. Identify significant events and learnings from the conversation
3. Extract context about what was accomplished
4. Create memories with appropriate scope and keywords

This structure enables chronological replay and context-aware memory generation.
