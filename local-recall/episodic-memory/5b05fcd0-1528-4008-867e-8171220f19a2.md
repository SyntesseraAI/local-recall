# Claude Project Transcripts Location and Naming

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with forward slashes replaced by dashes:
- Example: Working directory `/Users/joe/Code/Syntessera/local-recall` becomes folder name `Users-joe-Code-Syntessera-local-recall`
- Full path to transcripts: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with the format:
- File naming: `<session-id>.jsonl`
- Location: `~/.claude/projects/<project-folder>/transcripts/`
- Content: JSON lines format, each line is a complete JSON object representing a conversation turn

This information is crucial for the transcript-collector component which needs to locate and parse Claude Code session transcripts.
