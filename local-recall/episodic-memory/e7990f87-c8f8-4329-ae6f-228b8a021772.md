# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (JSON Lines), one event per line. Each file contains a single transcript session with events like `UserPromptSubmit`, `ToolUse`, `ToolResult`, etc.

## Relevant Code

See `src/utils/transcript-collector.ts` for the transcript collection logic that reads from these paths.
