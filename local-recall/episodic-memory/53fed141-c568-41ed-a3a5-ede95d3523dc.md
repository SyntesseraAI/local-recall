# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` → folder `Users-joe-Code-Syntessera-local-recall`
- Project at `/Users/alice/Projects/my-app` → folder `Users-alice-Projects-my-app`

## File Structure

Each project folder contains `transcripts.jsonl` with one JSON object per line, each representing a Claude Code session.

## Implementation Detail

When building the transcript collector, use path-based folder naming to match Claude's convention. This is documented in the transcript-collector component.

## Related Code

See `src/core/transcript-collector.ts` for implementation details of converting project paths to Claude's folder naming scheme.
