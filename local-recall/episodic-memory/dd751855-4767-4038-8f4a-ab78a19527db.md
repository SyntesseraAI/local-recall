When adding UUID filename validation to the transcript collector, the validation must be applied in THREE separate locations for consistency:

1. `findClaudeProjectDir()` - When locating the Claude project directory by checking for transcript files
2. `listSourceTranscripts()` - When filtering transcript files from the source directory
3. `listLocalTranscripts()` - When filtering local transcript files (if applicable)

This ensures the entire pipeline consistently rejects non-UUID `.jsonl` files and prevents false positives or inconsistent behavior.
