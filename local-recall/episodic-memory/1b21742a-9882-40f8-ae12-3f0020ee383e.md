# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory with slashes replaced by underscores:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files with format: `transcript-<timestamp>.jsonl`

Example: `transcript-20241201T143022Z.jsonl`

## Key Implementation Detail

When collecting transcripts, the `transcript-collector.ts` needs to:
1. Get the project's working directory (via `cwd` from stdin or process.cwd())
2. Hash the absolute path to get the folder name
3. Point to `~/.claude/projects/<hashed-folder-name>/transcripts/`

The transcript path calculation must account for:
- Different working directories producing different transcript locations
- Timestamp-based transcript file naming
- The need to handle edge cases where the directory might not exist
