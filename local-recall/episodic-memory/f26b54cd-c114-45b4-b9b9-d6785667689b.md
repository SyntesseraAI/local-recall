The MCP server in `src/mcp-server/server.ts` runs a background daemon that syncs and processes transcripts every 5 minutes. It calls `runTranscriptProcessing()` from memory-extractor.js for episodic memory and now also calls `runThinkingExtraction()` from thinking-extractor.js for thinking memories. Both run in parallel with separate processing flags to prevent concurrent runs of the same type.
