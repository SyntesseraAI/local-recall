# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path of the project directory, URL-encoded with forward slashes converted to `%2F`.

### Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- URL-encoded: `%2FUsers%2Fjoe%2FCode%2FSyntessera%2Flocal-recall`
- Full path: `~/.claude/projects/%2FUsers%2Fjoe%2FCode%2FSyntessera%2Flocal-recall/transcripts/`

This is important for the transcript-collector and background daemon to locate transcripts for processing and memory extraction.
