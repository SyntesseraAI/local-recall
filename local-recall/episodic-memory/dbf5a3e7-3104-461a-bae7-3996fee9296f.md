# Using Claude CLI for Transcript Processing

The transcript collector uses the `claude -p` command (prompt mode) to send transcript content to Claude for memory extraction. This is invoked as a shell command from within the Node.js application.

**Key implementation details:**
- Command: `claude -p` - sends prompt via stdin
- Input: Condensed transcript events and extraction guidelines
- Output: JSON array of extracted memories
- Used in: `src/core/transcript-collector.ts`

This approach leverages Claude's capability to analyze transcripts and extract meaningful memories following specific guidelines about what should and shouldn't be saved.
