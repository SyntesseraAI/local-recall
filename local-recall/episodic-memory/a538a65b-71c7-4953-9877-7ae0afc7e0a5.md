Each memory is stored as a separate markdown file in either `local-recall/episodic-memory/` or `local-recall/thinking-memory/`. Each file contains YAML frontmatter with metadata (id, subject, keywords, applies_to, occurred_at, content_hash) followed by the memory content in markdown format.
