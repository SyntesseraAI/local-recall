# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with leading slash removed and remaining slashes converted to underscores.

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:

- Transcript folder: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`
- Transcript files: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/*.jsonl`

## Transcript Format

Transcript files are JSONL (JSON Lines) format, where each line is a JSON object representing a single event in the session. Events include user messages, assistant responses, and tool invocations.

## Key Implementation Details

- The path transformation is critical for correctly locating transcripts programmatically
- This path format is used by `transcript-collector.ts` when syncing transcripts from Claude's cache
- Understanding this structure is essential for building tools that need to parse session history
