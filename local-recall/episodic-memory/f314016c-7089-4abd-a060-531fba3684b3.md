# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the project's absolute path:
- Forward slashes `/` are replaced with dashes `-`
- The path includes the full absolute path of the project
- Example: Project at `/Users/joe/Code/Syntessera/local-recall` maps to folder `Users-joe-Code-Syntessera-local-recall`

## Files

Transcripts are stored as `.jsonl` (JSON Lines) files with format `<session-id>.jsonl`

## Related Code

The `TranscriptCollector` in `src/core/transcript-collector.ts` handles reading these transcript files.
