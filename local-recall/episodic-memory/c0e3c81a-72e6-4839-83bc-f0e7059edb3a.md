Memories are deduplicated using two fields: `occurred_at` (when the event happened) and `content_hash` (SHA-256 prefix of content). The `findDuplicate()` function checks for existing memories with matching occurred_at and content_hash to prevent duplicate memories when transcripts are reprocessed.
