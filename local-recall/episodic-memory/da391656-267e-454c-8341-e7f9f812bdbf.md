The plugin configuration expects the MCP server at `${CLAUDE_PLUGIN_ROOT}/scripts/mcp-server/server.js` as specified in .mcp.json. The build script must output to this exact path, and the server.js file must be self-contained with all dependencies bundled (no separate node_modules directory).
