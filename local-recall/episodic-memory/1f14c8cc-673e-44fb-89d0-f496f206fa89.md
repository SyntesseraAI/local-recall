# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL (JSON Lines) files:
- File naming: `transcript-<timestamp>.jsonl`
- Each line is a separate JSON object representing a transcript event
- Files accumulate in the transcripts directory

## Windows Path Encoding

On Windows, drive letters and UNC paths are also encoded with dashes:
- `C:\Users\name\project` → `C-Users-name-project`
- `\\server\share\project` → `server-share-project`
