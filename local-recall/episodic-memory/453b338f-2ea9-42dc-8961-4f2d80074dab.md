# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), with the filename being the session ID followed by `.jsonl`:
- Example: `01HQY8jXfJm7zh35BcYYiE1u.jsonl`

## Session ID

The session ID is available in Claude Code via the `session_id` field passed to hooks.

## Usage

This information is critical for the transcript-collector to:
1. Locate the correct project directory based on the current working directory
2. Find the current session's transcript file
3. Parse JSONL format to extract transcript events
