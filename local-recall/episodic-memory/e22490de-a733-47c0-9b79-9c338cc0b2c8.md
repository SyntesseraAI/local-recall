The `transcript-collector.ts` module provides functionality to locate and parse Claude Code project transcripts. It reads `.jsonl` files from the Claude projects directory, parses them as JSON lines, and extracts useful information from each transcript entry. This is used by the MCP server daemon to process transcripts and extract memories.
