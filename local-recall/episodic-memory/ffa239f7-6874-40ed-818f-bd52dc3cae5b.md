Changed `episodicEnabled` default from `false` to `true` in the config schema.

Added separate configuration options for both episodic and thinking memories:
- `episodicMaxTokens` (default: 1000) - controls max tokens of episodic memories injected per prompt
- `episodicMinSimilarity` (default: 0.8) - minimum similarity threshold for episodic memory search
- `thinkingMaxTokens` (default: 1000) - controls max tokens of thinking memories injected per prompt
- `thinkingMinSimilarity` (default: 0.8) - minimum similarity threshold for thinking memory search

Environment variables:
- `LOCAL_RECALL_EPISODIC_ENABLED` (default: true)
- `LOCAL_RECALL_EPISODIC_MAX_TOKENS` (default: 1000)
- `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY` (default: 0.8)
- `LOCAL_RECALL_THINKING_MAX_TOKENS` (default: 1000)
- `LOCAL_RECALL_THINKING_MIN_SIMILARITY` (default: 0.8)
