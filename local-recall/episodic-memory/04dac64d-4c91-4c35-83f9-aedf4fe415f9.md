Memory markdown files in `local-recall/episodic-memory/` are checked into git and version-controlled. The SQLite database (`local-recall/memory.sqlite`) and embedding cache are gitignored as they are auto-generated. The `.gitignore` file in the memory directory excludes `memory.sqlite` and `recall.log`.
