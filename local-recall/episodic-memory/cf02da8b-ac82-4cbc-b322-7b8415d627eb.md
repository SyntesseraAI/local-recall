Claude stores project transcripts in `~/.claude/projects/` directory. Project directories use URL-encoded paths with dashes replacing slashes (e.g., `-Users-joe-Code-Syntessera-local-recall/` for `/Users/joe/Code/Syntessera/local-recall`). Individual transcript files are stored as JSONL format in `transcripts/` subdirectory. This is crucial for the transcript-collector component which syncs these transcripts for memory extraction.
