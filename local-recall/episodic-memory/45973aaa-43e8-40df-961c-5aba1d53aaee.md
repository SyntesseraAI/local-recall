# Claude Project Transcripts Location and Naming Convention

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path by:
1. Removing leading slashes
2. Replacing all forward slashes with dashes
3. Converting to lowercase

### Example
Absolute path: `/Users/joe/Code/Syntessera/local-recall`
Transcript folder: `~/.claude/projects/users-joe-code-syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in `transcripts/` subdirectory with names like:
- `2024-12-01T12-34-56Z-random-suffix.jsonl`

Each line in the JSONL file is a JSON object representing a transcript event.

## Importance

This convention is critical for the MCP server's background daemon which:
- Syncs transcripts from Claude's cache
- Processes transcripts to extract memories
- Tracks processed transcripts using content hashes
- Runs every 5 minutes
