Each memory markdown file must include YAML frontmatter with: `id` (UUID), `subject` (brief description), `keywords` (searchable terms), `applies_to` (scope: global/file/area), `occurred_at` (ISO-8601 timestamp), `content_hash` (SHA-256 prefix for deduplication). The content_hash and occurred_at fields are critical for detecting and preventing duplicate memories.
