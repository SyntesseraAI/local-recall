The project has two core modules for processing transcripts:

1. **transcript-condenser.ts** (src/core/transcript-condenser.ts) - Condenses verbose Claude transcripts into a structured format
2. **memory-extractor.ts** (src/core/memory-extractor.ts) - Extracts memories from condensed transcripts

These modules are part of the core memory management system and handle the key task of converting raw transcripts into extractable memory events.
