The memory extraction process is designed to analyze full session transcripts including:
- User prompts and requests
- Assistant responses and explanations
- Tool invocations and their results
- The outcomes and learnings from interactions

This comprehensive analysis allows extraction of memories about what was learned, problems solved, and patterns discovered during the session.
