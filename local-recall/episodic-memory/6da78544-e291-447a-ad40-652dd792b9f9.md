MCP server configured differently for Claude Code vs Claude Desktop. For Claude Code: use relative paths like `./scripts/mcp-server/server.js`. For Claude Desktop: use absolute paths in `claude_desktop_config.json`. Both require `LOCAL_RECALL_DIR` environment variable pointing to memory directory. Server exposes tools: `memory_create`, `memory_delete`, `memory_search`, `memory_list`, `index_rebuild`.
