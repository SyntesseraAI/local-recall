# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files:
- Each line is a valid JSON object
- File extension: `.jsonl`
- Each event in the transcript is a separate JSON object on its own line
- Events include tool calls, results, and messages

This discovery is important for the `transcript-collector` module which needs to read and parse these transcripts for memory extraction.
