# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with:
- Sequential numbering (e.g., `1.jsonl`, `2.jsonl`, `3.jsonl`)
- Each line contains a JSON object representing a message or event
- Multiple sessions create multiple files in the same directory

This is used by the MCP server to sync and process transcripts for memory extraction.
