The MCP server exposes memory tools (memory_create, memory_get, memory_search, memory_list, index_rebuild) and runs a background daemon that: syncs transcripts from Claude cache, processes them using 'claude -p' for memory extraction, tracks processed transcripts with content hashes for change detection, deletes/recreates memories on transcript changes, and runs every 5 minutes. This replaces the disabled Stop hook for asynchronous transcript processing.
