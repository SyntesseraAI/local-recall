The memory extraction system in `src/core/memory-extractor.ts` uses Claude Haiku (`claude-haiku-4-5-20251001`) to analyze session transcripts and generate memories. It processes condensed transcripts (with events like `[User]`, `[Assistant]`, `[Tool: Name]`, `[Result: OK/ERROR]`) through a prompt-based system. The extraction prompt is defined in `src/prompts/memory-extraction.ts` and returns a JSON response with a `memories` array containing structured memory objects.
