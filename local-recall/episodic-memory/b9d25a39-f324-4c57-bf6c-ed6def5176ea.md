# Storage Format Migration

The codebase is migrating to JSONL storage format for memories. Key files modified:

- `src/core/episodic-jsonl-store.ts` - New JSONL storage for episodic memories
- `src/core/thinking-jsonl-store.ts` - New JSONL storage for thinking memories  
- `src/core/jsonl-store.ts` - Base JSONL storage implementation
- `tests/unit/core/episodic-jsonl-store.test.ts` - Tests for new storage

This represents a format change from the existing markdown-based storage described in CLAUDE.md. The JSONL format likely provides better performance for large memory collections and easier batch operations.
