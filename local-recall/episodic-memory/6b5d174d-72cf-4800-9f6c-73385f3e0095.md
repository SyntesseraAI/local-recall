# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `users-joe-code-syntessera-local-recall` (slashes replaced with hyphens, lowercase)
- Full path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (`.jsonl` extension), one per session.

## Relevance

The transcript-collector needs to locate and parse these files to extract memories from Claude Code sessions.
