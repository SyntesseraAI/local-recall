Configuration system in `src/utils/config.ts`:

- Reads from both `.local-recall.json` file and environment variables
- Available configuration options:
  - `memoryDir`: Directory for memory storage (default: `./local-recall`)
  - `maxMemories`: Maximum number of memories (default: 1000) - currently not enforced
  - `indexRefreshInterval`: Seconds between index refreshes (default: 300)
  - `fuzzyThreshold`: Minimum fuzzy match score 0-1 (default: 0.6)
- Environment variables override file configuration
- Accessed via `getConfig()` function
