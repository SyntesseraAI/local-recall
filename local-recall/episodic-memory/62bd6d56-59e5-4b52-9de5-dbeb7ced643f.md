# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores. For example:

- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users_joe_Code_Syntessera_local-recall`
- Transcripts location: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript File Format

Transcript files are stored in JSONL format (`.jsonl` extension), with each line being a complete JSON object representing a single event/message in the conversation.

## Key Implementation Details

- Used in `src/core/transcript-collector.ts` to discover and read transcripts
- Path transformation uses string replace: `cwd.replace(/\//g, '_')`
- Transcripts are read and parsed line-by-line as JSONL
- This enables Local Recall to sync and process Claude's session history for memory extraction
