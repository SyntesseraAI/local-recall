# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the transcripts subfolder:
- Location: `~/.claude/projects/<project-folder>/transcripts/`
- Files are named like `20250101T120000Z.jsonl`
- Each line in the JSONL file is a separate transcript event

## Event Structure

Each line contains a JSON object with:
- `type`: Event type (e.g., "user_message", "assistant_message", etc.)
- `content`: The message or event content
- `timestamp`: When the event occurred

This information is critical for the transcript-collector to locate and read Claude's transcripts for memory extraction.
