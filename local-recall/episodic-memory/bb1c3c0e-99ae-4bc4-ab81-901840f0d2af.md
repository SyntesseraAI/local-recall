The index system in `src/core/index.ts` maintains a searchable keyword index of all memories stored at `local-recall/index.json` (gitignored). The index enables fast fuzzy keyword searching. Functions include:
- `buildIndex()`: Scans memory files and builds keyword index
- `getIndex()`: Retrieves current index
- `refreshIndex()`: Rebuilds index from scratch

Default fuzzy match threshold is 0.6 (configurable). This index is essential for the SessionStart and UserPromptSubmit hooks to quickly find relevant memories.
