Both episodic and thinking memory retrieval can be independently controlled through configuration: `episodicEnabled` and `thinkingEnabled` flags. Each memory type also has independent token budgets (`episodicMaxTokens`, `thinkingMaxTokens`) and similarity thresholds (`episodicMinSimilarity`, `thinkingMinSimilarity`). This allows for fine-grained control over which memory types are active.
