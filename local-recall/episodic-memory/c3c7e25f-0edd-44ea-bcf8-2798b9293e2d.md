Hooks are configured in .claude/settings.json. SessionStart hook loads 5 most recent memories on session begin (full reload, doesn't use vector store). UserPromptSubmit hook performs semantic search on user prompt before Claude processes it. Stop hook is disabled - memory extraction happens via MCP server daemon every 5 minutes instead. Command paths differ: 'node ./node_modules/local-recall/dist/hooks/...' for npm package vs 'node ./dist/hooks/...' when running locally.
