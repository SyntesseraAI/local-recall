# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Files are named with session identifiers
- Each line is a complete JSON object representing a session event
- Files grow incrementally as sessions progress

## Purpose

These transcripts are used by the transcript-collector to extract memories and build the memory index for future sessions.
