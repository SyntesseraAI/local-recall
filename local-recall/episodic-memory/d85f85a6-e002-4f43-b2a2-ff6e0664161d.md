The `MemoryExtractor` class in `src/core/memory-extractor.ts` orchestrates the transcript-to-memory workflow:
- Processes transcript files from Claude's transcript cache
- Uses Claude Haiku to extract memory suggestions
- Creates memory files with extracted metadata
- Tracks processed transcripts to avoid re-processing

Key methods:
- `extractFromTranscript(transcriptPath)` - Main processing method
- `processTranscriptFile(path)` - Handles individual transcript files
- Uses content hashes for deduplication and change detection
