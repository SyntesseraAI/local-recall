Memories include `occurred_at` (ISO-8601 timestamp) and `content_hash` (SHA-256 prefix, 16 chars) fields for deduplication. The `createMemory()` function returns existing memory if duplicate detected via `findDuplicate(occurredAt, contentHash)`. This enables idempotent memory creation - calling createMemory multiple times with same content won't create duplicates.
