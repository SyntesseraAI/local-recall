The memory extractor (src/core/memory-extractor.ts) uses Claude Haiku through MCP server calls to analyze transcripts and extract valuable memories. It processes transcripts line-by-line, uses the MCP call_tool method to invoke 'memory_create' with the memory data generated by Claude. This pattern allows for asynchronous memory extraction during background daemon runs.
