# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project with forward slashes replaced by dashes. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Stored as: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) in the `transcripts/` subdirectory with filenames like `session-1733116800000.jsonl`.

## Timestamp Format

Session identifiers and filenames use milliseconds since epoch for timestamps.
