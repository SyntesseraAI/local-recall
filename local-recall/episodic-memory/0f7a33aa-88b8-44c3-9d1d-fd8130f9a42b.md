First run may take 30-60 seconds while the BGE-small-en-v1.5 model (~133MB) downloads to `local_cache/`. Subsequent runs load from cache. If you see 'Tokenizer file not found' errors, the cache is corrupted - delete `local_cache/fast-bge-small-en-v1.5*` and it will re-download automatically.
