# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Becomes folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Storage

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a JSON object representing a single message event.

This is important for the transcript-collector module which needs to locate and process these files for memory extraction.
