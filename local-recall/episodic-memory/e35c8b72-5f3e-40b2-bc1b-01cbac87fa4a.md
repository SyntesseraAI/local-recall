## Claude Transcript File Location Pattern

Claude Code stores session transcripts in `~/.claude/projects/` with a specific naming convention:

- Project directories use URL-encoded paths with dashes replacing slashes
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored as JSONL files in `~/.claude/projects/<encoded-path>/transcripts/`
- Each transcript file contains an array of turn objects with `type` and `content` fields
- Transcript file paths are provided to hooks via the `transcript_path` parameter in JSON input
