Memory files use YAML frontmatter containing: `id` (UUID), `subject` (brief description), `keywords` (array of searchable terms), `applies_to` (scope), `occurred_at` (ISO-8601 timestamp), and `content_hash` (SHA-256 prefix for deduplication). The actual memory content follows the frontmatter in markdown format.
