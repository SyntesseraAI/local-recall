# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the sanitized project path:
- Slashes become dashes: `/Users/joe/Code/MyProject` â†’ `Users-joe-Code-MyProject`
- All characters are lowercase
- Special characters are removed or converted

Example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `users-joe-code-syntessera-local-recall`
- Full transcript path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

## Implementation Details

The `transcript-collector.ts` needs to:
1. Get the current working directory (cwd)
2. Convert the cwd to the expected folder name format
3. Build the path: `~/.claude/projects/{folder-name}/transcripts/`
4. List transcript files in that directory

The conversion should handle:
- Expanding `~` to user's home directory
- Converting `/` to `-`
- Converting to lowercase
- Removing or replacing special characters
