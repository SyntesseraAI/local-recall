# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and colons removed. For example:

- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files in the `transcripts/` subdirectory.

## Implementation Context

This is relevant to the `src/core/transcript-collector.ts` which needs to sync transcripts from Claude's cache for the MCP server's background daemon that processes transcripts every 5 minutes to extract memories.
