# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with forward slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: Named as `*.jsonl` files

## Accessing Transcripts

The transcript-collector needs to expand `~` to the user's home directory using `os.path.expanduser()` to correctly locate and read these files.
