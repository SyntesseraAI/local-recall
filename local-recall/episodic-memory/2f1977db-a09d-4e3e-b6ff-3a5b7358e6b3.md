# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses the absolute path with slashes replaced by underscores:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format:
- Pattern: `*.jsonl`
- Multiple session transcripts may exist
- Each line is a complete JSON object representing an event in the session

## Implementation Notes

The `transcript-collector.ts` needs to handle:
1. URL-encoding/decoding of project paths
2. Reading and parsing JSONL files from the transcripts directory
3. Handling multiple transcript files for different sessions

This is critical for the memory extraction pipeline that processes Claude Code session transcripts.
