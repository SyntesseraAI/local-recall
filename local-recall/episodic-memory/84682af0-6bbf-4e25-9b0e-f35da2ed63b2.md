# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), each containing:
- `type`: Event type (e.g., "user_message", "assistant_response")
- `timestamp`: ISO 8601 timestamp
- `content`: The actual message or event data

## Usage

This information is useful for:
- Processing transcripts to extract memories
- Syncing transcript data from Claude's cache
- Building tools that work with Claude Code session history
- Tracking what was learned across sessions
