The MCP server tools have been restructured:

### Removed Tools
- `index_rebuild` - No longer exposed to prevent unintended index rebuilds
- `memory_list` - Removed to limit LLM power

### Renamed Tools (episodic_* prefix)
- `episodic_create` - Create new episodic memory
- `episodic_get` - Retrieve episodic memory by ID
- `episodic_search` - Search episodic memories via vector similarity

### New Tools (thinking_*)
- `thinking_get` - Retrieve thinking memory by ID
- `thinking_search` - Search thinking memories via vector similarity

This naming convention makes it clear whether operations are on episodic or thinking memories. See `src/mcp-server/tools.ts` for implementation.
