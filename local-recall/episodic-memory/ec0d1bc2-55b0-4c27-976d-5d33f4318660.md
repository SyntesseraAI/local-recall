The memory extractor includes a `stripMarkdownCodeBlocks` utility method that removes markdown code block syntax from Claude CLI responses. When using Claude CLI via `callClaudeCLI`, responses may come wrapped in triple-backtick code blocks, and the existing utility already handles this normalization, making it compatible with different model outputs and response formats.
