The BGE-small-en-v1.5 embedding model is cached in `local_cache/fast-bge-small-en-v1.5/`. First run downloads the ~133MB model which takes 30-60 seconds. Subsequent runs load from cache. If 'Tokenizer file not found' error occurs, delete the cache directory and re-run to force re-download.
