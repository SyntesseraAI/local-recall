Memory creation is idempotent - `createMemory()` returns existing memory if duplicate found. Deduplication uses `findDuplicate(occurredAt, contentHash)` which checks for memories with same content hash and timestamp. Content hash is a 16-character SHA-256 prefix stored in frontmatter.
