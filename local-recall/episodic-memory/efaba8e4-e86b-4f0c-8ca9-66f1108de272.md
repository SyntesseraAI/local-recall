# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `transcripts.jsonl`

## File Format

Transcripts are stored in JSONL format (one JSON object per line), where each line represents an event in the conversation.

## Usage in Local Recall

The transcript collector should use this path format when syncing transcripts from Claude's cache to the local-recall project directory.
