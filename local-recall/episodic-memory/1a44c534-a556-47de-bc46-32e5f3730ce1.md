# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

The project folder name is derived from the absolute working path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` (JSON Lines) files, where each line is a separate JSON object representing a single message or event in the conversation.

## Key Implementation Details

The `transcript-collector.ts` module reads from this location to process and extract memories from transcripts.
