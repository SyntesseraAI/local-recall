The local-recall project IS the plugin root. Directory structure:
- `.claude-plugin/plugin.json` - Plugin metadata (name, version, description)
- `hooks.json` - Hook event handlers (uses ${CLAUDE_PLUGIN_ROOT})
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (build output, gitignored)
  - `hooks/` - Bundled hook scripts
  - `mcp-server/` - Bundled MCP server

To install, add to project's `.claude/settings.json` plugins list or add hooks/MCP config directly.
