When extracting memories from transcripts, condense the full transcript into a structured format using events like `[User]`, `[Assistant]`, `[Tool: Name]`, and `[Result]` rather than storing raw transcript text. This reduces token overhead when processing transcripts for memory extraction while maintaining all essential information needed to identify valuable memories.
