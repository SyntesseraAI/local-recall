# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Converting the path to lowercase
2. Replacing forward slashes `/` with dashes `-`
3. Removing the leading dash

**Example:**
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `users-joe-code-syntessera-local-recall`
- Full transcripts path: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`

Transcripts are stored as `.jsonl` files in this directory.

## Implementation Context

This is important for the transcript-collector component which reads transcripts from this location to extract memories.
