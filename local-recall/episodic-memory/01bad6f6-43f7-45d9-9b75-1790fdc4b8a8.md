The Local Recall project uses `claude -p` (project/paid Claude mode) to extract memories from session transcripts. The MCP server daemon processes transcripts asynchronously every 5 minutes, syncing from Claude's transcript cache (~/.claude/projects/<project>/transcripts/), extracting structured memories using Claude's analysis, and tracking processed transcripts with content hashes for deduplication. This approach allows intelligent memory extraction without requiring manual intervention.
