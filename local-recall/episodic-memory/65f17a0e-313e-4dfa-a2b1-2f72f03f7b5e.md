# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project, with special characters replaced:
- Forward slashes `/` → underscores `_`
- Colons `:` → underscores `_`

## Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`
- Transcripts are stored as JSONL files

## Implementation Location

This path conversion logic is implemented in `src/core/transcript-collector.ts` and is critical for the transcript daemon to locate and process transcripts correctly.
