# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts are stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Each transcript is a `.jsonl` file (JSON Lines format)

## Transcript Format

Transcripts are stored as JSONL (one JSON object per line) containing:
- Conversation events
- Tool invocations
- Results and outputs
- Timestamps

This location is important for the transcript-collector component which syncs transcripts from Claude's cache.
