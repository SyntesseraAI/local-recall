SessionStart and UserPromptSubmit hooks are configured in hooks.json. They receive JSON input containing session_id, transcript_path, cwd (and prompt for UserPromptSubmit). SessionStart loads the index, while UserPromptSubmit extracts keywords from the prompt and searches memories. Both output matching memories to stdout for injection into Claude's context.
