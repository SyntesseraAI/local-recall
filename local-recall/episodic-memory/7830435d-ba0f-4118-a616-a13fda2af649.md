# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each transcript is a `.jsonl` file (JSON Lines format) with one JSON object per line, containing the session transcript.

## Relevance

This is critical for the MCP server's background daemon which needs to sync transcripts from Claude's cache to extract memories.
