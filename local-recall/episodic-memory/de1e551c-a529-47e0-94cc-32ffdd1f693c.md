# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the repository path with slashes replaced by dashes:
- Repository: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Implementation Details

When implementing transcript collection, use this path derivation formula:
1. Get the full cwd path
2. Replace all `/` with `-`
3. Construct `~/.claude/projects/{derived-name}/transcripts/`

Transcripts are stored as JSONL files in this directory.
