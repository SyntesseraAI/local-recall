The project root is the plugin root following Claude Code's plugin format:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook event handlers using `${CLAUDE_PLUGIN_ROOT}`
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (build output, gitignored)

Hooks receive `cwd` via stdin to operate on user's project directory. The plugin can be installed directly or hooks configured manually in project's `.claude/settings.json`.
