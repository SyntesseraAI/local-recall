# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the project path with slashes replaced by dashes. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as `.jsonl` files in the `transcripts/` subdirectory.

This is important for `transcript-collector.ts` which needs to construct the correct path to locate Claude's transcript cache.
