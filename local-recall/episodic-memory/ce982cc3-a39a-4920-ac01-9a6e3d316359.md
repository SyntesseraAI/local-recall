# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored in JSONL format with:
- File naming: `transcript-YYYYMMDD-HHMMSS.jsonl`
- Each line is a JSON object representing an event
- Events include user prompts, assistant responses, and tool invocations
- Useful for extracting memories and analyzing session history

## Implementation Notes

When building transcript collection functionality:
1. Convert the CWD path by replacing `/` with `-` to get the project folder name
2. Build the full path using the standard `~/.claude/projects/` base directory
3. Handle the JSONL format properly - each line is a separate JSON object
4. Consider platform differences (macOS path format may differ from other OSes)
