# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with forward slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files stored as: `transcripts/*.jsonl`

## Transcript Format

Transcript files are in JSONL format (JSON Lines), where each line contains a complete JSON object representing a transcript event.
