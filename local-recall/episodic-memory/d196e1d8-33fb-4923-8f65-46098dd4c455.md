## Content Hash Implementation

### Algorithm
- Uses SHA-256 hashing of the memory content
- Hash is truncated to 16 characters (using `substring(0, 16)`)
- Stored in `content_hash` field in memory frontmatter

### Purpose
- Primary deduplication mechanism (combined with `occurred_at`)
- Allows detecting exact duplicate content
- 16-character hash provides good collision resistance for practical use

### Method
`computeContentHash(content: string): string` in memory.ts computes this hash
