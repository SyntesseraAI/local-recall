The user-prompt-submit hook uses the Claude Haiku model (via `claude -p` CLI) for fast keyword extraction from user prompts. This is a pragmatic choice that balances speed and accuracy - Haiku is fast enough for real-time hook execution while providing sufficient intelligence for keyword extraction. The extracted keywords are then used for fuzzy searching the memory index.
