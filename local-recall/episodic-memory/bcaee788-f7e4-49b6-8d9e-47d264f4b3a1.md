# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with names like:
- `20250101T120000Z_unique-id.jsonl`
- Each line is a JSON object representing a transcript event
- Contains user messages, assistant responses, and tool invocations

## Implementation Details

The `TranscriptCollector` class in `src/core/transcript-collector.ts` handles:
- Converting working directory paths to Claude's folder naming convention
- Reading JSONL transcript files
- Parsing transcript events for memory extraction

This path conversion is critical for the MCP server daemon to locate and process transcripts correctly.
