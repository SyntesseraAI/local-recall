Local Recall integrates with Claude Code via three hooks configured in .claude/settings.json:
1. **SessionStart**: Loads 5 most recent memories from disk (fast, no vector store)
2. **UserPromptSubmit**: Performs semantic search using vector embeddings to find relevant memories
3. **Stop**: Disabled (moved to MCP daemon processing)

Hooks execute as shell commands receiving JSON input (session_id, transcript_path, cwd, prompt) and output memory content to stdout for context injection.
