# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `_Users_joe_Code_Syntessera_local-recall`
- Transcripts stored in `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a separate JSON object representing a message or event in the conversation.

## Implementation Considerations

When building transcript collectors:
1. Convert absolute paths to the Claude naming convention (replace `/` with `_`, add leading `_`)
2. Look for transcripts in `~/.claude/projects/<converted-path>/transcripts/`
3. Parse JSONL format - each line is a separate JSON object
4. Handle the case where project paths contain special characters or multiple slashes
