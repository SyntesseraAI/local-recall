Local Recall functions as a Claude Code plugin:
- Project root IS the plugin root
- Hooks configured in `hooks.json` receive JSON via stdin
- Hooks use `${CLAUDE_PLUGIN_ROOT}` for script references
- Receive `cwd` parameter to operate on user's project directory
- SessionStart and UserPromptSubmit hooks are active
- Stop hook currently disabled - memory extraction handled by MCP server daemon
- MCP server runs background daemon processing transcripts every 5 minutes
