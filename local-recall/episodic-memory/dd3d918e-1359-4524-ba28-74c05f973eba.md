When a transcript contains the [LOCAL_RECALL_INTERNAL] marker at the beginning, it indicates this is an automated memory extraction session analyzing a prior user session to extract valuable memories. These sessions are driven by the MCP server's background daemon that processes transcripts and creates memories for future AI assistants.
