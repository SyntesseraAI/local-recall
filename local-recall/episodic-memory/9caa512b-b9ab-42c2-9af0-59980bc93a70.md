The MCP server exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild. A background daemon processes transcripts from Claude's cache every 5 minutes, extracting memories using `claude -p`. It tracks processed transcripts by content hash for change detection and deletes/recreates memories when transcripts change.
