The memory extractor module uses the Claude SDK (`@anthropic-ai/sdk`) to invoke Claude Haiku for analyzing transcripts and extracting memories. The `extractMemories` function takes a transcript string and calls `client.messages.create()` with the memory extraction prompt to identify valuable insights from session activity.
