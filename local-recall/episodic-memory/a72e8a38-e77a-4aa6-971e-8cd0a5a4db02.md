# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and special characters removed:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall` (slashes â†’ dashes, dots removed from domain-like segments)

## File Structure

Inside each project folder:
- `transcripts/` - Directory containing transcript files
  - Files named with timestamps: `<timestamp>.jsonl`
  - Format: JSONL (JSON Lines) with transcript events
  - Each line is a JSON object representing a transcript event

## Transcript Event Format

Each event in the JSONL file contains:
- `type`: Event type (e.g., 'UserMessage', 'AssistantMessage', 'ToolUse', 'ToolResult')
- `content`: The actual content/message
- `timestamp`: When the event occurred
- Additional metadata depending on event type

This information is critical for the `transcript-collector.ts` module which needs to locate and parse these files to extract memories.
