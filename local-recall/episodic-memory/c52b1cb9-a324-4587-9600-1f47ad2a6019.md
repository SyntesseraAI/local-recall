# Transcript Format

Claude Code stores transcripts as JSON Lines (`.jsonl`) files.

## Format Details

- Each line is a separate JSON object
- Objects represent individual transcript events
- No array wrapper - files are not standard JSON arrays
- Allows streaming and incremental reading

## Implication for Code

The transcript-collector must parse `.jsonl` files line-by-line rather than treating them as standard JSON arrays.
