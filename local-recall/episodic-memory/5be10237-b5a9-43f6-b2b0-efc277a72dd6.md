# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Taking the absolute path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replacing slashes with dashes
3. Removing the leading dash

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript Structure

Transcripts are stored as JSONL files (one JSON object per line) in:
`~/.claude/projects/{project-name}/transcripts/`

Each line contains a transcript event with:
- `type`: Event type (e.g., 'message', 'tool', etc.)
- `timestamp`: When the event occurred
- Other fields depending on event type

This is useful for the MCP server background daemon which processes transcripts to extract memories every 5 minutes.
