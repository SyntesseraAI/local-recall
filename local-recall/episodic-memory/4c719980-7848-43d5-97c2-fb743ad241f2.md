# Transcript Condensing for Memory Extraction

Local Recall implements transcript condensing to reduce token usage when extracting memories from sessions.

## Implementation

- **Location:** `src/core/memory-extractor.ts`
- **Purpose:** Summarize transcript events into a condensed format before sending to Claude for memory extraction
- **Benefit:** Significantly reduces API tokens consumed while maintaining important context

## Process

1. Raw transcript is parsed from JSONL format
2. Events are condensed (summarized/filtered)
3. Condensed transcript is sent to Claude API for memory extraction
4. Claude analyzes and identifies valuable memories to persist

## Related Components

- `src/core/transcript-collector.ts` - Collects and syncs transcripts from Claude's cache
- `src/mcp-server/server.ts` - MCP server background daemon that triggers memory extraction

This approach balances context retention with cost efficiency.
