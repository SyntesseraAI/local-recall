The `.gitignore` file created in the `local-recall` folder should exclude:
- `transcripts/` - Temporary transcript files that shouldn't be version-controlled
- `processed-log.json` - Log file tracking which transcripts have been processed

The gitignore template is defined in two places that must stay in sync:
1. `src/core/index.ts` - Template used when creating the gitignore
2. `src/core/memory.ts` - Duplicate template definition

The actual `.gitignore` file at `local-recall/.gitignore` won't be automatically regenerated, so manual updates are needed there.
