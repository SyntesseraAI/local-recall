The logger utility (`src/utils/logger.ts`) exports log functions that accept a single string argument, not multiple parameters like traditional logging libraries. Calling logger functions with multiple arguments (e.g., `logger.debug('message', data)`) causes TypeScript compilation errors.

When logging complex data, pass it as a single JSON string: `logger.debug(JSON.stringify({ message, data }))`
