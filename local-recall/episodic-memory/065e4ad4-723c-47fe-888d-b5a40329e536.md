# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using hash encoding:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude project folder: `~/.claude/projects/syntessera_local-recall_01HQY8jXfJm7zh35BcYYiE1u/`

Each project contains:
- `transcripts/` - Directory with JSONL files
- `transcript_*.jsonl` - Individual transcript files

## Implementation Details

The `TranscriptCollector` in `src/core/transcript-collector.ts` handles finding and parsing these transcripts. Key implementation notes:
1. Uses a hash-based folder naming scheme that encodes the project path
2. Transcripts are stored as JSONL (JSON Lines) format
3. Folder structure: `~/.claude/projects/<project-hash>/transcripts/`
