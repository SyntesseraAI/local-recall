The local-recall project root is the plugin root following Claude Code's plugin format:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook event handlers using `${CLAUDE_PLUGIN_ROOT}` variable
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (build output, gitignored)

Hooks receive `cwd` via stdin to operate on the user's project directory.
