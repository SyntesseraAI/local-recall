# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with:
- Colons (`:`) replaced with underscores (`_`)
- Slashes (`/`) replaced with dashes (`-`)

### Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `_Users-joe-Code-Syntessera-local-recall`
- Transcripts stored at: `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files are JSONL format with `.jsonl` extension

This is important for the transcript-collector to locate and process Claude's transcript files correctly.
