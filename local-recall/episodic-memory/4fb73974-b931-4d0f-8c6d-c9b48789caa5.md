The vector store is SQLite-backed and uses the fastembed library with BGE-small-en-v1.5 model (~133MB) for generating embeddings. Scores are cosine distance similarity (0.0-1.0), rounded to 2 decimal places. Results are sorted by score descending with recency tie-breaker when scores are equal.
