# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and forward slashes appended as underscores.

## Example

For project at `/Users/joe/Code/Syntessera/local-recall`:
- Encoded path: `Users-joe-Code-Syntessera-local-recall_`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall_/transcripts/`
- File pattern: `*.jsonl` (JSON Lines format)

## Key Details

- Each transcript is a separate `.jsonl` file
- Files contain newline-delimited JSON objects
- Useful for the `TranscriptCollector` to locate and process transcripts
- On other platforms (Windows, Linux), the path encoding may differ
