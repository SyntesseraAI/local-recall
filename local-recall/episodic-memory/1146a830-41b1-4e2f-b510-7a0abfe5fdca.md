The user-prompt-submit hook in src/hooks/user-prompt-submit.ts:1-265 extracts keywords from the user's prompt using the Claude API (claude -p command). These keywords are then used to perform fuzzy search against the memory index to find and inject relevant memories into the context before Claude processes the prompt. This enables dynamic context injection based on what the user is asking about.
