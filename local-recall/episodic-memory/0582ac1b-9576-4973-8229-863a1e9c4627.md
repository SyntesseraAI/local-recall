# TranscriptCollector Implementation

The `TranscriptCollector` class handles discovering and parsing Claude Code project transcripts.

## Key Responsibilities

1. **Finding Transcripts**: Locates transcript files in `~/.claude/projects/<project>/transcripts/`
2. **Parsing JSONL**: Reads and parses transcript JSONL files
3. **Extracting Content**: Converts transcript events into structured format

## File Location

Implementation is in `src/core/transcript-collector.ts`

## Integration

Used by:
- MCP Server daemon for background memory extraction
- Stop hook for immediate transcript processing
- Manual transcript analysis

## Notes

- Transcripts are JSONL format (JSON Lines)
- One event per line in transcript files
- Used to extract memories for long-term retention
