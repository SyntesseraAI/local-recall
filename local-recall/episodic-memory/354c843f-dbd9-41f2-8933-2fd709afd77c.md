# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Forward slashes `/` are replaced with underscores `_`
- Colons `:` are replaced with underscores `_`
- The resulting string becomes the folder name

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`
- Individual transcripts: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/*.jsonl`

## Transcript File Format

Transcript files are in JSONL format (JSON Lines), with each line containing a single transcript event.

## Implementation Note

The `transcript-collector` module needs to handle this path transformation when locating and reading transcript files from the Claude cache.
