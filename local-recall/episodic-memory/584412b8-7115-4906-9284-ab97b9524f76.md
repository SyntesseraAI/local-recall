Development prerequisites and conventions:

**Requirements:**
- Node.js 18+
- TypeScript 5.9+

**Build scripts:**
- `npm run build` - Compile TypeScript
- `npm run dev` - Watch mode
- `npm run test` - Run all tests
- `npm run lint` - Lint code
- `npm run mcp:start` - Start MCP server

**Testing tiers:**
- `npm test` - All tests
- `npm run test:unit` - Unit tests only
- `npm run test:integration` - Integration tests

**Development requirement:** All new features must have corresponding unit tests in `tests/unit/` and integration tests in `tests/integration/`. Documentation updates required in `docs/` folder when modifying functionality.
