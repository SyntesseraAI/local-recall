# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files where each line is a JSON object representing a transcript event.

## Implementation Notes

When implementing transcript collection:
1. Replace forward slashes with dashes in the absolute path
2. Look in `transcripts/` subdirectory
3. Parse as JSONL format, not plain JSON
4. This location is used by Local Recall's transcript-collector to sync transcripts from Claude's cache
