# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall` (leading underscore, forward slashes become underscores)
- Transcript files are stored in: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Usage in MCP Server

The MCP server background daemon syncs transcripts from this location to extract memories asynchronously. The path construction is critical for the daemon to locate and process transcripts correctly.
