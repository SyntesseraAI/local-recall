# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Path Format

Project folder names use the absolute working directory with slashes replaced by dashes and leading slashes removed.

### Example
For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Files are in JSONL format (`.jsonl`)

This is important for the MCP server daemon which syncs transcripts from Claude's cache to process for memory extraction.
