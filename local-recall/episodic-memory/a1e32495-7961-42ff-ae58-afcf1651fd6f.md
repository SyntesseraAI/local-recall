# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the working directory path:
- Working dir: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `users-joe-code-syntessera-local-recall` (all path separators `/` converted to `-`)
- Transcripts stored in: `~/.claude/projects/users-joe-code-syntessera-local-recall/transcripts/`
- Each transcript is a `.jsonl` file (JSON Lines format)

## Implementation Note

The `transcript-collector.ts` module uses this path pattern to locate and read transcripts. The conversion from path separators to dashes is consistent across all projects.

## Related Files

- `src/core/transcript-collector.ts` - Implements transcript discovery and reading
- Transcripts are processed by the MCP daemon to extract memories
