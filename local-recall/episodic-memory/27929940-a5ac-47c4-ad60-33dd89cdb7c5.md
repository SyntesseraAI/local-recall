Three hooks integrate Local Recall with Claude Code:
1. **SessionStart**: Loads 5 most recent memories when session begins
2. **UserPromptSubmit**: Performs semantic search on user prompt to find relevant memories
3. **Stop**: Disabled - memory extraction now handled by MCP daemon (every 5 minutes)

Hooks are configured in `.claude/settings.json` and output memory content to stdout for injection into Claude's context.
