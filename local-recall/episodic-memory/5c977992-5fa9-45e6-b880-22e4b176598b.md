The memory extractor module (src/core/memory-extractor.ts) uses the Claude API (claude-sdk) to analyze transcripts and extract memories. It processes condensed transcripts containing [User], [Assistant], and [Tool] events, then calls Claude's API to identify and create memories from the session. The extraction process handles edge cases like network timeouts and invalid responses.
