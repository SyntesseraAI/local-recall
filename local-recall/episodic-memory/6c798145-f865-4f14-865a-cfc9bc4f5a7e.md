Claude Code stores project transcripts in `~/.claude/projects/` directory. The folder names are path-encoded versions of the project working directory, using dashes instead of slashes.

## Example
- Project at: `/Users/joe/Code/Syntessera/local-recall`
- Transcripts stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files where each line is a complete JSON object representing a transcript event.
