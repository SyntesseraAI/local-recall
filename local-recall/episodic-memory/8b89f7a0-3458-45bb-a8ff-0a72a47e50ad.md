Local Recall uses a structured directory layout:
- `src/` - Implementation source code (TypeScript)
- `dist/` - Compiled output (gitignored)
- `local-recall/` - Memory storage root
  - `.gitignore` - Auto-generated, excludes `memory.sqlite` and `recall.log`
  - `memory.sqlite` - SQLite database with embeddings (gitignored, auto-generated)
  - `recall.log` - Debug log file (gitignored)
  - `episodic-memory/` - Individual memory markdown files (version-controlled)
- `local_cache/` - Embedding model cache (gitignored)

Memory files are version-controlled; derived artifacts (database, logs) are gitignored.
