Hook scripts in the plugin need to reference the compiled JavaScript files (.js extension), not the TypeScript source files.

The build process compiles `.ts` files to `.js` and places them in the `dist/` folder. When referencing these from plugin configuration or startup scripts, always use the `.js` extension:
- ✓ `node ./dist/hooks/session-start.js`
- ✗ `node ./src/hooks/session-start.ts`

This applies to all hook references in plugin initialization code.
