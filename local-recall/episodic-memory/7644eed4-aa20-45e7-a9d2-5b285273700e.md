The logger.ts utility functions (debug, info, warn, error) expect a single string argument, not multiple parameters or template literals with separate args. Correct usage is `logger.debug('message')` not `logger.debug('message', data)`. This requires string formatting before calling the logger function. Fix any calls that pass multiple parameters to the logger.
