# Transcript File Format

Claude Code transcripts are stored as JSON Lines files with the `.jsonl` extension.

## Format Details

- **Extension**: `.jsonl`
- **Structure**: One JSON object per line
- **Location**: `~/.claude/projects/<project-path>/transcripts/`
- **Naming**: Sequential numeric files like `0.jsonl`, `1.jsonl`, `2.jsonl`

## Processing

When processing transcripts for memory extraction, each line should be parsed as a separate JSON object. Tools like `jq` or custom parsers can process the file line-by-line.
