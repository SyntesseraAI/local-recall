# Transcript Collector JSONL Parsing

The transcript collector needs to parse JSONL format files where each line contains a separate JSON event object representing a transcript event.

## Implementation Requirements

- Read transcript files as JSONL (one JSON object per line)
- Parse each line into an event object
- Handle malformed JSON gracefully
- Process events sequentially to maintain order
- Extract memories from the parsed transcript events based on the transcript analysis questions
