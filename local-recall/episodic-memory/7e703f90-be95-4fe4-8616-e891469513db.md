# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Directory Structure

Project folders use the absolute path with slashes replaced by dashes:
- Example working directory: `/Users/joe/Code/Syntessera/local-recall`
- Becomes: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with `.jsonl` extension.

## Relevance to Local Recall

The transcript processing in local-recall needs to read from this location to extract memories from Claude Code sessions.
