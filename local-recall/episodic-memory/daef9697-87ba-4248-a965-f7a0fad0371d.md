# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path by:
1. Replacing forward slashes `/` with dashes `-`
2. Prepending `projects-` to the result
3. Result: `~/.claude/projects/projects-<path-with-dashes>/transcripts/`

## Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `projects-Users-joe-Code-Syntessera-local-recall`
- Transcript location: `~/.claude/projects/projects-Users-joe-Code-Syntessera-local-recall/transcripts/`

This is important for the transcript collector when syncing transcripts from Claude's cache.
