# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the filesystem path by replacing slashes with dashes and encoding special characters. For example:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall`

## File Format

Transcripts are stored as JSONL (JSON Lines) files in the `transcripts/` subdirectory:
- Location: `~/.claude/projects/<encoded-path>/transcripts/<timestamp>.jsonl`
- Each line is a JSON object representing a transcript event

## Relevance

The transcript collector in `src/core/transcript-collector.ts` needs to read from this location to extract memories from Claude Code sessions. Understanding the exact path format and file location is critical for correctly syncing transcripts.

## Reference

Implemented in `src/core/transcript-collector.ts` - the `getProjectTranscriptsPath()` function builds this path dynamically.
