# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons replaced with underscores. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `_Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a separate JSON object representing a transcript event.

## Relevance to Local Recall

The transcript-collector needs to read from these locations to sync transcripts from Claude's cache for memory extraction.
