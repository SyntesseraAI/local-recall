# Local Recall MCP Server Background Daemon

The Local Recall MCP server runs a background daemon process that:

1. **Polling Interval**: Processes transcripts every 5 minutes
2. **Source**: Syncs transcripts from `~/.claude/projects/<project>/transcripts/`
3. **Processing**: Uses `claude -p` command to extract memories from transcripts
4. **Change Detection**: Tracks processed transcripts with content hashes to identify changes
5. **Memory Management**: Deletes and recreates memories when transcripts change

This enables asynchronous memory extraction without blocking Claude Code sessions.
