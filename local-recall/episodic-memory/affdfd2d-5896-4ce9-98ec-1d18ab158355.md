The BGE-small-en-v1.5 model (~133MB) is automatically downloaded to local_cache/ on first use. First startup may take 30-60 seconds. Subsequent runs load from cache. If tokenizer file is not found, it indicates corrupted cache that should be deleted and will re-download automatically.
