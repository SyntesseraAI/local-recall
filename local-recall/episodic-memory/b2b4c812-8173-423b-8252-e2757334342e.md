**Memory Manager** (src/core/memory.ts): CRD operations (no update - memories are idempotent)
- `createMemory(data)` - Returns existing if duplicate
- `deleteMemory(id)` - Delete by ID
- `getMemory(id)` - Retrieve specific memory
- `listMemories(filter?)` - List with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Check for existing duplicates

**Index Manager** (src/core/index.ts): Maintains searchable keyword index
- `buildIndex()` - Scan all memory files and build index
- `getIndex()` - Retrieve current index
- `refreshIndex()` - Rebuild from scratch

**Search** (src/core/search.ts): Fuzzy search implementation
- `searchByKeywords(query)` - Find memories with fuzzy matching
- `searchBySubject(query)` - Search by subject line
- `searchByScope(scope)` - Find memories for specific scope
