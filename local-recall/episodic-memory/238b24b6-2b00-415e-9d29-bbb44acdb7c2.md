Search module provides fuzzy search capabilities:
- `searchByKeywords(query)` - Find memories matching keywords with fuzzy matching
- `searchBySubject(query)` - Search by subject line
- `searchByScope(scope)` - Find all memories for specific scope

Fuzzy threshold is configurable (default 0.6). Uses ripgrep for efficient searching.
