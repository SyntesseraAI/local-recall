Memory records include fields for unique identification (id), searchable keywords, scope specification (global/file/area), timestamps (occurred_at), and content hashing (content_hash) for deduplication. These metadata fields enable proper memory organization, duplicate detection, and scope-aware retrieval across different file contexts and project areas.
