# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with:
- Filename: ISO-8601 timestamp (e.g., `2025-12-02T00_00_00.jsonl`)
- Each line: Complete message object with tool invocations
- Size: Can grow to multiple MB over a session

## Implications

This location is critical for the transcript-collector to find and process Claude's session transcripts when building memories.
