Each memory file uses YAML frontmatter with fields: id (UUID), subject, keywords array, applies_to (scope), occurred_at (ISO-8601), and content_hash (SHA-256 prefix). The actual content is stored as markdown below the frontmatter. This format enables deduplication, semantic search, and scoped organization of memories.
