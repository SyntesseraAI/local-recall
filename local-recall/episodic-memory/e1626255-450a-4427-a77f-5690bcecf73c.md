MCP server can be installed as npm package or run from local-recall project directory. Claude Code configuration differs: npm package uses './node_modules/local-recall/dist/mcp-server/server.js', local directory uses './dist/mcp-server/server.js'. Claude Desktop requires absolute paths. Requires LOCAL_RECALL_DIR environment variable pointing to memory storage directory.
