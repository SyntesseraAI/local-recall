The MemoryManager.findDuplicate(occurredAt, contentHash) method checks for existing duplicates. Memory creation with createMemory(data) returns existing memory if duplicate detected. The content_hash field stores SHA-256 hash prefix (16 chars) of content for deduplication and sorting.
