# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with a timestamp prefix:
- Filename format: `YYYYMMDD_HHmmss.jsonl`
- Example: `20250122_143022.jsonl`

## Usage in transcript-collector.ts

The transcript collector needs to construct this path dynamically:
1. Take the working directory (cwd)
2. Replace all `/` with `-`
3. Construct the path: `~/.claude/projects/{transformed-cwd}/transcripts/`
4. Expand `~` to the user's home directory

This is necessary because `~` is a shell expansion and is not available in JavaScript context.
