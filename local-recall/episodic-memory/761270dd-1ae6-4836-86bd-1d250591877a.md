# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Path Encoding Convention

Project folder names encode the absolute path using URL/percent encoding:
- Forward slashes `/` → `%2F`
- Colons `:` → `%3A`

**Example:**
Project path: `/Users/joe/Code/Syntessera/local-recall`
Transcript folder: `~/.claude/projects/Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall/`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files:
- One JSON object per line
- Each line contains a complete event (message or tool invocation)
- Files are named `*.jsonl`

## Key Discovery

The `transcript-collector.ts` module should leverage this path encoding pattern to reliably locate and access Claude's cached transcripts without requiring user configuration.
