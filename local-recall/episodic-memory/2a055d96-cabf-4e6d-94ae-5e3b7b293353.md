The memory extraction process in Local Recall uses Claude Haiku (`claude -p --model haiku`) to extract keywords from transcripts. This is part of the asynchronous transcript processing that runs every 5 minutes in the MCP server daemon. The extracted keywords are used to create searchable memory files with proper categorization.
