The Memory Manager in src/core/memory.ts provides: createMemory(data) - creates new memory file, returns existing if duplicate; deleteMemory(id) - deletes memory by ID; getMemory(id) - retrieves specific memory; listMemories(filter?) - lists all memories with optional filtering; findDuplicate(occurredAt, contentHash) - checks for existing duplicate. No update operation exists because memories are idempotent.
