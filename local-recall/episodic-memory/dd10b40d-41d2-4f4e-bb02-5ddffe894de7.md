Memory files stored in episodic-memory/ use markdown with YAML frontmatter containing:
- id: UUID
- subject: One-line description
- keywords: Array of searchable terms
- applies_to: Scope (global, file:<path>, or area:<name>)
- occurred_at: ISO-8601 timestamp
- content_hash: SHA-256 prefix (16 chars) for deduplication

Scoring uses cosine distance (0.0-1.0, rounded to 2 decimals), with recency as tie-breaker.
