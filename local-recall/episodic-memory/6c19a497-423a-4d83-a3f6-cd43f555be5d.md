The `.gitignore` file is created automatically using a template in `ensureGitignore()`. The template must include:
- `orama-*-index.json` - Vector store indexes
- `recall.log` - Debug log file
- `transcripts/` - Transcript directory
- `processed-log.json` - Processed transcript tracking
- `orama-thinking-index.json` - Thinking memory index
- `thinking-processed-log.jsonl` - Thinking transcript tracking

This template is duplicated in `src/core/memory.ts` and both copies must be kept in sync when making changes.
