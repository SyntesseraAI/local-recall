# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory by:
1. Taking the absolute path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replacing `/` with `-` to create the folder name (e.g., `Users-joe-Code-Syntessera-local-recall`)
3. Storing transcripts in `~/.claude/projects/{folder-name}/transcripts/`

## Transcript File Format

Each transcript is stored as a single `.jsonl` file (JSON Lines format) containing an array of chat events.

## Implementation

The `transcript-collector.ts` module handles discovering and collecting transcripts from this location. When processing transcripts, remember this path format to correctly locate files.
