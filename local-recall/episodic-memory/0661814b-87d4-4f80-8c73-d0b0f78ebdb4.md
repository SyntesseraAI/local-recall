# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

The project folder name uses the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files are in JSONL format with structure: `{"event": "...", "content": "...", "timestamp": "..."}`

## Usage in Local Recall

The transcript-collector uses this path to:
1. Find and read project transcripts from Claude's cache
2. Process transcript events to extract memories
3. Track processed transcripts using content hashes for change detection
4. Create or update memories based on transcript content
