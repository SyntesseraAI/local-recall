# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Forward slashes `/` are converted to underscores `_`
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `_Users_joe_Code_Syntessera_local-recall`

## Transcript Structure

Transcripts are stored as `.jsonl` files (JSON Lines format):
- Directory: `~/.claude/projects/<project-folder-name>/transcripts/`
- Each line in the file is a separate JSON object representing a transcript event
- Events include user messages, assistant responses, and tool calls

## Implementation Notes

When implementing transcript collection:
1. Use `path.resolve()` to get absolute paths
2. Replace all forward slashes with underscores for folder naming
3. Parse `.jsonl` files line-by-line, not as a single JSON array
4. Handle the case where transcript files may not exist (new projects)
5. Transcripts accumulate across sessions in the same project

This is critical for `src/core/transcript-collector.ts` to properly locate and parse Claude's transcript files.
