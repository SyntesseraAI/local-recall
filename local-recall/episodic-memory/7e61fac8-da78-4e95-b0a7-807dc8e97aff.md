# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `_Users_joe_Code_Syntessera_local-recall`
- Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

This conversion is handled in `src/core/transcript-collector.ts` via the `convertPathToDashes` function.

## Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one entry per line.
