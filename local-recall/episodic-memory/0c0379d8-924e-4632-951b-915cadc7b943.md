# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with leading slash removed and forward slashes replaced with underscores.

### Example

Working directory: `/Users/joe/Code/Syntessera/local-recall`

Transcript location: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) with one JSON object per line, each representing a message or event in the conversation.

## Location in Code

The `transcript-collector.ts` module handles finding and reading these transcripts. The path conversion logic should account for this specific format when building paths dynamically.
