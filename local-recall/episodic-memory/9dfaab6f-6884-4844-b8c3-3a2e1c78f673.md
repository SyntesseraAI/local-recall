When the local-recall MCP server is properly configured as a plugin in Claude Code, tools defined in the MCP server are automatically discovered and available to Claude.

The current available tools are:
- `episodic_create` - Create episodic memories
- `episodic_get` - Retrieve specific episodic memories
- `episodic_search` - Search episodic memories
- `thinking_get` - Retrieve thinking memories
- `thinking_search` - Search thinking memories

These tools handle both episodic and thinking memory management automatically.
