## Claude Projects Directory Location

Claude Code stores project transcripts in a specific directory structure:

- **Base path**: `~/.claude/projects/`
- **Project folder naming**: Uses the full path as identifier with `/` replaced by `_`
- **Transcripts stored**: In `transcripts/` subdirectory within each project folder
- **File format**: JSONL (JSON Lines) - one JSON object per line
- **Filename pattern**: `{timestamp}-{uuid}.jsonl`

For the local-recall project:
- **Project directory**: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`
- **Transcripts path**: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

Each JSONL file contains an array of messages with `role` (user/assistant), `content`, `tool_use` blocks, and other metadata.
