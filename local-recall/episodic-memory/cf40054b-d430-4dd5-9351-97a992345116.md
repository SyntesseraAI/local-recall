# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored at `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Implementation

When converting a file path to a Claude project folder name:
1. Remove the leading slash
2. Replace all forward slashes with dashes
3. This matches the format Claude uses internally

Relevant code: `src/core/transcript-collector.ts`
