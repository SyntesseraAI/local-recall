Added separate token count limits for episodic and thinking memories:
- `episodicMaxTokens`: Max tokens for episodic memories (default: 1000, env: `LOCAL_RECALL_EPISODIC_MAX_TOKENS`)
- `thinkingMaxTokens`: Max tokens for thinking memories (default: 1000, env: `LOCAL_RECALL_THINKING_MAX_TOKENS`)

Both configured in `src/core/types.ts` and `src/utils/config.ts`
