# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) with names like `session-<timestamp>.jsonl`.

Each line contains an event object with:
- `type`: Event type (e.g., "user_message", "assistant_message", "tool_call", "tool_result")
- `timestamp`: ISO-8601 timestamp
- Other fields depending on event type

## Accessing Transcripts

The MCP server background daemon reads from this directory to extract memories asynchronously.
