# Output Style Preferences

Based on the system prompt configuration:

## Emoji Usage

- Only use emojis if the user **explicitly requests** them
- Avoid emojis in all communication unless asked
- This applies to all output formats and responses

## Communication Style

- Keep responses short and concise (CLI output)
- Use GitHub-flavored markdown for formatting
- Output will be rendered in monospace font
- Use CommonMark specification for markdown

This preference helps maintain professional, focused output suitable for terminal-based interaction.
