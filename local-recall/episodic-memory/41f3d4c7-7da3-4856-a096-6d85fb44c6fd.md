The local-recall system has two independent memory systems that can be configured separately:

**Episodic Memories** (`user-prompt-submit.ts` hook):
- Store facts and contextual information
- Default similarity threshold: 0.5
- Default token budget: 1000
- Configuration: `episodicEnabled`, `episodicMaxTokens`, `episodicMinSimilarity`

**Thinking Memories** (`user-prompt-submit-thinking.ts` hook):
- Store thoughtâ†’output examples
- Default similarity threshold: 0.8 (higher bar for relevance)
- Default token budget: 1000
- Configuration: `thinkingEnabled`, `thinkingMaxTokens`, `thinkingMinSimilarity`

Both can be enabled/disabled independently and have separate vector stores with different Orama indexes.
