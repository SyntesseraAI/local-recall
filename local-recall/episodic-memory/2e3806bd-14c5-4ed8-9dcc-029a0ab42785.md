# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format:
- Named with ISO-8601 timestamps: `YYYY-MM-DDTHH-MM-SS.jsonl`
- Each line is a complete JSON object representing a session event
- Contains all messages, tool calls, and results from Claude Code sessions

This information is essential for the transcript-collector to locate and process transcripts for memory extraction.
