# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with naming convention:
- `transcript-YYYYMMDD-HHMMSS.jsonl`
- Example: `transcript-20250102-153045.jsonl`

## Usage in MCP Server

The transcript collector reads from this location to extract memories asynchronously. The path is constructed using the project's working directory provided to the MCP server.
