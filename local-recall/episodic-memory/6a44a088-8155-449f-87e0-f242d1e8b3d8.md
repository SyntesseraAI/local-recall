TypeScript source is in `src/` with subdirectories:
- `core/`: Memory management (memory.ts, index.ts, search.ts)
- `hooks/`: Hook implementations
- `mcp-server/`: MCP server code
- `utils/`: Utilities for markdown, transcript parsing, logging

Bundled output goes to `scripts/` (gitignored). Plugin root IS the project root.
