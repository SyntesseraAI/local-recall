Configuration options:
- `memoryDir`: Default `./local-recall`
- `maxMemories`: Default 1000
- `indexRefreshInterval`: Default 300 seconds
- `fuzzyThreshold`: Default 0.6 (0-1 range)

Can be set via environment variables or `.local-recall.json` file in project root.
