Each memory file has YAML frontmatter with fields:
- `id`: UUID
- `subject`: Brief one-line description
- `keywords`: Searchable keywords array
- `applies_to`: Scope (global, file:<path>, or area:<name>)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix for deduplication

Memory files are named with UUIDs and stored in `local-recall/episodic-memory/`
