The MemoryExtractor class processes transcripts to create memories with built-in deduplication:

- Extracts memories from transcripts using the Memory class
- Uses `content_hash` and `occurred_at` fields for deduplication via `findDuplicate()`
- Returns idempotent memory creation (same input = same result, no duplicates)
- Handles both new memory creation and detection of existing duplicate memories

This ensures that re-processing the same transcript won't create duplicate memories in the system.
