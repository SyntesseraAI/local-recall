The `TranscriptCondenser` class in `src/core/transcript-condenser.ts` is responsible for converting full session transcripts into a condensed event-based format. This condensed format is used by the memory extractor to work with tokens more efficiently. The format includes markers like `[User]`, `[Assistant]`, `[Tool: Name]`, and `[Result: OK/ERROR]` to represent the key elements of a coding session.
