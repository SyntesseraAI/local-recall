# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

Project folders use the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are JSONL format (newline-delimited JSON), one event per line.

## Relevance to local-recall

The MCP server and transcript collector components need to:
1. Construct the project path from the working directory
2. Read JSONL transcript files from the `.claude/projects/` location
3. Process transcript events to extract memories

This is essential for the background daemon that syncs and processes transcripts every 5 minutes.
