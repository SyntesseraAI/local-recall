Local Recall is installed as a Claude Code plugin with root directory as plugin root. Contains .claude-plugin/plugin.json, hooks.json, and .mcp.json. Scripts in scripts/ directory are built output (gitignored). When installed as npm package, scripts reference node_modules/local-recall/scripts/. Can be added to .claude/settings.json plugins array or hooks array directly.
