Claude's transcript JSONL files contain events with these main types:

- **user_message**: Contains `type: "user_message"` and `content` field with the user's input
- **assistant_message**: Contains `type: "assistant_message"`, `content` field, and potentially `tool_use` array
- **tool_use**: Nested within assistant messages, contains `id`, `name`, `input` (JSON), and execution details
- **tool_result**: Separate events with `type: "tool_result"` containing tool output

The transcript-collector should parse these events to extract session information and understand what happened during a Claude session for memory extraction purposes.
