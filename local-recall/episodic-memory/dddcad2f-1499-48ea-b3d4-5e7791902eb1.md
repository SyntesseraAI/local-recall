The local-recall directory is the plugin root following Claude Code's plugin format:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook configuration
- `.mcp.json` - MCP server configuration
- `src/` - TypeScript source code
  - `core/` - Memory management (memory.ts, vector-store.ts, embedding.ts, search.ts, types.ts)
  - `hooks/` - Claude Code hooks (session-start.ts, user-prompt-submit.ts, stop.ts)
  - `mcp-server/` - MCP server (server.ts, tools.ts)
  - `utils/` - Utilities (markdown.ts, transcript.ts, logger.ts, fuzzy.ts)
- `scripts/` - Bundled executables (build output, gitignored)
- `local-recall/` - Memory storage (version-controlled except .gitignore, memory.sqlite, recall.log)
