Added `cleanupTranscripts()` method (lines 353-403) that:
1. Validates file format - expects UUID.jsonl naming pattern
2. Removes files that don't match format (e.g., .DS_Store, readme.txt, malformed names)
3. Removes synthetic transcripts (containing '<synthetic>' model marker)
4. Returns cleanup stats: {invalidFormat: number, synthetic: number}

The cleanup is automatically called at the start of `syncTranscripts()` so malformed and synthetic files are cleaned before any processing begins.
