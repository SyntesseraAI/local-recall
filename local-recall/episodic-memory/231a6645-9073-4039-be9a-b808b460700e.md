# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons replaced with underscores. For example:

- Project at `/Users/joe/Code/Project` → folder named `_Users_joe_Code_Project`
- Project at `/home/user/projects/app` → folder named `_home_user_projects_app`

## Transcript File Format

Each session creates a `.jsonl` file (newline-delimited JSON) with transcript data. The transcript collector reads these files to extract memory-worthy content.

## Implementation Note

The `transcript-collector.ts` module handles reading and processing these transcripts. It needs to correctly convert project paths to the folder naming convention when locating transcript files.
