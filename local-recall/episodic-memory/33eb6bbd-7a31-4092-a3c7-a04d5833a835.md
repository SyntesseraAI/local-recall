Claude stores project transcripts in `~/.claude/projects/` directory. Project directories are named with URL-encoded paths where slashes become dashes, matching the repo folder name. Example: `/Users/joe/.claude/projects/Users-joe-Code-Syntessera-local-recall/` for the local-recall project. Each project contains a `transcripts/` subdirectory with JSONL files containing Claude session transcripts.
