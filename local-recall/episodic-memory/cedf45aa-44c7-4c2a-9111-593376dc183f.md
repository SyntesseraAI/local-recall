Memory operations are idempotent - no update operation exists:
- `createMemory(data)` - Returns existing memory if duplicate detected
- `deleteMemory(id)` - Remove by ID
- `getMemory(id)` - Retrieve specific memory
- `listMemories(filter?)` - List with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Check for existing duplicate

Deduplication uses occurred_at timestamp and content_hash (SHA-256 prefix) to detect and prevent duplicate memories.
