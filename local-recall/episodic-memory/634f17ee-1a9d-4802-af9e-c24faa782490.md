# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (one JSON object per line) with a `.jsonl` extension.

Each line contains a message object with properties like `role`, `content`, `timestamp`, etc.

## Implementation Notes

- The path conversion happens in `TranscriptCollector.getTranscriptPath()` in `src/core/transcript-collector.ts`
- This pattern applies to all projects, making transcripts discoverable and retrievable
- The transcript path is predictable given a working directory path
