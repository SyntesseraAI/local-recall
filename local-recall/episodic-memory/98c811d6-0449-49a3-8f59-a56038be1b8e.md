Development setup for Local Recall:

- **Build system:** npm with TypeScript compilation
- **Test framework:** Tests located in tests/unit/ and tests/integration/
- **Development mode:** npm run dev for watch mode
- **Plugin deployment:** dev-marketplace/local-recall-plugin/ directory contains plugin configuration
- **Hook configuration:** hooks.json defines SessionStart, UserPromptSubmit, and Stop hook handlers
- **Memory extraction:** Uses transcript analysis to automatically create memories

Project follows TypeScript best practices with type definitions and test coverage requirements for new features.
