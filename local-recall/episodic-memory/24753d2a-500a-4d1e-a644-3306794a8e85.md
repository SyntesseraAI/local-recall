Memory files are stored as structured markdown files in `local-recall/episodic-memory/` with YAML frontmatter containing:

- **id**: Unique identifier (UUID)
- **subject**: Brief one-line description
- **keywords**: Array of searchable keywords
- **applies_to**: Scope string - `global`, `file:<path>`, or `area:<component-name>`
- **occurred_at**: ISO-8601 timestamp for deduplication and sorting
- **content_hash**: SHA-256 hash prefix (16 chars) for deduplication

Memories are idempotent - duplicate detection is based on `occurred_at` and `content_hash`.
