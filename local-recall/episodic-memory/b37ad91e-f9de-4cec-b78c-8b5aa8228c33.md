The `episodicEnabled` configuration defaults to `false` as of this session. Users must explicitly enable episodic memory retrieval by either:
1. Setting environment variable `LOCAL_RECALL_EPISODIC_ENABLED=true`
2. Adding `"episodicEnabled": true` to their `.local-recall.json` config file

This default is defined in `src/core/types.ts:118` in the Zod schema definition.

Documentation for this configuration option is maintained in `CLAUDE.md` in the Configuration Options table.
