Changed the default value of `episodicEnabled` from `true` to `false` in the configuration schema. This means episodic memory retrieval is now disabled by default unless explicitly enabled via `LOCAL_RECALL_EPISODIC_ENABLED=true` environment variable or `"episodicEnabled": true` in `.local-recall.json` config file.

Modified files:
- `src/core/types.ts:118` - schema default in Zod schema
- `CLAUDE.md` - configuration table documentation

Users must explicitly opt-in to episodic memory retrieval.
