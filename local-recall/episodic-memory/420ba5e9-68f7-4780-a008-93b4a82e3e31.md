# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project with `/` replaced by `_`. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with names like `transcript_123456.jsonl`, containing timestamped events and Claude's responses.

This is crucial for the MCP server's background daemon which:
1. Syncs transcripts from this location
2. Extracts memories using Claude Haiku
3. Tracks processed transcripts with content hashes
4. Deletes and recreates memories when transcripts change
