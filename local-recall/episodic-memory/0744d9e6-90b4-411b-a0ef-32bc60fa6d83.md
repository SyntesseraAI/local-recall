# Claude Project Transcripts Storage

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Project Folder Naming

The project folder name is derived from the absolute working directory path:
- Path separators (`/`) are converted to underscores (`_`)
- The leading underscore from the root `/` is removed
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users_joe_Code_Syntessera_local-recall`

## Transcript Format

- Transcripts are stored as JSONL files (one JSON object per line)
- Located in: `~/.claude/projects/{project-folder}/transcripts/`
- Filename format: `{timestamp}.jsonl` (ISO-8601 timestamp)

## Content Structure

Each transcript line contains:
- `role`: Either "user" or "assistant"
- `content`: The message text
- Additional metadata fields for tracking context

This is critical for the MCP server background daemon which syncs and processes transcripts every 5 minutes.
