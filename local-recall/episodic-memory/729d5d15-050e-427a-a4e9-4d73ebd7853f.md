# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with sequential numbering:
- Format: `transcript_<number>.jsonl` (e.g., `transcript_0.jsonl`, `transcript_1.jsonl`)
- Each line contains a complete message object from the Claude Code session

## Key Details

- The transcript directory may not exist until the first Claude Code session
- Each session appends to the transcript files
- Transcripts are stored locally on the user's machine in the Claude cache directory
- This location is used by the MCP server daemon to collect transcripts for memory extraction
