Memory files are markdown files with YAML frontmatter stored in local-recall/episodic-memory/ and ARE version-controlled (tracked in git). The following are gitignored:
- local-recall/memory.sqlite (SQLite database with vector embeddings)
- local-recall/recall.log (debug log file)
- local_cache/ (embedding model cache, ~133MB for BGE-small-en-v1.5)
- dist/ (build output)

This separation allows team members to share memory files while keeping auto-generated files local.
