Each memory file has required fields: id (UUID), subject (brief description), keywords (array), applies_to (scope), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix). The occurred_at and content_hash fields enable deduplication and help track when memories were created.
