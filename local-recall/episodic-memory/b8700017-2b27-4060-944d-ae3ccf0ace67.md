# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format):
- Each line is a complete JSON object representing a message in the conversation
- Files use naming pattern like `<session-id>.jsonl`
- Multiple transcript files may exist for different sessions

## Key Files

- `~/.claude/projects/` - Base directory for all Claude projects
- Transcripts are cached locally for quick access
- The MCP server daemon syncs these transcripts to process them for memory extraction
