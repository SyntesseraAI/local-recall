# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores and colons replaced by hyphens.

### Example

For a project at `/Users/joe/Code/Syntessera/local-recall`:
- Encoded path: `Users-joe-Code-Syntessera-local-recall`
- Full transcript directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Individual transcripts: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/<session_id>.md`

## Usage

This encoding scheme is used by Local Recall's MCP server daemon to locate and sync transcripts from Claude's cache for memory extraction.
