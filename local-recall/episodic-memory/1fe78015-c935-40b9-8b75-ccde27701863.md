Local Recall stores memories in JSONL (JSON Lines) format with files like `episodic-000001.jsonl` and `thinking-000001.jsonl` in addition to individual markdown files in `episodic-memory/` and `thinking-memory/` directories. The JSONL files appear to use 6-digit padding for sequencing. This format allows for efficient appending of new memories while maintaining backward compatibility with the markdown-based storage system.
