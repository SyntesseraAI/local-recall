# Transcript Path Resolution

The transcript collector needs to:

1. Receive the working directory (`cwd`) from the hook or context
2. Convert the absolute path to Claude's project folder format (replace `/` with `-`)
3. Look for transcripts in `~/.claude/projects/<converted-path>/transcripts/`
4. Read and parse `.jsonl` transcript files

## Important Detail

The conversion removes the leading slash, so:
- Input: `/Users/joe/Code/Syntessera/local-recall`
- Output: `Users-joe-Code-Syntessera-local-recall`

This is critical for correctly locating the transcript directory on macOS.
