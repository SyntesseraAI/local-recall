If getting 'Tokenizer file not found at local_cache/fast-bge-small-en-v1.5/tokenizer.json' error, cache is corrupted (usually from interrupted download). Fix by removing local_cache/fast-bge-small-en-v1.5* directory - model will re-download automatically on next run.
