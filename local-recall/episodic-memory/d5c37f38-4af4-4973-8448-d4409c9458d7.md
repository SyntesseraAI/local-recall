Directory structure:
- `.claude-plugin/plugin.json`: Plugin metadata
- `hooks.json`: Hook configuration (uses `${CLAUDE_PLUGIN_ROOT}`)
- `.mcp.json`: MCP server configuration
- `scripts/`: Bundled executables (build output, gitignored)
  - `hooks/`: Bundled hook scripts
  - `mcp-server/`: Bundled MCP server

Hooks receive `cwd` via stdin to operate on user's project directory.
