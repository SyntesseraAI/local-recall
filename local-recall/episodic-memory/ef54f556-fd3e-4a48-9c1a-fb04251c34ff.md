The local-recall project uses this structure:

- `src/` - TypeScript source organized by feature (core, hooks, mcp-server, utils, prompts)
- `dist/` - Compiled JavaScript output (completely gitignored)
- `local-recall/` - Memory storage with auto-generated .gitignore that:
  - Excludes `memory.sqlite` (SQLite database with embeddings)
  - Excludes `recall.log` (debug log)
  - Tracks `episodic-memory/` directory (individual *.md memory files)
- `local_cache/` - Embedding model cache (gitignored, auto-downloaded on first use)

This separation allows memory files to be version-controlled while keeping generated databases and logs local.
