# Memory Extraction Output Format Preference

When analyzing transcripts for memory extraction, the user prefers:
- **Direct JSON output only** - no markdown code blocks, no explanations
- **Raw JSON object** - can be parsed directly without removing markdown syntax
- **No extra formatting** - just the JSON structure as specified

This is important for automation and programmatic processing of extracted memories.
