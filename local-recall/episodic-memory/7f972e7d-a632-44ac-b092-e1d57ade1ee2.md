# Transcript Collector Implementation

The transcript collector in `src/core/transcript-collector.ts` uses the Claude CLI (`claude -p` command) to summarize and extract memories from raw transcripts.

## Key Details

- Uses `claude -p --model haiku` to process transcripts
- Passes transcript content to Claude for summarization
- The Haiku model is chosen for cost-efficiency in memory extraction
- Output from Claude is used to generate memory files
- This approach allows for intelligent, context-aware memory creation from unstructured transcript data
