The TranscriptCollector is responsible for finding and syncing transcripts from Claude's cache. It searches ~/.claude/projects for folders matching the current repository's path pattern, then looks for .jsonl transcript files directly in the project folder. The findClaudeProjectDir method tries multiple fallback strategies: first the dash-separated path convention, then scanning the projects directory if needed.
