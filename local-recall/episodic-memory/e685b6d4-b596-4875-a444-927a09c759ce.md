Episodic memory processing is disabled by default in local-recall. It is controlled by the `LOCAL_RECALL_EPISODIC_ENABLED` environment variable in the MCP server configuration. To enable it, add `"LOCAL_RECALL_EPISODIC_ENABLED": "true"` to the `env` section of the MCP server config in `.claude/settings.json` (or `.claude/settings.local.json`). The configuration is loaded through `src/utils/config.ts` and checked in `src/mcp-server/server.ts` to control whether the daemon runs memory extraction.
