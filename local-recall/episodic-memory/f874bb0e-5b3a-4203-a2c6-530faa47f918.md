The project is organized as:
- `src/hooks/` - Claude Code hooks (session-start, stop)
- `src/mcp-server/` - MCP server for tool exposure
- `src/core/` - Core memory management (memory.ts, search.ts, vector-store.ts, types.ts)
- `src/utils/` - Utilities (transcript parsing, markdown, config, logging)
- `local-recall/` - Memory storage directory (version-controlled markdown files)
- `dev-marketplace/` - Plugin configuration and marketplace metadata
