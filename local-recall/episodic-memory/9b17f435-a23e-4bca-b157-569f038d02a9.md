Each memory file is a markdown file with YAML frontmatter containing: `id` (UUID), `subject` (brief description), `keywords` (array for searchability), `applies_to` (global|file:<path>|area:<name>), `occurred_at` (ISO-8601 timestamp), and `content_hash` (SHA-256 prefix for deduplication). The content follows after the frontmatter closing `---`.
