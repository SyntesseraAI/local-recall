Added new configuration options:
- `episodicMaxTokens` (env: `LOCAL_RECALL_EPISODIC_MAX_TOKENS`, default: 1000) - Controls maximum tokens for episodic memories in context
- `episodicMinSimilarity` (env: `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY`, default: 0.8) - Controls minimum similarity threshold for episodic memories (80%)
- `thinkingMaxTokens` (env: `LOCAL_RECALL_THINKING_MAX_TOKENS`, default: 1000) - Controls maximum tokens for thinking memories in context
- `thinkingMinSimilarity` (env: `LOCAL_RECALL_THINKING_MIN_SIMILARITY`, default: 0.8) - Controls minimum similarity threshold for thinking memories (80%)

Both episodic and thinking memory have separate token budgets and similarity thresholds, allowing fine-grained control over how much context is injected for each memory type.
