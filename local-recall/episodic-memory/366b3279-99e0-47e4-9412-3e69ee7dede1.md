The memory system prevents duplicates by checking `occurred_at` timestamp and `content_hash` (16-character SHA-256 prefix). The `findDuplicate()` method in MemoryManager detects existing memories before creation, making memory operations idempotent. This enables safe re-processing of transcripts without creating duplicate memory entries.
