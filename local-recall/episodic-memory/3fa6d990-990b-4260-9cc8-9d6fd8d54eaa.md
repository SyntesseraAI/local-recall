The memory extractor (`src/core/memory-extractor.ts`) is responsible for analyzing Claude Code transcripts and automatically creating memory files. It uses Claude's API (via `claude -p`) to process transcripts through a structured prompt system defined in `src/prompts/memory-extraction.ts`. The extractor handles transcript parsing and converts the analysis results into memory file format with proper YAML frontmatter.
