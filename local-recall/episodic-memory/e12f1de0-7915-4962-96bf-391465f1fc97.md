# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path by replacing `/` with `-`. For example:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript File Format

Transcripts are stored as JSONL (JSON Lines) files at:
`~/.claude/projects/{project-folder}/transcripts/`

Each line is a JSON object representing a message/event in the transcript. This format is used by the transcript-collector component to parse and analyze Claude Code sessions.

## Implications for transcript-collector

When implementing transcript collection functionality, the path conversion logic must account for this naming scheme to correctly map from project working directory to the Claude transcripts directory.
