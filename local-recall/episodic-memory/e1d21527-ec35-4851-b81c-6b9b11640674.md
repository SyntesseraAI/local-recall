# TranscriptCollector Class

The `TranscriptCollector` class in `src/utils/transcript.ts` provides utilities for:

1. Locating transcript files at the Claude-specific path: `~/.claude/projects/<project-id>/transcripts/`
2. Reading and parsing JSONL transcript format
3. Extracting relevant events and content from transcripts

This class is essential for:
- The MCP server's transcript syncing daemon
- Hook implementations that need access to session context
- Memory extraction workflows

The implementation needs to handle the macOS-specific path format where slashes are converted to dashes.
