The `local-recall/.gitignore` file itself is version-controlled and tracked in git (unlike the index files it specifies). The file should be added to git commits so that all developers get the same gitignore configuration.

Current .gitignore pattern: Ignores index files (`orama-*-index.json`), processed logs (`*-processed-log.jsonl`), and debug logs (`recall.log`).
