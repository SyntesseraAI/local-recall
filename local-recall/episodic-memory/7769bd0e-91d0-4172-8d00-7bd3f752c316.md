All new features must have:
- Corresponding unit tests in tests/unit/
- Integration tests in tests/integration/
- Documentation placed in docs/ folder
- Updates to existing docs when modifying functionality
- Tests covering both happy paths and edge cases

This is a firm requirement for this project per CLAUDE.md.
