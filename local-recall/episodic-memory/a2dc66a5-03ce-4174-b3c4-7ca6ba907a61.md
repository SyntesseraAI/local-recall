Memory storage in Local Recall preserves full message content with role prefixes rather than summarizing. Messages are stored as:
- `**user**:` prefix for user messages
- `**assistant**:` prefix for assistant messages

This approach maintains complete information and allows future sessions to understand the full context of previous conversations.
