The transcript collector uses `findClaudeProjectDir()` to discover the Claude project directory through multiple strategies:
1. Check environment variable `CLAUDE_PROJECT_DIR`
2. Search parent directories for `.claude/` folder containing project settings
3. Search parent directories for `transcripts/` folder containing transcript files

When searching for transcript directories, only files with valid UUID filenames should be considered to avoid matching on other `.jsonl` files that might exist in those folders.
