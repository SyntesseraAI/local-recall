# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with `/` replaced by `-` and a unique session ID appended.

### Example
For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/`
- Transcripts directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-01HQY8jXfJm7zh35BcYYiE1u/transcripts/`
- Transcript files: `*.jsonl` format

## Implementation Details

The `transcript-collector.ts` module implements transcript collection from this location. The path format with dashes is critical for correctly locating and parsing transcript files.

## Reference
- File: `src/core/transcript-collector.ts`
- Used by: MCP server daemon for automatic transcript processing
