# Memory Extraction Output Format Preference

When extracting memories from transcripts, the user prefers:
- **ONLY raw JSON output** - no markdown formatting, no code blocks, no explanation
- **Direct object return** - not wrapped in markdown code blocks
- **Immediate response** - no preamble or additional commentary

This streamlines the memory extraction process and makes the output directly machine-readable.
