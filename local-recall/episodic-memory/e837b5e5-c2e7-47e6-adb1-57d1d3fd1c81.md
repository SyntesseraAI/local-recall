# Claude Haiku for Keyword Extraction

The memory extraction pipeline uses Claude Haiku (`claude -p --model haiku`) to extract keywords from transcripts and user prompts.

## CLI Usage

The CLI accepts text input and returns structured keyword data. This approach leverages Claude's language understanding while using a smaller, faster model for efficiency.

## Benefits

- Fast keyword extraction for real-time prompt processing
- Structured output suitable for memory indexing
- Prompt caching enables efficient batch processing
- Lower latency and cost compared to larger models

## Implementation Notes

When implementing keyword extraction, ensure the Claude CLI is properly invoked with the haiku model specified and prompt caching enabled for batch operations.
