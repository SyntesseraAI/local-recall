The `src/utils/markdown.ts` module provides utilities for:
- Parsing YAML frontmatter from markdown files
- Extracting metadata fields (id, subject, keywords, applies_to, occurred_at, content_hash)
- Serializing memory objects back to markdown format
- Handling markdown content separation from frontmatter

These utilities are essential for reading and writing memory files consistently across the codebase.
