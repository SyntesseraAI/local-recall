# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and prefixed with an underscore.

Example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files (newline-delimited JSON) in the `transcripts/` subdirectory:
- `~/.claude/projects/_Users-joe-Code-Syntessera-local-recall/transcripts/`

Each line in a JSONL file is a complete JSON event object representing a single transcript event (user message, assistant response, tool invocation, etc.).

## Related Code

The `transcript-collector.ts` module in `src/core/` handles reading and processing these JSONL transcript files.
