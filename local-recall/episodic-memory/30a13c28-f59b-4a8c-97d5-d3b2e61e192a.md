The transcript condensing feature (implemented in transcript-collector.ts) summarizes verbose session transcripts before memory extraction to:

- Reduce token usage when calling Claude API for memory extraction
- Remove redundant or debugging-only information
- Preserve essential information about decisions, learnings, and solutions
- Improve quality of extracted memories by filtering noise
- Make transcripts more efficient for processing by the MCP background daemon
