# Cross-Platform Transcript Collection

The transcript collector must handle different transcript storage locations across operating systems:

## macOS
- Location: `~/.claude/projects/`
- Example: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Windows
- Location: `%APPDATA%\Claude\projects\`
- Example: `%APPDATA%\Claude\projects\C_Users_joe_Code_Syntessera_local-recall\transcripts\`

The transcript path is passed via the session hook input (`transcript_path` field), which can be used directly. However, when collecting transcripts from the user's project directory, the code needs to determine the correct system-specific path.

Implementation should:
1. Detect the OS using `process.platform`
2. Resolve home directory using `os.homedir()`
3. Construct the appropriate transcript collection path
4. Handle path separators correctly (/ on macOS/Linux, \\ on Windows)
