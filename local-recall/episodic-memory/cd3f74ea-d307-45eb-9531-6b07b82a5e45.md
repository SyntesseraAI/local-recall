Claude Code stores project transcripts in `~/.claude/projects/` directory on macOS. The project folder name is derived from the full path with slashes replaced by dashes, e.g., `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`. Individual transcripts are stored as JSONL files with naming pattern `transcript-{uuid}.jsonl`.
