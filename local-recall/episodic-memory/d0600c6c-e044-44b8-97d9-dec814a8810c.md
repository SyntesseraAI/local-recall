# MCP Server Background Daemon

## Configuration

The MCP server runs a background daemon that processes transcripts asynchronously.

## Workflow

1. **Sync**: Copies transcripts from `~/.claude/projects/<project>/transcripts/` to local directory
2. **Process**: Uses `claude -p` to extract memories from transcript content
3. **Track**: Maintains content hashes to detect transcript changes
4. **Manage**: Deletes and recreates memories when transcripts are updated
5. **Schedule**: Runs every 5 minutes

## Purpose

Allows automatic memory extraction without relying on the Stop hook, providing asynchronous memory collection that doesn't block the Claude Code session.
