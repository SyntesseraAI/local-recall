# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts are stored as `.jsonl` files

## Location Implementation

This path transformation is critical for the transcript-collector to locate and process Claude's transcripts. The mechanism is implemented in `src/core/transcript-collector.ts`.

## Related

The MCP server daemon processes transcripts from this location every 5 minutes to extract memories.
