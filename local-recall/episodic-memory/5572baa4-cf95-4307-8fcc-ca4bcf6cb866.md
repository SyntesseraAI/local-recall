The memory extractor (`src/core/memory-extractor.ts`) processes transcripts using Claude's API to identify and structure memories. It uses prompt templates from `src/prompts/memory-extraction.ts` that guide Claude to extract actionable memories from session transcripts. The extractor handles transcript condensing, deduplication via content hashes and timestamps, and returns structured memory objects with metadata (subject, keywords, applies_to, occurred_at, content_hash).
