Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (`.jsonl`) where each line is a JSON object representing a single message or event in the conversation.

## Key Files

- Located in: `src/core/transcript-collector.ts`
- Responsible for reading and parsing Claude's transcript cache
- Used by the memory extraction process to identify memories from conversations

## Use Cases

This knowledge is essential for:
1. The MCP server daemon that processes transcripts asynchronously every 5 minutes
2. Memory extraction hooks that parse conversations to create memories
3. Understanding how Claude Code persists conversation history
