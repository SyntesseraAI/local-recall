# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (JSON Lines format) with names like:
- `20250102_120530.jsonl`
- `20250102_144015.jsonl`

Each line in the JSONL file is a complete JSON object representing a single event in the transcript.

## Implementation Details

When implementing transcript collection, use `path.resolve(process.env.HOME, '.claude/projects/', projectFolderName, 'transcripts/')` to construct the correct path.
