# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with the naming pattern `session-{id}.jsonl` where:
- Each line is a JSON object representing an event
- Events have a `type` field (e.g., "user_message", "assistant_response")
- Content is stored in a `content` field

## Implications for Transcript Collection

When building transcript collection functionality:
1. Use `~/.claude/projects/` as the base directory
2. Convert absolute paths to the dash-separated format
3. Read from the `transcripts/` subdirectory
4. Parse JSONL files line by line
5. Handle the specific event structure Claude uses
