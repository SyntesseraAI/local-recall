## Claude Projects Directory Location

Claude Code stores project transcripts in a specific directory structure:

- **Base path**: `~/.claude/projects/`
- **Project folder naming**: Uses the full path of the project with slashes replaced by hyphens
- **Example**: Project at `/Users/joe/Code/Syntessera/local-recall` maps to `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- **Transcripts location**: Within each project folder: `transcripts/` subdirectory
- **File format**: Transcripts are stored as `.jsonl` files (JSON Lines format)
- **File naming**: Uses `transcript-<timestamp>.jsonl` format

This is crucial for the transcript-collector component to locate and process session transcripts for memory extraction.
