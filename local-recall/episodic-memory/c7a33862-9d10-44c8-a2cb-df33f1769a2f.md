# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with timestamps:
- File naming: `transcript_<ISO-8601-timestamp>.jsonl`
- Example: `transcript_2024-12-19T10-30-45.jsonl`
- Each line in the file is a valid JSON object representing a message event

## Usage Context

This information is critical for the transcript-collector component which needs to:
1. Locate Claude's cached transcripts
2. Parse transcript files to extract information
3. Monitor for new or updated transcripts

The transcript collector can use this path to sync transcripts and extract memories asynchronously.
