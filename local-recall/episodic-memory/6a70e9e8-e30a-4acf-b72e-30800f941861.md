Local Recall enables persistent memory storage across AI coding sessions. Memories are stored as markdown files with YAML frontmatter in the `local-recall/episodic-memory/` directory. The system supports semantic search through vector embeddings using the BGE-small-en-v1.5 model via fastembed.
