The project root is the plugin root. Plugin structure requires:
- .claude-plugin/plugin.json (plugin metadata: name, version, description)
- hooks.json (hook event handlers using ${CLAUDE_PLUGIN_ROOT})
- .mcp.json (MCP server configuration)
- scripts/ directory (bundled executables from build output, gitignored)

Install via .claude/settings.json plugins section with absolute path.
