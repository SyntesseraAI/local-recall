Local Recall is a markdown-powered memory system for AI coding assistants. Key architectural components:

- **Memory Storage**: YAML frontmatter markdown files in `local-recall/episodic-memory/` (version-controlled)
- **Vector Store**: SQLite with sqlite-vec extension using BGE-small-en-v1.5 embeddings (~133MB model)
- **Core Components**:
  - `src/core/memory.ts` - CRD operations for memory files
  - `src/core/vector-store.ts` - Semantic search with cosine distance scoring
  - `src/core/embedding.ts` - Embedding generation via fastembed
  - `src/hooks/` - Claude Code integration hooks (SessionStart, UserPromptSubmit)
  - `src/mcp-server/` - Exposes memory tools via MCP protocol
  - `src/core/transcript-collector.ts`, `transcript-condenser.ts`, `memory-extractor.ts` - Transcript processing

- **Memory File Format**: Markdown with YAML metadata (id, subject, keywords, applies_to, occurred_at, content_hash)
- **Build Output**: TypeScript compiled to `dist/` (gitignored)
- **Model Cache**: `local_cache/` stores embedding model (gitignored)
- **Database**: `local-recall/memory.sqlite` (gitignored, auto-generated)
