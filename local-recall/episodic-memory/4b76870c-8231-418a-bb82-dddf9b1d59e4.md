# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) with numbered filenames:
- Format: `transcript_{number}.jsonl`
- Example: `transcript_1.jsonl`, `transcript_2.jsonl`
- Each line is a JSON object representing a single message or event in the session

## Key Details

- Transcripts are location-sensitive: the exact absolute path matters for folder naming
- `.jsonl` format means one JSON object per line, not a single JSON array
- This is useful for streaming and incremental processing
- Transcripts are stored in the user's home directory cache, not in the project repository
