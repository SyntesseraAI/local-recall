Local Recall provides:
- Structured markdown-based memory storage in `local-recall/episodic-memory/`
- YAML frontmatter for memory metadata (id, subject, keywords, applies_to, occurred_at, content_hash)
- Memory index system for fast keyword searching
- Claude Code hooks for SessionStart, UserPromptSubmit, and Stop events
- MCP server exposing memory tools (create, delete, search, list, index_rebuild)
- Uses fuzzy search for finding relevant memories
- Supports semantic search with fastembed BGE-small-en-v1.5 embedding model (~133MB)
