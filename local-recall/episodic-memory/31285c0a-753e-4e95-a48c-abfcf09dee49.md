Memory files are stored as markdown with YAML frontmatter containing: `id` (UUID), `subject` (brief description), `keywords` (searchable array), `applies_to` (scope: global/file:<path>/area:<name>), `occurred_at` (ISO-8601 timestamp), and `content_hash` (SHA-256 prefix). This structure enables deduplication, scoping, and efficient searching.
