The MCP server runs a background daemon that processes Claude transcripts asynchronously every 5 minutes. It syncs transcripts from `~/.claude/projects/<project>/transcripts/`, uses content hashes to detect changes, and extracts memories using `claude -p`. The Stop hook is disabled in favor of this asynchronous approach.
