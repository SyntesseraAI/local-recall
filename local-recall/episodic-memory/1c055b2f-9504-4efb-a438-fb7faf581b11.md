# Claude Project Transcripts Location and Format

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a separate JSON object representing a conversation event.

## Implementation Details

The `transcript-collector.ts` module handles reading and parsing these transcripts for memory extraction purposes. Future memory extraction needs to account for this file format and location structure.
