# Transcript JSONL Format

Claude's transcript files use JSONL (JSON Lines) format where each line is a separate JSON object representing an event.

## Typical Event Structure

Transcript events include metadata like:
- Session ID
- User interaction type (e.g., "message", "tool_use")
- Timestamp information
- Content (user messages, assistant responses, tool invocations)

This line-delimited format allows efficient streaming and incremental processing of large transcripts.
