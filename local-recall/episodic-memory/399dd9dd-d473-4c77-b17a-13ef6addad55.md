# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Structure

Transcripts are stored as JSONL files (one JSON object per line):
- Each session has its own transcript file
- Files are named with timestamps or session identifiers
- Format: One JSON object per line (JSONL format)

## Implications for Transcript Collection

When implementing transcript collection:
1. Use the absolute path from Claude's context to construct the project folder name (replace `/` with `-`)
2. Look for JSONL files in the transcripts subdirectory
3. Parse line-by-line as JSON objects
4. Handle multiple transcript files from different sessions

This path computation is essential for the `transcript-collector.ts` implementation.
