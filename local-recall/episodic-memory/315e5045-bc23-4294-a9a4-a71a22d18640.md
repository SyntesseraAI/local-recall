Claude Code stores project transcripts at `~/.claude/projects/<project>/transcripts/` on macOS. This path is used by the MCP server daemon to sync and process transcripts asynchronously. The daemon tracks processed transcripts with content hashes to detect changes and avoid reprocessing.
