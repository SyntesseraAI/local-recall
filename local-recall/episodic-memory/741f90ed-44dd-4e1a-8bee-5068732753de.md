Local Recall can be installed as npm package or cloned locally. Plugin uses ${CLAUDE_PLUGIN_ROOT} variable for hook paths. Alternative direct hook configuration in .claude/settings.json available. MCP server exposed via command/args format with LOCAL_RECALL_DIR environment variable. Both npm package and local installation patterns documented.
