# Transcript Collector Implementation

The transcript collector in `src/core/transcript-collector.ts` finds Claude Code transcripts by:

1. Using the working directory to construct the Claude projects folder path
2. On macOS, the path format is: `~/.claude/projects/{projectNameWithDashes}/transcripts/`
3. Using glob pattern `**/*.jsonl` to find all transcript files
4. Processing each transcript as newline-delimited JSON

This direct file system approach allows the collector to access and process Claude Code transcripts without relying on Claude's APIs.
