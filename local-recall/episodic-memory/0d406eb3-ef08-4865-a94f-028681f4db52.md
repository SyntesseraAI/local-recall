# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are stored as JSONL format within this directory.

## Relevance

This is important for the transcript-collector component which needs to locate and process Claude Code session transcripts for memory extraction.
