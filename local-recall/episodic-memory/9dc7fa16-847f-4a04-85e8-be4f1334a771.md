# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `transcripts/` subdirectory
- File format: `.jsonl` (JSON Lines)

## Transcript Structure

Each transcript contains condensed session events:
- `[User]` - User requests or questions
- `[Assistant]` - Claude's responses and explanations
- `[Tool: Name]` - Tool invocations with parameters
- `[Result: OK/ERROR]` - Tool outcomes

This structure is useful for programmatically parsing transcripts to extract meaningful memories about what was accomplished during sessions.
