# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are stored in JSONL format with `.jsonl` extension.

## Implementation in transcript-collector.ts

The `transcript-collector.ts` file handles:
1. Converting the working directory path to the Claude projects folder name (replace `/` with `-`)
2. Locating the transcripts directory
3. Reading and parsing JSONL transcript files
4. Extracting session information and events from transcripts

This is critical for the MCP server daemon which needs to find and process transcripts every 5 minutes.
