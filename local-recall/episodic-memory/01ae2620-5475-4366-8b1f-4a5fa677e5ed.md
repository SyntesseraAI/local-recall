Memory extraction output must return a valid JSON object with a `memories` array. Each memory object must include:

- `subject`: Brief one-line description (max 200 chars)
- `keywords`: Array of 1-10 searchable keywords (lowercase, specific)
- `applies_to`: Scope string - either `global`, `file:<path>`, or `area:<name>`
- `content`: Detailed memory content in markdown format

If no valuable memories exist, return `{ "memories": [] }`. Output should be raw JSON with no markdown formatting or code blocks.
