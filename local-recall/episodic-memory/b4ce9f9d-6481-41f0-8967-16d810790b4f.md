The function `parseTranscriptForMemories(rawContent: string): MemorySuggestion[]` (lines 377-413) is the primary interface for extracting memories from transcript files:

1. Takes raw JSONL string as input
2. Parses each line into transcript entries
3. Filters for user/assistant messages only
4. Extracts thinking and text from content blocks
5. Calls `analyzeForMemories()` to apply filtering rules
6. Returns array of memory suggestions ready to save

This is what `stop.ts` should call instead of reimplementing parsing.
