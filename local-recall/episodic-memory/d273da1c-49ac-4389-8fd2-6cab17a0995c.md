Local Recall can be configured through environment variables or a `.local-recall.json` file in the project root. Available options: `memoryDir` (default: `./local-recall`) - directory for memory storage; `maxMemories` (default: 1000) - maximum number of memories to retain. Configuration is handled by `src/utils/config.ts`.
