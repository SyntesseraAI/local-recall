## MCP Tool Design

### User Preference
Removed `memory_update` and `memory_delete` from MCP tools.

### Rationale
- Memories are immutable episodic records
- Users can manually remove memories by deleting files
- AI assistants should not have delete/update capabilities

### Available Tools (after changes)
- `memory_create` - Create new memory (idempotent)
- `memory_get` - Retrieve memory by ID
- `memory_list` - List all memories with optional filter
- `memory_search` - Semantic search

This enforces the immutable memory model at the API level.
