# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the project's working directory path:
- The path is converted to base64 encoding
- Special characters in the base64 string are replaced: `+` → `-`, `/` → `_`
- This creates a unique, filesystem-safe identifier for each project

Example:
- Project directory: `/Users/joe/Code/Syntessera/local-recall`
- Encoded folder name: `L1VzZXJzL2pvZS9Db2RlL1N5bnRlc3NlcmEvbG9jYWwtcmVjYWxs` (with base64 replacements)

## Transcript Files

Transcript files are stored in JSONL format at:
`~/.claude/projects/<encoded-path>/transcripts/<timestamp>.jsonl`

Each line contains a JSON event object representing a step in the conversation.

## Implementation Notes

When building transcript collection systems, account for this encoding scheme to properly map local project directories to their Claude transcript storage locations.
