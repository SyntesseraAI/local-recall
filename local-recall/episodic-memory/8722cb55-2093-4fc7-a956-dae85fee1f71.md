If error 'Tokenizer file not found at local_cache/fast-bge-small-en-v1.5/tokenizer.json' occurs, cache is corrupted (usually interrupted download). Fix: `rm -rf local_cache/fast-bge-small-en-v1.5*`. Model re-downloads automatically on next run.
