# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Storage

Transcripts are stored as JSONL files in the project directory at `transcripts/` subdirectory:
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Format: `.jsonl` (JSON Lines - one JSON object per line)
- Each line represents a transcript event

## Implications for Transcript Collection

When implementing transcript collection for the Local Recall memory system, the transcript collector needs to:
1. Determine the project folder name by converting the current working directory path
2. Access the `~/.claude/projects/[project-folder]/transcripts/` directory
3. Parse `.jsonl` files to extract transcript data
4. Handle the case where transcripts may not exist or the directory structure may vary

This pattern is specific to macOS; Windows paths may use different separators and the claude cache location is different (`%APPDATA%\Claude`).
