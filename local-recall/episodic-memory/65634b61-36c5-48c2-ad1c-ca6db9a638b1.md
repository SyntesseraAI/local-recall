Memories are deduplicated using two fields:
- `occurred_at`: ISO-8601 timestamp of when the original event occurred
- `content_hash`: SHA-256 hash prefix (16 characters) of the memory content

The `findDuplicate(occurredAt, contentHash)` function checks for existing memories with matching hash and timestamp, preventing duplicate memory creation. This allows transcript reprocessing without creating duplicate memories.
