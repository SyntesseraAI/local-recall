Local Recall uses the `fastembed` library with BGE-small-en-v1.5 model (~133MB) for semantic search. Key details:

- Model automatically downloads to `local_cache/` on first use
- First startup takes 30-60 seconds (model download only)
- Subsequent runs load from cache
- Configuration options:
  - `indexRefreshInterval` - Seconds between index refreshes (default 300)
  - `fuzzyThreshold` - Minimum fuzzy match score 0-1 (default 0.6)

Troubleshooting:
- If "Tokenizer file not found" error occurs, corrupted cache is likely cause
- Fix: Remove `local_cache/fast-bge-small-en-v1.5*` directory
- Model will re-download automatically on next run
