# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory:
- Replace `/` with `-` in the path
- Store transcripts as JSONL files in `transcripts/` subdirectory
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Integration

This path format is used by `src/core/transcript-collector.ts` to sync and process Claude Code transcripts for memory extraction.

## Practical Location

For the local-recall project specifically:
`~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
