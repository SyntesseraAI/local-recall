# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using URL encoding:
- Forward slashes (`/`) become underscores (`_`)
- Colons (`:`) become hyphens (`-`)
- Other special characters use percent-encoding

## Example

Working directory: `/Users/joe/Code/Syntessera/local-recall`
Encoded path: `_Users_joe_Code_Syntessera_local-recall`
Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcript files are named with ISO timestamps and have `.jsonl` extension:
- Format: `YYYY-MM-DDTHH-mm-ss.jsonl`
- Example: `2025-01-15T10-30-45.jsonl`

Each line in the JSONL file is a complete JSON object representing a transcript event.

## Key Implementation Details

- The transcript collector needs to handle the path encoding when locating transcripts
- Transcripts can be found by iterating through the encoded project directory
- The timestamp format helps with sorting and identifying recent transcripts
