The memory extractor in `memory-extractor.ts` uses Claude Haiku via the `claude` command-line tool to analyze transcripts and extract memories. It sends a structured prompt defined in `src/prompts/memory-extraction.ts` to Haiku for processing. The extractor handles parsing the JSON response from Claude and manages error handling for API failures or parsing errors.
