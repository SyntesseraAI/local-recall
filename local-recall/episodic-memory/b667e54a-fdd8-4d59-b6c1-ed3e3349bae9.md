# Transcript Collector Path Mapping

The transcript-collector module must map from the current working directory to Claude's project transcript location.

## Mapping Logic

1. Get the current working directory (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replace forward slashes with dashes to get the project folder name
3. Look in `~/.claude/projects/<project-folder-name>/transcripts/` for JSONL transcript files

Example mapping:
- CWD: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This is essential for locating session transcripts to process for memory extraction.
