The vector store implementation:
- Uses `better-sqlite3` with `sqlite-vec` extension for vector similarity
- Generates embeddings using `fastembed` with BGE-small-en-v1.5 model (~133MB)
- Model automatically downloads to `local_cache/` on first use
- Methods: `initialize()`, `add(memory)`, `remove(id)`, `search(query, options)`, `sync(memories)`
- First run may take 30-60 seconds for model download; subsequent runs load from cache
