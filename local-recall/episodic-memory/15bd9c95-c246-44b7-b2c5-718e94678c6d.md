# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), with filenames based on session timestamps:
- Format: `{timestamp}.jsonl`
- Location: `~/.claude/projects/{project-folder}/transcripts/`

## Events in Transcript

Each line in a transcript JSONL file contains an event object with:
- `type`: Type of event (e.g., 'user', 'assistant', 'tool')
- `timestamp`: ISO-8601 timestamp
- `content`: Event content (varies by type)
- Additional fields depending on event type

## Usage in Local Recall

The transcript-collector syncs these transcripts to extract memories via the MCP daemon (runs every 5 minutes). The daemon processes transcript changes and creates/deletes memories based on content hashes.
