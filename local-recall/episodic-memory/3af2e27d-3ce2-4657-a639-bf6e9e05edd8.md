# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory `/Users/joe/Code/Syntessera/local-recall` becomes folder name `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line).

This information is useful for the `transcript-collector.ts` which syncs transcripts from Claude's cache.
