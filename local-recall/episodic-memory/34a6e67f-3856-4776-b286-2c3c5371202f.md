MCP server provides tools: memory_create, memory_get, memory_search, memory_list, and index_rebuild. Runs a background daemon every 5 minutes that syncs transcripts from Claude's cache, processes them using 'claude -p', tracks processed transcripts with content hashes, and manages memory lifecycle.
