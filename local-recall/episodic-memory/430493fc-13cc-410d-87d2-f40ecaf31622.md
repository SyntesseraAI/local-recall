Configuration options:
- `LOCAL_RECALL_DIR`: Directory for memory storage (default: ./local-recall)
- `maxMemories`: Maximum number of memories (default: 1000)
- `memoryDir`: Can also be set in .local-recall.json config file

The MCP server expects LOCAL_RECALL_DIR environment variable to locate the memory directory.
