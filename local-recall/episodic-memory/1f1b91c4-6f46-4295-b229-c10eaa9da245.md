Vector store has been migrated from SQLite with sqlite-vec extension to Orama (pure JavaScript). The src/core/vector-store.ts now uses Orama instead of sqlite-vec, eliminating SQLite mutex issues. However, this doesn't solve the ONNX runtime mutex problem from fastembed.
