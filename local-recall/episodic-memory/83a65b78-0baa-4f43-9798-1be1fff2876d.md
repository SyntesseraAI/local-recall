# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory by replacing slashes with underscores.

**Example:** For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `_Users_joe_Code_Syntessera_local-recall`
- Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`
- Each transcript: `<session-id>.jsonl` format

## Integration with Local Recall

The `TranscriptCollector` in `src/core/transcript-collector.ts` uses this knowledge to:
1. Construct the correct path to Claude's transcript cache
2. Load JSONL-formatted transcripts from sessions
3. Process them for memory extraction in the MCP daemon

This is critical for the background transcript daemon that runs every 5 minutes to extract memories from active Claude Code sessions.
