Each memory is stored as a markdown file with YAML frontmatter containing: id (UUID), subject (brief description), keywords (array), applies_to (scope), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix). These fields enable deduplication, searchability, and proper memory organization.
