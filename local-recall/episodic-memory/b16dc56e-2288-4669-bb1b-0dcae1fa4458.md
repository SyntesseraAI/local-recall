# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with names like `transcript-<id>.jsonl`. Each line is a JSON object representing one message event.

## Important for Transcript Collection

When building transcript collectors, remember:
1. The project name transformation (replace `/` with `-`)
2. Transcripts are stored in a `transcripts/` subdirectory
3. Files use the `.jsonl` format (one JSON object per line)
4. This path is consistent across sessions for the same project

This is crucial for the MCP server's background daemon which syncs transcripts to extract memories.
