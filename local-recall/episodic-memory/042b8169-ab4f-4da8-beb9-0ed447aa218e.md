# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` becomes `Users_joe_Code_Syntessera_local-recall`
- Full transcript directory: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line. This is the format that the transcript-collector needs to parse.

## Integration Notes

The transcript-collector needs to:
1. Expand `~` to the user's home directory
2. Build the path with underscores replacing slashes
3. Look for `.jsonl` files in the transcripts directory
4. Parse each line as a separate JSON object

This is critical for the MCP server daemon to correctly locate and process transcripts.
