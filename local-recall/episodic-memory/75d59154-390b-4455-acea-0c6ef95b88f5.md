Important for understanding persistence model:
- Memory markdown files in `local-recall/episodic-memory/*.md` ARE tracked in git (can be committed)
- `local-recall/index.json` is gitignored (auto-generated keyword index)
- `local-recall/recall.log` is gitignored (debug log output)
- `.gitignore` in local-recall/ is auto-generated

This allows memories to be shared among team members while keeping generated artifacts and logs local.
