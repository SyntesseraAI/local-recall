The `local-recall/` directory contains auto-generated files that are gitignored:
- `index.json` - The keyword index cache used for fast memory searches
- `recall.log` - Debug log file written by the logger utility
- `.gitignore` - Excludes both index.json and recall.log from version control

Only the episodic-memory markdown files are tracked in git, not the generated index or logs.
