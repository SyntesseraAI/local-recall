# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Becomes folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

- Stored as `.jsonl` (JSON Lines) format
- One JSON object per line
- Named with timestamps: `transcript-YYYYMMDD-HHMMSS.jsonl`
- Can be accessed via `transcript_path` in hook inputs

This convention is important for the transcript-collector and MCP server to locate and process transcripts correctly.
