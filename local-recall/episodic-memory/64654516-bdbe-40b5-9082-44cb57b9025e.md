# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

Project folder names replace forward slashes with dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Format: `transcript-[timestamp].jsonl`
- Example: `transcript-20250101-120000.jsonl`
- Each line contains a JSON event from the session

## Relevance

This is crucial for the MCP server's background daemon which syncs transcripts from Claude's cache to extract memories asynchronously every 5 minutes.
