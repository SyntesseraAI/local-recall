Integration mechanisms:
1. **Hooks**: Configured in `hooks.json` to execute as shell commands receiving JSON via stdin
   - SessionStart: Loads memory index on session begin
   - UserPromptSubmit: Searches memories before user prompt processing
   - Stop: Processes transcripts (currently disabled, handled by MCP daemon)
2. **MCP Server**: Exposes memory tools (memory_create, memory_delete, memory_search, memory_list, index_rebuild)
3. **Background Daemon**: Syncs transcripts from `~/.claude/projects/<project>/transcripts/`, extracts memories using Claude Haiku, and tracks changes with content hashes
