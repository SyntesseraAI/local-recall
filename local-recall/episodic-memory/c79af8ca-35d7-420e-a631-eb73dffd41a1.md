# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path to the project with slashes replaced by dashes. For example:

- Project at `/Users/joe/Code/Syntessera/local-recall`
- Transcripts at `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files (one JSON object per line), with each line representing a transcript event.

## Usage in local-recall

The `transcript-collector.ts` module uses this to:
1. Construct the path to Claude's transcript cache
2. Read JSONL files containing session transcripts
3. Extract memories from transcript content using semantic analysis
