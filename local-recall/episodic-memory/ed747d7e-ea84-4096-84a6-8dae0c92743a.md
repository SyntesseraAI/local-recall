# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes, and colons removed.

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension.

Each line is a complete JSON object representing a turn in the conversation:
- Contains message content, timestamps, and metadata
- Can be parsed line-by-line for streaming or batch processing
- Useful for transcript analysis and memory extraction

## Implications

When implementing transcript collection or processing, account for:
- Path transformation logic (replace `/` with `-`, remove `:`)
- JSONL format parsing (parse line-by-line, not as single JSON document)
- Multiple transcript files may exist in the same project directory
