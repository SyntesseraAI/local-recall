Claude Code plugins require version synchronization across three files:
- `package.json` - npm package version
- `.claude-plugin/plugin.json` - plugin manifest
- `.claude-plugin/marketplace.json` - marketplace distribution metadata

All three must be kept in sync when bumping versions. Missing any of these can cause version mismatch issues in marketplace distribution.
