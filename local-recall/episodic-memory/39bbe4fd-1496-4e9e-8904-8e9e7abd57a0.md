# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with names like `transcript-<timestamp>.jsonl`.

Each line is a JSON object representing a single event in the transcript:
- User messages
- Assistant responses
- Tool invocations
- Tool results

## Relevance

This is important for the transcript collector feature which needs to:
1. Locate project transcripts
2. Parse JSONL format
3. Extract meaningful events
4. Identify memory-worthy content

The transcript collector reads from this location to build memories for the local-recall system.
