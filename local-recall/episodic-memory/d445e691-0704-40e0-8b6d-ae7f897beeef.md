# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the repository path, replacing slashes with dashes and removing special characters. For example:

- Repository: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line. Each line contains a JSON object representing a transcript event.

## Implementation Details

When collecting transcripts from Claude's cache, the path must be constructed by:
1. Taking the current working directory (received via stdin in hook input)
2. Converting it to the standardized project folder name format (replace `/` with `-`)
3. Looking for transcripts in `~/.claude/projects/<project-folder>/transcripts/`

This is important for the transcript-collector to find and process transcripts correctly in the background daemon.
