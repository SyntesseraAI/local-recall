# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by underscores:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a separate JSON object representing a message or event in the conversation.

## Discovery Context

This was discovered while implementing `transcript-collector.ts` to automatically locate and process Claude transcripts for memory extraction. The path format is consistent and predictable, allowing the tool to programmatically build the correct path given a working directory.

## Implementation Detail

The transcript collector uses this path to find transcripts without requiring user configuration. This enables automatic memory extraction from ongoing Claude Code sessions.
