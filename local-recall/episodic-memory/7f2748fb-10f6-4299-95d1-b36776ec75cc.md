# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Directory Naming

Project directories are named using the absolute path with forward slashes converted to dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript Files

Transcripts are stored as JSONL files within each project directory:
- File format: `.jsonl` (JSON Lines - one JSON object per line)
- Located in: `~/.claude/projects/{project-dir-name}/transcripts/`

## Important Implementation Detail

When implementing transcript collection, handle the path-to-directory-name conversion correctly: replace `/` with `-` and handle the leading slash appropriately (no leading dash in the directory name).
