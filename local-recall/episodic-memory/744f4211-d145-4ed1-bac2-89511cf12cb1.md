Memory creation is idempotent - `createMemory()` returns existing memories if duplicates are detected. Deduplication checks use two fields: `occurred_at` (ISO-8601 timestamp when the original event occurred) and `content_hash` (SHA-256 prefix of content). The `findDuplicate()` function checks for existing memories using these identifiers to prevent duplicate memory files.
