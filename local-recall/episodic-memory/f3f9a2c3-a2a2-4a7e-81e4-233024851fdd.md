The memory-extractor.ts uses Claude Haiku 4.5 model for transcript analysis and memory extraction. This is a deliberate choice for cost-efficiency when processing transcripts. The `extractMemoriesFromTranscript` function uses this model to identify and extract valuable memories from session transcripts.
