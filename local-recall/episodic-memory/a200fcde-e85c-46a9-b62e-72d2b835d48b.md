# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall`
- Transcripts directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a separate JSON object representing an event in the transcript.

## Event Structure

Each line in a JSONL transcript file contains a JSON object with:
- `type`: Event type (e.g., "user_message", "assistant_message", "tool_call", etc.)
- `content`: The actual event data
- `timestamp`: ISO-8601 timestamp of when the event occurred

This information is crucial for the transcript collector and memory extraction systems to properly locate and parse Claude Code session transcripts.
