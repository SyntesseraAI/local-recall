# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored in JSONL format with one JSON object per line:
- File naming: `transcript_<timestamp>.jsonl`
- Each line contains a complete message event with `role` and `content` fields
- Transcripts accumulate over time within a project folder

## Usage in Memory Extraction

The transcript collector should:
1. Use the `cwd` to determine the project folder name (convert `/` to `-`)
2. Read all JSONL files in the transcripts directory
3. Parse each line as a separate JSON object
4. Extract memory-relevant information from `content` field
