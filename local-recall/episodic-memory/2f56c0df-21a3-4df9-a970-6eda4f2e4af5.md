# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with:
- File extension: `.jsonl`
- Each line contains a JSON object representing a transcript entry
- Files follow naming pattern: `session-{timestamp}.jsonl`

## Usage in Local Recall

The transcript collector uses this path structure to:
1. Locate transcript files for the current project
2. Parse transcript entries to extract memory-worthy information
3. Track which transcripts have been processed using content hashes

This is critical for the MCP server's background daemon that syncs and processes transcripts every 5 minutes.
