# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a complete JSON object representing a single tool invocation or message event.

## Key Details

- Folder name uses dashes to replace slashes in the absolute path
- Transcripts accumulate over multiple sessions
- Each line is a separate JSON object (not an array)
- The transcript-collector in local-recall needs to parse this format correctly
