Local Recall can be configured via:
1. Environment variables
2. `.local-recall.json` file in project root

Key options:
- `memoryDir` (default: `./local-recall`) - Directory for memory storage
- `maxMemories` (default: `1000`) - Maximum number of memories to retain

The `LOCAL_RECALL_DIR` environment variable is used by MCP server and hooks to locate the memory directory.
