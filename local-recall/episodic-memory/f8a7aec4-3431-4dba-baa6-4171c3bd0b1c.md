# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `-Users-joe-Code-Syntessera-local-recall`

## Storage Structure

Each project has a `transcripts/` folder containing JSONL files:
- `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files are in JSONL format (one JSON object per line)

## Accessing Transcripts

The `src/core/transcript-collector.ts` file reads transcripts from this location to extract memories.

This is the canonical source for finding session transcripts when implementing memory extraction or analysis features.
