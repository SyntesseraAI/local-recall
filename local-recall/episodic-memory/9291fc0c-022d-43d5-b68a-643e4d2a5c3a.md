# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files. Each session creates files like:
- `<session-id>.jsonl`

Each line is a complete JSON object representing a message or tool invocation in the conversation.

## Relevance to Local Recall

The MCP server daemon syncs transcripts from this location as a source for memory extraction. Understanding this path structure is critical for implementing transcript-collector functionality and ensuring the daemon can locate and process transcripts correctly.
