# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line contains a JSON event object representing a message or interaction in the session.

## Implementation Notes

The `transcript-collector.ts` module is responsible for discovering and reading these transcript files. When building the transcript path, it should:
1. Get the absolute working directory path
2. Replace all slashes with dashes
3. Look in `~/.claude/projects/{path}/transcripts/`
