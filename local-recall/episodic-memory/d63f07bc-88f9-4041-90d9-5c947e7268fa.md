# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (newline-delimited JSON). Each line is a separate JSON object representing a message or tool invocation in the session.

## Usage

This path is used by the MCP background daemon to:
1. Monitor for new transcript files
2. Extract memories from completed sessions
3. Track processed transcripts with content hashes
4. Regenerate memories when transcripts change

## Related Code

The transcript path is referenced in:
- `src/mcp-server/server.ts` - Background daemon that monitors and processes transcripts
- Hook scripts receive transcript path via stdin for real-time memory extraction
