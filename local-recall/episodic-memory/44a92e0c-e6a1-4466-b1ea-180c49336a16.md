## MCP Tool Changes

The MCP server tools were refactored to improve clarity and add thinking memory support:

### Removed Tools
- `index_rebuild` - Removed to prevent excessive LLM power
- `memory_list` - Removed to limit LLM capabilities

### Renamed Tools
All episodic memory tools now use the `episodic_` prefix:
- `episodic_create` - Create episodic memories
- `episodic_get` - Retrieve specific episodic memory by ID
- `episodic_search` - Semantic search of episodic memories

### New Thinking Memory Tools
Added parallel tools for thinking memories:
- `thinking_get` - Retrieve specific thinking memory by ID
- `thinking_search` - Semantic search of thinking memories

**Location**: `src/mcp-server/tools.ts`

This change clearly delineates memory types and prevents the LLM from having access to potentially dangerous operations like index rebuilding or listing all memories.
