Session transcripts are condensed into a standardized format with event notation:
- `[User]` - User's questions or requests
- `[Assistant]` - Claude's responses or explanations
- `[Tool: Name]` - Tool invocations (Read, Edit, Write, Bash, Grep, etc.)
- `[Result: OK/ERROR]` - Outcome of tool invocations

This format makes it easier for Claude to analyze and extract memories from the session history.
