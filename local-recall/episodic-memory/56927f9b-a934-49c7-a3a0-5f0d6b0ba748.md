Added new configuration options:
- LOCAL_RECALL_THINKING_MAX_TOKENS: Maximum tokens of thinking memories to inject (default: 1000)
- LOCAL_RECALL_THINKING_MIN_SIMILARITY: Minimum similarity threshold 0.0-1.0 (default: 0.8)
- LOCAL_RECALL_EPISODIC_MAX_TOKENS: Maximum tokens of episodic memories to inject (default: 1000)
- LOCAL_RECALL_EPISODIC_MIN_SIMILARITY: Minimum similarity threshold for episodic (default: 0.8)
Both memory types now use token-based retrieval with configurable similarity filtering.
