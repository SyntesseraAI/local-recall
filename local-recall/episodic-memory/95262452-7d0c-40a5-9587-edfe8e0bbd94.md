# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project at `/Users/joe/Code/Syntessera/local-recall` â†’ folder name `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a separate JSON object representing a transcript event.

## Relevance

This is critical for the `transcript-collector.ts` module which needs to locate and process Claude Code session transcripts for memory extraction.
