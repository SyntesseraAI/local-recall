# Fuzzy Search Threshold

Local Recall supports a configurable fuzzy search threshold via:
- Environment variable: `FUZZY_THRESHOLD`
- Configuration file: `.local-recall.json`
- Default value: `0.6` (60% match confidence)

## Range

Accepts values between 0 and 1:
- Lower values: More permissive matching (more false positives)
- Higher values: Stricter matching (may miss some results)
- Default 0.6 is a reasonable balance

This is used by the search implementation in `src/core/search.ts`.
