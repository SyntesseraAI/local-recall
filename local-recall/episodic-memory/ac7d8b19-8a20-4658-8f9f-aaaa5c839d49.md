# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with forward slashes converted to dashes. For example:

- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcript location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a separate JSON object representing an event in the transcript.

## Relevance

The transcript-collector module needs to account for this path transformation when discovering and processing transcripts. The dashe conversion is critical for correctly identifying and accessing the right transcript files.
