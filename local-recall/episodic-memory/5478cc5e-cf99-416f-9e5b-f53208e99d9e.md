# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with the naming pattern `{timestamp}.jsonl`, where timestamp is the session start time in ISO-8601 format.

## Use Cases

This location is important for:
- The transcript-collector module to locate and process transcripts
- Building the memory extraction daemon that monitors transcript changes
- Understanding Claude's internal storage structure for integration purposes
