Local Recall integrates with Claude Code via three hooks:
1. **SessionStart**: Loads memory index and retrieves relevant memories on session start
2. **UserPromptSubmit**: Searches memories for matches before Claude processes user prompts
3. **Stop**: Processes transcripts asynchronously via MCP server daemon (every 5 minutes)

Hooks are configured in `hooks.json` and bundled scripts are in `scripts/hooks/`
