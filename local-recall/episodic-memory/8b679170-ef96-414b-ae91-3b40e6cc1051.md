# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes.

Example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/latest.jsonl`

## File Format

Transcripts are stored as JSONL (JSON Lines) where each line is a complete JSON object representing a transcript event.

## Relevance

This is important for the `transcript-collector.ts` module which needs to locate and read Claude's transcript files for memory extraction.
