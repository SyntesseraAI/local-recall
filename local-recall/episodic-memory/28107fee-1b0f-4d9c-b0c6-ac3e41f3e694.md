# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

The project folder name is the absolute path with slashes replaced by dashes:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) format:
- One JSON object per line
- Each object represents a transcript event
- Can be parsed line-by-line for processing

## Relevance

This is critical for the transcript-collector and MCP server daemon which sync transcripts from Claude's cache to extract memories.
