Changed the default value of `episodicEnabled` from `true` to `false` in the schema. This means episodic memory retrieval is now disabled by default.

Users can enable it explicitly by:
- Setting environment variable `LOCAL_RECALL_EPISODIC_ENABLED=true`
- Adding `"episodicEnabled": true` to `.local-recall.json` config file
