Each memory file must have frontmatter with:
- `id`: UUID
- `subject`: Brief one-line description
- `keywords`: Array of searchable keywords
- `applies_to`: Scope (global, file:<path>, or area:<name>)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 hash prefix (16 chars)

Followed by markdown content starting with # Content header.
