# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Claude cache folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts are stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), where each line is a complete JSON object representing a transcript event.
