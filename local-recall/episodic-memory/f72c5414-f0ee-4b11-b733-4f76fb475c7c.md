Local Recall can be installed in Claude Code in three ways:

1. **As a plugin** - Add to `.claude/settings.json` with plugin path pointing to local-recall directory
2. **As npm package hooks** - Reference `./node_modules/local-recall/scripts/hooks/*` directly
3. **As local project** - Reference `./scripts/hooks/*` when developing locally

**MCP Server configuration:**
- For npm package: `"command": "node", "args": ["./node_modules/local-recall/scripts/mcp-server/server.js"]`
- For local development: `"command": "node", "args": ["./scripts/mcp-server/server.js"]`
- Requires `LOCAL_RECALL_DIR` environment variable pointing to memory storage directory

**Important:** Hooks use `${CLAUDE_PLUGIN_ROOT}` for plugin installations and receive `cwd` via stdin to operate on user's project directory.
