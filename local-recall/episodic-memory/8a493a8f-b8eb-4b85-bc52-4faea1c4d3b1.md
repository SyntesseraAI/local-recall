# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Each project folder contains:
- `transcripts/` - Directory with JSONL transcript files
- Each JSONL file contains multiple events: `[User]`, `[Assistant]`, `[Tool: ...]`, `[Result: ...]`

## Key Implementation Details

When building transcript collection utilities:
1. Use `path.resolve(homedir(), '.claude/projects/')` to get the base directory
2. Encode project paths by replacing `/` with `-`
3. Read JSONL files and parse each line as JSON
4. Handle both `UserPromptSubmit` and `ToolCall` events for comprehensive transcript collection
