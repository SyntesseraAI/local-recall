To extract memories from Claude sessions, transcripts must be read from the Claude cache at `~/.claude/projects/<encoded-path>/transcripts/`. The MCP server daemon processes these transcripts asynchronously every 5 minutes, using Claude Haiku to extract memories and track processed transcripts with content hashes for deduplication.
