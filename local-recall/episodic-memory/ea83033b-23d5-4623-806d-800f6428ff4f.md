The UserPromptSubmit hook uses `claude -p --model haiku` to extract keywords from user prompts for memory search. The `-p` flag enables prompt caching which is critical for performance when repeatedly processing similar content. This pattern should be used whenever extracting keywords or summarizing content in Local Recall.
