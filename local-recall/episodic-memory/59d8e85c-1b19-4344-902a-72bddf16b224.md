# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by hyphens and special characters removed. For example:

- `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## Transcript Files

Transcripts are stored as JSONL files with one event per line:
- Format: `~/.claude/projects/{folder-name}/transcripts/*.jsonl`
- Each line contains a JSON object representing an event in the session
- Events include: `[User]`, `[Assistant]`, `[Tool: Name]`, `[Result: OK/ERROR]`

This is important for the `transcript-collector.ts` module which syncs and processes transcripts for memory extraction.
