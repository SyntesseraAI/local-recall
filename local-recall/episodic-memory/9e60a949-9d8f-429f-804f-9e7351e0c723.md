Two main hooks for Claude Code integration:

1. **SessionStart**: Loads 5 most recent memories on session start (full reload, no vector store for speed)
2. **UserPromptSubmit**: Performs semantic search using vector embeddings when user submits prompt

Hooks are configured in `hooks.json` using `${CLAUDE_PLUGIN_ROOT}` for relative paths and receive `cwd` via stdin. Stop hook is disabled - transcript processing runs via MCP server daemon every 5 minutes.
