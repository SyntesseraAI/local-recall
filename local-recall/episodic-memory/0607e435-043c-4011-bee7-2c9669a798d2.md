# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall` (forward slashes replaced with hyphens)
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` files (JSON lines format) with naming pattern: `[session-id]-transcript.jsonl`

## Importance

Knowing this path format is crucial for the transcript collector to locate and process project transcripts for memory extraction from the MCP server or daemon process.
