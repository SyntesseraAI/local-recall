# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with names like `transcript-2024-12-01-164523.jsonl`.

## Content Structure

Each line in the JSONL file is a JSON object representing a transcript event with fields:
- `type`: Event type (e.g., 'tool_use', 'message', etc.)
- `role`: 'user' or 'assistant'
- `content`: Event content/message
- Additional metadata depending on event type

## Usage

The `transcript-collector.ts` module reads these transcripts to extract memories. Understanding this structure is essential for the MCP server's background daemon that processes transcripts asynchronously every 5 minutes.
