The memory system prevents duplicates by checking both the `occurred_at` timestamp and `content_hash` (SHA-256 prefix, 16 chars) of memory content. The `createMemory()` function uses `findDuplicate()` to check for existing memories before creating new ones, ensuring idempotent memory creation.
