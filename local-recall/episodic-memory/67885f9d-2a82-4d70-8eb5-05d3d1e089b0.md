# Git Branch Naming Convention

## Format
Branch names follow the pattern: `type/description-UUID`

## Example
`claude/refactor-memory-collection-01HQY8jXfJm7zh35BcYYiE1u`

## Structure
- `type`: Action type (e.g., `claude`, `feat`, `fix`)
- `description`: Hyphenated description of the work
- `UUID`: Unique identifier (appears to be ULID format) to prevent naming conflicts

## Purpose
The UUID suffix ensures branch names are globally unique even when multiple developers or AI agents work on similar tasks simultaneously.
