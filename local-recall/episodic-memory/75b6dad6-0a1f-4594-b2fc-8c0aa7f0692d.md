# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the normalized file path:
- Forward slashes replaced with hyphens
- Characters like colons are converted
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall`

## Transcript Format

Transcripts are stored as JSONL files in `~/.claude/projects/<project-folder>/transcripts/`

Each line is a JSON object with:
- `type`: 'user' or 'assistant'
- `content`: The message text
- `timestamp`: ISO-8601 timestamp
- Additional metadata as needed

This is used by the transcript-collector to sync and process transcripts for memory extraction.
