# Transcript Collector Module

## Purpose

The `transcript-collector.ts` module is responsible for discovering, reading, and parsing Claude Code session transcripts stored in `~/.claude/projects/`.

## Key Responsibilities

1. **Path Discovery**: Converts absolute working directory paths to Claude's folder naming convention (slashes to dashes)
2. **File Reading**: Reads JSONL transcript files from the discovered project directory
3. **JSONL Parsing**: Parses line-delimited JSON to extract individual session events
4. **Event Processing**: Provides structured access to transcript events for downstream processing

## Integration Point

This module feeds into the MCP server's background daemon which:
- Syncs transcripts periodically (every 5 minutes)
- Extracts memories from transcript content
- Tracks processed transcripts using content hashes for change detection
