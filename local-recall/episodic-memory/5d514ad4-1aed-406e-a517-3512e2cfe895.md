# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path. The folder name is created by:
1. Taking the absolute path (e.g., `/Users/joe/Code/Syntessera/local-recall`)
2. Replacing slashes with underscores (e.g., `_Users_joe_Code_Syntessera_local-recall`)
3. Storing transcripts in `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format), with one event per line.

## Relevance to transcript-collector.ts

The `transcript-collector.ts` module handles reading and processing these transcripts from Claude's standard location. When collecting transcripts, account for this path structure and use the working directory to compute the correct folder name.
