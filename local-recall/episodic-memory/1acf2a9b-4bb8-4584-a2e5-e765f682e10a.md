# User Communication Preference

Based on interaction patterns:

## Preferred Style

- Concise, direct responses
- Focus on actionable information
- Avoid unnecessary elaboration
- Provide clear next steps
- Include specific file paths and line numbers when referencing code

## Application

When assisting with local-recall development:
- Get to the point quickly
- Explain what was done and why
- Provide concrete examples with file references
- Keep explanations focused on the current task
