Local-recall follows Claude Code's plugin format where the project root directory serves as the plugin root. The structure includes:
- `.claude-plugin/plugin.json` - Plugin metadata
- `hooks.json` - Hook event handlers
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (build output, gitignored)

Hooks use `${CLAUDE_PLUGIN_ROOT}` to reference scripts relative to the plugin installation and receive `cwd` via stdin to operate on the user's project directory.
