# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is created by replacing slashes in the absolute path with dashes. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension, where each line is a separate JSON object representing a transcript event.
