# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Input path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one entry per line, with timestamps and messages.

## Important Details

- The transcripts directory may not exist if no sessions have been started yet
- Path conversion uses dashes as separators for all slashes in the absolute path
- This is critical for the transcript-collector feature to correctly locate and parse transcripts
