# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files: Named with timestamps (e.g., `1733072893.jsonl`)

## Transcript Format

Each transcript is a JSONL file containing:
- `role`: Either "user" or "assistant"
- `content`: The message text
- `timestamp`: Unix timestamp of the message

This is useful for the transcript-collector and MCP server daemon which need to sync and process transcripts from Claude's cache.
