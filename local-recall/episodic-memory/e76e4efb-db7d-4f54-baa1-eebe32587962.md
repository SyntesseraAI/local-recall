# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as `.jsonl` (JSON Lines) files in the `transcripts/` directory. Each line is a JSON object representing a message or event in the session.

## Implementation

The `transcript-collector.ts` module handles:
- Resolving the project folder path from the working directory
- Reading and processing `.jsonl` transcript files
- Extracting events and conversation data

Key function: `getProjectTranscriptPath(cwd)` - converts a working directory to the actual transcript file path on disk.
