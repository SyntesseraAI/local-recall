When extracting memories from transcripts, the output must be a valid JSON object with a 'memories' array. Each memory should contain subject, keywords, applies_to, and content fields. The output should be raw JSON only - no markdown formatting, code blocks, or explanatory text. This is important for proper integration with the local-recall memory system.
