Memory files are stored in `local-recall/episodic-memory/` as individual markdown files. Each file is version-controlled in git.

**Structure:**
- `local-recall/episodic-memory/*.md` - Individual memory files with YAML frontmatter
- `local-recall/index.json` - Auto-generated keyword index (gitignored)
- `local-recall/recall.log` - Debug log file (gitignored)

This design allows memories to be:
- Version-controlled alongside the codebase
- Reviewed and shared via git history
- Individually tracked and modified
