# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one JSON object per line, each containing a message event.

## Parsing Pattern

When processing transcripts for memory extraction, parse `.jsonl` by reading lines and parsing each as a separate JSON object.
