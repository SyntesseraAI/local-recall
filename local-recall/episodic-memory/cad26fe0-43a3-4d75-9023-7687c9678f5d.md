Development requirements for this project:

**Setup:**
- Node.js 18+ required
- TypeScript 5.9+ required
- Install with `npm install` and build with `npm run build`

**Build and Development Scripts:**
- `npm run build`: Compile TypeScript
- `npm run dev`: Watch mode
- `npm run test`: Run all tests
- `npm run lint`: Lint code
- `npm run mcp:start`: Start MCP server
- `npm run test:unit`: Unit tests only
- `npm run test:integration`: Integration tests

**Code Requirements:**
- ALL new features must have corresponding unit tests in `tests/unit/`
- Integration tests go in `tests/integration/`
- Documentation must be placed in `docs/` folder
- Update existing docs when modifying functionality
- Tests should cover both happy paths and edge cases
- Memory files should have relevant, specific keywords
- Consider scope carefully: `global` vs `file:` vs `area:`
