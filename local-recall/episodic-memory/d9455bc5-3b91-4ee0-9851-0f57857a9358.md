# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with entries containing:
- `type`: Event type (e.g., `user_message`, `assistant_message`, `tool_call`)
- `timestamp`: ISO-8601 timestamp
- `content`: Message or event content
- Additional event-specific fields

## Usage in Local Recall

The MCP server reads from this location to:
1. Detect new or modified transcripts
2. Process them with Claude Haiku to extract memories
3. Track processed transcripts by content hash for change detection
4. Automatically extract memories every 5 minutes
