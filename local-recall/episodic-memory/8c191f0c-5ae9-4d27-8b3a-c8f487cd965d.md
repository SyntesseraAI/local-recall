# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes.

### Example
For project at: `/Users/joe/Code/Syntessera/local-recall`

Transcript location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (`.jsonl` extension) where each line is a JSON object.

This path format is crucial for the `transcript-collector.ts` module to locate and process transcripts for memory extraction.
