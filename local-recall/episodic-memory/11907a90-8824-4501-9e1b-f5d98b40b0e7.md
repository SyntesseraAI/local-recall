The MCP server includes a background daemon that runs every 5 minutes to process transcripts. Instead of using the Stop hook (which is disabled), transcript processing happens asynchronously via this daemon. The daemon syncs transcripts from Claude's cache, extracts memories using Claude Haiku, and manages deduplication via content hashes.
