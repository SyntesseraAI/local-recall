The embedding model is automatically downloaded on first use (takes 30-60 seconds) and cached in `local_cache/fast-bge-small-en-v1.5/`. If the tokenizer file is missing (corrupted cache), delete the directory to force re-download. Subsequent runs load from cache with no delay.
