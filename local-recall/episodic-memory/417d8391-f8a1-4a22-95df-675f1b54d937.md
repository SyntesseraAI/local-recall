# Embedding Model for Semantic Search

Local Recall implements semantic search using embeddings with the BGE-small-en-v1.5 model.

## Model Details

- **Model**: BGE-small-en-v1.5
- **Library**: fastembed (Python)
- **Size**: ~133MB
- **Cache Location**: `local_cache/fast-bge-small-en-v1.5/`

## Features

- Semantic search beyond keyword matching
- Vector store for efficient similarity queries
- Automatic model download on first use
- Cached for subsequent runs

## Performance Characteristics

- First run: 30-60 seconds for model download
- Subsequent runs: Fast (loaded from cache)
- Model is lightweight enough for local deployment

## Files

- `src/core/embedding.ts` - Embedding generation
- `src/core/vector-store.ts` - Vector storage and similarity search
