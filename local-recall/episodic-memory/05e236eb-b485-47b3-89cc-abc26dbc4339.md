# Keyword Extraction from Transcript Text

When extracting keywords from transcript content, use Claude's API (haiku model) to identify relevant search terms.

## Implementation Pattern

The process:
1. Send the transcript text or event content to Claude
2. Request extraction of keywords as a JSON array of strings
3. Parse the response to get the keyword list
4. Use keywords for memory creation and search indexing

## Benefits

- More accurate and contextually relevant keywords than simple text parsing
- Captures semantic meaning rather than just word frequency
- Can identify domain-specific terms and concepts
- Returns structured JSON format for easy integration
