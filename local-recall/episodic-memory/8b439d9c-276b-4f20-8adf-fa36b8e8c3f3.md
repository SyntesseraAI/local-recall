# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS/Linux.

## Folder Naming

Project folders are named using MD5 hash of the absolute file path:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `~/.claude/projects/0e39d8f4f7c8a2b1d9e3f4a5b6c7d8e9/`
- Hash is lowercase hexadecimal
- Consistent across sessions for the same project path

## Transcript Storage

- Transcripts stored in `.jsonl` format (JSON Lines - one JSON object per line)
- Located at: `~/.claude/projects/<hash>/transcripts/`
- File naming: `<timestamp>.jsonl` or similar pattern
- Each line represents one turn in the conversation

## Practical Usage

When collecting transcripts programmatically:
1. Calculate MD5 hash of project's absolute path
2. Look in `~/.claude/projects/<hash>/transcripts/`
3. Parse `.jsonl` files line-by-line for transcript events

This is critical for the MCP background daemon that syncs transcripts every 5 minutes.
