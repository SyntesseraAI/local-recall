# Transcript Collector Implementation

`src/core/transcript-collector.ts` handles loading and converting Claude Code transcripts to a standardized format for memory extraction.

## Key Functionality

- **loadTranscripts()**: Reads JSONL files from `~/.claude/projects/<project>/transcripts/`
- **Converts transcripts to events**: Each transcript entry becomes an event object
- **Standardizes format**: Creates consistent event structure with type, content, and metadata
- **Enables memory extraction**: Prepares transcript data for the MCP server daemon to extract memories

## Event Format

Transcripts are converted to events with:
- `type`: Event type (e.g., "user-prompt", "assistant-response", "tool-call")
- `content`: The actual content of the event
- `timestamp`: When the event occurred
- Additional metadata as needed

## Usage

The MCP server daemon calls this utility to process transcripts and extract memories asynchronously every 5 minutes.
