The `MemoryExtractor` class in src/core/memory-extractor.ts uses the Claude API (via `claude -p`) to analyze transcripts and extract memories. It sends transcript chunks to Claude with a structured prompt, parses the JSON response, and creates memory objects. The extractor handles batching of transcripts and deduplication via content hashes.
