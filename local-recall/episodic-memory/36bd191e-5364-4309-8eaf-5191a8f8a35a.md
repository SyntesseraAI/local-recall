# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/`

## Transcript Organization

Transcripts are stored in subdirectories within the project folder:
- `transcripts/` - Contains individual transcript files
- Each session generates a new transcript file

This is important for the `transcript-collector.ts` module which needs to sync transcripts from Claude's cache.
