# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Forward slashes `/` are replaced with dashes `-`
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`

## Transcript Storage

Transcripts are stored as JSONL files in the project folder:
- Location: `~/.claude/projects/[project-name]/transcripts/`
- File format: JSONL (JSON Lines - one JSON object per line)
- File naming: Multiple transcript files may exist for different sessions

## Implementation Notes

The `transcript-collector.ts` module handles reading and processing these transcripts. When building paths to transcript directories, use dash-replacement instead of expecting the exact directory name.
