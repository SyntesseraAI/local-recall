# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with special character transformations:
- All forward slashes `/` are converted to dashes `-`
- All spaces are converted to underscores `_`
- All dots `.` are converted to underscores `_`

## Example

For the project at `/Users/joe/Code/Syntessera/local-recall`:
- Folder name becomes: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcript files are JSONL format with `.jsonl` extension, where each line is a complete JSON event object.

## Notes for Implementation

When implementing `transcript-collector.ts`, use this path transformation logic to locate and read transcript files for the current project.
