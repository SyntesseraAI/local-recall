# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Becomes: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcript files are in JSONL format (`.jsonl` extension):
- Each line is a separate JSON object
- Format: `{ "role": "user" | "assistant", "content": "..." }`
- Files are named with timestamps or session identifiers

## Implementation Details

The `transcript-collector.ts` module uses this path to:
1. Locate transcripts for the current project
2. Parse JSONL files to extract conversation history
3. Sync transcripts with the memory system

## Key Files

- `src/core/transcript-collector.ts` - Handles transcript discovery and parsing
