# Claude Project Transcripts Storage

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name encodes the absolute path by replacing forward slashes with dashes:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This encoding allows multiple projects to be tracked independently while maintaining a consistent storage structure.
