# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by hyphens and colons removed:
- Absolute path: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line), with filenames like:
- `transcript-20251202-143022.jsonl` (timestamp format: YYYYMMDD-HHMMSS)

Each line in the JSONL file contains a message event with roles like `user` and `assistant`.

## Key Implementation Detail

When implementing transcript collection, the path conversion logic must:
1. Take the absolute working directory
2. Remove colons (from drive letters on Windows or time values)
3. Replace forward slashes with hyphens
4. Use this as the folder name under `~/.claude/projects/`
