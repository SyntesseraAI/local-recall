The BGE-small-en-v1.5 embedding model (~133MB) is automatically downloaded on first use to local_cache/ and cached for subsequent runs. Initial startup may take 30-60 seconds. If corrupted, remove local_cache/fast-bge-small-en-v1.5* directory to force re-download.
