The MCP server provides tools (memory_create, memory_delete, memory_search, memory_list, index_rebuild) and runs a background daemon that syncs transcripts from Claude's cache, processes them with `claude -p`, extracts memories, and tracks changes using content hashes. The daemon runs every 5 minutes to keep memories current.
