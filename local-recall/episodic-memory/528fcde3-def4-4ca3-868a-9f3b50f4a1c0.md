## JSONL Format Handling

Transcript files in Claude's cache are stored as JSONL (JSON Lines) format:

- **Format**: Each line is a separate JSON object (newline-delimited)
- **Parsing requirement**: Must split by newlines and parse each line as individual JSON
- **Error handling**: Invalid lines should be skipped or logged appropriately
- **File extension**: `.jsonl` files in the `~/.claude/projects/*/transcripts/` directory

The transcript collector must correctly handle this format when reading and processing transcript files for memory extraction.
