Memory extraction answers these key questions during analysis:
1. **What was learned?** - New knowledge or insights from the session
2. **What is now known?** - Important facts about codebase/architecture/conventions
3. **What is specific to this codebase?** - Unique patterns, configurations, or quirks
4. **What problems were solved?** - Bugs fixed, issues resolved, and how
