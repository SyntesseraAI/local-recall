# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with `.jsonl` extension, where each line is a JSON object representing a transcript event.

## Usage in Local Recall

The transcript collector reads from this location to extract and process session data for memory creation. This is referenced in `src/core/transcript-collector.ts` for locating and parsing Claude Code transcripts.
