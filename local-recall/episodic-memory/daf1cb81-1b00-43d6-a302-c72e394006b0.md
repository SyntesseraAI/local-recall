# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute work directory path:
- Absolute path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/<transcript>.jsonl`

## Path Conversion Algorithm

1. Take the absolute working directory path
2. Remove leading slash `/`
3. Replace all forward slashes `/` with hyphens `-`
4. Result is the project folder name in `~/.claude/projects/`

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

This is essential for the transcript collector to locate transcript files when processing memories.
