# Transcript JSONL Format

Claude stores session transcripts as JSONL files containing structured events.

## Event Structure

Each line in a JSONL file is a JSON object representing a transcript event with fields like:
- `type`: Event type (e.g., 'user_message', 'assistant_response', 'tool_call')
- `timestamp`: When the event occurred
- `content`: Event payload
- Other event-specific fields

## Import Implementation

The `transcript-collector.ts` module handles reading and parsing these JSONL files to extract memories and context.
