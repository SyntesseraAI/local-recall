The MCP server exposes these memory management tools:

| Tool | Purpose |
|------|----------|
| `memory_create` | Create a new memory (idempotent - returns existing if duplicate) |
| `memory_get` | Retrieve a specific memory by ID |
| `memory_search` | Semantic search using vector embeddings |
| `memory_list` | List all memories with optional filtering |
| `index_rebuild` | Sync vector store with memory files (add/remove as needed) |

These tools can be used by any MCP-compatible client, not just Claude Code.
