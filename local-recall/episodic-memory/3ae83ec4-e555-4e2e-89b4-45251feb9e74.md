# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in:
`~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Each line is a JSON object representing a single transcript event with:
- `type`: Event type (e.g., 'user_message', 'assistant_message', 'tool_call')
- `timestamp`: ISO-8601 timestamp
- `content`: The actual message or tool data

This location is used by the MCP server daemon to sync and process transcripts every 5 minutes.
