# Transcript Collector Module

The `transcript-collector.ts` module is responsible for:
1. Discovering and syncing JSONL transcript files from Claude's cache
2. Detecting changes using content hashes
3. Processing transcripts to extract memories using Claude Haiku
4. Managing memory lifecycle (create on new transcript, update on changes, delete on removal)

## Key Functionality

- Reads from `~/.claude/projects/<project>/transcripts/` 
- Uses content hashing to detect transcript changes
- Processes transcripts asynchronously via MCP daemon
- Creates memories from transcript summaries
- Cleans up memories when transcripts are deleted
