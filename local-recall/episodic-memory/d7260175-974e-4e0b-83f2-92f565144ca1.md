# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are stored as `.jsonl` files (JSON Lines format) in this directory.

## Relevance to Local Recall

The transcript-collector module needs to construct this path correctly when syncing Claude transcripts for memory extraction. This path format is critical for the MCP server's background daemon to locate and process transcripts.

## Implementation Notes

When constructing paths:
1. Replace all `/` with `-` in the absolute working directory path
2. Account for the leading `/` being converted to a dash (results in `Users-` not `-Users-`)
3. Use `~/.claude/projects/` as the base directory
4. Look for `.jsonl` files within the `transcripts/` subdirectory
