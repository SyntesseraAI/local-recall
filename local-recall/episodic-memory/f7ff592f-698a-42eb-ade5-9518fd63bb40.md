Claude Code stores project transcripts at `~/.claude/projects/` with project directories named using a path-encoding pattern.

**Pattern:** `/Users/<user>/.claude/projects/-Users-<user>-<path-segments>/`

For example, a project at `/Users/joe/Code/Syntessera/local-recall` stores transcripts at:
`/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files (one JSON object per line) where each line contains:
- `type`: Event type (user message, assistant message, tool call, etc.)
- `content`: The actual content of that event
- `timestamp`: When the event occurred

This is important for the transcript-collector which needs to find and parse these files to extract memories.
