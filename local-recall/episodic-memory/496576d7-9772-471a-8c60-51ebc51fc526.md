The Memory Manager (`src/core/memory.ts`) handles CRD operations (no update - memories are idempotent):
- `createMemory(data)` - Creates new memory, returns existing if duplicate
- `deleteMemory(id)` - Deletes memory by ID
- `getMemory(id)` - Retrieves specific memory
- `listMemories(filter?)` - Lists all memories with optional filtering
- `findDuplicate(occurredAt, contentHash)` - Checks for existing duplicates
