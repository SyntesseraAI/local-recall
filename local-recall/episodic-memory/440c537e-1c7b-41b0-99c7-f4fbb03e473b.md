Memory markdown files in `local-recall/episodic-memory/` ARE tracked in git and should be committed. The SQLite database (`local-recall/memory.sqlite`) and vector embedding cache are gitignored and auto-generated. A `.gitignore` file in `local-recall/` auto-excludes `memory.sqlite` and `recall.log`.
