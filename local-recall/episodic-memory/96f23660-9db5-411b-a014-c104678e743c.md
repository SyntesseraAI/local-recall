# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path using a hash-based naming scheme:
- Format: `~/.claude/projects/<hash-based-folder-name>/transcripts/`
- Example: For working directory `/Users/joe/Code/Syntessera/local-recall`, transcripts are stored in `~/.claude/projects/` with a hashed folder name

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), where each line is a complete JSON object representing a transcript event.

## Usage in transcript-collector

When implementing transcript collection, the path resolution must:
1. Get the absolute working directory (via `cwd` from hook input)
2. Hash the directory path to generate the project folder name
3. Look for `.jsonl` files in the `~/.claude/projects/<hashed-folder>/transcripts/` directory
4. Parse each line as a separate JSON object

This is essential for the transcript-collector to properly locate and parse Claude Code session transcripts.
