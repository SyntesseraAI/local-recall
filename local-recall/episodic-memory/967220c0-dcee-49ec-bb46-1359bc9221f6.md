# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line:
- Each line is a complete JSON object representing a transcript event
- Events include timestamps, types, and content
- Files can be parsed line-by-line without loading entire file into memory

## Usage in Local Recall

The transcript-collector module uses this location to sync transcripts from Claude's cache for memory extraction processing.
