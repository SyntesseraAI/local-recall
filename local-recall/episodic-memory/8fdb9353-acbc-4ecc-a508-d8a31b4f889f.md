# Transcript Schema Types

The `src/types/transcript-schema.ts` file defines TypeScript interfaces for parsing Claude Code JSONL transcripts.

## Entry Types
- **UserEntry** - User prompts with text content
- **AssistantEntry** - Claude responses with content blocks (text, code, thinking)
- **SystemEntry** - System messages for events/context
- **FileHistorySnapshotEntry** - File state snapshots
- **QueueOperationEntry** - Tool queue operations

Each entry includes:
- `type` - Discriminator for union type
- `timestamp` - ISO-8601 timestamp
- `content` - Entry-specific payload

These types are essential for parsing and condensing transcripts efficiently.
