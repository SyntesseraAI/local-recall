# TranscriptCollector Class

The `TranscriptCollector` class is responsible for reading and parsing Claude Code session transcripts from the local filesystem.

## Key Features

- Converts working directory paths to Claude's project folder name format (slashes → dashes, dots → underscores)
- Reads JSONL transcript files from `~/.claude/projects/<project>/transcripts/`
- Parses each line as a JSON object representing a transcript event
- Handles missing transcript files gracefully
- Async/await pattern for file operations

## Implementation Details

- Located in `src/core/transcript-collector.ts`
- Used by the MCP server and background transcript processing daemon
- Should be tested with real Claude transcript formats to ensure compatibility
- Error handling should account for file permissions and missing directories
