Transcript files use JSONL (JSON Lines) format where each line is a complete, valid JSON object representing a single transcript event. This format allows for streaming parsing and processing of transcripts without loading the entire file into memory. Each event contains information about the interaction (user input, assistant response, tool usage) that can be parsed to extract memories.
