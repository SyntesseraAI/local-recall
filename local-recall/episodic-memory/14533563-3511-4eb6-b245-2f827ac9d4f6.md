# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS.

## Directory Naming

The project directory name uses the project path with `/` replaced by `--` (double dashes).

Example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Directory name: `Users--joe--Code--Syntessera--local-recall`
- Full path: `~/.claude/projects/Users--joe--Code--Syntessera--local-recall/`

## Transcript Files

Transcript files are stored as JSONL (JSON Lines) format in `transcripts/` subdirectory within the project folder.

This path structure is important for the transcript collector to properly locate and process session transcripts.
