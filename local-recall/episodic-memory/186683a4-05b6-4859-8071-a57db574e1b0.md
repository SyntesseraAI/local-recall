Added four new configuration options in src/core/types.ts:
- `episodicMaxTokens` (env: `LOCAL_RECALL_EPISODIC_MAX_TOKENS`, default: 1000) - max tokens of episodic memories to inject
- `episodicMinSimilarity` (env: `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY`, default: 0.8) - minimum similarity threshold for episodic results
- `thinkingMaxTokens` (env: `LOCAL_RECALL_THINKING_MAX_TOKENS`, default: 1000) - max tokens of thinking memories to inject
- `thinkingMinSimilarity` (env: `LOCAL_RECALL_THINKING_MIN_SIMILARITY`, default: 0.8) - minimum similarity threshold for thinking results

These allow independent control of memory injection behavior for each memory type.
