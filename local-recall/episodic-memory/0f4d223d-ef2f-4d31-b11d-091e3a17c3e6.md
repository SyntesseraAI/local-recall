# Build Configuration

## TypeScript Setup

The project uses TypeScript with CommonJS output target.

## Build Output

- Compiled files go to `dist/` directory
- Hook scripts are compiled from `src/hooks/` to `dist/hooks/`
- MCP server is compiled from `src/mcp-server/` to `dist/mcp-server/`
- Bundled scripts go to `scripts/` directory (from dist)

## Development Workflow

- `npm run build` compiles TypeScript
- `npm run dev` runs TypeScript in watch mode
- Changes to TypeScript files should trigger recompilation
- Test the bundled scripts in `scripts/` after building

## Configuration Files

- `tsconfig.json` defines the compilation settings
- Check tsconfig for output directory configuration
