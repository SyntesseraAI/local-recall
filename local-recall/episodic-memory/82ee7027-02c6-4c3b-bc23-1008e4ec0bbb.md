The memory-extractor module uses the Claude API (claude-3-5-haiku model) to process transcripts and generate structured memories. It sends transcript content with a system prompt to Claude, which extracts relevant memories in JSON format. The extractor handles transcript processing asynchronously and includes error handling for API failures.
