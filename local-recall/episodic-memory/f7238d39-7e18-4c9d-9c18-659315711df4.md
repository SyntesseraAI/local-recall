Each memory file in `local-recall/episodic-memory/*.md` has YAML frontmatter with fields: `id` (UUID), `subject`, `keywords` (array), `applies_to` (global|file:<path>|area:<name>), `occurred_at` (ISO-8601), `content_hash` (SHA-256 prefix). The memory files are version-controlled in git while the SQLite database (`memory.sqlite`) and embedding model cache (`local_cache/`) are gitignored.
