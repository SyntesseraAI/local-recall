Local Recall integrates with Claude Code through two mechanisms:

**Plugin System:**
- Project root is the plugin root (follows Claude Code plugin format)
- Requires `.claude-plugin/plugin.json` with metadata
- Hooks configured in `hooks.json` using `${CLAUDE_PLUGIN_ROOT}` variable
- Bundled scripts in `scripts/` (build output, gitignored)

**Installation:**
- Add to `.claude/settings.json` under `plugins.installed`
- Hooks and MCP server can also be configured directly in `.claude/settings.json` if not using plugin system

**MCP Server Tools Exposed:**
- `memory_create` - Create new memory (idempotent)
- `memory_get` - Retrieve memory by ID
- `memory_search` - Semantic search
- `memory_list` - List memories with optional filtering
- `index_rebuild` - Sync vector store with memory files
