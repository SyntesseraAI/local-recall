# Transcript Collector JSONL Processing

The `transcript-collector.ts` module processes JSONL (JSON Lines) format files from Claude Code transcripts.

## Key Features

- Reads `.jsonl` files line-by-line
- Each line is a separate JSON object representing a transcript event
- Parses events to extract relevant information for memory creation
- Handles streaming to avoid memory issues with large transcripts

This format allows efficient processing of transcript data without loading entire files into memory.
