The memory-extractor module (`src/core/memory-extractor.ts`) is responsible for:

1. **Processing transcripts**: Reads and analyzes Claude Code session transcripts
2. **Using Claude API**: Calls Claude with a memory extraction prompt via streaming
3. **Memory creation**: Extracts structured memories (subject, keywords, applies_to, content) from transcript analysis
4. **Streaming response**: Uses the `@anthropic-ai/sdk` with streaming to process responses as they arrive
5. **Memory storage**: Integrates with the memory management system to persist extracted memories

Key files:
- `src/core/memory-extractor.ts` - Main extraction logic
- `src/prompts/memory-extraction.ts` - Contains the prompt template used for extraction
