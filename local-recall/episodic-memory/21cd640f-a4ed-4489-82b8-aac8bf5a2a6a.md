# Transcript Collector Implementation Needed

The MCP server needs a transcript collector to:
1. Read Claude Code transcripts from `~/.claude/projects/<project>/transcripts/`
2. Parse JSONL format transcript files
3. Extract memories from transcript content
4. Handle transcript change detection via content hashing
5. Process transcripts asynchronously in the background (every 5 minutes)

## Current Status

The transcript collector functionality is referenced in CLAUDE.md but not yet fully implemented. The `src/core/transcript-collector.ts` file exists and may contain partial implementation.

## Related Components

- `src/core/memory-extractor.ts` - Extract memories from transcript content
- `src/core/search.ts` - Find relevant memories
- `src/mcp-server/server.ts` - Manages background daemon that runs transcript collector
