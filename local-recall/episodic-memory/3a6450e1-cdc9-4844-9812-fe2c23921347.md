# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcript files are stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Each transcript is a `.jsonl` file with ISO-8601 timestamp in filename

## File Format

Transcripts are stored as JSONL (JSON Lines) where each line is a separate JSON event with:
- Chronological ordering within a session
- Tool invocations and results
- User prompts and assistant responses
- Structured format suitable for programmatic parsing
