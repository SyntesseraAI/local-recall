The memory extraction system in `memory-extractor.ts` uses Claude Haiku model to:
1. Condense transcript content using the `condenseTranscript` function
2. Generate keywords for memories using the `generateKeywords` function
3. Extract memories from the condensed transcript using the `extractMemories` function

The system processes transcripts by calling Claude Haiku with specific prompts defined in `src/prompts/memory-extraction.ts`. This allows for intelligent extraction of valuable memories without needing to manually parse transcript content.
