# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), where each line is a separate JSON object representing an event in the transcript.

## Importance

This path format is critical for the transcript collector to locate and process Claude Code transcripts for memory extraction. The transcript collector needs to know this exact path structure to sync transcripts from Claude's cache.
