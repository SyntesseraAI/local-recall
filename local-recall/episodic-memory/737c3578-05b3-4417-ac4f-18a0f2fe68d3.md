Memory files are stored in `local-recall/episodic-memory/` as markdown with YAML frontmatter containing: `id` (UUID), `subject` (one-line description), `keywords` (array), `applies_to` (scope: global/file:/path/area:name), `occurred_at` (ISO-8601), and `content_hash` (SHA-256 prefix). Memory files are version-controlled in git.
