Memories are deduplicated using two fields: `occurred_at` (ISO-8601 timestamp of when the event occurred) and `content_hash` (SHA-256 prefix of 16 chars). The `findDuplicate()` function in memory.ts checks for existing duplicates before creating new memories, ensuring idempotency.
