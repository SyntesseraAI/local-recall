# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts JSONL file: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/transcripts.jsonl`

## File Format

Transcripts are stored as JSONL (JSON Lines) where each line is a complete JSON object representing a single message in the conversation.

## Relevance

This is critical for the transcript-collector to correctly locate and read Claude's session transcripts for memory extraction.
