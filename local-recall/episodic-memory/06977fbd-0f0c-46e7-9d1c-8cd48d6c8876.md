The embedding model BGE-small-en-v1.5 is approximately 133MB and is automatically downloaded on first use to `local_cache/fast-bge-small-en-v1.5/`. First startup may take 30-60 seconds during download. If tokenizer file corruption occurs (incomplete download), delete the cache directory to force re-download: `rm -rf local_cache/fast-bge-small-en-v1.5*`
