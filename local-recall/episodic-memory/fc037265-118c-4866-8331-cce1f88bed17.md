# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and suffixed with a hash:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users-joe-code-syntessera-local-recall-<hash>`
- The `<hash>` is a deterministic hash of the full path

## File Structure

Inside each project folder:
- `transcripts.jsonl` - Contains all session transcripts in JSONL format (JSON Lines)
- Each line is a complete JSON object representing one transcript event

## Transcript Events

Transcript events include:
- Tool invocations with their arguments
- Tool results with success/error status
- User messages
- Claude messages
- Session metadata

This is the source data that the transcript-collector analyzes to extract memories.
