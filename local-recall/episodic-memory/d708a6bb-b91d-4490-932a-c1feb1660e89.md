The transcript-collector.ts uses different log levels:
- Older debug-level messages for transcript discovery
- Newer info-level messages for project directory searches (e.g., 'Searching for Claude project transcripts...')

This mismatch can help identify if stale code is running. Check the compiled dist/core/transcript-collector.js for the current log messages.
