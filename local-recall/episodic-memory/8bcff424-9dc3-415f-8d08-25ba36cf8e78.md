# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcripts are stored as JSONL files with ISO-8601 timestamps in the filename.

## Relevance to Local Recall

The `transcript-collector.ts` module needs to correctly format the path to find and process Claude transcripts for memory extraction. This is critical for the background daemon that syncs and processes transcripts.
