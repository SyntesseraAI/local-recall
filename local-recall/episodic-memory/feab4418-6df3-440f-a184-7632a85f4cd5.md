# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory:
- Full path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users_joe_Code_Syntessera_local-recall` (slashes replaced with underscores)
- Full path to transcripts: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with names like:
- `transcript-1735808401435.jsonl`
- Each line is a JSON object representing a transcript event

This is used by `transcript-collector.ts` to find and process transcripts from Claude Code sessions.
