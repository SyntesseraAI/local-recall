The MCP server (`src/mcp-server/server.ts`) exposes tools: `memory_create`, `memory_get`, `memory_search`, `memory_list`, `index_rebuild`. It runs a background daemon that syncs transcripts from Claude's cache, processes them every 5 minutes using `claude -p` to extract memories, and tracks processed transcripts with content hashes for change detection.
