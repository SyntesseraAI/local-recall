# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in:
`~/.claude/projects/<project-folder>/transcripts/`

Each line is a JSON object representing a conversation event or message.

## Relevance to Local Recall

The transcript-collector module reads from this location to extract memories from Claude Code sessions.
