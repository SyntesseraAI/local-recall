The logger utility in logger.ts expects a single string argument for debug/error/warn/info calls. When logging structured data, concatenate strings with template literals or use `JSON.stringify()` rather than passing multiple arguments.

Incorrect: `logger.error('error:', error, 'data:', data)`
Correct: `logger.error(`error: ${JSON.stringify(error)}`)
