Claude stores projects in ~/.claude/projects using the full path with slashes replaced by dashes.

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `-Users-joe-Code-Syntessera-local-recall`

Transcripts are stored directly in the project folder as `.jsonl` files, NOT in a `transcripts` subfolder.

Fix applied in `src/core/transcript-collector.ts`:
- Prioritize searching ~/.claude/projects for directories matching the path-to-dashes pattern
- Look for .jsonl files directly in the project directory
- Use path conversion: `cwd.replace(/\//g, '-')` to match Claude's naming scheme
