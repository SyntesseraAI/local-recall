Default configuration in `types.ts` has both memory types enabled by default:
- `episodicEnabled: true` - Episodic memories are active by default
- `thinkingEnabled: true` - Thinking memories are active by default

These are the base defaults that can be overridden via configuration files or environment variables. Users don't need to explicitly enable these features.
