# Transcript File Format

Transcripts stored in `~/.claude/projects/<project-name>/` are in JSONL format (JSON Lines - one JSON object per line).

Each line represents a transcript event/message. This is important for:
- Parsing transcripts in `transcript-collector.ts`
- Processing with Claude's API using `-p` flag
- Building the transcript sync mechanism in the MCP server
