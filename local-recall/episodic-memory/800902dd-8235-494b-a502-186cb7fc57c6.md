The fuzzy search matching in Local Recall can be configured with:

- **fuzzyThreshold** configuration option (default: 0.6, range: 0-1)
- Can be set via environment variables or `.local-recall.json` configuration file
- Lower threshold = more permissive matching (catches more distant matches)
- Higher threshold = stricter matching (only very similar keywords)

Other configurable options include:
- `memoryDir`: Directory for memory storage (default: `./local-recall`)
- `maxMemories`: Maximum number of memories (default: 1000)
- `indexRefreshInterval`: Seconds between index refreshes (default: 300)
