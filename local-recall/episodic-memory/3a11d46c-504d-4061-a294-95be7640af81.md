**IMPORTANT for version control:**
- Memory files in `local-recall/episodic-memory/*.md` ARE git-tracked and should be committed
- The SQLite database `local-recall/memory.sqlite` is gitignored and auto-generated
- The embedding model cache `local_cache/` is gitignored
- A `.gitignore` file in `local-recall/` auto-generates exclusions for memory.sqlite and recall.log

This allows team members to share memories while keeping generated data out of version control.
