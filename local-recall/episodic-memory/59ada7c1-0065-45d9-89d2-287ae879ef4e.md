# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format):
- One JSON object per line
- Each object represents a transcript event
- Files are gzip-compressed
- Events include timestamps and message content

## Key Implementation Details

- When processing transcripts in transcript-collector, account for the path transformation
- macOS uses `~/.claude/` as the base directory for Claude configuration and session data
- This path is critical for the MCP server daemon's background transcript processing
