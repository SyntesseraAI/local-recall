Claude Code stores session transcripts in `~/.claude/projects/` directory. The project folder name is the URL-encoded absolute path to the project directory (slashes become dashes).

Example path pattern:
- Project at `/Users/joe/Code/Syntessera/local-recall`
- Stored in `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are JSONL format (one JSON object per line), where each line represents a single event in the session.

This is important for the transcript-collector to find and process transcripts correctly.
