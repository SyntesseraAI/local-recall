# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a complete JSON object representing a message or event in the transcript.

## Key Implementation Detail

The `transcript-collector.ts` module needs to handle this path transformation when locating and reading transcript files from the Claude cache.
