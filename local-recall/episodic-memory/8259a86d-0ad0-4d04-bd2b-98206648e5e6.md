# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall`
- Transcripts location: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files:
- Format: One JSON object per line
- Each line contains: `{"role": "user"|"assistant", "content": "..."}`
- Files named with timestamps: `transcript_YYYY-MM-DD_HH-MM-SS.jsonl`

## Key Implementation Detail

The `transcript-collector.ts` module needs to:
1. Convert the current working directory to the Claude projects format (replace `/` with `-` and prefix with `-`)
2. Build the full transcript path: `~/.claude/projects/{converted_path}/transcripts/`
3. Handle platform differences (currently implements macOS logic)
4. Read and parse JSONL files to extract transcript events
