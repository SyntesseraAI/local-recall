The MCP server provides tools for memory management:
- `memory_create`, `memory_delete`, `memory_search`, `memory_list`, `index_rebuild`

It also runs a background daemon that:
- Syncs transcripts from Claude's cache
- Processes transcripts every 5 minutes
- Tracks changes using content hashes
- Deletes and recreates memories when transcripts change
