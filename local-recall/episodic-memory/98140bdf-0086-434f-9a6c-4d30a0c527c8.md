# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name encodes the absolute path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` becomes `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

- Format: JSONL (JSON Lines)
- Each line is a complete JSON object representing a transcript event
- Multiple transcript files may exist, numbered sequentially
- Files contain session history with timestamps and content

## Integration Point

The transcript-collector in Local Recall needs to read from this location to process transcripts for memory extraction. This is critical for the MCP server's background daemon that processes transcripts every 5 minutes.
