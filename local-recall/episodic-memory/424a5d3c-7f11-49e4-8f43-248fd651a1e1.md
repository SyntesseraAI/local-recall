MCP server configuration is defined in `.claude/settings.local.json`. Environment variables are passed via the `env` object in the MCP server configuration. Key environment variables include `LOCAL_RECALL_DIR` for the memory directory path and feature flags like `LOCAL_RECALL_EPISODIC_ENABLED`.
