# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Full transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files where each line is a valid JSON object representing a session event.

## Relevance to Local Recall

The `transcript-collector.ts` module is responsible for:
1. Calculating the project folder name from the working directory
2. Locating transcript files in the Claude projects directory
3. Reading and parsing JSONL transcript files
4. Extracting memories from transcript content
