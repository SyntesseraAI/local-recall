Vector embeddings are generated via Ollama with the `nomic-embed-text` model (768 dimensions, ~274MB). Configuration: `OLLAMA_BASE_URL` (default: http://localhost:11434), `OLLAMA_EMBED_MODEL` (default: nomic-embed-text). Ollama must be running (`ollama serve`). Index files (orama-episodic-index.json, orama-thinking-index.json) are gitignored and auto-generated.
