# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the working directory path by:
1. Converting absolute paths to lowercase
2. Replacing path separators (`/`) with underscores (`_`)
3. Removing leading underscores

Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `users_joe_code_syntessera_local-recall`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format) where each line is a JSON object representing an event in the session.

## Relevance to local-recall

The `transcript-collector.ts` module uses this information to locate and process Claude transcripts for memory extraction via the MCP server daemon.
