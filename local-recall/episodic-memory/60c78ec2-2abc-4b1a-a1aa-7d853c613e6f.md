Local Recall is a markdown-powered memory system for AI coding assistants. The project root IS the plugin root following Claude Code's plugin format. Key directories:
- `.claude-plugin/` - Plugin metadata (plugin.json)
- `hooks.json` - Hook configuration (SessionStart, UserPromptSubmit, Stop)
- `.mcp.json` - MCP server configuration
- `scripts/` - Bundled executables (hooks and mcp-server, gitignored)
- `src/` - Source TypeScript files
- `local-recall/` - Memory storage with episodic-memory subdirectory for markdown files

The plugin uses `${CLAUDE_PLUGIN_ROOT}` environment variable to reference scripts and receives `cwd` via stdin to operate on user's project directory.
