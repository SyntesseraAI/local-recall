# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and a 16-character hash suffix appended.

### Example
For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Project folder: `Users-joe-Code-Syntessera-local-recall-[16-char-hash]`
- Transcripts stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall-[16-char-hash]/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension.

Each line contains a JSON event object with:
- `timestamp`: ISO-8601 timestamp
- `role`: Either "user" or "assistant"
- `content`: The message content
- `type`: Event type (e.g., "message")

## Relevance

This information is critical for the `transcript-collector.ts` which needs to locate and parse Claude's transcript files to extract memories asynchronously.
