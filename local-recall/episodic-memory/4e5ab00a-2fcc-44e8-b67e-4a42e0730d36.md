# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the project's file path. For example:
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files in the `transcripts/` subdirectory:
- Filename: `<timestamp>.jsonl`
- One JSON object per line
- Each object contains turn data with `role` ('user' or 'assistant') and `content`

## Important Notes

- The path encoding replaces `/` with `_` in folder names
- This is critical for `transcript-collector.ts` to locate and process transcripts
- The transcript directory may not exist if no sessions have been run yet
