The `ensureGitignore()` function checks if gitignore exists and skips creation if it does, preventing updates to existing gitignore files. Since the gitignore is auto-generated, it should always be written to ensure new patterns (like memory.sqlite) are included for all users. Changed the function to always write the gitignore file.
