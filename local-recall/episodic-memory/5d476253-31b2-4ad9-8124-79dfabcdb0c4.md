Transcripts in this project use a standardized event-based format:
- `[User]` - User requests or questions
- `[Assistant]` - Claude's responses
- `[Tool: Name]` - Tool invocations (Read, Edit, Bash, etc.)
- `[Result: OK/ERROR]` - Outcomes of tool invocations

This structured format is used by the memory extractor to parse and analyze sessions consistently.
