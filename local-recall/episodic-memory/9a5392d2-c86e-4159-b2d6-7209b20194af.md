# Claude Transcripts Path Format

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name converts the absolute working directory path by replacing slashes with dashes:
- Original path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

This is important for understanding where Claude stores transcript JSONL files for memory extraction and analysis.
