# Transcript File Naming Convention

Claude stores transcripts with sequential numbering:
- Format: `transcript-{number}.md`
- Examples: `transcript-1.md`, `transcript-2.md`, `transcript-3.md`
- Located in: `~/.claude/projects/{project-folder}/transcripts/`

When collecting transcripts, list all files matching this pattern and process them in order.
