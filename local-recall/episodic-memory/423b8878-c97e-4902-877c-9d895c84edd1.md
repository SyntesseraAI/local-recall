# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes converted to underscores. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`

## Transcript Files

Transcripts are stored as JSONL files in the `transcripts/` subdirectory:
- Format: One JSON object per line
- Each line contains: `{"type": "...", "content": "...", ...}`
- File naming follows Claude's internal naming convention

## Implementation Details

The `transcript-collector.ts` module handles:
- Converting project paths to folder names (replacing `/` with `_`)
- Reading JSONL files from the transcripts directory
- Parsing transcript data for memory extraction
- Handling both absolute and relative path conversions

This is critical for the MCP server's background daemon that syncs and processes transcripts every 5 minutes.
