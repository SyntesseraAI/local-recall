Each memory is a markdown file with YAML frontmatter containing:
- `id`: UUID
- `subject`: One-line description
- `keywords`: Array of searchable keywords
- `applies_to`: Scope (global, file:<path>, area:<name>)
- `occurred_at`: ISO-8601 timestamp for deduplication and sorting
- `content_hash`: SHA-256 prefix for deduplication

Memories are stored in `local-recall/episodic-memory/` and are version-controlled.
