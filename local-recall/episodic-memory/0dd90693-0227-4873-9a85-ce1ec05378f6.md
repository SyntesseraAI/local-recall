# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line.

## Key Implementation Detail

When building transcript collector functionality, use `~/.claude/projects/<project-folder>/transcripts/` as the base path where the project folder name has all path separators replaced with dashes.
