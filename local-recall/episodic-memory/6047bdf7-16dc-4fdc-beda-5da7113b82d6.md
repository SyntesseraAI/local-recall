Each memory file must include the following YAML frontmatter fields:
- `id`: UUID for unique identification
- `subject`: Brief one-line description
- `keywords`: Array of searchable keywords
- `applies_to`: Scope (global, file:<path>, or area:<name>)
- `occurred_at`: ISO-8601 timestamp for sorting and deduplication
- `content_hash`: SHA-256 prefix (16 chars) for detecting duplicates

This structure enables automatic deduplication based on `occurred_at` and `content_hash` combination.
