The memory extraction system uses the Claude API via `anthropic` package to analyze session transcripts and extract valuable memories. The MemoryExtractor class processes transcripts and calls Claude to identify and structure memories from the condensed transcript format.

Key implementation details:
- Uses `createMemory()` from the memory module for persistence
- Processes condensed transcripts with special markers like `[User]`, `[Assistant]`, `[Tool: Name]`, `[Result: OK/ERROR]`
- Returns extracted memories in structured format for deduplication and storage
