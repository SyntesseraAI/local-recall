- Stored in `local-recall/episodic-memory/` directory
- Files ARE tracked in git (version-controlled)
- SQLite database (`local-recall/memory.sqlite`) is gitignored and auto-generated
- Each file has unique ID (UUID), subject, keywords array, scope (global/file:/path/area:name), timestamp, and content hash for deduplication
- Memories should be atomic (one concept per file) with specific keywords for searchability
