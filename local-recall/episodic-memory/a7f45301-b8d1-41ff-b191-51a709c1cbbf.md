# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Replace `/` with `_` (slash to underscore)
- Replace `:` with `_` (colon to underscore, important for macOS paths)
- Use the transformed path as the project folder name

## Example

Working directory: `/Users/joe/Code/Syntessera/local-recall`

Transforms to: `Users_joe_Code_Syntessera_local-recall`

Transcripts are stored in: `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, one event per line, in chronological order.
