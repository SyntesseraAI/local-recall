Version 0.1.3 of the local-recall plugin was released with significant improvements:

**New Skills Added:**
- `check-memories` - For searching and retrieving past context and decisions
- `proactive-recall` - For checking memory before making architectural decisions

**Improvements:**
- MCP tool descriptions enhanced with usage guidance
- Plugin metadata updated for better discoverability in marketplaces
- 8 new thinking memory files were added to the repository

**Location:** `dev-marketplace/local-recall-plugin/`

These skills help users leverage the memory system by prompting them to check relevant memories before taking action, improving decision-making and preventing repeated mistakes across sessions.
