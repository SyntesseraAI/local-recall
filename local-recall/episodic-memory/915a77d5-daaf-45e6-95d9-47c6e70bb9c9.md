UUID validation for transcript filenames should use a regex pattern that matches standard UUID format (with hyphens). The pattern can be defined as a constant and used with a helper function like `isUuidFilename(filename: string): boolean` that returns true only if the filename (without extension) matches the UUID pattern. This provides a reusable way to validate transcript filenames across multiple methods in the TranscriptCollector class.
