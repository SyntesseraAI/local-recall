The codebase implements transcript condensing before memory extraction. This preprocessing step reduces the volume of transcript data sent to the Claude API, improving both the quality of memory extraction and reducing token usage. The condensed format includes events like [User], [Assistant], [Tool: Name], and [Result: OK/ERROR] tags to maintain essential context while removing redundancy.
