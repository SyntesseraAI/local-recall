# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute path of the project directory:
- Path: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/Users/joe/Code/Syntessera/local-recall/`
- Transcripts stored at: `~/.claude/projects/Users/joe/Code/Syntessera/local-recall/transcripts/`

## File Format

Transcript files are named with `.jsonl` extension and contain JSON Lines format data where each line is a separate JSON object representing a transcript event.

## Discovery Context

This path format was discovered while investigating the transcript-collector module to understand how to locate and access Claude's transcript data for memory extraction.
