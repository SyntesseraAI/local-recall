# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL files (one JSON object per line) in:
`~/.claude/projects/<project-folder>/transcripts/`

Each line contains a transcript event with:
- `type`: Event type (e.g., 'user', 'assistant', 'tool')
- `content`: The actual content
- `timestamp`: When the event occurred

## Usage in local-recall

The `src/core/transcript-collector.ts` module reads transcripts from this location to extract memories. The background MCP daemon syncs these transcripts every 5 minutes to process and create memories from them.
