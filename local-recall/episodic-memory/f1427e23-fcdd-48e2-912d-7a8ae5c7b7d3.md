Configuration options for Local Recall:
- memoryDir: Directory for memory storage (default: ./local-recall)
- maxMemories: Maximum number of memories (default: 1000)

Configuration can be set via:
1. .local-recall.json file
2. Environment variables

For MCP server, LOCAL_RECALL_DIR environment variable specifies the memory directory location.
