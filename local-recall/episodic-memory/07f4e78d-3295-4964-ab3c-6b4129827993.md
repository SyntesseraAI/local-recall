Three hooks available: SessionStart (loads 5 most recent memories on session begin), UserPromptSubmit (performs semantic search on user prompt before Claude processes), and Stop (disabled - memory extraction handled by MCP server daemon). All hooks receive JSON via stdin and output to stdout.
