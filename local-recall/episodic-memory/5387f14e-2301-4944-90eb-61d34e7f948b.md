Plugin-layer hooks (SessionStart:Callback, UserPromptSubmit:Callback) require stdout to be in JSON format with a `hookSpecificOutput` object containing `additionalContext` field. Plain `console.log()` output won't work - the hook output must follow the plugin hook response schema with `hookSpecificOutput.additionalContext` for injecting context into Claude's session.
