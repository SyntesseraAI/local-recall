Condensed transcripts use a specific event marker format:
- `[User]` - User requests
- `[Assistant]` - Claude responses
- `[Tool: Name]` - Tool invocations (Read, Edit, Write, Bash, Grep, etc.)
- `[Result: OK/ERROR]` - Tool outcomes

This format is used by the memory extraction system to parse and understand session flow. The transcript-collector needs to properly handle these markers when extracting memories.
