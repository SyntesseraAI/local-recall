# Claude Project Transcripts Location on macOS

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Encoding

The project folder name encodes the absolute path with forward slashes replaced by dashes:
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts stored in: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` (JSON Lines) files, where each line is a JSON object representing a transcript event.

## Implementation Note

The `transcript-collector.ts` module should use this path encoding scheme when constructing paths to Claude's transcript cache directory.
