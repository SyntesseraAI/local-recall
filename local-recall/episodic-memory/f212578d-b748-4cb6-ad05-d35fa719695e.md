# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with leading slash removed and remaining slashes replaced with dashes.

**Example:**
- Project path: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcript files: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/*.jsonl`

## File Format

Transcript files are in JSONL format (JSON Lines), with each line being a complete JSON object representing a transcript event.

## Implementation Location

The `transcript-collector.ts` file should implement path handling that:
1. Takes the project directory as input
2. Converts absolute path to Claude's folder naming convention (dashes instead of slashes)
3. Constructs the full path to the transcripts directory
4. Reads and parses JSONL transcript files
