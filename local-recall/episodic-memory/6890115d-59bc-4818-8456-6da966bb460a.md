Individual memory files in `local-recall/episodic-memory/*.md` are committed to git. However, `local-recall/memory.sqlite` (vector store) is gitignored and auto-generated. The `.gitignore` file in `local-recall/` excludes `memory.sqlite` and `recall.log` (debug log). This allows teams to share memories while keeping database artifacts local.
