# Transcript Collector Implementation

## Key Functions

The `transcript-collector.ts` module handles reading and processing transcript files from Claude's transcript directory.

## Important Implementation Details

- The transcript collector needs to handle `.jsonl` format (JSON Lines)
- Each line in a transcript file is a separate JSON object
- The module should track content hashes to detect transcript changes
- The path to transcripts comes from Claude's `~/.claude/projects/<project>/transcripts/` directory
- The project directory name uses the dashed version of the working directory path

## Related Code

The module works in conjunction with the memory extraction pipeline that processes transcripts asynchronously (every 5 minutes in the MCP server).
