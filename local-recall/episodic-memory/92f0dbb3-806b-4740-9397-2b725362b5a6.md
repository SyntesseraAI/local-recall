# JSONL Transcript Format

Claude Code stores session transcripts in JSONL (JSON Lines) format.

## Format Details

- File extension: `.jsonl`
- Each line is a complete, valid JSON object
- Lines represent sequential events/turns in a conversation
- No outer array wrapper - just newline-separated objects

## Parsing Implications

When processing transcripts, must parse line-by-line rather than treating the entire file as a single JSON array.
