# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path with slashes replaced by hyphens and special characters removed.

### Example

For working directory: `/Users/joe/Code/Syntessera/local-recall`

1. Take the full path: `/Users/joe/Code/Syntessera/local-recall`
2. Remove leading slash: `Users/joe/Code/Syntessera/local-recall`
3. Replace slashes with hyphens: `Users-joe-Code-Syntessera-local-recall`

Transcripts are then stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

Transcript files are named with timestamps and stored as `.jsonl` format.

## Relevant Code

The `transcript-collector.ts` module handles locating and reading these transcript files.
