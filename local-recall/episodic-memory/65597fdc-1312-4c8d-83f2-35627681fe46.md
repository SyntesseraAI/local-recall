The transcript-collector component needs to implement logic for discovering transcript files from Claude's cache directory. Based on the transcript location information:

1. Construct the project path encoding: `/Users/joe/Code/Syntessera/local-recall` becomes `-Users-joe-Code-Syntessera-local-recall`
2. Look for transcripts in `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`
3. Parse `.jsonl` files where each line is a JSON message event
4. Handle multiple transcript files and session management

This discovery mechanism is essential for the automatic memory extraction that runs in the background MCP server daemon every 5 minutes.
