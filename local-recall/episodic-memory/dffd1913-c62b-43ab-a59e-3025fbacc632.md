Two installation approaches:

1. **Clone and build**: Install dependencies and run `npm run build`, then reference in `.claude/settings.json`
2. **Direct hooks**: Use npm package with hooks configuration in `.claude/settings.json`

For MCP server: either npm package reference or direct path. Always use absolute paths for Claude Desktop configuration.
