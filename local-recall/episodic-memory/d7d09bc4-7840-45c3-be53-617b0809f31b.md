# Transcript Encoding Format

Claude project transcripts use BASE64 encoding for transcript content fields.

## Structure

Each transcript entry in the `.jsonl` file contains fields with BASE64-encoded content. This encoding is applied by Claude itself when storing transcripts.

## Decoding

When parsing transcripts:
1. Extract the BASE64-encoded string from the relevant field
2. Decode it using `Buffer.from(encoded, 'base64').toString('utf-8')`
3. Parse the resulting JSON or text content

## Related Code

The transcript-collector component in `src/core/transcript-collector.ts` handles reading these transcripts from `~/.claude/projects/*/transcripts/`.
