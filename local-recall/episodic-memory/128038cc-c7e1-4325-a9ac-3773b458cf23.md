# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with slashes replaced by dashes and the entire string prefixed with a hash.

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Hash the path: `sha256` of `/Users/joe/Code/Syntessera/local-recall`
- Create folder: `~/.claude/projects/` with the hash as folder name
- Transcripts stored in: `~/.claude/projects/<hash>/transcripts/`
- Files are JSONL format with numeric names: `0.jsonl`, `1.jsonl`, etc.

## Implementation Notes

When collecting transcripts in `src/core/transcript-collector.ts`:
1. Use Node.js `crypto.createHash('sha256')` to hash the absolute path
2. The hash serves as a stable, unique identifier for the project
3. Transcripts are stored sequentially numbered as JSONL files
4. Each JSONL entry is a complete transcript event with timestamp and content
