The project root IS the plugin root following Claude Code's plugin format. Hooks.json uses `${CLAUDE_PLUGIN_ROOT}` to reference bundled scripts (session-start.js, user-prompt-submit.js, stop.js) for proper path resolution when the plugin is installed in different locations. Scripts receive `cwd` via stdin to operate on the user's project directory.
