MCP server exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild. It runs a background daemon that syncs transcripts from `~/.claude/projects/<project>/transcripts/`, processes them to extract memories, tracks processed transcripts with content hashes, and regenerates memories when transcripts change. Daemon runs every 5 minutes.
