# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files where each line is a JSON event:
- Files are named by session timestamp
- Each line represents a tool call, result, or message
- Messages contain `role` (user/assistant) and `content`
- Tool events contain `tool` name and `input`/`output` data

## Useful for

The transcript-collector can read these JSONL files to extract session history and analyze tool usage patterns.
