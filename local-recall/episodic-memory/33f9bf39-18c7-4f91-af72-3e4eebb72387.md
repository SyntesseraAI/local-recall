# Fuzzy Matching Implementation

The fuzzy matching utility in `src/utils/fuzzy.ts` uses Levenshtein distance algorithm to calculate string similarity.

## Key Implementation Details

- Uses Levenshtein distance to measure edit distance between strings
- Converts distance to a similarity score: `score = 1 - (distance / maxLength)`
- Configurable threshold (default 0.6) determines if a match is valid
- Score ranges from 0 (no match) to 1 (perfect match)
- The threshold comparison is: `score >= threshold`

## Search Usage

Fuzzy matching is used in `searchByKeywords()` function to find memories that partially match search terms, enabling flexible keyword-based discovery even with typos or partial keywords.

## Configuration

The fuzzy threshold can be customized via `.local-recall.json` or environment variables (default: 0.6).
