Transcript discovery logs (searching for Claude projects, finding transcript directories) use INFO-level logging, not DEBUG. This helps distinguish transcript collection activity in recall.log. When upgrading the MCP server, ensure you rebuild with `npm run build` to get the latest logging changes, otherwise old cached versions may show different log messages.
