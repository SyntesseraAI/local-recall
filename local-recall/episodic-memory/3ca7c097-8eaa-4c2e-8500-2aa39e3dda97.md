# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with forward slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files (one JSON object per line) with filenames like:
- `claude_20251202_123456.jsonl`

Each line contains a transcript event with metadata like session_id, timestamp, and message content.

## Relevance to Local Recall

This information is critical for the transcript-collector component which needs to:
1. Locate Claude's transcript cache
2. Parse the path correctly to find the right project folder
3. Read JSONL files to extract transcript data
4. Process transcripts for memory extraction
