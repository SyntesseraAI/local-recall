Local Recall integrates with Claude Code through three hooks defined in `hooks.json`: SessionStart hook loads all memories from disk and returns the 5 most recent memories (sorted by occurred_at) to inject into Claude's context. UserPromptSubmit hook performs semantic search using vector embeddings and injects matching memories. The Stop hook is disabled (asynchronous MCP daemon handles memory extraction instead). All hooks receive input via stdin and output via stdout.
