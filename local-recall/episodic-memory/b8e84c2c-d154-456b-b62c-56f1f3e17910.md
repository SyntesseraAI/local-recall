The Local Recall codebase has a specific directory structure:

- `src/hooks/` - Contains hook implementations (session-start, stop, etc.)
- `src/core/` - Core functionality (memory.ts, index.ts, search.ts)
- `src/utils/` - Utility functions including `transcript.ts` for parsing transcripts
- `dev-marketplace/local-recall-plugin/` - Plugin packaging and configuration
- `dev-marketplace/local-recall-plugin/config/hooks.json` - Hook configuration file
- Tests organized in `tests/integration/` for integration testing

The transcript parsing and memory extraction uses `analyzeForMemories()` function in `src/utils/transcript.ts` (line 105).
