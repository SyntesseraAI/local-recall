Memory files are stored as markdown with YAML frontmatter containing: id (UUID), subject (one-line description), keywords (searchable terms), applies_to (scope: global/file:/area:), occurred_at (ISO-8601 timestamp), and content_hash (SHA-256 prefix). The actual content goes after the frontmatter. This structure enables deduplication via content_hash and occurred_at fields.
