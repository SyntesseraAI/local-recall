# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Slashes are replaced with hyphens
- Example: `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored as JSONL files in `~/.claude/projects/{folder-name}/transcripts/`

## Relevance

The transcript collector in `src/core/transcript-collector.ts` needs to handle this path conversion when syncing transcripts from Claude's cache to the local-recall memory system.
