# Claude Transcript Storage Location

Claude Code stores project transcripts in `~/.claude/projects/` where the project directory path is URL-encoded into the subdirectory name.

## Structure
- Location: `~/.claude/projects/<url-encoded-project-path>/transcripts/`
- Format: JSONL files (one JSON object per line)
- Each transcript file contains session events with timestamps

## Example
For project `/Users/joe/Code/Syntessera/local-recall`, transcripts are stored at:
`~/.claude/projects/Users%2Fjoe%2FCode%2FSyntessera%2Flocal-recall/transcripts/`

## Key Points
- Path encoding: Forward slashes become `%2F`, colons become `%3A`
- One JSONL file per session
- Each line is a complete JSON event object
- Timestamps are in ISO-8601 format
