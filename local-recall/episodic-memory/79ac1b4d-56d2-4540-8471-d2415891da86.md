The `condenseTranscript()` function in memory-extractor.ts compresses transcript content before sending it to Claude for memory extraction. This reduces token usage significantly, making the memory extraction pipeline more cost-effective. The condensed format removes redundancy while preserving essential information needed for memory creation.
