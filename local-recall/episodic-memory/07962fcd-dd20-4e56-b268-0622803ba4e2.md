Local Recall integrates with Claude Code through three hooks (SessionStart, UserPromptSubmit, Stop) and an MCP server. SessionStart hook loads memory index on session begin, UserPromptSubmit searches memories based on user prompts, and Stop hook parses transcripts. The MCP server runs as a background daemon processing transcripts every 5 minutes.
