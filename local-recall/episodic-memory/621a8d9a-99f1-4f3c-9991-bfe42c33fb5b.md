Each memory file is markdown with YAML frontmatter containing:
- `id`: UUID
- `subject`: Brief description
- `keywords`: Array for searchability
- `applies_to`: Scope (global, file:path, or area:name)
- `occurred_at`: ISO-8601 timestamp
- `content_hash`: SHA-256 prefix (16 chars) for deduplication

Content hash and occurred_at are used together to detect and prevent duplicate memories.
