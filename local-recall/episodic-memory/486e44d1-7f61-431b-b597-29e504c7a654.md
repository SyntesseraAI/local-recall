The BGE-small-en-v1.5 embedding model (~133MB) is automatically downloaded on first use to local_cache/fast-bge-small-en-v1.5/. Initial startup takes 30-60 seconds. If tokenizer file is missing (cache corrupted), delete local_cache/fast-bge-small-en-v1.5* and the model will re-download automatically.
