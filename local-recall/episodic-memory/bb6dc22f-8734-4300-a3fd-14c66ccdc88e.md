Local Recall uses Ollama for generating vector embeddings. Prerequisites:
1. Install Ollama: https://ollama.com
2. Pull model: `ollama pull nomic-embed-text`
3. Run server: `ollama serve`

Configuration:
- `OLLAMA_BASE_URL`: Default `http://localhost:11434`
- `OLLAMA_EMBED_MODEL`: Default `nomic-embed-text`

Model info: nomic-embed-text produces 768-dimensional vectors. Significantly larger than previous fastembed model (BGE-small-en-v1.5: 384 dimensions), so vector indexes must be rebuilt when migrating.
