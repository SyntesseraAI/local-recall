The memory extraction process uses the Claude API (via Anthropic SDK) with streaming enabled for efficient token usage. The implementation includes:

- Streaming responses to handle large transcripts
- Automatic fallback to non-streaming if streaming fails
- Error handling for API failures with logging
- Configuration of model parameters (temperature, max_tokens)

This approach allows processing of transcripts without loading entire responses into memory at once.
