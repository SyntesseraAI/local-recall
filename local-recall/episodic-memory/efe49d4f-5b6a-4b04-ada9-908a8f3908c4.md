Added separate minimum similarity score thresholds for episodic and thinking memories:
- `episodicMinSimilarity`: Minimum similarity threshold for episodic memories (default: 0.8 or 80%, env: `LOCAL_RECALL_EPISODIC_MIN_SIMILARITY`)
- `thinkingMinSimilarity`: Minimum similarity threshold for thinking memories (default: 0.8 or 80%, env: `LOCAL_RECALL_THINKING_MIN_SIMILARITY`)

These replace the single combined threshold that was previously used, allowing fine-grained control over filtering for each memory type.

Configured in `src/core/types.ts` and `src/utils/config.ts`
