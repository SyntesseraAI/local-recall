# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL (JSON Lines) files:
- One JSON object per line
- Each line is a complete transcript entry
- Filename format: `<timestamp>.jsonl`
- Tracks session history and assistant responses

## Usage in local-recall

The MCP server and transcript-collector components use this path to:
1. Sync transcripts from Claude's cache
2. Extract memories from transcript content
3. Track processed transcripts with content hashes
4. Delete and recreate memories when transcripts change
