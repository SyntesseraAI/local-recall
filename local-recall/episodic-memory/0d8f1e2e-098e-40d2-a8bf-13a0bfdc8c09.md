# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension. Each line is a valid JSON object representing a transcript event.

## Integration Point

The `TranscriptCollector` class in `src/utils/transcript-collector.ts` uses this path pattern to locate and sync transcripts for memory extraction. This is critical for the MCP server's background daemon that processes transcripts every 5 minutes.

## Key Implementation Detail

When collecting transcripts, the code must:
1. Resolve `~` to the user's home directory
2. Replace `/` with `-` when constructing the folder name
3. Look for `.jsonl` files in the `transcripts/` subdirectory
4. Parse JSONL format correctly (one JSON object per line)
