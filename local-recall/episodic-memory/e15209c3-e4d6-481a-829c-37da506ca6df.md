The memory extraction process must return strictly formatted JSON with a `memories` array. Each memory object requires: `subject` (max 200 chars, one-line description), `keywords` (array of 1-10 lowercase searchable terms), `applies_to` (scope: 'global', 'file:<path>', or 'area:<name>'), and `content` (markdown-formatted detailed memory). The extraction should return only the JSON object with no explanation, no markdown formatting, and no code blocks.
