Each memory file contains YAML frontmatter with: `id` (UUID), `subject` (brief description), `keywords` (array for searchability), `applies_to` (scope: global/file:/path/area:name), `occurred_at` (ISO-8601 timestamp for deduplication), and `content_hash` (SHA-256 prefix for duplicate detection). Memory files are idempotent - no updates, only create/delete operations.
