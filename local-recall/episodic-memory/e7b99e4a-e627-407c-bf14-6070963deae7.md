# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and colons removed. For example:
- `/Users/joe/Code/Syntessera/local-recall` â†’ `Users-joe-Code-Syntessera-local-recall`

## File Structure

- Location: `~/.claude/projects/<project-folder>/transcripts/`
- File format: JSONL (JSON Lines)
- Each line is a complete JSON object representing a conversation turn
- Files are named with ISO timestamps: `YYYY-MM-DDTHH-MM-SS.jsonl`

## Transcript Content

Each JSON object in the transcript contains:
- `role`: Either "user" or "assistant"
- `content`: The actual message text
- `timestamp`: When the message was created

This is relevant for `transcript-collector.ts` which needs to sync and process these transcripts for memory extraction.
