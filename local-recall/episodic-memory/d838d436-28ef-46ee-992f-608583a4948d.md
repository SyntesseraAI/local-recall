# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path with:
- Leading slashes removed
- All forward slashes (`/`) replaced with dashes (`-`)

### Example

For working directory `/Users/joe/Code/Syntessera/local-recall`:
- Transcript path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`
- Transcript files are JSONL format

## Impact on Code

The `transcript-collector.ts` module needs to:
1. Get the absolute working directory (`cwd`)
2. Strip leading slashes
3. Replace all `/` with `-` to construct the project folder name
4. Read transcript files from the appropriate location

This path transformation is critical for the memory extraction system to locate and process Claude Code transcripts correctly.
