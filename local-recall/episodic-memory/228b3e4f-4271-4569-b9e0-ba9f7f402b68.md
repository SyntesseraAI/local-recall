# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is derived from the absolute working directory path:
- Slashes `/` are replaced with dashes `-`
- Dots `.` are removed
- Leading dash is removed

### Example

Working directory: `/Users/joe/Code/Syntessera/local-recall`
Transcript location: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension.
Each line contains a JSON object representing a transcript event or message.

## Relevance

This information is critical for the `transcript-collector` module which syncs transcripts from Claude's cache to build memories from session history.
