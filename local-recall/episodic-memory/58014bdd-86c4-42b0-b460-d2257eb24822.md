# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files in the `transcripts/` directory. Each line is a JSON object representing a message or tool call.

## Key Information

- Transcripts are cumulative (append-only)
- Each transcript file can be used to extract memories
- The transcript collector feature depends on this path format for reliable transcript location
- This path format allows distinguishing between projects with similar names in different directories
