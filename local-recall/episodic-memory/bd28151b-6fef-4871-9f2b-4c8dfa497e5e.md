The memory extraction process uses transcript condensing to improve efficiency. Instead of processing raw transcripts, the system condenses transcripts into events with categories like [User], [Assistant], [Tool], and [Result]. This reduces token usage and improves processing speed when extracting memories from Claude Code sessions.
