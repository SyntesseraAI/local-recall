# MCP Server Tool Updates

## Removed Operations
- Removed `memory_update` tool (no longer supported)
- Removed `memory_delete` tool (no longer supported)

## Retained Operations
- `memory_create` - Create new episodic memories (idempotent)
- `memory_list` - List all memories
- `memory_get` - Retrieve specific memory by ID
- `memory_search` - Semantic search over memories

## Why
Memories are now immutable. Deduplication at creation time ensures no duplicates are created. This aligns with the episodic memory model where memories are historical records, not a mutable knowledge base.
