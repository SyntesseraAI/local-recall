The transcript collection system handles:

1. **Responsibility**:
   - Collects transcript data from Claude Code sessions
   - Manages storage and access of transcripts
   - Provides interface for memory extraction to access transcript content

2. **Integration Pattern**:
   - Memory extractor depends on transcript collector for input data
   - Separation allows independent testing and development
   - Transcript data flows from collector → extractor → memory manager

3. **File Location**:
   - Core implementation in `src/core/transcript-collector.ts`
   - Modified recently with standardized string quotes and improved logging
