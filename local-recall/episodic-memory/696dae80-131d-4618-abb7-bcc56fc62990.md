# Transcript Collector Path Resolution

The `transcript-collector.ts` module must handle two scenarios:

1. **Absolute path input**: When given a working directory path like `/Users/joe/Code/Syntessera/local-recall`
2. **UUID-based folder pattern**: Claude stores transcripts in `~/.claude/projects/UUID-dash-converted-path/transcripts/`

## Implementation Note

The collector needs to:
- Convert absolute paths to Claude's folder naming convention (replace slashes with dashes)
- Handle the UUID prefix that Claude adds
- Locate the correct transcript directory even when only the working directory is provided
- Account for the `.jsonl` file format of transcript entries
