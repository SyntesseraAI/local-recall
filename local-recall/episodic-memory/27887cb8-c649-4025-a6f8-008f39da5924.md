The search module uses Orama vector store with Ollama embeddings (nomic-embed-text model, 768 dimensions) for semantic search. Results are ranked by cosine similarity score (0.0-1.0) with a recency tie-breaker: when scores are equal, more recent memories (`occurred_at`) rank higher.
