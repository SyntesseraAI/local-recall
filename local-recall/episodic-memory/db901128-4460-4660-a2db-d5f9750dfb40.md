# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude project folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`
- Transcripts stored at: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with names like:
- `chatcmpl-...session-id....jsonl`

Each line is a JSON object representing a single event in the session.

## Key Observations

1. The path transformation uses the absolute working directory with all slashes replaced by dashes
2. This is used by `transcript-collector.ts` to locate and process transcripts
3. Important for the MCP server background daemon that processes transcripts every 5 minutes
