Updated `src/core/memory-extractor.ts` to integrate the new transcript condenser:

**Changes made:**
1. Added import for `TranscriptCondenser`
2. Reads raw JSONL transcript content
3. Condenses transcript using `TranscriptCondenser.condense(jsonlContent)`
4. Passes condensed data to `buildMemoryExtractionPrompt()` instead of raw content
5. Claude processes only the essential information, reducing token usage significantly

**Impact:**
- Extraction prompts are much smaller
- Faster processing
- Lower API costs
- Better focus on actual memory-worthy content

The integration is seamless and backwards-compatible with existing memory extraction logic.
