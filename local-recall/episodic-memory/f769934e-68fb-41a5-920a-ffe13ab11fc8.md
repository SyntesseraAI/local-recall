# Transcript Event Structure

When processing transcript JSONL files, events may have null values for:
- `user_prompt` - Can be null/undefined
- `assistant_response` - Can be null/undefined

This should be handled gracefully when extracting or processing transcript data. Check for null/undefined before attempting to read or parse these fields.
