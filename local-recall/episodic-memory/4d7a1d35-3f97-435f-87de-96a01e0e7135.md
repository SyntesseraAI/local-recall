The MCP server includes a background daemon that syncs transcripts from Claude's cache (~/.claude/projects/<project>/transcripts/), processes them using `claude -p` to extract memories, and tracks processed transcripts with content hashes for change detection. The daemon runs every 5 minutes and deletes/recreates memories when transcripts change. Exposes tools: memory_create, memory_get, memory_search, memory_list, index_rebuild.
