The UserPromptSubmit hook receives input containing:
- `session_id`: unique session identifier (UUID)
- `transcript_path`: path to jsonl transcript file
- `cwd`: current working directory of the project
- `permission_mode`: user permission level

Transcripts are stored in jsonl format at paths like `/Users/joe/.claude/projects/-Users-joe-Code-Syntessera-local-recall/[session_id].jsonl`. Each line contains a JSON event object. The hook processes these events to extract keywords and search memories.
