# LOCAL_RECALL_INTERNAL Marker

The `[LOCAL_RECALL_INTERNAL]` message in the system message indicates that Claude is in the memory extraction phase, analyzing a condensed transcript of the user's session.

## What This Means

- A Claude Code session has been recorded and transcribed
- The transcript has been condensed into a structured format
- Claude is analyzing this condensed transcript to identify valuable memories
- The goal is to extract actionable insights that will help future AI assistants

## Extraction Guidelines

When extracting memories, focus on:
- What was learned or discovered
- Problems that were solved and how
- Code patterns or architectural decisions
- User preferences or project conventions
- Specific file locations or configuration details

Avoid extracting:
- Generic programming knowledge
- Temporary debugging steps
- Obvious or trivial information
- Sensitive data
