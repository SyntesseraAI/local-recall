# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full transcript directory: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files with naming convention:
- Pattern: `{iso-timestamp}_{random-id}.jsonl`
- Each line is a JSON object representing an event/message in the transcript

## Transcript Structure

Each JSONL file contains transcript events with:
- `role`: 'user' or 'assistant'
- `content`: The message content
- Additional metadata fields

This is used by the transcript-collector to process transcripts and extract memories.
