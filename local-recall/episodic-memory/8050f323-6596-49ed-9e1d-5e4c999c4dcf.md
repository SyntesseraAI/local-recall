# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes and encoded in lowercase. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `~/.claude/projects/users-joe-code-syntessera-local-recall/`

## Transcript Files

Transcripts are stored as JSONL (JSON Lines) files in the `transcripts/` subdirectory:
- Location: `~/.claude/projects/<project-id>/transcripts/`
- Files are numbered sequentially: `0.jsonl`, `1.jsonl`, `2.jsonl`, etc.
- Each line is a valid JSON object representing a single message/event
- Files can be read and parsed line-by-line

## Transcript Structure

Each transcript file contains events with metadata including:
- Message content and type
- Tool invocations and results
- Timestamps
- Session information
