# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as JSONL files with `.jsonl` extension in the `transcripts/` subdirectory. Each line in the file is a valid JSON object representing a single transcript event.

## Usage Context

This location is used by the transcript-collector to sync transcripts from Claude's cache into the local-recall memory system for processing and memory extraction.
