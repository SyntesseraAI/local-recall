Extracted memories must follow a JSON structure with a 'memories' array. Each memory object contains: subject (brief one-line description, max 200 chars), keywords (array of 1-10 lowercase searchable terms), applies_to (scope: 'global', 'file:<path>', or 'area:<name>'), and content (markdown formatted detailed memory). If no memories can be extracted, return {"memories": []}.
