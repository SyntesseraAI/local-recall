# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name uses the full absolute path with `/` replaced by `-`. For example:
- Original path: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/-Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` (JSON Lines) files, with each line being a complete JSON object representing a single turn in the conversation.

## Implementation

The `transcript-collector.ts` module handles reading and processing these transcripts for memory extraction by the MCP server daemon.
