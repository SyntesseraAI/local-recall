# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Naming Convention

The project folder name is derived from the absolute working directory path with slashes converted to underscores. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/`
- Transcripts location: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

Transcripts are stored as `.jsonl` files in this directory.

## Relevance

This path convention is critical for the MCP server's background daemon which syncs transcripts from Claude's cache. The daemon needs to correctly construct the path to find transcripts for memory extraction.
