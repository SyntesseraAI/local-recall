Claude Code stores project transcripts at `~/.claude/projects/` on macOS in a specific directory structure:

## Path Format

The project folder name uses the absolute path with slashes replaced by dashes. For example:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Transcript folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/`

## File Structure

Inside each project folder:
- `transcripts/` - Contains JSONL files with transcript data
- Each transcript file follows a JSONL format (JSON Lines)
- Files contain session transcripts with user messages, assistant responses, and tool invocations

This path structure is important for the `transcript-collector` module which syncs transcripts and extracts memories from them.
