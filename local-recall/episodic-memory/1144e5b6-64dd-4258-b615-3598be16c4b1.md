# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Claude projects folder: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall`
- Transcripts are stored as `.jsonl` files in this directory

## Accessing Transcripts

When implementing transcript collection in `src/core/transcript-collector.ts`, use:
- Get home directory via `os.homedir()`
- Construct path: `${home}/.claude/projects/<project-folder>/`
- Replace slashes with dashes in the full working directory path
- Use glob patterns to find `.jsonl` files: `**/*.jsonl`
