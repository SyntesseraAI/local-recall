# Claude Code Transcript JSONL Format

Claude Code stores transcripts as JSONL (JSON Lines) files where each line is a complete JSON object representing an event.

## Event Types

- `type: 'user-message'` - User input
- `type: 'assistant-message'` - Claude's response
- `type: 'tool-call'` - Tool invocation
- `type: 'tool-result'` - Tool output

## Relevant Fields

- `content` - The actual message/response text
- `tool_name` - Name of the tool being called
- `tool_use_id` - Identifier for tracking tool use
- `timestamp` - When the event occurred

This structure is used for parsing and extracting memories from session history.
