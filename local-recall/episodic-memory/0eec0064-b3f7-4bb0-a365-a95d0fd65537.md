# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes converted to underscores. For example:
- Project: `/Users/joe/Code/Syntessera/local-recall`
- Folder: `_Users_joe_Code_Syntessera_local-recall`
- Full path: `~/.claude/projects/_Users_joe_Code_Syntessera_local-recall/transcripts/`

## Transcript Files

Transcripts are stored as JSONL files with names like:
- `1733082461_0ec3ecfd.jsonl`
- `1733082700_6a3d4681.jsonl`

Each file contains a sequence of JSON objects representing events in the conversation.

## Key Discovery

For the `transcript-collector.ts` utility, the path construction needs to handle the underscore conversion properly when reading transcripts from Claude's cache.
