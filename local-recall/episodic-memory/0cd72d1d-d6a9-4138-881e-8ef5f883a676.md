# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL files with:
- Filename format: `<timestamp>-<random-suffix>.jsonl`
- Example: `2025-12-01T10-30-45-abc123.jsonl`
- Each line is a JSON object representing a turn in the conversation

## Important for Transcript Processing

When collecting and processing transcripts for memory extraction:
1. Replace slashes with dashes to construct the project folder name
2. Look for all `.jsonl` files in the transcripts directory
3. Each JSONL file contains multiple turns/events in the session
4. Files are named with ISO timestamps for chronological ordering
