# Transcript Path Mapping Issue

## Problem
The `transcript-collector.ts` needs to correctly map a working directory path to Claude's project transcript location.

## Solution
When given a working directory (e.g., `/Users/joe/Code/Syntessera/local-recall`), convert it to Claude's project folder name by:
1. Taking the absolute path
2. Replacing all forward slashes `/` with dashes `-`
3. Using this as the folder name under `~/.claude/projects/`

Final path: `~/.claude/projects/{path-with-dashes-instead-of-slashes}/transcripts/`

## File Format
Transcripts are JSONL files where each line is a complete JSON event object.
