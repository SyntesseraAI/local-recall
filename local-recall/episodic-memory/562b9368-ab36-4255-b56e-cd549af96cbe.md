# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files, one event per line:
- Each line is a valid JSON object
- Multiple transcript files exist in the directory
- Files can be read sequentially to reconstruct session history

## Key for Memory Extraction

This information is critical for the transcript-collector to:
1. Locate transcript files for the current working directory
2. Read and parse JSONL format correctly
3. Extract memories from transcript events for creating episodic memories
