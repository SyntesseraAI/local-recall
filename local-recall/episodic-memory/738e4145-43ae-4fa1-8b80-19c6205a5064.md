The project maintains separate user-prompt-submit hooks:

- `src/hooks/user-prompt-submit.ts` - Handles episodic memory search and injection
- `src/hooks/user-prompt-submit-thinking.ts` - Handles thinking memory search and injection

Both hooks now check their respective enabled flags before executing. This modular approach allows each memory type to be independently configured and controlled, making it easier to debug issues and adjust behavior for each system separately.
