Project requires Node.js 18+ and TypeScript 5.9+. Key npm scripts:
- `npm run build`: Compile TypeScript
- `npm run dev`: Watch mode
- `npm run test`: Run tests
- `npm run lint`: Lint code
- `npm run mcp:start`: Start MCP server

All memory-related development must include comprehensive tests and documentation in `tests/` and `docs/` folders.
