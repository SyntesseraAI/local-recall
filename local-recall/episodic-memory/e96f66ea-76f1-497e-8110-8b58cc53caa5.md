# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## Transcript Format

Transcripts are stored as `.jsonl` files (JSON Lines format) with the filename pattern `transcript-<timestamp>.jsonl`.

Each line in a transcript file is a JSON object representing a single event or message in the conversation.

## Impact on transcript-collector.ts

The `transcript-collector.ts` module uses this path to sync transcripts from Claude's cache into the local-recall project for memory extraction.
