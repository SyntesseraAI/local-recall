# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Structure

Project folders use the absolute path with slashes replaced by dashes:
- Project at `/Users/joe/Code/Syntessera/local-recall`
- Maps to folder: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as JSONL (JSON Lines) files with `.jsonl` extension, one transcript per line, allowing efficient appending and parsing of session transcripts.
