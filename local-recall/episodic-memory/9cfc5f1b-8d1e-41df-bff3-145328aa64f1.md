# Claude Project Transcripts Location

Claude Code stores project transcripts in `~/.claude/projects/` on macOS. The project folder name is derived from the absolute path with slashes replaced by underscores and some hashing. For example:

- Project at `/Users/joe/Code/Syntessera/local-recall` may have transcripts at `~/.claude/projects/Users_joe_Code_Syntessera_local-recall/transcripts/`

This is important for the transcript-collector to locate and sync transcripts from Claude's cache into the local-recall memory system.

**Related files:**
- `src/core/transcript-collector.ts` - Handles fetching transcripts from this location
- `src/mcp-server/server.ts` - Background daemon that processes transcripts
