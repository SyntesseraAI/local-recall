Memory creation is idempotent - createMemory returns existing memory if duplicate found. Deduplication uses findDuplicate(occurredAt, contentHash) method. Content hash is SHA-256 prefix (16 chars) of memory content. This allows memories to be created safely without worrying about duplicates.
