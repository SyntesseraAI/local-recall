# Claude Project Transcripts Location

Claude Code stores project transcripts at `~/.claude/projects/` on macOS.

## Path Format

The project folder name is the absolute path with slashes replaced by dashes:
- Working directory: `/Users/joe/Code/Syntessera/local-recall`
- Project folder name: `Users-joe-Code-Syntessera-local-recall`
- Full path: `~/.claude/projects/Users-joe-Code-Syntessera-local-recall/transcripts/`

## File Format

Transcripts are stored as `.jsonl` files (JSON Lines format), one event per line. Each line is a valid JSON object representing a single transcript event.

## Key Implementation Detail

When implementing transcript collection in `src/core/transcript-collector.ts`, the path construction uses:
1. Home directory from environment (`$HOME` or via `os.homedir()`)
2. Replace path separators (forward slashes) with dashes
3. Append `/transcripts/` directory

This is critical for the background daemon and MCP server to correctly locate and process transcripts.
