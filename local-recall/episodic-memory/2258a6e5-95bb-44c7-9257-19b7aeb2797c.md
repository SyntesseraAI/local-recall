The LocalRecallDaemon class in the MCP server implements background transcript processing:

1. **Interval**: Runs every 5 minutes (300 seconds)
2. **Process**: 
   - Scans Claude's project transcript cache
   - Processes unprocessed or changed transcripts
   - Extracts memories using Claude's API
   - Creates memory files in episodic-memory/
3. **Deduplication**: Uses content hashes to detect changes and avoid duplicate memory creation
4. **Error handling**: Continues processing even if individual transcripts fail
5. **Status**: Logs processing status for debugging and monitoring
