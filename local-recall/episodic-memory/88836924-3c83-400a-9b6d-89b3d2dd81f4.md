## Claude Projects Directory Location

Claude Code stores project transcripts in a specific directory structure:

- **Base path**: `~/.claude/projects/`
- **Project folder naming**: Uses the full path with slashes replaced by dashes, prefixed with a UUID
- **Example**: For `/Users/joe/Code/Syntessera/local-recall`, the folder is `~/.claude/projects/da-<UUID>-Users-joe-Code-Syntessera-local-recall/`
- **Transcript storage**: Inside the project folder at `transcripts/` subdirectory
- **File format**: `.jsonl` (JSON Lines) files containing transcript events

This directory structure is important for the transcript collector to locate and sync transcripts from Claude's cache for memory extraction.
