The BGE-small-en-v1.5 embedding model is automatically downloaded to `local_cache/` on first use (~133MB, takes 30-60 seconds). Subsequent runs load from cache. If tokenizer file not found error occurs, remove the corrupted cache directory and the model will re-download automatically on next run.
