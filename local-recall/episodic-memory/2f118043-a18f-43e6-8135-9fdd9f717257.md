Claude Code plugins have multiple version references that must be kept in sync:

1. `package.json` - The npm package version
2. `.claude-plugin/plugin.json` - Claude Code plugin manifest
3. `.claude-plugin/marketplace.json` - Marketplace distribution metadata

When bumping versions, all three files must be updated together. Missing any of these can cause version mismatches and prevent proper marketplace detection of updates. Use grep to find all `0.X.X` references before committing version bumps.
