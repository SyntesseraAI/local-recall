The `TranscriptCollector` class (`src/core/transcript-collector.ts`) is responsible for:
1. Syncing transcripts from Claude's cache directory (`~/.claude/projects/`)
2. Converting raw JSONL transcripts to condensed event format
3. Extracting memories from condensed transcripts using Claude's API
4. Tracking processed transcripts via content hashes to detect changes
5. Managing memory lifecycle (creation, updating, deletion)

Key methods include `syncTranscripts()`, `condenseTranscript()`, `extractMemories()`, and helper methods for transcript processing. The class manages deduplication using content hashes and handled `occurred_at` timestamps to prevent duplicate memory creation.
